### T3 quantifier — missing ∈ ℕ carrier annotation on bound variable i
**Class**: OBSERVE
**Foundation**: T0 (CarrierSetDefinition) — extensionality axiom writes `(A i ∈ ℕ : 1 ≤ i ≤ #a : aᵢ = bᵢ)` with explicit carrier.
**ASN**: T3 (CanonicalRepresentation) — body `(A a, b ∈ T : #a = #b ∧ (A i : 1 ≤ i ≤ #a : aᵢ = bᵢ) ≡ a = b)` and Formal Contract postcondition `a = b ⟺ #a = #b ∧ (A i : 1 ≤ i ≤ #a : aᵢ = bᵢ)`.
**Issue**: The quantifier over `i` omits the `∈ ℕ` carrier annotation. T0's extensionality, T1's agreement condition `(A i ∈ ℕ : 1 ≤ i < k : aᵢ = bᵢ)`, and TA-Pos's index quantifiers all carry the annotation explicitly. T3's formulation of the same range leaves the domain of `i` implicit. The intended domain is ℕ — unambiguous from context — but the notation is inconsistent with the rest of the ASN.
