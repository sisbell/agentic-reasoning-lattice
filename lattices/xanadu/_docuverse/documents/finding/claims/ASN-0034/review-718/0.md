### T3 formal statement drops explicit ℕ-annotation that T0 formal contract requires
**Class**: OBSERVE
**Foundation**: T0 (CarrierSetDefinition)
**ASN**: T3 (CanonicalRepresentation) — formal statement `(A a, b ∈ T : #a = #b ∧ (A i : 1 ≤ i ≤ #a : aᵢ = bᵢ) ≡ a = b)`, and the postcondition `a = b ⟺ #a = #b ∧ (A i : 1 ≤ i ≤ #a : aᵢ = bᵢ)`
**Issue**: T0's formal contract extensionality axiom writes `(A i ∈ ℕ : 1 ≤ i ≤ #a : aᵢ = bᵢ)` with explicit `i ∈ ℕ`. T3's forward direction applies that axiom, but T3's stated antecedent is `(A i : 1 ≤ i ≤ #a : aᵢ = bᵢ)`, dropping the ℕ-binding. The same omission appears in Divergence's formal contract at `(A i : 1 ≤ i < k : aᵢ = bᵢ)` and `(A i : 1 ≤ i ≤ #a ∧ i ≤ #b : aᵢ = bᵢ)`. Since `≤` is typed on ℕ (NAT-order), the implied domain is recoverable from context, and the claims are sound as written; but the document is not internally consistent about whether the index variable's membership in ℕ is stated or assumed.
