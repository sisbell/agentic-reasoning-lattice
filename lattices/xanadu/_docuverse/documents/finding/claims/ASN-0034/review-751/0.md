### T5 precondition `#p ≥ 1` is redundant
**Class**: OBSERVE
**Foundation**: T0 (CarrierSetDefinition)
**ASN**: T5 (ContiguousSubtrees) — Formal Contract *Preconditions:* `p is a tumbler prefix with #p ≥ 1`
**Issue**: T0's nonemptiness axiom `(A a ∈ T :: 1 ≤ #a)` already guarantees `#p ≥ 1` for every `p ∈ T`. Stating it as a precondition of T5 is vacuously satisfied and could suggest to a reader that there is a case where `#p = 0` requiring exclusion — a case T0 has already closed.

VERDICT: OBSERVE
