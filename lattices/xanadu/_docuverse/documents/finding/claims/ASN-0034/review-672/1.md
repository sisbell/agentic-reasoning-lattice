### T1 proof elides ≤-unfolding before ruling out its disjuncts
**Class**: OBSERVE
**Foundation**: T1 (LexicographicOrder)
**ASN**: Irreflexivity part (a) — "Case (ii) requires `k = m + 1 ≤ m`; NAT-addcompat gives `m < m + 1`, and NAT-order's trichotomy then excludes both `m + 1 < m` and `m + 1 = m`"; Trichotomy part (b) Case 3 — "`(β)` gives `m + 1 ≤ n`, hence `m < n` via NAT-addcompat's `m < m + 1`"
**Issue**: Both passages apply the cited tools directly to the conclusion without first unfolding `≤`. In irreflexivity, `m + 1 ≤ m` must be unfolded to `m + 1 < m ∨ m + 1 = m` before trichotomy can rule out each disjunct. In Case 3, deriving `m < n` from `m < m + 1` and `m + 1 ≤ n` requires unfolding `m + 1 ≤ n` to `m + 1 < n ∨ m + 1 = n` and handling each branch (< by <-transitivity, = by substitution). The conclusions are correct; the elided steps are derivable from the cited dependencies; but the inference gap is present in both locations. By contrast, the transitivity sub-case (ii,ii) in the same proof walks the analogous step explicitly ("via NAT-order transitivity or substitution"), making the inconsistency in proof granularity visible to a careful reader.
