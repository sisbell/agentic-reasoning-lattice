### T1 Case 2 no-reverse-witness includes unexplained aside for the wrong direction
**Class**: OBSERVE
**Foundation**: T1 (LexicographicOrder)
**ASN**: Trichotomy part (b) Case 2 — "case (ii) requires `k' = n + 1` **(or m + 1)**, contradicting `k' < k ≤ n` **(respectively k ≤ m)**"
**Issue**: In Case 2, the no-reverse-witness argument is eliminating candidates for a `b < a` witness. For `b < a` via T1(ii), the required form is `k' = n + 1 ≤ m` — only `n + 1` arises. The parenthetical "(or m + 1)" and "(respectively k ≤ m)" apply to the `a < b` T1(ii) form, which is the direction already established, not the reverse being ruled out. The parenthetical introduces an unexplained asymmetry into an argument that is otherwise tracking only `b < a` candidates, and the "respectively" clause has no referent in that local context.
