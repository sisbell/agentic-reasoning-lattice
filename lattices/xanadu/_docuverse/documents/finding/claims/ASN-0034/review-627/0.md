### Mixed `≥`/`≤` in T0 comprehension
**Class**: OBSERVE
**ASN**: T0 comprehension axiom opens `(A p ∈ ℕ : p ≥ 1 : ...)`, while the same axiom's inner ranges use `≤` (`1 ≤ j ≤ p`, `1 ≤ i ≤ p`) and the nonemptiness clause writes `1 ≤ #a`. T0's *Depends:* on NAT-order describes only "the non-strict relation `≤`".
**Issue**: A single appearance of `≥` against four uses of `≤` in the same axiom block. Soundness is preserved by NAT-order's *Definition* clause `m ≥ n ⟺ n ≤ m`, but the depend description mentions only `≤`, so a precise reader must consult NAT-order to confirm `≥` is licensed here. Normalizing the lone `p ≥ 1` to `1 ≤ p` aligns with the rest of the axiom and the depend description.
