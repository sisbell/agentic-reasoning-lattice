### T0 inner quantifiers omit index type
**Class**: OBSERVE
**Foundation**: T0 (CarrierSetDefinition) — comprehension and extensionality axioms
**ASN**: T0 formal contract — `(A i : 1 ≤ i ≤ p : tᵢ = r(i))` in the comprehension axiom and `(A i : 1 ≤ i ≤ #a : aᵢ = bᵢ)` in the extensionality axiom.
**Issue**: The bound variable `i` carries no declared type in these quantifiers. The component projection `tᵢ` is defined on `{j ∈ ℕ : 1 ≤ j ≤ #t}`, so `tᵢ` is only well-formed when `i ∈ ℕ`; the same holds for `r(i)` whose domain is `{j ∈ ℕ : 1 ≤ j ≤ p}`. By contrast, the projection-signature clause in the same contract writes the domain explicitly as `{j ∈ ℕ : 1 ≤ j ≤ #a}`, making the `ℕ` membership visible. The two styles are inconsistent within the same formal contract.
