### [NAT-closure: `0 < 1` is derivable, not independent]
**Class**: OBSERVE
**Foundation**: NAT-closure (NatArithmeticClosureAndIdentity)
**ASN**: Formal Contract *Axiom:* `0 < 1` (distinctness of the two named constants)
**Issue**: The body of NAT-closure itself shows `0 < 1` is derivable from the other axioms in the same group: instantiate `(A n ∈ ℕ :: 0 < n + 1)` at `n = 0` to get `0 < 0 + 1`, then apply left-identity at `n = 1` (requires `1 ∈ ℕ`, also in this group) to rewrite to `0 < 1`. The axiom set in the formal contract is therefore not independent — `0 < 1` is a consequence, not a primitive. The text acknowledges the derivation but lists it as an axiom regardless. This is not incorrect, but a precise reader notes the redundancy.
