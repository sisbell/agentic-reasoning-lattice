### NAT-closure axiom `0 < 1` is subsumed by the universal successor-positivity clause
**Class**: OBSERVE
**Foundation**: NAT-closure (NatArithmeticClosureAndIdentity)
**ASN**: Axiom clause `0 < 1` alongside `(A n ∈ ℕ :: 0 < n + 1)`
**Issue**: Instantiating `(A n ∈ ℕ :: 0 < n + 1)` at `n := 0` yields `0 < 0 + 1`; NAT-closure's own left-identity at `n := 1` rewrites this to `0 < 1`. The distinctness axiom is therefore derivable from the universal within the same formal contract. It is stated as a separate axiom clause, creating a redundancy the prose itself acknowledges.

VERDICT: REVISE
