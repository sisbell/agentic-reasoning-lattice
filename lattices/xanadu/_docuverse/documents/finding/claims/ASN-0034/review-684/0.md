### T1 Case 3: trichotomy branch to clause without ruling out the other
**Class**: REVISE
**Foundation**: T1 (LexicographicOrder) — proof of part (b), Case 3
**ASN**: T1 trichotomy, Case 3 prose: *"If `m < n`, then `k = m + 1 ≤ n`, so `a` is a proper prefix of `b` and `k` witnesses `a < b` via T1(ii)."*
**Issue**: Case 3's premise is that the first divergence position `k` satisfies clause (β) **or** (γ). The proof establishes (β) ⟹ `m < n` and (γ) ⟹ `n < m`, then invokes NAT-order's trichotomy to split on `m < n ∨ n < m`, and immediately asserts `k = m + 1 ≤ n` in the `m < n` branch. This assertion is that `k` satisfies (β), but the proof does not eliminate (γ) in this branch. The elimination requires: (γ) ⟹ `n < m` (established) ⟹ `¬(m < n)` (by trichotomy's `¬(m < n ∧ n < m)`) — contradicting the branch hypothesis — so (γ) is impossible and (β) must hold. This step is absent. The same gap appears in the symmetric `n < m` branch.
**What needs resolving**: Within the `m < n` branch of the trichotomy split, the proof must establish that (γ) is impossible — citing that (γ) ⟹ `n < m` and NAT-order's `¬(m < n ∧ n < m)` together yield a contradiction — before concluding `k = m + 1 ≤ n`.
