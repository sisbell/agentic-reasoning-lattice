### T10a.3 Depends missing NAT-addassoc; telescoping cites wrong form
**Class**: REVISE
**Foundation**: NAT-addassoc (NatAdditionAssociative) — `(m + n) + p = m + (n + p)` for every `m, n, p ∈ ℕ`
**ASN**: T10a.3 (LengthSeparation) — local-monotonicity paragraph and Formal Contract *Depends:* section
**Issue**: The local-monotonicity proof requires `#output(B) = #output(A) + (k'_{d_A+1} + … + k'_{d_B})` at two points: (1) the strict-inequality step, which applies NAT-addcompat at `m = #output(A)` and concludes `#output(A) + (k'_{d_A+1} + … + k'_{d_B}) = #output(B)`; and (2) the telescoping step that computes the exact difference. Getting from the inductive formula `γ + k'₁ + … + k'_{d_B}` (left-associated accumulation) to the split form `#output(A) + (k'_{d_A+1} + … + k'_{d_B})` requires NAT-addassoc applied at the depth-`d_A` boundary. NAT-addassoc is absent from T10a.3's Depends.

The same gap appears one step earlier: "Left order-compatibility at `m = γ` delivers `γ + d ≤ γ + k'₁ + … + k'_d`." The NAT-addcompat conclusion is `γ + d ≤ γ + (k'₁ + … + k'_d)`, where the right operand is the partial sum treated as a unit. Identifying `γ + (k'₁ + … + k'_d)` with the left-associated accumulation `γ + k'₁ + … + k'_d` again uses NAT-addassoc.

Additionally, the proof names right-telescoping `(m + n) − n = m` with `m = k'_{d_A+1} + … + k'_{d_B}` and `n = #output(A)`. Right-telescoping at those values requires the premise `#output(B) = (k'_{d_A+1} + … + k'_{d_B}) + #output(A)` — the sum in the opposite order from what the inductive formula provides. Reaching that order from the left-associated formula additionally needs commutativity of addition, which is not a stated claim in this ASN. Left-telescoping `(n + m) − n = m` at `n = #output(A)`, `m = k'_{d_A+1} + … + k'_{d_B}` avoids commutativity; it still requires NAT-addassoc for the premise, but not an unlisted axiom. TA5a, which faces the same regrouping challenge, correctly lists NAT-addassoc in its own Depends.
**What needs resolving**: NAT-addassoc must be added to T10a.3's Depends. The telescoping reference in the local-monotonicity paragraph should name left-telescoping rather than right-telescoping, and the premise `#output(B) = #output(A) + (k'_{d_A+1} + … + k'_{d_B})` must be explicitly grounded by NAT-addassoc applied to the inductive length formula at the split depth.

VERDICT: REVISE
