### TA7a.2: "since o ≠ w, the zero-padded sequences disagree" is unjustified
**Class**: REVISE
**Foundation**: ZPD (ZeroPaddedDivergence) — Partiality: "zpd(a, w) is undefined iff a and w are zero-padded-equal"; the explicit counterexample a = [3, 0], w = [3] shows a ≠ w with zpd undefined.
**ASN**: TA7a.2 (SubspaceDivergenceResidue), proof body: "Since o ≠ w, the zero-padded sequences disagree at some position, so zpd(o, w) is defined; write d = zpd(o, w)."
**Issue**: The claim o ≠ w → zpd(o, w) is defined is false in general. ZPD's own documentation names the failure mode: a proper prefix whose trailing extension is all zeros (e.g., [3, 0] vs [3]) gives ô = ŵ even though the tumblers differ. The proof invokes only o ≠ w, which is insufficient. The correct argument splits on the two sub-cases already established at that point in the proof (from #w ≤ #o): sub-case α (#o = #w), where padded sequences equal native sequences — if they agreed everywhere, T3 would yield o = w, contradicting the precondition; sub-case γ (#w < #o), where positions i with #w < i ≤ #o have ôᵢ = oᵢ > 0 (from o ∈ S) and ŵᵢ = 0 (ZPD padding), so ôᵢ ≠ ŵᵢ and zpd is defined. Neither T3 (for α) nor the S-membership clause (for γ) is cited.
**What needs resolving**: The proof must replace the one-line invocation of o ≠ w with the two-sub-case argument: in sub-case α, derive zpd defined from T3 and o ≠ w; in sub-case γ, derive zpd defined from o ∈ S's universal positivity clause at positions past #w against ZPD's zero-padding.

VERDICT: REVISE
