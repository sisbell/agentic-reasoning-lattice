### TA3-strict mis-cites NAT-sub's strict positivity as yielding `> 0`; NAT-closure absent from Depends
**Class**: REVISE
**Foundation**: NAT-sub (NatPartialSubtraction) — strict positivity: `(A m, n ∈ ℕ : m > n : m − n ≥ 1)`. NAT-closure (NatArithmeticClosureAndIdentity) — Consequence: `0 < 1`.

**Issue**: NAT-sub's strict positivity states `m > n ⟹ m − n ≥ 1`, not `m − n > 0`. NAT-closure is absent from TA3-strict's Depends.
**What needs resolving**: NAT-closure must be added to TA3-strict's Depends, annotated for the `≥ 1` to `> 0` bridge in Cases A and 3. The Depends entry for NAT-sub must be corrected to state that strict positivity yields `≥ 1` and that NAT-closure's `0 < 1` closes the remaining step.
