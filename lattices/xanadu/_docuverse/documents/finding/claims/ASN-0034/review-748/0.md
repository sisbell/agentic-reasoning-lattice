### Prefix uses `∀` where the document convention is `A`
**Class**: OBSERVE
**Foundation**: —
**ASN**: Prefix (PrefixRelation) — definition body and Formal Contract definition line: `p ≼ q` iff `#p ≤ #q ∧ (∀i : 1 ≤ i ≤ #p : qᵢ = pᵢ)`
**Issue**: Every bounded quantifier in this ASN — NAT-order, NAT-zero, NAT-closure, NAT-wellorder, T0, T3, T10 — uses Dijkstra's `(A ...)` notation. Prefix alone uses standard `∀`. The inconsistency appears in both the body and the Formal Contract.
