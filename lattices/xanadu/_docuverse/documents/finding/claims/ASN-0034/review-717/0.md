### T3 inner quantifier drops carrier annotation
**Class**: OBSERVE
**Foundation**: T0 (CarrierSetDefinition)
**ASN**: T3 (CanonicalRepresentation) — formal statement and Postcondition
**Issue**: T0's extensionality clause writes `(A i ∈ ℕ : 1 ≤ i ≤ #a : aᵢ = bᵢ)` — the carrier `∈ ℕ` makes the type of `i` explicit. T3's formal statement and Postcondition both write `(A i : 1 ≤ i ≤ #a : aᵢ = bᵢ)`, dropping the annotation. The type is recoverable from context (the bounds `1`, `#a` are in ℕ), but the notation is inconsistent with T0's precision level, which a Dafny translation will need to make explicit.
