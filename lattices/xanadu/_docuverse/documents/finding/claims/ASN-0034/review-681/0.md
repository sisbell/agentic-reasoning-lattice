### NAT-discrete no-interval proof — incorrect structural description of case analysis
**Class**: OBSERVE
**Foundation**: NAT-discrete (NatDiscreteness), no-interval consequence derivation
**ASN**: "Both branches of the split terminate in `n = m`, closing the case analysis."
**Issue**: The Case `m < n` branch does not terminate in `n = m` — it terminates in a contradiction. The axiom at `m < n` yields `m + 1 ≤ n`, which unfolds to `m + 1 < n ∨ m + 1 = n`; paired with the hypothesis `n < m + 1`, both disjuncts are contradictory (the first via `¬(a < b ∧ b < a)`, the second via irreflexivity after rewriting). The branch is discharged by absurdity, contributing nothing to the conclusion. Only the Case `m = n` branch directly yields `n = m`. The conclusion is correct, but the prose misrepresents the proof structure: it says both branches reach the same terminus when one is eliminated by contradiction.
