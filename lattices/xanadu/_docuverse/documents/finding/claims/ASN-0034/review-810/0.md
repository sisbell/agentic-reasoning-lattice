### T10a.3 postcondition uses undefined function `output`
**Class**: REVISE
**Foundation**: T10a (AllocatorDiscipline) — the only defined function from allocators to tumblers is `dom(A) = {tₙ : n ≥ 0}`, a set; no function named `output` is introduced anywhere in this ASN.

**Issue**: `output` is applied to allocators as though it is a total function `𝒯 → T`, but it is never defined. `dom(A)` produces a set of tumblers, not a single representative tumbler. The informal prose in Consequence 3 establishes the correct claim (child domain lengths strictly exceed parent domain lengths), but the formal postcondition as written is not derivable or usable in its current form.
