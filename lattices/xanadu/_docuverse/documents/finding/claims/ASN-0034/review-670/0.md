### NAT-closure: `0 < 1` is a redundant axiom
**Class**: OBSERVE
**Foundation**: NAT-closure (NatArithmeticClosureAndIdentity)
**ASN**: Formal Contract, Axiom bullet — `0 < 1 (distinctness of the two named constants)`
**Issue**: The body text itself states that `0 < 1` is "recovered as the case n := 0 together with the left-identity rewrite `0 + 1 = 1`." That derivation is correct: left-identity at n=1 gives `0 + 1 = 1`; successor-positivity at n=0 gives `0 < 0 + 1`; substitution gives `0 < 1`. So `0 < 1` is derivable from the other axioms in the same contract and carries no independent axiomatic force. The body acknowledges this but lists it as an axiom anyway.
