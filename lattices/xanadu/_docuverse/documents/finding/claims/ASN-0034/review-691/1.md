### T1 transitivity — component-existence reasoning runs backwards
**Class**: OBSERVE
**Foundation**: T1 (LexicographicOrder), Postcondition (c) Transitivity
**ASN**: T1 — transitivity proof, Case k₁ < k₂: *"the existence of `cₖ₁` gives `k₁ ≤ p`"*
**Issue**: The sentence infers the index bound from the component's existence, but a component exists precisely when the index is in range — so the sentence is circular as stated. The correct direction is: from the witness structure of `b < c`, either T1(i) gives `k₂ ≤ p`, whence `k₁ < k₂ ≤ p` and `k₁ ≤ p`; or T1(ii) gives `k₂ = n + 1 ≤ p`, so `k₁ < n + 1` and `k₁ ≤ n < p`, hence `k₁ ≤ p`. The same gap appears in the T1(ii) sub-case ("and `cₖ₁` exists, so `m + 1 ≤ p`"). The conclusion is correct in both cases; the direction of the argument is inverted.
