### T3 main statement: biconditional notation parses as conditional
**Class**: OBSERVE
**Foundation**: T3 (CanonicalRepresentation)
**ASN**: `(A a, b ∈ T : #a = #b ∧ (A i : 1 ≤ i ≤ #a : aᵢ = bᵢ) ≡ a = b)`
**Issue**: In Dijkstra notation, `(A x : R : B)` reads "for all x satisfying R, B holds." The single `:` before `≡ a = b` parses the guard as the entire predicate `#a = #b ∧ ... ≡ a = b`, making the body empty — or alternatively, parses it as the forward direction only with `a = b` as body. The intent is a biconditional over all pairs, and the formal contract postcondition (`a = b ⟺ …`) is unambiguous. The headline statement is not.
