**Class**: OBSERVE
**Foundation**: T4 (HierarchicalParsing) — *Definition* slot, *Canonical written form*

> "`k = 3`: `t = N₁. ... .Nₐ . 0 . U₁. ... .Uᵦ . 0 . D₁. ... .Dᵧ . 0 . E₁. ... .Eδ`, with `0 < Nᵢ, 0 < Uⱼ, 0 < Dₖ, 0 < Eₗ` at every position present."
