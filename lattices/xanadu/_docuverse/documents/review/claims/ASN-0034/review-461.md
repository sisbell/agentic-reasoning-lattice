### Both-sided coverage applied to inverses and compatibility but not to telescoping or addition-dominance
**Class**: REVISE
**Foundation**: n/a
**ASN**: NAT-sub axiom ‚Äî right telescoping `(A m, n ‚àà ‚Ñï :: (m + n) ‚àí n = m)` is listed without a left counterpart `(A m, n ‚àà ‚Ñï :: (n + m) ‚àí n = m)`. NAT-addbound ‚Äî `(A m, n ‚àà ‚Ñï :: m + n ‚â• n)` is stated right-operand-only, with no `m + n ‚â• m` companion. Meanwhile NAT-sub's prose explicitly declares the principle: "Both-sided coverage is the foundation's deliberate substitute for the absent commutativity axiom," and cites its use for inverse characterisations and for NAT-addcompat's compatibility pair and NAT-cancel's cancellation pair.
**Issue**: The both-sided principle is applied inconsistently. Inverse (`(m ‚àí n) + n = m` and `n + (m ‚àí n) = m`), compatibility (`p ‚â§ n ‚üπ m + p ‚â§ m + n` and `p + m ‚â§ n + m`), and cancellation (`m + n = m + p ‚üπ n = p` and `n + m = p + m ‚üπ n = p`) are all doubled. But telescoping and dominance are one-sided. Without commutativity, `(n + m) ‚àí n = m` is not derivable from `(m + n) ‚àí n = m`, and in fact its well-formedness isn't even establishable from the foundation ‚Äî NAT-addbound gives `n + m ‚â• m` but not `n + m ‚â• n`, so `(n + m) ‚àí n` cannot even be shown to lie in `‚àí`'s domain of definition. A downstream consumer that needs `(n + m) ‚àí n = m` ‚Äî a natural obligation when working with sums whose left operand is the subtrahend ‚Äî has no clause to cite and cannot derive it. The same asymmetry applies to NAT-addbound: no `m + n ‚â• m` is exported, and without commutativity `m + n ‚â• n` does not lift to it.
**What needs resolving**: Either add the missing both-sided counterparts (`(n + m) ‚àí n = m` to NAT-sub, `m + n ‚â• m` to NAT-addbound ‚Äî the latter likely requires a left-dominance proof parallel to NAT-addbound's current right-dominance derivation), or state in each prose body that the one-sided form is all the foundation provides and justify why no downstream consumer needs the other side. As written, the declared "deliberate substitute" principle is applied in three places and silently abandoned in two, and the abandonment creates real expressivity gaps (not even statable, let alone provable).
