**TA3-strict (OrderPreservationUnderSubtractionStrict).** `(A a, b, w : a < b вҲ§ a вүҘ w вҲ§ b вүҘ w вҲ§ #a = #b : a вҠ– w < b вҠ– w)`.

Subtracting a common lower bound from two equal-length tumblers preserves strict order.

*Proof.* Given `a, b, w вҲҲ T` with `a < b`, `a вүҘ w`, `b вүҘ w`, `#a = #b`, show `a вҠ– w < b вҠ– w`.

**The form of `a < b`.** Since `#a = #b`, T1 case (ii) (which requires `#a < #b`) is impossible. So `a < b` holds by case (i): there exists a least `j` with `1 вүӨ j вүӨ #a вҲ§ j вүӨ #b` such that `aбөў = bбөў` for all `i < j` and `aвұј < bвұј`. Fix this `j`.

**Well-formedness and common length.** By TA2, `a вҠ– w, b вҠ– w вҲҲ T`. NAT-order's trichotomy on `(#a, #w)` selects one of (Оұ) `#a = #w`, (ОІ) `#a < #w`, (Оі) `#w < #a`, naming `L_{a,w}`; with `#a = #b`, the trichotomy on `(#b, #w)` falls in the same sub-case, naming `L_{b,w} = L_{a,w}`. Write `L = L_{a,w} = L_{b,w}`; ZPD's padded projections `Гў`, `bМӮ`, `Еө` are defined on `{1, ..., L}`. TumblerSub's length postcondition gives `#(a вҠ– w) = #(b вҠ– w) = L`. From the chosen sub-case, `#a вүӨ L` and `#b вүӨ L`, so the T1 witness satisfies `j вүӨ L`.

Proceed by case analysis on the divergence structure of `(a, w)` and `(b, w)`.

If `b` were zero-padded-equal to `w`, then `bМӮбөў = Еөбөў` for all `1 вүӨ i вүӨ L`. Instantiating at `i := d_a` (legal since `d_a вүӨ L`) gives `bМӮ_{d_a} = Еө_{d_a}`. Lifting via padded-projection equality (legal since `i < d_a вүӨ #a = #b`): `aбөў = bбөў` for `1 вүӨ i < d_a`. At `d_a`: TumblerSub's `Гў_{d_a} > Еө_{d_a} = bМӮ_{d_a}` lifts via padded-native equality to `a_{d_a} > b_{d_a}`. So the pair `(b, a)` satisfies T1 case (i) at witness `j' = d_a`, yielding `b < a`; this contradicts `a < b` via T1's trichotomy disjointness clause `В¬(a < b вҲ§ b < a)`. Hence `b` is not zero-padded-equal to `w`, so `d_b = zpd(b, w)` is well-defined with `1 вүӨ d_b вүӨ L`, and TumblerSub's exported postcondition under `b вүҘ w` gives `bМӮ_{d_b} > Еө_{d_b}`. The same padding-zone contradiction (now substituting `bМӮ_{d_b} = 0` from the supposition `d_b > #b` into `bМӮ_{d_b} > Еө_{d_b}` to clash with NAT-zero's lower bound) yields `d_b вүӨ #b`, hence `d_b вүӨ #a`.

By NAT-order trichotomy on `(d_a, d_b) вҲҲ в„• Г— в„•`, exactly one of `d_a = d_b`, `d_a < d_b`, `d_a > d_b` holds.

**Case 1: `d_a = d_b = d`.** Both `d вүӨ #a` and `d вүӨ #b`, so padded projections coincide with native at `d` and earlier (`Гў_d = a_d`, `bМӮ_d = b_d`, and `Гўбөў = aбөў`, `bМӮбөў = bбөў` for `1 вүӨ i вүӨ d`).
