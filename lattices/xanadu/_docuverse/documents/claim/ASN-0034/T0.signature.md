- `#·` — length operator on T; the function `#· : T → ℕ` returning the number of components of a tumbler
- `·ᵢ` — component projection; for each `a ∈ T`, the operator yielding `aᵢ ∈ ℕ` at index `i` in the index domain `{j ∈ ℕ : 1 ≤ j ≤ #a}`
- `T` — the carrier set of tumblers; the set of all nonempty finite sequences over ℕ
