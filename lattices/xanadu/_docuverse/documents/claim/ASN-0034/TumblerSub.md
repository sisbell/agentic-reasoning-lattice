## Tumbler subtraction

**TumblerSub (TumblerSub).** Given two tumblers `a` (minuend) and `w` (subtrahend), compute their component-wise difference at the first point of zero-padded divergence. NAT-order's trichotomy on `(#a, #w)` selects exactly one of: (őĪ) `#a = #w` with `L = #a`; (ő≤) `#a < #w` with `L = #w`; (ő≥) `#w < #a` with `L = #a`. All component references at indices in `{1, ..., L}` use ZPD's *padded projections* `√Ę`, `ŇĶ` (`√ĘŠĶĘ = aŠĶĘ` for `1 ‚Č§ i ‚Č§ #a` and `√ĘŠĶĘ = 0` for `#a < i ‚Č§ L`; symmetrically for `ŇĶ`), so that the bare native symbols `aŠĶĘ`, `wŠĶĘ` retain their T0 meaning on the native domain only. When `√Ę` and `ŇĶ` agree at every position, the result is the zero tumbler of length `L`: `a ‚äĖ w = [0, ..., 0]`. Otherwise, `zpd(a, w)` is defined (ZPD) ‚ÄĒ write `k = zpd(a, w)`. The result is:

```
         ‚éß 0             if i < k
rŠĶĘ   =  ‚é® √Ę‚āĖ - ŇĶ‚āĖ      if i = k
         ‚é© √ĘŠĶĘ           if i > k
```

The result has length `L`.

**Precondition:** `a ‚Č• w` (T1). We prove that when `zpd(a, w)` is defined, this entails `√Ę‚āĖ > ŇĶ‚āĖ` at `k = zpd(a, w)` ‚ÄĒ stated on the padded projections so the inequality is well-typed regardless of whether `k` lies in the native domain of both operands. Since zpd is defined, `a` and `w` are not zero-padded-equal (ZPD): the negation of `(A i : 1 ‚Č§ i ‚Č§ L : √ĘŠĶĘ = ŇĶŠĶĘ)` supplies an index `i` with `1 ‚Č§ i ‚Č§ L` and `√ĘŠĶĘ ‚Č† ŇĶŠĶĘ`. T3's contrapositive `a ‚Č† b ‚üļ #a ‚Č† #b ‚ą® (‚ąÉ j : 1 ‚Č§ j ‚Č§ #a : a‚Īľ ‚Č† b‚Īľ)` speaks of native components and lengths only, so the padded disagreement does not by itself license `a ‚Č† w`; we discharge `a ‚Č† w` by case analysis on whether `i` lies in the shared native domain. (A) `i ‚Č§ #a ‚ąß i ‚Č§ #w`: ZPD's padded-projection definition gives `√ĘŠĶĘ = aŠĶĘ` and `ŇĶŠĶĘ = wŠĶĘ`, hence `aŠĶĘ ‚Č† wŠĶĘ`; T3's contrapositive instantiated at `(a, w)` with witness `j := i` (legal since `1 ‚Č§ i ‚Č§ #a`) fires its existential disjunct, yielding `a ‚Č† w`. (B) `i > #a ‚ą® i > #w` with `i ‚Č§ L`. Sub-case B1 (`i > #a`): ZPD's padding clause gives `√ĘŠĶĘ = 0`, so `√ĘŠĶĘ ‚Č† ŇĶŠĶĘ` forces `ŇĶŠĶĘ ‚Č† 0`, ruling out the padding clause `ŇĶŠĶĘ = 0` (which would apply for `i > #w`) and fixing `i ‚Č§ #w`; combined with `i > #a`, this yields `#a < i ‚Č§ #w`, hence `#a ‚Č† #w`. Sub-case B2 (`i > #w`) is the mirror image: `ŇĶŠĶĘ = 0` and the disagreement force `i ‚Č§ #a`, giving `#w < #a` and `#a ‚Č† #w`. In both sub-cases, T3's contrapositive instantiated at `(a, w)` fires its length disjunct, yielding `a ‚Č† w`. In either case (A) or (B), `a ‚Č† w`; combined with `a ‚Č• w`, this yields `w < a` (T1). Two Divergence cases arise for the pair `(w, a)` with `w ‚Č† a`:

  (i) Component divergence at position `k` with `k ‚Č§ #w ‚ąß k ‚Č§ #a` and `w‚āĖ ‚Č† a‚āĖ` (native projections, well-defined since `k` lies in both native domains). ZPD's Relationship-to-Divergence gives `zpd(a, w) = divergence(a, w) = k`. From `w < a` (T1), a witness `j` exists in one of T1's two cases for the pair `(w, a)`. T1 case (ii) is eliminated: it would require `j = #w + 1 ‚Č§ #a` with `wŠĶĘ = aŠĶĘ` for `1 ‚Č§ i < j`, i.e., for `1 ‚Č§ i ‚Č§ #w`; instantiating this prefix agreement at `i := k` ‚ÄĒ legal since `k ‚Č§ #w` from Divergence case (i) ‚ÄĒ yields `w‚āĖ = a‚āĖ`, contradicting `w‚āĖ ‚Č† a‚āĖ`. So T1 case (i) holds: `j ‚Č§ #w ‚ąß j ‚Č§ #a`, `w‚Īľ < a‚Īľ`, and `wŠĶĘ = aŠĶĘ` for `1 ‚Č§ i < j`. NAT-order's disjointness-of-`<`-and-`=` at `(w‚Īľ, a‚Īľ)` converts `w‚Īľ < a‚Īľ` into `w‚Īľ ‚Č† a‚Īľ`, so the position `j` satisfies Divergence case (i)'s conjunction `1 ‚Č§ j ‚ąß j ‚Č§ #w ‚ąß j ‚Č§ #a ‚ąß w‚Īľ ‚Č† a‚Īľ ‚ąß (A i : 1 ‚Č§ i < j : wŠĶĘ = aŠĶĘ)` for the pair `(w, a)`; Divergence's uniqueness clause for case (i) identifies `j = k`. Hence `w‚āĖ < a‚āĖ`, whence `a‚āĖ > w‚āĖ`. ZPD's padded projections coincide with native here (`√Ę‚āĖ = a‚āĖ` from `k ‚Č§ #a`, `ŇĶ‚āĖ = w‚āĖ` from `k ‚Č§ #w`), lifting the inequality to `√Ę‚āĖ > ŇĶ‚āĖ`.

  (ii) Prefix divergence splits via NAT-order's trichotomy on `(#w, #a)` into sub-case (ii-a) `#w < #a` with `wŠĶĘ = aŠĶĘ` for `1 ‚Č§ i ‚Č§ #w` (native projections on the shared native domain), and sub-case (ii-b) `#a < #w` with `wŠĶĘ = aŠĶĘ` for `1 ‚Č§ i ‚Č§ #a`. Sub-case (ii-b) is eliminated: its prefix hypothesis yields `a < w` via T1 case (ii), contradicting `w < a`. In sub-case (ii-a), `w` is a proper prefix of `a`, so `L = #a`. ZPD's padded projection sets `ŇĶŠĶĘ = 0` for `#w < i ‚Č§ L` (and `ŇĶŠĶĘ = wŠĶĘ = aŠĶĘ = √ĘŠĶĘ` for `1 ‚Č§ i ‚Č§ #w` by the prefix agreement). Since zpd is defined, `√Ę` and `ŇĶ` disagree somewhere (ZPD, contrapositive); the prefix agreement rules out positions `1 ‚Č§ i ‚Č§ #w`, so the disagreement lies at some `i > #w`. By ZPD's minimality, `k > #w`, whence `ŇĶ‚āĖ = 0` by zero-padding and `√Ę‚āĖ ‚Č† 0`. From NAT-zero's `0 ‚Č§ √Ę‚āĖ` and NAT-order's `m ‚Č§ n ‚üļ m < n ‚ą® m = n`, the divergence `√Ę‚āĖ ‚Č† 0` leaves `0 < √Ę‚āĖ`; hence `√Ę‚āĖ > 0 = ŇĶ‚āĖ`.

In both cases `√Ę‚āĖ > ŇĶ‚āĖ` at `k = zpd(a, w)`. When zpd is undefined, the consequence is vacuous.  ‚ąé

Each component of the result is a natural number: for `i < k`, `rŠĶĘ = 0 ‚ąą ‚Ąē` by NAT-zero; at the divergence point, `r‚āĖ = √Ę‚āĖ ‚ąí ŇĶ‚āĖ ‚ąą ‚Ąē` by NAT-sub, whose precondition `√Ę‚āĖ ‚Č• ŇĶ‚āĖ` follows from `√Ę‚āĖ > ŇĶ‚āĖ` via NAT-order; for `i > k`, `rŠĶĘ = √ĘŠĶĘ`, which is `aŠĶĘ ‚ąą ‚Ąē` (T0) when `i ‚Č§ #a` and `0 ‚ąą ‚Ąē` (NAT-zero) when `i > #a`, by ZPD's padded-projection definition. In the no-divergence case every component is `0 ‚ąą ‚Ąē`. The length `L ‚Č• 1` since T0 gives `#a ‚Č• 1` and `#w ‚Č• 1`, and `L` is named by the trichotomy as one of `#a` or `#w`. Hence **`a ‚äĖ w ‚ąą T`** by T0.

When `zpd(a, w)` is defined ‚ÄĒ write `k = zpd(a, w)` ‚ÄĒ components before `k` are zero by construction. To discharge TA-Pos we exhibit `k` as an index with `¬¨(r‚āĖ = 0)`. NAT-sub's right-inverse characterisation, at `(√Ę‚āĖ, ŇĶ‚āĖ)` with precondition `√Ę‚āĖ ‚Č• ŇĶ‚āĖ` already established above from `√Ę‚āĖ > ŇĶ‚āĖ` via NAT-order, yields `(√Ę‚āĖ ‚ąí ŇĶ‚āĖ) + ŇĶ‚āĖ = √Ę‚āĖ` ‚ÄĒ that is, `r‚āĖ + ŇĶ‚āĖ = √Ę‚āĖ`. Suppose, for contradiction, `r‚āĖ = 0`: NAT-closure's left additive identity instantiated at `n := ŇĶ‚āĖ` rewrites the left-hand side via `0 + ŇĶ‚āĖ = ŇĶ‚āĖ`, giving `ŇĶ‚āĖ = √Ę‚āĖ`. But `√Ę‚āĖ > ŇĶ‚āĖ` unfolds through NAT-order's `>` definition `m > n ‚üļ n < m` to `ŇĶ‚āĖ < √Ę‚āĖ`, and NAT-order's disjointness-of-`<`-and-`=` clause applied at `(ŇĶ‚āĖ, √Ę‚āĖ)` then forces `ŇĶ‚āĖ ‚Č† √Ę‚āĖ`, contradicting `ŇĶ‚āĖ = √Ę‚āĖ`. Hence `¬¨(r‚āĖ = 0)`. ZPD's codomain places `k` in `{1, ..., L}`, and `#(a ‚äĖ w) = L` gives `1 ‚Č§ k ‚Č§ #(a ‚äĖ w)`, so `k` is a valid existential witness; whence **`Pos(a ‚äĖ w)`** (TA-Pos). ActionPoint names `actionPoint(a ‚äĖ w)` as the unique `m ‚ąą S := {i : 1 ‚Č§ i ‚Č§ #(a ‚äĖ w) ‚ąß rŠĶĘ ‚Č† 0}` with `(A n ‚ąą S :: m ‚Č§ n)`; to identify this minimum with `k` we discharge both membership and the least-element clause. Membership: `1 ‚Č§ k ‚Č§ #(a ‚äĖ w)` and `r‚āĖ ‚Č† 0` just established jointly give `k ‚ąą S`. Least-element: take any `n ‚ąą S` and suppose for contradiction `n < k`. The Definition fixes `rŠĶĘ = 0` for every `i` with `1 ‚Č§ i < k`; instantiating at `i := n` yields `r‚āô = 0`, contradicting `r‚āô ‚Č† 0` from `n ‚ąą S`. Hence `¬¨(n < k)`, and NAT-order's at-least-one trichotomy at `(k, n)` gives `k < n ‚ą® k = n ‚ą® n < k`; the excluded disjunct drops out, and NAT-order's defining clause `k ‚Č§ n ‚üļ k < n ‚ą® k = n` folds the remainder to `k ‚Č§ n`. Hence `(A n ‚ąą S :: k ‚Č§ n)`, and ActionPoint's uniqueness clause names `k` as the action point: **`actionPoint(a ‚äĖ w) = k = zpd(a, w)`**.

When `zpd(a, w)` is undefined, the Definition fixes `a ‚äĖ w = [0, ‚Ä¶, 0]` of length `L`, so every component satisfies `rŠĶĘ = 0` for `1 ‚Č§ i ‚Č§ L`. Using the postcondition `#(a ‚äĖ w) = L` to retype the index range, the universal `(A i ‚ąą ‚Ąē : 1 ‚Č§ i ‚Č§ #(a ‚äĖ w) : rŠĶĘ = 0)` holds, discharging **`Zero(a ‚äĖ w)`** (TA-Pos). TA-Pos's complementarity `Pos(t) ‚üļ ¬¨Zero(t)` then yields `¬¨Pos(a ‚äĖ w)`, so the previous paragraph's Pos witness and ActionPoint identification are inapplicable here; ActionPoint's `Pos(w)` precondition fails at `w := a ‚äĖ w`, leaving `actionPoint(a ‚äĖ w)` unspecified.
