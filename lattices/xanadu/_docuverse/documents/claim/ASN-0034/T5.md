## Contiguous subtrees

**T5 (ContiguousSubtrees).** For any tumbler prefix `p`, the set `{t ‚àà T : p ‚âº t}` forms a contiguous interval under T1:

  `[p ‚âº a ‚àß p ‚âº c ‚àß a ‚â§ b ‚â§ c ‚üπ p ‚âº b]`

*Proof.* Let `p` be a tumbler prefix with `#p ‚â• 1`, let `a, b, c ‚àà T` with `p ‚âº a`, `p ‚âº c`, and `a ‚â§ b ‚â§ c` under T1. We must show `p ‚âº b`.

Since `p ‚âº a`, by Prefix, `#a ‚â• #p` and `a·µ¢ = p·µ¢` for all `1 ‚â§ i ‚â§ #p`. Likewise, from `p ‚âº c`, Prefix gives `#c ‚â• #p` and `c·µ¢ = p·µ¢` for all `1 ‚â§ i ‚â§ #p`. We proceed by case analysis on the length of `b`.

*Case 1: `#b ‚â• #p`.* We show `b·µ¢ = p·µ¢` for all `1 ‚â§ i ‚â§ #p`, which is `p ‚âº b`.

Suppose, for contradiction, that `b` diverges from `p` at some position. The set of indices in `{1, ..., #p}` at which `b‚Çñ ‚âÝ p‚Çñ` is nonempty; NAT-wellorder's least-element principle delivers its minimum `k`, giving `b·µ¢ = p·µ¢` for all `1 ‚â§ i < k` together with `b‚Çñ ‚âÝ p‚Çñ`. NAT-order's trichotomy at `(b‚Çñ, p‚Çñ)` then resolves `b‚Çñ ‚âÝ p‚Çñ` into exactly one of `b‚Çñ < p‚Çñ` or `b‚Çñ > p‚Çñ`.

*Subcase 1a: `b‚Çñ < p‚Çñ`.* Since `p ‚âº a`, `a‚Çñ = p‚Çñ`, so `b‚Çñ < a‚Çñ`. For all `i < k`, `b·µ¢ = p·µ¢ = a·µ¢`. Position `k` is the first divergence between `a` and `b`, with `b‚Çñ < a‚Çñ`, and `k ‚â§ #p ‚â§ min(#a, #b)`. By T1 case (i), `b < a`. Now `a ‚â§ b` abbreviates `a < b ‚à® a = b`: the disjunct `a < b` together with `b < a` violates T1 postcondition (b) trichotomy's clause `¬¨(a < b ‚àß b < a)`; the disjunct `a = b` substitutes into `b < a` to yield `a < a`, violating T1 postcondition (a) irreflexivity. Either disjunct produces a contradiction.

*Subcase 1b: `b‚Çñ > p‚Çñ`.* Since `p ‚âº c`, `c‚Çñ = p‚Çñ`, so `b‚Çñ > c‚Çñ`. For all `i < k`, `b·µ¢ = p·µ¢ = c·µ¢`, and `k ‚â§ #p ‚â§ min(#b, #c)`. By T1 case (i), `c < b`. Now `b ‚â§ c` abbreviates `b < c ‚à® b = c`: the disjunct `b < c` together with `c < b` violates T1 postcondition (b) trichotomy's clause `¬¨(a < b ‚àß b < a)` instantiated at `(b, c)`; the disjunct `b = c` substitutes into `c < b` to yield `b < b`, violating T1 postcondition (a) irreflexivity. Either disjunct produces a contradiction.

Both subcases yield contradictions, so `b·µ¢ = p·µ¢` for all `1 ‚â§ i ‚â§ #p`, giving `p ‚âº b`.

*Case 2: `#b < #p`.* We derive a contradiction.

From `p ‚âº a`, `#a ‚â• #p > #b`. By T3, `#a > #b` implies `a ‚âÝ b`. Since `#a > #b`, `a` cannot be a proper prefix of `b`. Thus `a ‚â§ b` means `a < b`, which by T1 requires a witness `k ‚â• 1` with `a·µ¢ = b·µ¢` for all `i < k` and either: (i) `k ‚â§ min(#a, #b) = #b` and `a‚Çñ < b‚Çñ`, or (ii) `k = #a + 1 ‚â§ #b`. Case (ii) supplies `#a + 1 ‚â§ #b`; NAT-addcompat's strict successor inequality `#a < #a + 1` chains with this through NAT-order's definition of `‚â§` ‚Äî unfolding `#a + 1 ‚â§ #b` into `#a + 1 < #b ‚à® #a + 1 = #b`, the first disjunct closing with `#a < #a + 1` under NAT-order's `<`-transitivity and the second substituting into `#a < #a + 1` to give `#a < #b` directly ‚Äî yielding `#a < #b`, which contradicts `#a > #b` via NAT-order's exactly-one trichotomy clause `¬¨(m < n ‚àß n < m)` at `(#a, #b)`. So case (i) holds: there exists `k ‚â§ #b` with `a·µ¢ = b·µ¢` for all `1 ‚â§ i < k` and `a‚Çñ < b‚Çñ`.

Since `k ‚â§ #b < #p ‚â§ #a`, position `k` lies within `p`, so `a‚Çñ = p‚Çñ`. Therefore `b‚Çñ > a‚Çñ = p‚Çñ`. Likewise, since `k < #p ‚â§ #c`, `c‚Çñ = p‚Çñ`, so `b‚Çñ > c‚Çñ`.

For all `i < k`: `b·µ¢ = a·µ¢ = p·µ¢ = c·µ¢`. At position `k`, `b‚Çñ > c‚Çñ`, and `k ‚â§ min(#b, #c)`. By T1 case (i), `c < b`. Now `b ‚â§ c` abbreviates `b < c ‚à® b = c`: the disjunct `b < c` together with `c < b` violates T1 postcondition (b) trichotomy's clause `¬¨(a < b ‚àß b < a)` instantiated at `(b, c)`; the disjunct `b = c` substitutes into `c < b` to yield `b < b`, violating T1 postcondition (a) irreflexivity. Either disjunct produces a contradiction.

Since Case 2 is impossible, `p ‚âº b` holds in all cases. ‚àé

*Formal Contract:*
- *Preconditions:* `a, b, c ‚àà T`; `p` is a tumbler prefix with `#p ‚â• 1`; `p ‚âº a`; `p ‚âº c`; `a ‚â§ b ‚â§ c` under T1.
- *Depends:*
  - T0 (CarrierSetDefinition) ‚Äî supplies the length operator `#¬∑` on `b` for the Case 1 hypothesis `#b ‚â• #p` and the Case 2 hypothesis `#b < #p`, and the component projection `i ‚Ü¶ b·µ¢` used in Case 1 to construct the divergence-index set `{k ‚àà {1, ..., #p} : b‚Çñ ‚âÝ p‚Çñ}`. Neither operator is supplied for arbitrary `b ‚àà T` by Prefix (which only unfolds prefix relations on `a`, `c`) or T3 (which speaks to equality from component agreement, not the projection itself).
  - Prefix (PrefixRelation) ‚Äî unfolds `p ‚âº a`, `p ‚âº c` into length and component-wise equalities; re-folds component-wise agreement into `p ‚âº b`.
  - T1 (LexicographicOrder), case (i) ‚Äî derives contradictions `b < a` and `c < b` from divergence-position witnesses.
  - T1 (LexicographicOrder), case (ii) ‚Äî supplies `k = #a + 1 ‚â§ #b` in Case 2; combined with NAT-addcompat's `#a < #a + 1` this yields `#a < #b`, which contradicts `#a > #b` and excludes case (ii).
  - T1 (LexicographicOrder), postcondition (a) irreflexivity ‚Äî closes the `a = b` disjunct of `a ‚â§ b` (Subcase 1a) and the `b = c` disjunct of `b ‚â§ c` (Subcase 1b and Case 2) by substituting the derived `b < a` / `c < b` into `a < a` / `b < b`.
  - T1 (LexicographicOrder), postcondition (b) trichotomy ‚Äî closes the `a < b` disjunct of `a ‚â§ b` (Subcase 1a) and the `b < c` disjunct of `b ‚â§ c` (Subcase 1b and Case 2) via the clause `¬¨(a < b ‚àß b < a)` at `(a, b)` and `(b, c)` respectively.
  - T3 (CanonicalRepresentation) ‚Äî distinct lengths imply distinct tumblers, giving `a ‚âÝ b` in Case 2.
  - NAT-order (NatStrictTotalOrder) ‚Äî trichotomy at `(b‚Çñ, p‚Çñ)` in Case 1 dichotomizes `b‚Çñ ‚âÝ p‚Çñ` into `b‚Çñ < p‚Çñ ‚à® b‚Çñ > p‚Çñ`; the `‚â§`/`<` clauses on ‚Ñï underwrite the length reasoning throughout (`#a ‚â• #p`, `#c ‚â• #p`, `#b < #p`, `k ‚â§ #p ‚â§ min(#a, #b)`); the exclusion of T1(ii) in Case 2 unfolds `#a + 1 ‚â§ #b` by NAT-order's definition of `‚â§` and applies either `<`-transitivity (on `#a < #a + 1 < #b`) or equality substitution (on `#a + 1 = #b`) to reach `#a < #b`, then closes via exactly-one trichotomy's `¬¨(m < n ‚àß n < m)` clause at `(#a, #b)` against `#a > #b`.
  - NAT-addcompat (NatAdditionOrderAndSuccessor) ‚Äî supplies the strict successor inequality `#a < #a + 1`, used in Case 2 to promote T1(ii)'s `#a + 1 ‚â§ #b` to `#a < #b` and derive the contradiction with `#a > #b`.
  - NAT-wellorder (NatWellOrdering) ‚Äî least-element principle in Case 1, invoked on the nonempty set of indices `k ‚àà {1, ..., #p}` with `b‚Çñ ‚âÝ p‚Çñ` to select the first divergence index.
- *Postconditions:* `p ‚âº b` ‚Äî `b` extends `p` and belongs to the same subtree as `a` and `c`.

Nelson: "The first point of a span may designate a server, an account, a document or an element; so may the last point. There is no choice as to what lies between; this is implicit in the choice of first and last point." A span between two endpoints under the same prefix captures exactly the addresses under that prefix between those endpoints ‚Äî no addresses from unrelated subtrees can interleave.

Because the hierarchy is projected onto a flat line (T1), containment in the tree corresponds to contiguity on the line. Nelson: "A span may be visualized as a zone hanging down from the tumbler line ‚Äî what is called in computer parlance a depth-first spanning tree." Every subtree maps to a contiguous range, and every contiguous range within a subtree stays within the subtree.
