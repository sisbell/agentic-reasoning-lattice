**NAT-order (NatStrictTotalOrder).** Posited on the carrier `вДХ` вАФ the set introduced by NAT-carrier вАФ the binary relation `<` is a strict total order, with non-strict companion `вЙ§` defined by `m вЙ§ n вЯЇ m < n вИ® m = n` and reverse companions `вЙ•` and `>` defined by `m вЙ• n вЯЇ n вЙ§ m` and `m > n вЯЇ n < m`.

Strict total order on вДХ means three clauses hold jointly:
- Irreflexivity: `¬ђ(n < n)` for every `n вИИ вДХ`
- Transitivity: `m < n вИІ n < p вЯє m < p` for every `m, n, p вИИ вДХ`
- At-least-one trichotomy: for any `m, n вИИ вДХ`, at least one of `m < n`, `m = n`, `n < m` holds

Together the three clauses yield *exactly-one trichotomy*: for any `m, n вИИ вДХ`, exactly one of `m < n`, `m = n`, `n < m` holds. Exactly-one is the at-least-one disjunction conjoined with three pairwise mutual-exclusion clauses. `¬ђ(m < n вИІ n < m)` follows from transitivity вАФ which chains `m < n` and `n < m` to `m < m` вАФ against irreflexivity. `¬ђ(m < n вИІ m = n)` follows by substituting `m = n` into `m < n` via indiscernibility of `=`, rewriting to `m < m`, which irreflexivity at `n := m` rules out. `¬ђ(m = n вИІ n < m)` follows by the same substitution applied to `n < m`: rewriting under `m = n` yields `m < m`, again against irreflexivity.

The non-strict companion `вЙ§` inherits transitivity from `<` and `=` jointly: `m вЙ§ n вИІ n вЙ§ p вЯє m вЙ§ p`. Unfolding each hypothesis against the defining disjunction `x вЙ§ y вЯЇ x < y вИ® x = y` yields four cases, and each discharges to `m вЙ§ p`. If both are strict вАФ `m < n вИІ n < p` вАФ `<`-transitivity gives `m < p`, so `m вЙ§ p` by the definition. If the first is strict and the second is equality вАФ `m < n вИІ n = p` вАФ indiscernibility of `=` substitutes `n = p` into `m < n` to yield `m < p`. The symmetric case `m = n вИІ n < p` substitutes `m = n` into `n < p` to yield `m < p`. If both are equalities вАФ `m = n вИІ n = p` вАФ transitivity of `=` gives `m = p`, so `m вЙ§ p`.

Unfolding each hypothesis by the definition `x вЙ§ y вЯЇ x < y вИ® x = y` yields four cases. `m < n вИІ n < p` gives `m < p` by `<`-transitivity, hence `m вЙ§ p`. `m < n вИІ n = p` gives `m < p` by substituting `n = p` into `m < n` via indiscernibility of `=`, hence `m вЙ§ p`. `m = n вИІ n < p` gives `m < p` by substituting `m = n` into `n < p`, hence `m вЙ§ p`. `m = n вИІ n = p` gives `m = p` by transitivity of `=`, hence `m вЙ§ p`.
