**Divergence (Divergence).** For tumblers `a, b ‚ąą T` with `a ‚Č† b`, the *divergence* `divergence(a, b)` is defined by two cases reflecting the two ways distinct finite sequences over ‚Ąē can fail to coincide: at a shared position by component mismatch, or by one being a proper prefix of the other.

  (i) If there exists `k` with `1 ‚Č§ k ‚ąß k ‚Č§ #a ‚ąß k ‚Č§ #b` and `a‚āĖ ‚Č† b‚āĖ`, then `divergence(a, b)` is the least such `k` ‚ÄĒ equivalently, the unique `k` satisfying `1 ‚Č§ k ‚ąß k ‚Č§ #a ‚ąß k ‚Č§ #b ‚ąß a‚āĖ ‚Č† b‚āĖ ‚ąß (A i : 1 ‚Č§ i < k : aŠĶĘ = bŠĶĘ)`, the universal conjunct restating minimality ‚ÄĒ component divergence at a shared position.

  (ii) If `#a ‚Č† #b ‚ąß (A i : 1 ‚Č§ i ‚Č§ #a ‚ąß i ‚Č§ #b : aŠĶĘ = bŠĶĘ)` ‚ÄĒ equivalently, lengths differ and case (i) does not apply ‚ÄĒ then NAT-order's trichotomy applied to `(#a, #b)` rules out the `#a = #b` branch and splits on which length is shorter. In sub-case (ii-a), `#a < #b`. To see that `i ‚Č§ #a` entails `i ‚Č§ #b`, unfold `i ‚Č§ #a` against NAT-order's definition `x ‚Č§ y ‚üļ x < y ‚ą® x = y` into two branches: in the strict branch `i < #a`, NAT-order's `<`-transitivity at `(i, #a, #b)` with `#a < #b` gives `i < #b`; in the equality branch `i = #a`, indiscernibility of `=` substitutes `i = #a` into `#a < #b` to yield `i < #b`. Either way `i < #b`, hence `i ‚Č§ #b` by the same definition. The range `1 ‚Č§ i ‚Č§ #a ‚ąß i ‚Č§ #b` therefore reduces to `1 ‚Č§ i ‚Č§ #a`, yielding `(A i : 1 ‚Č§ i ‚Č§ #a : aŠĶĘ = bŠĶĘ)`, whence `divergence(a, b) = #a + 1`. In sub-case (ii-b), `#b < #a`. To see that `i ‚Č§ #b` entails `i ‚Č§ #a`, unfold `i ‚Č§ #b` similarly: in the strict branch `i < #b`, `<`-transitivity at `(i, #b, #a)` with `#b < #a` gives `i < #a`; in the equality branch `i = #b`, indiscernibility of `=` substitutes `i = #b` into `#b < #a` to yield `i < #a`. Either way `i < #a`, hence `i ‚Č§ #a` by the definition. The range `1 ‚Č§ i ‚Č§ #a ‚ąß i ‚Č§ #b` therefore reduces to `1 ‚Č§ i ‚Č§ #b`, yielding `(A i : 1 ‚Č§ i ‚Č§ #b : aŠĶĘ = bŠĶĘ)`, whence `divergence(a, b) = #b + 1`. In either sub-case the divergence lies one position past the shorter tumbler's last component ‚ÄĒ prefix divergence, where one tumbler is a proper prefix of the other.

Exactly one case applies for any `a ‚Č† b`. Mutual exclusivity: if case (i) holds, some `a‚āĖ ‚Č† b‚āĖ` with `k ‚Č§ #a ‚ąß k ‚Č§ #b` falsifies case (ii)'s universal agreement at shared positions. Exhaustiveness: if neither case applies, all shared components agree and `#a = #b`, so by T3, `a = b`, contradicting `a ‚Č† b`.

The function is symmetric: `divergence(a, b) = divergence(b, a)`. In case (i), the qualifying conjunction is invariant under operand swap ‚ÄĒ `1 ‚Č§ k` mentions neither operand, `k ‚Č§ #a ‚ąß k ‚Č§ #b` by `‚ąß`-commutativity, `a‚āĖ ‚Č† b‚āĖ` by `‚Č†`-symmetry, `(A i : 1 ‚Č§ i < k : aŠĶĘ = bŠĶĘ)` by `=`-symmetry ‚ÄĒ so the same `k` witnesses case (i) under swap. In case (ii), swapping `(a, b)` exchanges sub-cases (ii-a) and (ii-b); both select one-plus the shorter tumbler's length.

*Formal Contract:*
- *Preconditions:* a ‚ąą T, b ‚ąą T, a ‚Č† b
- *Definition:* (i) if `(‚ąÉ k : 1 ‚Č§ k ‚ąß k ‚Č§ #a ‚ąß k ‚Č§ #b : a‚āĖ ‚Č† b‚āĖ)`, then `divergence(a, b)` is the least `k` satisfying `1 ‚Č§ k ‚ąß k ‚Č§ #a ‚ąß k ‚Č§ #b ‚ąß a‚āĖ ‚Č† b‚āĖ ‚ąß (A i : 1 ‚Č§ i < k : aŠĶĘ = bŠĶĘ)` (equivalently, the unique such `k`, the universal conjunct being the minimality condition restated); (ii) if `#a ‚Č† #b ‚ąß (A i : 1 ‚Č§ i ‚Č§ #a ‚ąß i ‚Č§ #b : aŠĶĘ = bŠĶĘ)`, then `divergence(a, b) = #a + 1` when `#a < #b` (sub-case (ii-a)) and `divergence(a, b) = #b + 1` when `#b < #a` (sub-case (ii-b)).
- *Depends:*
  - T0 (CarrierSetDefinition) ‚ÄĒ supplies `a, b ‚ąą T`, lengths `#a, #b`, and component projections `a‚āĖ, b‚āĖ, aŠĶĘ, bŠĶĘ` as ‚Ąē-valued, making component (in)equalities well-formed.
  - T3 (CanonicalRepresentation) ‚ÄĒ exhaustiveness: if neither case applies, all shared components agree and `#a = #b`, so `a = b`, contradicting `a ‚Č† b`.
  - NAT-order (NatStrictTotalOrder) ‚ÄĒ trichotomy at length pair `(#a, #b)` splits case (ii) into sub-cases (ii-a)/(ii-b); the definition `x ‚Č§ y ‚üļ x < y ‚ą® x = y` and `<`-transitivity together discharge the mixed `‚Č§`-`<` chain showing `i ‚Č§ #a ‚áí i ‚Č§ #b` under `#a < #b` (and the symmetric chain under `#b < #a`) by case-splitting on the unfolded `‚Č§` and applying `<`-transitivity in the strict branch and `=`-substitution in the equality branch.
  - NAT-wellorder (NatWellOrdering) ‚ÄĒ existence of a least element in the nonempty subset `{k ‚ąą ‚Ąē : 1 ‚Č§ k ‚ąß k ‚Č§ #a ‚ąß k ‚Č§ #b ‚ąß a‚āĖ ‚Č† b‚āĖ}` grounds case (i)'s designating description, so "the least such `k`" is non-vacuous.
  - NAT-closure (NatArithmeticClosureAndIdentity) ‚ÄĒ addition closure instantiated at `(#a, 1)` and `(#b, 1)`, with `1 ‚ąą ‚Ąē` from the same axiom, well-types case (ii)'s values `#a + 1` and `#b + 1` as ‚Ąē.
- *Postconditions:* `divergence(a, b) ‚ąą ‚Ąē`; exactly one of case (i) or case (ii) applies; in case (ii), `divergence(a, b) = #a + 1` in sub-case (ii-a) and `divergence(a, b) = #b + 1` in sub-case (ii-b); `divergence(a, b) = divergence(b, a)` for all `a ‚Č† b`.
