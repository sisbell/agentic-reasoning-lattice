- `<` — lexicographic strict total order on T; defined by component-wise ℕ comparison with the proper-prefix convention (prefix is less than any proper extension)
- `≤` — non-strict companion order on T; abbreviates `a < b ∨ a = b` for tumblers
- `≥` — reverse non-strict order on T; abbreviates `b ≤ a` for tumblers
- `>` — reverse strict order on T; abbreviates `b < a` for tumblers
