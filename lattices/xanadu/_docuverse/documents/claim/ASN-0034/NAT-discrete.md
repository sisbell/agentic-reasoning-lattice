**NAT-discrete (NatDiscreteness).** For every `m, n ‚ąą ‚Ąē`, `m < n ‚üĻ m + 1 ‚Č§ n`.

No natural number lies strictly between `n` and its successor `n + 1`: whenever `m < n`, the successor `m + 1` is already bounded above by `n`, so `m + 1 ‚Č§ n`. This is the discreteness of ‚Ąē.

The axiom has a familiar no-interval reformulation `(A m, n ‚ąą ‚Ąē :: m ‚Č§ n < m + 1 ‚üĻ n = m)` ‚ÄĒ "nothing strictly between `m` and `m + 1`". The derivation (axiom ‚üĻ no-interval): assume `m ‚Č§ n < m + 1`; unfolding `m ‚Č§ n` by the NAT-order definition `m ‚Č§ n ‚üļ m < n ‚ą® m = n` splits into two cases, each of which we walk to `n = m`. In the case `m < n`, the axiom gives `m + 1 ‚Č§ n`, which unfolds to `m + 1 < n ‚ą® m + 1 = n`; paired with the hypothesis `n < m + 1`, either disjunct is contradictory ‚ÄĒ `m + 1 < n` together with `n < m + 1` is excluded by NAT-order's exactly-one trichotomy (specifically its `¬¨(a < b ‚ąß b < a)` clause, instantiated at `(m + 1, n)`), while `m + 1 = n` rewrites `n < m + 1` to `n < n`, contradicting irreflexivity. The case `m < n` is therefore impossible; it contributes nothing to the conclusion and is discharged by contradiction. In the case `m = n`, symmetry of `=` gives `n = m` directly. Both branches of the split terminate in `n = m`, closing the case analysis. ‚ąé

*Formal Contract:*
- *Axiom:* `(A m, n ‚ąą ‚Ąē :: m < n ‚üĻ m + 1 ‚Č§ n)` (discreteness).
- *Consequence:* `(A m, n ‚ąą ‚Ąē :: m ‚Č§ n < m + 1 ‚üĻ n = m)` (no-interval form) ‚ÄĒ derived from the axiom together with NAT-order (the `‚Č§`-definition used to split `m ‚Č§ n`, the exactly-one-trichotomy clause `¬¨(a < b ‚ąß b < a)` instantiated at `(m + 1, n)`, and irreflexivity `¬¨(n < n)` after rewriting `n < m + 1` to `n < n` via `m + 1 = n`) via the forward walk in the preceding prose.
- *Depends:*
  - NAT-carrier (NatCarrierSet) ‚ÄĒ supplies `‚Ąē` as the underlying set appearing in the carrier-side clause `m, n ‚ąą ‚Ąē` of the bounded universal `(A m, n ‚ąą ‚Ąē :: m < n ‚üĻ m + 1 ‚Č§ n)` in the Axiom and likewise of the bounded universal `(A m, n ‚ąą ‚Ąē :: m ‚Č§ n < m + 1 ‚üĻ n = m)` in the no-interval Consequence, over which the bound variables `m, n` range before being further restricted by the term-side hypotheses.
  - NAT-order (NatStrictTotalOrder) ‚ÄĒ supplies the non-strict companion `‚Č§` (defined by `m ‚Č§ n ‚üļ m < n ‚ą® m = n`) used in the axiom's consequent `m + 1 ‚Č§ n` and in the Consequence derivation's case split on `m ‚Č§ n`, the exactly-one-trichotomy clause `¬¨(a < b ‚ąß b < a)` instantiated at `(m + 1, n)` in the derivation, and irreflexivity `¬¨(n < n)` used to discharge the rewritten `n < n`.
  - NAT-closure (NatArithmeticClosureAndIdentity) ‚ÄĒ posits `1 ‚ąą ‚Ąē` and closes ‚Ąē under addition, so the successor `m + 1 ‚ąą ‚Ąē` and the inequalities `m + 1 ‚Č§ n` and `n < m + 1` are comparisons of two ‚Ąē-elements.
