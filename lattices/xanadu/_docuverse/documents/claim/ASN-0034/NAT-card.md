**NAT-card (NatFiniteSetCardinality).** For every `n ‚ąą ‚Ąē` and every subset `S ‚äÜ {j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ n}`, we postulate `|¬∑|` as a primitive operator with `|S| ‚ąą ‚Ąē` the unique `k ‚ąą ‚Ąē` for which there exists a strictly increasing function `f : {j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ k} ‚Üí ‚Ąē` with `S = {f.j : 1 ‚Č§ j ‚Č§ k}` (at `k = 0` the domain `{j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ 0}` is empty, `f` is the empty function, vacuously strictly increasing with image `‚ąÖ`, forcing `S = ‚ąÖ` and `|‚ąÖ| = 0` without recourse to a convention on empty lists), and `|S| ‚Č§ n`.

*Formal Contract:*
- *Axiom:* `(A n ‚ąą ‚Ąē, S : S ‚äÜ {j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ n} :: |S|` is the unique `k ‚ąą ‚Ąē` such that `(E f :: f : {j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ k} ‚Üí ‚Ąē ‚ąß (A i, j : 1 ‚Č§ i < j ‚Č§ k : f.i < f.j) ‚ąß S = {f.j : 1 ‚Č§ j ‚Č§ k}))` ‚ÄĒ strictly-increasing-function characterisation, existence-and-uniqueness of `k` carried by "the unique"; `(A n ‚ąą ‚Ąē, S : S ‚äÜ {j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ n} :: |S| ‚Č§ n)` ‚ÄĒ upper bound.
- *Depends:*
  - NAT-carrier (NatCarrierSet) ‚ÄĒ supplies `‚Ąē` as the underlying set appearing in the outer membership clause `n ‚ąą ‚Ąē`, in the initial-segment domain `{j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ n}` whose subsets `S` ranges over, in the cardinality clause `|S| ‚ąą ‚Ąē` and the inner existential over `k ‚ąą ‚Ąē`, in the domain `{j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ k}` and codomain ‚Ąē of the enumerating function `f : {j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ k} ‚Üí ‚Ąē`, and in the upper-bound clause `|S| ‚Č§ n` over which `n` is the ‚Ąē-bound.
  - NAT-order (NatStrictTotalOrder) ‚ÄĒ supplies the strict order `<` (used in the strictly-increasing condition `(A i, j : 1 ‚Č§ i < j ‚Č§ k : f.i < f.j)` on the enumerating function `f`) and the non-strict companion `‚Č§` (used in the upper bound `|S| ‚Č§ n`, in the initial-segment domain `{j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ n}` bounding `S`, and in the domain `{j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ k}` of `f`); the strict-total-order discipline (irreflexivity, transitivity, trichotomy `m < n ‚ą® m = n ‚ą® n < m`) makes "strictly increasing function" a well-formed predicate on ‚Ąē-valued functions.
  - NAT-closure (NatArithmeticClosureAndIdentity) ‚ÄĒ supplies `1 ‚ąą ‚Ąē`, the lower bound in the initial segment `{j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ n}` and in the domain `{j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ k}` of the enumerating function; combined with NAT-order's `‚Č§` and the outer `n ‚ąą ‚Ąē`, this grounds `S ‚äÜ {j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ n}` with ‚Ąē-typed elements. Also supplies `0 < 1`, which forces `{j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ 0} = ‚ąÖ` and so renders the `k = 0` and `n = 0` cases of the axiom well-formed.
  - NAT-zero (NatZeroMinimum) ‚ÄĒ supplies `0 ‚ąą ‚Ąē` for the literal `0` appearing in the empty-domain case `k = 0` (where `{j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ 0}` is forced empty by `0 < 1`) and in the empty-set cardinality `|‚ąÖ| = 0`.
