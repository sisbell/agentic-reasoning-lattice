**T0 (CarrierSetDefinition).** `T` is the set of nonempty finite sequences over ‚Ąē, written `d‚āĀ.d‚āā. ... .d‚āô` with each `dŠĶĘ ‚ąą ‚Ąē`. For each `a ‚ąą T`, write `#a` for the length of `a`. The component positions of `a` form the index domain `{j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ #a}`, and component projection is the typed operator `i ‚Ü¶ aŠĶĘ : {j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ #a} ‚Üí ‚Ąē` ‚ÄĒ total and single-valued on the index domain ‚ÄĒ so `aŠĶĘ ‚ąą ‚Ąē` at each `i ‚ąą {j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ #a}`. `(A a ‚ąą T :: 1 ‚Č§ #a)` forces each tumbler to have at least one component, so the index domain is nonempty. Elements of T are *extensional*: two tumblers with equal length and identical components at every position are the same tumbler ‚ÄĒ there is no hidden structure beyond the length-and-components data. T is also *populated by every such sequence*: for every length `p ‚ąą ‚Ąē` with `p ‚Č• 1` and every component assignment `r : {j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ p} ‚Üí ‚Ąē`, there is a tumbler `t ‚ąą T` with `#t = p` and `tŠĶĘ = r(i)` at each `i` ‚ÄĒ extensionality makes this `t` unique, but existence is what licenses the construction of new tumblers from a length and a component map.

*Formal Contract:*
- *Axiom:* `T` is a set (the carrier of tumblers); `#¬∑ : T ‚Üí ‚Ąē` (length operator on T); `(A a ‚ąą T :: 1 ‚Č§ #a)` (nonemptiness ‚ÄĒ each tumbler has at least one component); `(A a ‚ąą T :: i ‚Ü¶ aŠĶĘ : {j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ #a} ‚Üí ‚Ąē)` (component projection signature ‚ÄĒ for each tumbler `a ‚ąą T`, the projection `i ‚Ü¶ aŠĶĘ` is a total, single-valued function from the index domain `{j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ #a}` into ‚Ąē; in particular `aŠĶĘ ‚ąą ‚Ąē` at each `i` in the index domain); `(A p ‚ąą ‚Ąē : p ‚Č• 1 : (A r : {j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ p} ‚Üí ‚Ąē :: (E t ‚ąą T :: #t = p ‚ąß (A i : 1 ‚Č§ i ‚Č§ p : tŠĶĘ = r(i)))))` (comprehension ‚ÄĒ every nonempty finite sequence of naturals, presented as a length `p ‚Č• 1` and a component map `r` from the index domain `{j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ p}` into ‚Ąē, is represented in T by some `t` with `#t = p` and `tŠĶĘ = r(i)`); `(A a, b ‚ąą T : #a = #b ‚ąß (A i : 1 ‚Č§ i ‚Č§ #a : aŠĶĘ = bŠĶĘ) : a = b)` (extensionality ‚ÄĒ tumblers with equal length and pointwise-equal components are identical).
- *Depends:*
  - NAT-carrier (NatCarrierSet) ‚ÄĒ supplies `‚Ąē` as the underlying set appearing as the codomain of the length operator `#¬∑ : T ‚Üí ‚Ąē`, as the codomain of the component-projection signature `i ‚Ü¶ aŠĶĘ : {j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ #a} ‚Üí ‚Ąē` and the comprehension's component map `r : {j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ p} ‚Üí ‚Ąē`, in the index-domain comprehensions `{j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ #a}` and `{j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ p}`, and over which the bounded quantifier `(A p ‚ąą ‚Ąē : p ‚Č• 1 : ...)` of the comprehension axiom ranges.
  - NAT-closure (NatArithmeticClosureAndIdentity) ‚ÄĒ supplies `1 ‚ąą ‚Ąē` for the lower bound of the nonemptiness clause `1 ‚Č§ #a` and for the lower bound `1` in the index domain `{j ‚ąą ‚Ąē : 1 ‚Č§ j ‚Č§ #a}` of the component-projection signature.
  - NAT-order (NatStrictTotalOrder) ‚ÄĒ supplies the non-strict relation `‚Č§` on ‚Ąē appearing in the nonemptiness clause `1 ‚Č§ #a` and in the index-domain bound `1 ‚Č§ j ‚Č§ #a` of the component-projection signature.
