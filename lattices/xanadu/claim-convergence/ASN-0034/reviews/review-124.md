### T1's NAT-discrete citation omits part (b) Case 3's use of `m < n ⟹ m + 1 ≤ n`
**Foundation**: (foundation ASN — internal consistency; per-step NAT-* citation convention)
**ASN**: T1's proof, part (b) Trichotomy, Case 3 reads: *"If m < n, then k = m + 1 ≤ n, so a is a proper prefix of b and k witnesses a < b via T1 case (ii)."* (The symmetric `m > n` branch uses the same implication.) Deducing `m + 1 ≤ n` from `m < n` for `m, n ∈ ℕ` is exactly NAT-discrete's contrapositive form — the axiom `m ≤ n < m + 1 ⟹ n = m` rearranged to say "no natural number lies strictly between `m` and `m + 1`," so `m < n` forces `m + 1 ≤ n`. Without NAT-discrete, a natural `n` with `m < n < m + 1` is not excluded, and the step `k = m + 1 ≤ n` cannot be closed from `m < n` alone. T1's Depends does list NAT-discrete, but the annotation narrows its consumption to a single site: *"part (c) Transitivity, sub-case (ii, ii), invokes the equivalence `m + 1 ≤ n ⟺ m < n` … to convert the case hypothesis `m + 1 ≤ n` into the strict comparison `m < n`."* The Case 3 (part (b)) site uses the *same* equivalence in the opposite direction and is not acknowledged.
**Issue**: The ASN's stated per-step citation convention requires each use site of a NAT-* axiom to be enumerated in the Depends annotation. T1 lists NAT-addcompat's strict successor inequality at two sites (part (a) Case (ii); part (c) Case `k₂ < k₁`) and NAT-order's trichotomy/transitivity at four sites — the convention is applied with full granularity to those axioms. NAT-discrete's citation here is single-site, but the axiom is in fact consumed twice: once in part (c) sub-case (ii, ii) (`m + 1 = n + 1` → `m < n` via the contrapositive), and once in part (b) Case 3 (`m < n` → `m + 1 ≤ n`, and its `m > n` mirror). A reviser tightening or re-stating NAT-discrete has Depends-backed visibility only into the Transitivity site, and the Trichotomy proof's proper-prefix conclusion — which is load-bearing for Case 3's "exactly one" outcome — is tacitly reliant on the axiom.
**What needs resolving**: T1's NAT-discrete Depends annotation must enumerate the part (b) Case 3 site alongside the part (c) sub-case (ii, ii) site, matching the granularity the same Depends applies to NAT-order and NAT-addcompat; absent that, the Trichotomy proof's proper-prefix step has no cited source for its successor-comparison step.
