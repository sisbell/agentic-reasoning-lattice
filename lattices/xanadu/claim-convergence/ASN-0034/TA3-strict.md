**The form of `a < b`.** Since `#a = #b`, T1 case (ii) (which requires `#a < #b`) is impossible. So `a < b` holds by case (i): there exists a least `j` with `1 ‚Č§ j ‚Č§ #a ‚ąß j ‚Č§ #b` such that `aŠĶĘ = bŠĶĘ` for all `i < j` and `a‚Īľ < b‚Īľ`. Fix this `j`.

**Well-formedness and common length.** By TA2, `a ‚äĖ w, b ‚äĖ w ‚ąą T`.
