**T4a (SyntacticEquivalence).** T4's field-segment constraint вҖ” stated positionally as (i) no two zeros are adjacent, (ii) `tвӮҒ вү  0`, (iii) `t_{#t} вү  0` вҖ” is equivalent to the condition that every *field segment* of `t` is non-empty, where the field segments are the `zeros(t) + 1` maximal contiguous sub-sequences of non-zero positions delimited by the zeros.

Let `t вҲҲ T` be a tumbler with `zeros(t) вүӨ 3`. Set `k = zeros(t)`; T4's Exhaustion Consequence вҖ” whose derivation uses only the bound `zeros(t) вүӨ 3` together with NAT-zero's `0 вүӨ zeros(t)`, NAT-order's trichotomy, and NAT-discrete, and thus applies under T4a's precondition without invoking the field-segment constraint вҖ” pins `k вҲҲ {0, 1, 2, 3}`. Enumerate the zeros of `t` at positions `sвӮҒ < sвӮӮ < вҖҰ < s_k` (strict increase by NAT-order; the length `k` of this enumeration equals `zeros(t)` by NAT-card's enumeration characterisation of `|В·|` applied at the zero-index subset `{i : 1 вүӨ i вүӨ #t вҲ§ tбөў = 0} вҠҶ {1, вҖҰ, #t}`, whose cardinality T4 identifies with `zeros(t)`). Set `sвӮҖ = 0` and `s_{k+1} = #t + 1` as sentinels. The arithmetic in what follows вҖ” the numerals `2` and `3`, the sums `s_i + 1`, `s_i + 2`, and `#t + 1`, and the last-segment upper bound `s_{k+1} вҲ’ 1` вҖ” is grounded thus: NAT-closure posits `1 вҲҲ в„•` and closes в„• under addition, so `2 := 1 + 1 вҲҲ в„•`, `3 := 2 + 1 вҲҲ в„•`, and each of `s_i + 1`, `s_i + 2`, `#t + 1` lies in в„•. NAT-sub's right-telescoping clause `(m + n) вҲ’ n = m`, instantiated at `m = #t, n = 1`, reduces `s_{k+1} вҲ’ 1 = (#t + 1) вҲ’ 1` to `#t` in в„•, so the last segment occupies the в„•-interval `[s_k + 1, #t]`. T0's Axiom includes the nonemptiness clause `(A a вҲҲ T :: 1 вүӨ #a)`; instantiating its universally quantified `a` at `t` (licensed by `t вҲҲ T`) yields `#t вүҘ 1` directly as a first-class axiom citation, and this bound underwrites the `k = 0` branches of Forward Conditions (ii) and (iii), where the single segment equals `t` itself. The `k + 1` field segments of `t` are the index ranges `(s_i, s_{i+1})` for `i = 0, 1, вҖҰ, k` вҖ” i.e. the component sub-sequences `t[s_i + 1 .. s_{i+1} - 1]`. A segment is non-empty precisely when `s_{i+1} вүҘ s_i + 2`; for the last segment, whose index range is `[s_k + 1, #t]` as shown above, this specialises to the native `+1` form `#t вүҘ s_k + 1` вҖ” equivalently `s_k + 1 вүӨ #t` вҖ” which is what the Reverse Last-segment argument lands on directly via NAT-discrete.

*Forward.* Assume every field segment is non-empty. We derive each positional condition.

*Condition (ii): `tвӮҒ вү  0`.* The first field segment is `t[1 .. sвӮҒ - 1]` when `k вүҘ 1`, or `t[1 .. #t]` when `k = 0`; in either case its first index is 1. Segment non-emptiness gives `sвӮҒ вүҘ 2` (when `k вүҘ 1`) or `#t вүҘ 1` (when `k = 0`), so index 1 lies in the segment and `tвӮҒ` is a non-zero component. Hence `tвӮҒ вү  0`.

*Condition (iii): `t_{#t} вү  0`.* The last field segment is `t[s_k + 1 .. #t]` when `k вүҘ 1`, or `t[1 .. #t]` when `k = 0`; in either case its last index is `#t`. Segment non-emptiness, in the last-segment `+1` specialisation stated above, gives `s_k + 1 вүӨ #t` (when `k вүҘ 1`) or `#t вүҘ 1` (when `k = 0`), so index `#t` lies in the segment and `t_{#t}` is a non-zero component. Hence `t_{#t} вү  0`.

*Condition (i): no adjacent zeros.* Suppose for contradiction that `tбөў = 0 вҲ§ tбөўвӮҠвӮҒ = 0` for some `i` with `1 вүӨ i < #t`. Then `i` and `i + 1` are both zero positions, so `i = s_j` and `i + 1 = s_{j'}` for some enumeration indices `j < j'` (strict monotonicity from `i < i + 1`); since `i + 1 вүӨ #t` places the second zero inside the enumerated range `{s_1, вҖҰ, s_k}`, we have `j' вүӨ k`, hence `j + 1 вүӨ j' вүӨ k` вҖ” so `s_{j+1}` is a real enumeration entry (not the sentinel `s_{k+1} = #t + 1`). We claim `s_{j+1} = i + 1`: if instead `j + 1 < j'`, strict monotonicity would give `s_j < s_{j+1} < s_{j'}`, i.e., `i < s_{j+1} < i + 1`, and NAT-discrete's no-interval Consequence `m вүӨ n < m + 1 вҹ№ n = m`, instantiated at `(m, n) := (i, s_{j+1})`, would force `s_{j+1} = i`, contradicting `s_j < s_{j+1}`. Hence `s_{j+1} = i + 1`, and the interior field segment `t[s_j + 1 .. s_{j+1} - 1] = t[i + 1 .. i]` has no indices, so it is empty. This contradicts segment non-emptiness. Hence no two zeros are adjacent.

*Reverse.* Assume conditions (i), (ii), and (iii) hold. We show every field segment is non-empty.

*First segment (`i = 0`).* The segment occupies indices 1 through `sвӮҒ - 1` (or 1 through `#t` when `k = 0`). If `k = 0` the segment equals `t` itself, which has `#t вүҘ 1` indices. If `k вүҘ 1`, condition (ii) forces `tвӮҒ вү  0`, so index 1 is not a zero position, and therefore `sвӮҒ вүҘ 2` вҖ” the segment has at least one index.

*Last segment (`i = k`).* The segment occupies indices `s_k + 1` through `#t` (when `k вүҘ 1`); non-emptiness of this index range is the `+1` inequality `s_k + 1 вүӨ #t`. Condition (iii) forces `t_{#t} вү  0`, so index `#t` is not among the zero positions `sвӮҒ < вҖҰ < s_k`, giving `s_k вү  #t`. Every zero position `s_j` is an index of `t`, hence `s_k вүӨ #t`; unfolding this by NAT-order's definition `s_k вүӨ #t вҹә s_k < #t вҲЁ s_k = #t` and excluding the `s_k = #t` branch by `s_k вү  #t` leaves `s_k < #t`. NAT-discrete's axiom `m < n вҹ№ m + 1 вүӨ n`, instantiated at `(m, n) := (s_k, #t)`, promotes this strict inequality directly to its native `+1` output `s_k + 1 вүӨ #t` вҖ” the segment has at least one index.

*Interior segments (`1 вүӨ i < k`).* Each such segment occupies indices `s_i + 1` through `s_{i+1} - 1`. Condition (i) forbids `s_{i+1} = s_i + 1`, and since the zero positions are strictly increasing we already have `s_{i+1} вүҘ s_i + 1`; together these give `s_{i+1} вүҘ s_i + 2`, so the segment has at least one index.

All segments вҖ” first, interior, and last вҖ” are non-empty. вҲҺ
