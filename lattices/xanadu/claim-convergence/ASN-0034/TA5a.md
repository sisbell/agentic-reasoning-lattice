**TA5a (IncrementPreservesT4).** The operation `inc(t, k)` on a T4-valid address `t` preserves T4 iff `k ‚ąą {0, 1}`, or `k = 2` with `zeros(t) ‚Č§ 2`. For `k ‚Č• 3`, T4 is violated.

*Proof.* Let `t` satisfy T4, `t' = inc(t, k)`. T4 requires: (i) `zeros(t) ‚Č§ 3`, (ii) no two zeros adjacent, (iii) `t‚āĀ ‚Č† 0`, (iv) `t_{#t} ‚Č† 0`.

*Case `k = 0`.* By TA5(c), `#t' = #t` and `t'_{sig(t)} = t_{sig(t)} + 1`; by TA5(b), `t'` agrees with `t` at every position `‚Č† sig(t)`. By TA5-SigValid, `sig(t) = #t`; by T4(iv), `t_{#t} ‚Č† 0`; hence `t_{sig(t)} ‚Č† 0`. By NAT-closure, `t_{sig(t)} + 1 ‚ąą ‚Ąē`; by NAT-zero and NAT-addcompat, `t_{sig(t)} + 1 > t_{sig(t)} ‚Č• 0`, so `t_{sig(t)} + 1 ‚Č† 0`. The zero-index set of `t'` is `{i : 1 ‚Č§ i ‚Č§ #t' ‚ąß t'ŠĶĘ = 0} = {i : 1 ‚Č§ i ‚Č§ #t ‚ąß t'ŠĶĘ = 0}` (using `#t' = #t`); at every `i ‚Č† sig(t)`, `t'ŠĶĘ = tŠĶĘ` by TA5(b), and at `i = sig(t)` neither `tŠĶĘ = 0` nor `t'ŠĶĘ = 0`, so position `sig(t)` lies outside both the original and the primed zero-index subsets. The two subsets coincide as sets: `{i : 1 ‚Č§ i ‚Č§ #t ‚ąß t'ŠĶĘ = 0} = {i : 1 ‚Č§ i ‚Č§ #t ‚ąß tŠĶĘ = 0}`. Both subsets lie inside `{1, ‚Ä¶, #t}`; by NAT-card's well-definedness of `|¬∑|` as a total function on subsets of that initial segment, equal subsets carry equal cardinalities, so `zeros(t') = zeros(t)`. For T4(ii), we split each `i` with `1 ‚Č§ i < #t'` (equivalently `1 ‚Č§ i < #t` by TA5(c)'s `#t' = #t`) on whether `sig(t) ‚ąą {i, i + 1}`: when `sig(t) ‚ąČ {i, i + 1}`, TA5(b)'s agreement at positions `‚Č† sig(t)` instantiated at `i` and `i + 1` gives `t'ŠĶĘ = tŠĶĘ` and `t'ŠĶĘ‚āä‚āĀ = tŠĶĘ‚āä‚āĀ`, and T4(ii) on `t` at index `i` (legal since `1 ‚Č§ i < #t`) gives `¬¨(tŠĶĘ = 0 ‚ąß tŠĶĘ‚āä‚āĀ = 0)`, hence `¬¨(t'ŠĶĘ = 0 ‚ąß t'ŠĶĘ‚āä‚āĀ = 0)`; when `i = sig(t)`, `t'ŠĶĘ = t'_{sig(t)} = t_{sig(t)} + 1 ‚Č† 0` (from the NAT-closure/NAT-zero/NAT-addcompat chain above) falsifies the conjunct `t'ŠĶĘ = 0`, and hence the conjunction `t'ŠĶĘ = 0 ‚ąß t'ŠĶĘ‚āä‚āĀ = 0`; when `i + 1 = sig(t)`, the same chain at position `sig(t) = i + 1` gives `t'ŠĶĘ‚āä‚āĀ ‚Č† 0`, falsifying the conjunct `t'ŠĶĘ‚āä‚āĀ = 0` and hence the conjunction. For T4(iv), TA5(c)'s `#t' = #t` and TA5-SigValid's `sig(t) = #t` identify `t'_{#t'} = t'_{#t} = t'_{sig(t)}`; combining TA5(c)'s `t'_{sig(t)} = t_{sig(t)} + 1` with the already-established `t_{sig(t)} + 1 ‚Č† 0` (from the NAT-closure/NAT-zero/NAT-addcompat chain above) gives `t'_{#t'} ‚Č† 0`. For T4(iii), we split on `sig(t)`: when `sig(t) = 1`, position `1 = sig(t)` places `t'‚āĀ = t'_{sig(t)} = t_{sig(t)} + 1` at the site of the NAT-closure/NAT-zero/NAT-addcompat chain above, so reusing that chain at position `1` gives `t'‚āĀ ‚Č† 0`; when `sig(t) ‚Č† 1`, TA5(b)'s agreement at every position `‚Č† sig(t)` instantiated at position `1` gives `t'‚āĀ = t‚āĀ`, and T4(iii) on `t` gives `t‚āĀ ‚Č† 0`, hence `t'‚āĀ ‚Č† 0`. T4 preserved unconditionally.

*Case `k = 1`.* By TA5(d), `#t' = #t + 1` and `t'_{#t+1} = 1`; by TA5(b), `t'` agrees with `t` on original positions. The zero-index set of `t'` is `{i : 1 ‚Č§ i ‚Č§ #t + 1 ‚ąß t'ŠĶĘ = 0}`; for `1 ‚Č§ i ‚Č§ #t` we have `t'ŠĶĘ = tŠĶĘ` by TA5(b), and at the new boundary `t'_{#t+1} = 1 ‚Č† 0` excludes index `#t + 1`, so the set equals `{i : 1 ‚Č§ i ‚Č§ #t ‚ąß tŠĶĘ = 0}`. Both subsets lie inside `{1, ‚Ä¶, #t + 1}`; by NAT-card's well-definedness on subsets of that initial segment, equal subsets carry equal cardinalities, so `zeros(t') = zeros(t)`. Left-flank `t'_{#t} = t_{#t} ‚Č† 0` by T4(iv), so no adjacent zeros; boundary `t'_{#t'} = 1 ‚Č† 0`. For T4(iii), TA5(b)'s original-position agreement `(A i : 1 ‚Č§ i ‚Č§ #t : t'ŠĶĘ = tŠĶĘ)` instantiated at `i = 1` ‚ÄĒ legal since `1 ‚Č§ #t` by T0 ‚ÄĒ gives `t'‚āĀ = t‚āĀ`, and T4(iii) on `t` gives `t‚āĀ ‚Č† 0`, hence `t'‚āĀ ‚Č† 0`. T4 preserved unconditionally.

*Case `k = 2`.* By TA5(d), `#t' = #t + 2`, `t'_{#t+1} = 0`, `t'_{#t+2} = 1`; by TA5(b), original positions agree. Let `S = {i : 1 ‚Č§ i ‚Č§ #t ‚ąß tŠĶĘ = 0}` (with `|S| = zeros(t)` by definition) and `S' = {i : 1 ‚Č§ i ‚Č§ #t + 2 ‚ąß t'ŠĶĘ = 0}`. For `1 ‚Č§ i ‚Č§ #t` original-position agreement gives `t'ŠĶĘ = tŠĶĘ`; `t'_{#t+1} = 0` admits index `#t + 1`; `t'_{#t+2} = 1 ‚Č† 0` excludes index `#t + 2`. Hence `S' = S ‚ą™ {#t + 1}`; since `S ‚äÜ {1, ‚Ä¶, #t}` and `#t + 1 > #t` by NAT-addcompat's strict successor inequality, `#t + 1 ‚ąČ S` and the union is disjoint. By NAT-card, `S` admits a unique strictly increasing enumeration `s‚āĀ < ‚Ä¶ < s_{|S|}` with each `s_j ‚Č§ #t < #t + 1`; appending `#t + 1` (which strictly exceeds `s_{|S|}` by the same bound) yields a strictly increasing enumeration of `S'` of length `|S| + 1`. By NAT-card's enumeration characterisation applied at `n = #t + 2` (so `S' ‚äÜ {1, ‚Ä¶, #t + 2}`), `|S'| = |S| + 1`, so `zeros(t') = zeros(t) + 1`. Left-flank `t'_{#t} = t_{#t} ‚Č† 0` by T4(iv); right-flank is `1`; boundary `t'_{#t'} = 1 ‚Č† 0`. For T4(iii), TA5(b)'s original-position agreement `(A i : 1 ‚Č§ i ‚Č§ #t : t'ŠĶĘ = tŠĶĘ)` instantiated at `i = 1` ‚ÄĒ legal since `1 ‚Č§ #t` by T0 ‚ÄĒ gives `t'‚āĀ = t‚āĀ`, and T4(iii) on `t` gives `t‚āĀ ‚Č† 0`, hence `t'‚āĀ ‚Č† 0`. T4 preserved iff `zeros(t) + 1 ‚Č§ 3`, i.e., `zeros(t) ‚Č§ 2`.

*Case `k ‚Č• 3`.* By TA5(d), positions `#t + 1` through `#t + k - 1` are zero. NAT-sub's conditional closure at `k ‚Č• 1` (implied by `k ‚Č• 3`) places `k ‚ąí 1 ‚ąą ‚Ąē`. To sharpen `k ‚Č• 3` to `k ‚ąí 1 ‚Č• 2`, NAT-order's `‚Č§` definition unfolds `3 ‚Č§ k` to `3 < k ‚ą® 3 = k`, splitting the hypothesis into two sub-branches. In the sub-branch `k = 3`, NAT-sub's right telescoping (unconditional) at `(m, n) = (2, 1)` gives `(2 + 1) ‚ąí 1 = 2`, i.e., `3 ‚ąí 1 = 2`, so `k ‚ąí 1 = 2`. In the sub-branch `3 < k`, NAT-sub's strict monotonicity at `(m, n, p) = (3, k, 1)` ‚ÄĒ with `3 ‚Č• 1` and `k ‚Č• 1` (the latter from `k ‚Č• 3`) ‚ÄĒ yields `3 ‚ąí 1 < k ‚ąí 1`, and right telescoping at `(2, 1)` rewrites the left side to `2`, so `2 < k ‚ąí 1`; NAT-order's `‚Č§` definition then lifts `2 < k ‚ąí 1` to `2 ‚Č§ k ‚ąí 1`, i.e., `k ‚ąí 1 ‚Č• 2`. In either sub-branch `k ‚ąí 1 ‚Č• 2`, so positions `#t + 1` and `#t + 2` lie within `{#t + 1, ‚Ä¶, #t + k ‚ąí 1}` and carry zero; that is, `t'_{#t+1} = t'_{#t+2} = 0`. Suppose for contradiction that T4 holds on `t'`. T4(ii) on `t'` is `(A i : 1 ‚Č§ i < #t' : ¬¨(t'ŠĶĘ = 0 ‚ąß t'ŠĶĘ‚āä‚āĀ = 0))`; with `#t' = #t + k` by TA5(d), the index `i = #t + 1` is legal ‚ÄĒ `1 ‚Č§ #t + 1` follows from T0's `1 ‚Č§ #t` and NAT-addcompat's strict successor inequality `#t < #t + 1` by NAT-order's transitivity, and `#t + 1 < #t + k` follows from the chain `#t + 1 < #t + 2 ‚Č§ #t + k`: NAT-addcompat's strict successor inequality at `n = #t + 1` together with NAT-addassoc's `(#t + 1) + 1 = #t + (1 + 1) = #t + 2` gives `#t + 1 < #t + 2`, and NAT-addcompat's left order compatibility at `2 ‚Č§ k` (a consequence of `3 ‚Č§ k` chained with the numeral `2 < 3` from NAT-addcompat's strict successor at `n = 2` via NAT-order's transitivity and `‚Č§`-definition) gives `#t + 2 ‚Č§ #t + k`. T4(ii) instantiated at `i = #t + 1` ‚ÄĒ with successor index `(#t + 1) + 1 = #t + 2` by the same NAT-addassoc/numeral identification ‚ÄĒ yields `¬¨(t'_{#t+1} = 0 ‚ąß t'_{#t+2} = 0)`, which directly contradicts the established `t'_{#t+1} = t'_{#t+2} = 0`. The supposition that T4 holds on `t'` is inconsistent. Witness: `inc([1], 3) = [1, 0, 0, 1]`. T4 violated. ‚ąé

- *Precondition:* `t` satisfies T4; `k ‚Č• 0`.
