## Tumbler subtraction

**TumblerSub (TumblerSub).** Given two tumblers `a` (minuend) and `w` (subtrahend), compute their component-wise difference at the first point of zero-padded divergence. When the operands have different lengths, zero-pad the shorter to the length of the longer: `a·µ¢ = 0` for `i > #a` and `w·µ¢ = 0` for `i > #w`. NAT-order's trichotomy on `(#a, #w)` selects exactly one of: (Œ±) `#a = #w` with `L = #a`; (Œ≤) `#a < #w` with `L = #w`; (Œ≥) `#w < #a` with `L = #a`. When the zero-padded sequences agree at every position, the result is the zero tumbler of length `L`: `a ‚äñ w = [0, ..., 0]`. Otherwise, `zpd(a, w)` is defined (ZPD) ‚Äî write `k = zpd(a, w)`. The result is:

```
         ‚éß 0             if i < k
r·µ¢   =  ‚é® a‚Çñ - w‚Çñ      if i = k
         ‚é© a·µ¢           if i > k
```

The result has length `L`.

**Precondition:** `a ‚â• w` (T1). We prove that when `zpd(a, w)` is defined, this entails `a‚Çñ > w‚Çñ` at `k = zpd(a, w)`. Since zpd is defined, `a` and `w` are not zero-padded-equal (ZPD), so by T3 (contrapositive) `a ‚âÝ w`; combined with `a ‚â• w`, this yields `w < a` (T1). Two Divergence cases arise for the pair `(w, a)` with `w ‚âÝ a`:

  (i) Component divergence at position `k` with `k ‚â§ #w ‚àß k ‚â§ #a` and `w‚Çñ ‚âÝ a‚Çñ`. ZPD's Relationship-to-Divergence gives `zpd(a, w) = divergence(a, w) = k`. From `w < a` (T1), a witness `j` exists in one of T1's two cases for the pair `(w, a)`. T1 case (ii) is eliminated: it would require `j = #w + 1 ‚â§ #a` with `w·µ¢ = a·µ¢` for `1 ‚â§ i < j`, i.e., for `1 ‚â§ i ‚â§ #w`; instantiating this prefix agreement at `i := k` ‚Äî legal since `k ‚â§ #w` from Divergence case (i) ‚Äî yields `w‚Çñ = a‚Çñ`, contradicting `w‚Çñ ‚âÝ a‚Çñ`. So T1 case (i) holds: `j ‚â§ #w ‚àß j ‚â§ #a`, `w‚±º < a‚±º`, and `w·µ¢ = a·µ¢` for `1 ‚â§ i < j`. NAT-order's disjointness-of-`<`-and-`=` at `(w‚±º, a‚±º)` converts `w‚±º < a‚±º` into `w‚±º ‚âÝ a‚±º`, so the position `j` satisfies Divergence case (i)'s conjunction `1 ‚â§ j ‚àß j ‚â§ #w ‚àß j ‚â§ #a ‚àß w‚±º ‚âÝ a‚±º ‚àß (A i : 1 ‚â§ i < j : w·µ¢ = a·µ¢)` for the pair `(w, a)`; Divergence's uniqueness clause for case (i) identifies `j = k`. Hence `w‚Çñ < a‚Çñ`, whence `a‚Çñ > w‚Çñ`.

  (ii) Prefix divergence splits via NAT-order's trichotomy on `(#w, #a)` into sub-case (ii-a) `#w < #a` with `w·µ¢ = a·µ¢` for `1 ‚â§ i ‚â§ #w`, and sub-case (ii-b) `#a < #w` with `w·µ¢ = a·µ¢` for `1 ‚â§ i ‚â§ #a`. Sub-case (ii-b) is eliminated: its prefix hypothesis yields `a < w` via T1 case (ii), contradicting `w < a`. In sub-case (ii-a), `w` is a proper prefix of `a`. The padded extension sets `w·µ¢ = 0` for `i > #w`. Since zpd is defined, the padded sequences disagree somewhere (ZPD, contrapositive); the prefix agreement rules out positions `1 ‚â§ i ‚â§ #w`, so the disagreement lies at some `i > #w`. By ZPD's minimality, `k > #w`, whence `w‚Çñ = 0` by zero-padding and `a‚Çñ ‚âÝ 0`. From NAT-zero's `0 ‚â§ a‚Çñ` and NAT-order's `m ‚â§ n ‚ü∫ m < n ‚à® m = n`, the divergence `a‚Çñ ‚âÝ 0` leaves `0 < a‚Çñ`; hence `a‚Çñ > 0 = w‚Çñ`.

In both cases `a‚Çñ > w‚Çñ` at `k = zpd(a, w)`. When zpd is undefined, the consequence is vacuous.  ‚àé

Each component of the result is a natural number: for `i < k`, `r·µ¢ = 0 ‚àà ‚Ñï` by NAT-zero; at the divergence point, `r‚Çñ = a‚Çñ ‚àí w‚Çñ ‚àà ‚Ñï` by NAT-sub, whose precondition `a‚Çñ ‚â• w‚Çñ` follows from `a‚Çñ > w‚Çñ` via NAT-order; for `i > k`, `r·µ¢` is the zero-padded value of `a`, either `a·µ¢ ‚àà ‚Ñï` or `0 ‚àà ‚Ñï` by NAT-zero. In the no-divergence case every component is `0 ‚àà ‚Ñï`. The length `L ‚â• 1` since T0 gives `#a ‚â• 1` and `#w ‚â• 1`, and `L` is named by the trichotomy as one of `#a` or `#w`. Hence **`a ‚äñ w ‚àà T`** by T0.

When `zpd(a, w)` is defined ‚Äî write `k = zpd(a, w)` ‚Äî components before `k` are zero by construction. To discharge TA-Pos we exhibit `k` as an index with `¬¨(r‚Çñ = 0)`. NAT-sub's right-inverse characterisation, at `(a‚Çñ, w‚Çñ)` with precondition `a‚Çñ ‚â• w‚Çñ` already established above from `a‚Çñ > w‚Çñ` via NAT-order, yields `(a‚Çñ ‚àí w‚Çñ) + w‚Çñ = a‚Çñ` ‚Äî that is, `r‚Çñ + w‚Çñ = a‚Çñ`. Suppose, for contradiction, `r‚Çñ = 0`: NAT-closure's left additive identity instantiated at `n := w‚Çñ` rewrites the left-hand side via `0 + w‚Çñ = w‚Çñ`, giving `w‚Çñ = a‚Çñ`. But `a‚Çñ > w‚Çñ` unfolds through NAT-order's `>` definition `m > n ‚ü∫ n < m` to `w‚Çñ < a‚Çñ`, and NAT-order's disjointness-of-`<`-and-`=` clause applied at `(w‚Çñ, a‚Çñ)` then forces `w‚Çñ ‚âÝ a‚Çñ`, contradicting `w‚Çñ = a‚Çñ`. Hence `¬¨(r‚Çñ = 0)`. ZPD's codomain places `k` in `{1, ..., L}`, and `#(a ‚äñ w) = L` gives `1 ‚â§ k ‚â§ #(a ‚äñ w)`, so `k` is a valid existential witness; whence **`Pos(a ‚äñ w)`** (TA-Pos). ActionPoint names `actionPoint(a ‚äñ w)` as the unique `m ‚àà S := {i : 1 ‚â§ i ‚â§ #(a ‚äñ w) ‚àß r·µ¢ ‚âÝ 0}` with `(A n ‚àà S :: m ‚â§ n)`; to identify this minimum with `k` we discharge both membership and the least-element clause. Membership: `1 ‚â§ k ‚â§ #(a ‚äñ w)` and `r‚Çñ ‚âÝ 0` just established jointly give `k ‚àà S`. Least-element: take any `n ‚àà S` and suppose for contradiction `n < k`. The Definition fixes `r·µ¢ = 0` for every `i` with `1 ‚â§ i < k`; instantiating at `i := n` yields `r‚Çô = 0`, contradicting `r‚Çô ‚âÝ 0` from `n ‚àà S`. Hence `¬¨(n < k)`, and NAT-order's at-least-one trichotomy at `(k, n)` gives `k < n ‚à® k = n ‚à® n < k`; the excluded disjunct drops out, and NAT-order's defining clause `k ‚â§ n ‚ü∫ k < n ‚à® k = n` folds the remainder to `k ‚â§ n`. Hence `(A n ‚àà S :: k ‚â§ n)`, and ActionPoint's uniqueness clause names `k` as the action point: **`actionPoint(a ‚äñ w) = k = zpd(a, w)`**. When `zpd(a, w)` is undefined, the result is the zero tumbler and neither conclusion holds.

*Formal Contract:*
- *Preconditions:* a ‚àà T, w ‚àà T, a ‚â• w (T1).
- *Depends:*
  - T0 (CarrierSetDefinition) ‚Äî carrier membership `a ‚äñ w ‚àà T` and per-operand length bounds `#a ‚â• 1`, `#w ‚â• 1`.
  - T1 (LexicographicOrder) ‚Äî precondition ordering `a ‚â• w`; trichotomy derives `w < a`; T1 case (ii) is eliminated in Divergence case (i) (its prefix-agreement range `1 ‚â§ i ‚â§ #w` instantiated at `i := k` would force `w‚Çñ = a‚Çñ`, contradicting `w‚Çñ ‚âÝ a‚Çñ`) and in Divergence case (ii) sub-case (ii-b) (it yields `a < w`, contradicting `w < a`); in Divergence case (i), T1 case (i)'s witness `j` satisfies `w‚±º < a‚±º` and, once identified with `k` via Divergence's uniqueness, supplies `w‚Çñ < a‚Çñ`.
  - T3 (CanonicalRepresentation) ‚Äî forward direction: `not-padded-equal ‚üπ a ‚âÝ w`; reverse direction: length inequality implies tumbler inequality.
  - Divergence ‚Äî case analysis on the pair `(w, a)`; uniqueness clause for case (i) identifies T1 case (i)'s witness position `j` with the divergence index `k`.
  - ZPD ‚Äî defines `zpd(a, w)`; Relationship-to-Divergence identifies `zpd = divergence` under case (i); case-split and minimality under case (ii).
  - TA-Pos (PositiveTumbler) ‚Äî defines `Pos` for the conditional postcondition.
  - ActionPoint ‚Äî characterises `actionPoint(a ‚äñ w)` as the unique least element of `S := {i : 1 ‚â§ i ‚â§ #(a ‚äñ w) ‚àß r·µ¢ ‚âÝ 0}`; membership `k ‚àà S` (from `1 ‚â§ k ‚â§ #(a ‚äñ w)` and `r‚Çñ ‚âÝ 0`) and the least-element clause `(A n ‚àà S :: k ‚â§ n)` (from the Definition's `r·µ¢ = 0` for `i < k`) jointly identify this minimum with `k`, yielding `actionPoint(a ‚äñ w) = zpd(a, w)`.
  - NAT-sub (NatPartialSubtraction) ‚Äî conditional closure `a‚Çñ ‚àí w‚Çñ ‚àà ‚Ñï` under `a‚Çñ ‚â• w‚Çñ`; right-inverse characterisation `(a‚Çñ ‚àí w‚Çñ) + w‚Çñ = a‚Çñ` under `a‚Çñ ‚â• w‚Çñ`, supplying the sum rewritten in the Pos derivation.
  - NAT-zero (NatZeroMinimum) ‚Äî `0 ‚àà ‚Ñï` for zero-padding, `r·µ¢ = 0` components, and the zero-tumbler branch; lower bound `0 ‚â§ a‚Çñ`.
  - NAT-order (NatStrictTotalOrder) ‚Äî trichotomy on `(#a, #w)` naming `L`; defining clause `‚â§ ‚ü∫ < ‚à® =` at `(0, a‚Çñ)`; conversion `>` to `‚â•` at `(a‚Çñ, w‚Çñ)` for NAT-sub; disjointness-of-`<`-and-`=` at `(w‚±º, a‚±º)` in Divergence case (i) converts T1 case (i)'s `w‚±º < a‚±º` into `w‚±º ‚âÝ a‚±º`, qualifying the witness `j` for Divergence case (i)'s conjunction (whose uniqueness then identifies `j` with `k`); the `>` definition `m > n ‚ü∫ n < m` at `(a‚Çñ, w‚Çñ)` and disjointness-of-`<`-and-`=` at `(w‚Çñ, a‚Çñ)` jointly discharge the Pos-derivation contradiction `w‚Çñ ‚âÝ a‚Çñ` from `a‚Çñ > w‚Çñ`; at-least-one trichotomy at `(k, n)` together with the defining clause `‚â§ ‚ü∫ < ‚à® =` at `(k, n)` discharges the least-element clause `(A n ‚àà S :: k ‚â§ n)` from `¬¨(n < k)`, identifying `k` as `actionPoint(a ‚äñ w)` within ActionPoint's uniqueness.
  - NAT-closure (NatArithmeticClosureAndIdentity) ‚Äî posits `1 ‚àà ‚Ñï` and closes ‚Ñï under `+`, and fixes `0 + n = n`. The left-identity clause is instantiated at `n := w‚Çñ` to rewrite `0 + w‚Çñ = w‚Çñ` in the Pos derivation, bridging the supposition `r‚Çñ = 0` to the contradiction `w‚Çñ = a‚Çñ`.
- *Definition:* NAT-order's trichotomy on `(#a, #w)` selects exactly one of: (Œ±) `#a = #w`, `L = #a`; (Œ≤) `#a < #w`, `L = #w`; (Œ≥) `#w < #a`, `L = #a`. a ‚äñ w is computed by case analysis on k = zpd(a, w) (ZPD), all component references using zero-padded values (a·µ¢ = 0 for i > #a, w·µ¢ = 0 for i > #w); r·µ¢ = 0 for i < k, r‚Çñ = a‚Çñ ‚àí w‚Çñ, r·µ¢ = a·µ¢ (zero-padded) for i > k; when zpd(a, w) is undefined, a ‚äñ w = [0, ‚Ä¶, 0]; #(a ‚äñ w) = L.
- *Postconditions:* a ‚äñ w ‚àà T, #(a ‚äñ w) = L (the longer of `#a` and `#w`, named by NAT-order trichotomy per the Definition); when zpd(a, w) is defined: Pos(a ‚äñ w) (TA-Pos), actionPoint(a ‚äñ w) = zpd(a, w) (ActionPoint).
