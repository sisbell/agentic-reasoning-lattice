## Tumbler subtraction

**TumblerSub (TumblerSub).** Given two tumblers `a` (minuend) and `w` (subtrahend), compute their component-wise difference at the first point of zero-padded divergence. NAT-order's trichotomy on `(#a, #w)` selects exactly one of: (Œ±) `#a = #w` with `L = #a`; (Œ≤) `#a < #w` with `L = #w`; (Œ≥) `#w < #a` with `L = #a`. All component references at indices in `{1, ..., L}` use ZPD's *padded projections* `√¢`, `≈µ` (`√¢·µ¢ = a·µ¢` for `1 ‚â§ i ‚â§ #a` and `√¢·µ¢ = 0` for `#a < i ‚â§ L`; symmetrically for `≈µ`), so that the bare native symbols `a·µ¢`, `w·µ¢` retain their T0 meaning on the native domain only. When `√¢` and `≈µ` agree at every position, the result is the zero tumbler of length `L`: `a ‚äñ w = [0, ..., 0]`. Otherwise, `zpd(a, w)` is defined (ZPD) ‚Äî write `k = zpd(a, w)`. The result is:

```
         ‚éß 0             if i < k
r·µ¢   =  ‚é® √¢‚Çñ - ≈µ‚Çñ      if i = k
         ‚é© √¢·µ¢           if i > k
```

The result has length `L`.

**Precondition:** `a ‚â• w` (T1). We prove that when `zpd(a, w)` is defined, this entails `√¢‚Çñ > ≈µ‚Çñ` at `k = zpd(a, w)` ‚Äî stated on the padded projections so the inequality is well-typed regardless of whether `k` lies in the native domain of both operands. Since zpd is defined, `a` and `w` are not zero-padded-equal (ZPD): the negation of `(A i : 1 ‚â§ i ‚â§ L : √¢·µ¢ = ≈µ·µ¢)` supplies an index `i` with `1 ‚â§ i ‚â§ L` and `√¢·µ¢ ‚âÝ ≈µ·µ¢`. T3's contrapositive `a ‚âÝ b ‚ü∫ #a ‚âÝ #b ‚à® (‚àÉ j : 1 ‚â§ j ‚â§ #a : a‚±º ‚âÝ b‚±º)` speaks of native components and lengths only, so the padded disagreement does not by itself license `a ‚âÝ w`; we discharge `a ‚âÝ w` by case analysis on whether `i` lies in the shared native domain. (A) `i ‚â§ #a ‚àß i ‚â§ #w`: ZPD's padded-projection definition gives `√¢·µ¢ = a·µ¢` and `≈µ·µ¢ = w·µ¢`, hence `a·µ¢ ‚âÝ w·µ¢`; T3's contrapositive instantiated at `(a, w)` with witness `j := i` (legal since `1 ‚â§ i ‚â§ #a`) fires its existential disjunct, yielding `a ‚âÝ w`. (B) `i > #a ‚à® i > #w` with `i ‚â§ L`. Sub-case B1 (`i > #a`): ZPD's padding clause gives `√¢·µ¢ = 0`, so `√¢·µ¢ ‚âÝ ≈µ·µ¢` forces `≈µ·µ¢ ‚âÝ 0`, ruling out the padding clause `≈µ·µ¢ = 0` (which would apply for `i > #w`) and fixing `i ‚â§ #w`; combined with `i > #a`, this yields `#a < i ‚â§ #w`, hence `#a ‚âÝ #w`. Sub-case B2 (`i > #w`) is the mirror image: `≈µ·µ¢ = 0` and the disagreement force `i ‚â§ #a`, giving `#w < #a` and `#a ‚âÝ #w`. In both sub-cases, T3's contrapositive instantiated at `(a, w)` fires its length disjunct, yielding `a ‚âÝ w`. In either case (A) or (B), `a ‚âÝ w`; combined with `a ‚â• w`, this yields `w < a` (T1). Two Divergence cases arise for the pair `(w, a)` with `w ‚âÝ a`:

  (i) Component divergence at position `k` with `k ‚â§ #w ‚àß k ‚â§ #a` and `w‚Çñ ‚âÝ a‚Çñ` (native projections, well-defined since `k` lies in both native domains). ZPD's Relationship-to-Divergence gives `zpd(a, w) = divergence(a, w) = k`. From `w < a` (T1), a witness `j` exists in one of T1's two cases for the pair `(w, a)`. T1 case (ii) is eliminated: it would require `j = #w + 1 ‚â§ #a` with `w·µ¢ = a·µ¢` for `1 ‚â§ i < j`, i.e., for `1 ‚â§ i ‚â§ #w`; instantiating this prefix agreement at `i := k` ‚Äî legal since `k ‚â§ #w` from Divergence case (i) ‚Äî yields `w‚Çñ = a‚Çñ`, contradicting `w‚Çñ ‚âÝ a‚Çñ`. So T1 case (i) holds: `j ‚â§ #w ‚àß j ‚â§ #a`, `w‚±º < a‚±º`, and `w·µ¢ = a·µ¢` for `1 ‚â§ i < j`. NAT-order's disjointness-of-`<`-and-`=` at `(w‚±º, a‚±º)` converts `w‚±º < a‚±º` into `w‚±º ‚âÝ a‚±º`, so the position `j` satisfies Divergence case (i)'s conjunction `1 ‚â§ j ‚àß j ‚â§ #w ‚àß j ‚â§ #a ‚àß w‚±º ‚âÝ a‚±º ‚àß (A i : 1 ‚â§ i < j : w·µ¢ = a·µ¢)` for the pair `(w, a)`; Divergence's uniqueness clause for case (i) identifies `j = k`. Hence `w‚Çñ < a‚Çñ`, whence `a‚Çñ > w‚Çñ`. ZPD's padded projections coincide with native here (`√¢‚Çñ = a‚Çñ` from `k ‚â§ #a`, `≈µ‚Çñ = w‚Çñ` from `k ‚â§ #w`), lifting the inequality to `√¢‚Çñ > ≈µ‚Çñ`.

  (ii) Prefix divergence splits via NAT-order's trichotomy on `(#w, #a)` into sub-case (ii-a) `#w < #a` with `w·µ¢ = a·µ¢` for `1 ‚â§ i ‚â§ #w` (native projections on the shared native domain), and sub-case (ii-b) `#a < #w` with `w·µ¢ = a·µ¢` for `1 ‚â§ i ‚â§ #a`. Sub-case (ii-b) is eliminated: its prefix hypothesis yields `a < w` via T1 case (ii), contradicting `w < a`. In sub-case (ii-a), `w` is a proper prefix of `a`, so `L = #a`. ZPD's padded projection sets `≈µ·µ¢ = 0` for `#w < i ‚â§ L` (and `≈µ·µ¢ = w·µ¢ = a·µ¢ = √¢·µ¢` for `1 ‚â§ i ‚â§ #w` by the prefix agreement). Since zpd is defined, `√¢` and `≈µ` disagree somewhere (ZPD, contrapositive); the prefix agreement rules out positions `1 ‚â§ i ‚â§ #w`, so the disagreement lies at some `i > #w`. By ZPD's minimality, `k > #w`, whence `≈µ‚Çñ = 0` by zero-padding and `√¢‚Çñ ‚âÝ 0`. From NAT-zero's `0 ‚â§ √¢‚Çñ` and NAT-order's `m ‚â§ n ‚ü∫ m < n ‚à® m = n`, the divergence `√¢‚Çñ ‚âÝ 0` leaves `0 < √¢‚Çñ`; hence `√¢‚Çñ > 0 = ≈µ‚Çñ`.

In both cases `√¢‚Çñ > ≈µ‚Çñ` at `k = zpd(a, w)`. When zpd is undefined, the consequence is vacuous.  ‚àé

Each component of the result is a natural number: for `i < k`, `r·µ¢ = 0 ‚àà ‚Ñï` by NAT-zero; at the divergence point, `r‚Çñ = √¢‚Çñ ‚àí ≈µ‚Çñ ‚àà ‚Ñï` by NAT-sub, whose precondition `√¢‚Çñ ‚â• ≈µ‚Çñ` follows from `√¢‚Çñ > ≈µ‚Çñ` via NAT-order; for `i > k`, `r·µ¢ = √¢·µ¢`, which is `a·µ¢ ‚àà ‚Ñï` (T0) when `i ‚â§ #a` and `0 ‚àà ‚Ñï` (NAT-zero) when `i > #a`, by ZPD's padded-projection definition. In the no-divergence case every component is `0 ‚àà ‚Ñï`. The length `L ‚â• 1` since T0 gives `#a ‚â• 1` and `#w ‚â• 1`, and `L` is named by the trichotomy as one of `#a` or `#w`. Hence **`a ‚äñ w ‚àà T`** by T0.

When `zpd(a, w)` is defined ‚Äî write `k = zpd(a, w)` ‚Äî components before `k` are zero by construction. To discharge TA-Pos we exhibit `k` as an index with `¬¨(r‚Çñ = 0)`. NAT-sub's right-inverse characterisation, at `(√¢‚Çñ, ≈µ‚Çñ)` with precondition `√¢‚Çñ ‚â• ≈µ‚Çñ` already established above from `√¢‚Çñ > ≈µ‚Çñ` via NAT-order, yields `(√¢‚Çñ ‚àí ≈µ‚Çñ) + ≈µ‚Çñ = √¢‚Çñ` ‚Äî that is, `r‚Çñ + ≈µ‚Çñ = √¢‚Çñ`. Suppose, for contradiction, `r‚Çñ = 0`: NAT-closure's left additive identity instantiated at `n := ≈µ‚Çñ` rewrites the left-hand side via `0 + ≈µ‚Çñ = ≈µ‚Çñ`, giving `≈µ‚Çñ = √¢‚Çñ`. But `√¢‚Çñ > ≈µ‚Çñ` unfolds through NAT-order's `>` definition `m > n ‚ü∫ n < m` to `≈µ‚Çñ < √¢‚Çñ`, and NAT-order's disjointness-of-`<`-and-`=` clause applied at `(≈µ‚Çñ, √¢‚Çñ)` then forces `≈µ‚Çñ ‚âÝ √¢‚Çñ`, contradicting `≈µ‚Çñ = √¢‚Çñ`. Hence `¬¨(r‚Çñ = 0)`. ZPD's codomain places `k` in `{1, ..., L}`, and `#(a ‚äñ w) = L` gives `1 ‚â§ k ‚â§ #(a ‚äñ w)`, so `k` is a valid existential witness; whence **`Pos(a ‚äñ w)`** (TA-Pos). ActionPoint names `actionPoint(a ‚äñ w)` as the unique `m ‚àà S := {i : 1 ‚â§ i ‚â§ #(a ‚äñ w) ‚àß r·µ¢ ‚âÝ 0}` with `(A n ‚àà S :: m ‚â§ n)`; to identify this minimum with `k` we discharge both membership and the least-element clause. Membership: `1 ‚â§ k ‚â§ #(a ‚äñ w)` and `r‚Çñ ‚âÝ 0` just established jointly give `k ‚àà S`. Least-element: take any `n ‚àà S` and suppose for contradiction `n < k`. The Definition fixes `r·µ¢ = 0` for every `i` with `1 ‚â§ i < k`; instantiating at `i := n` yields `r‚Çô = 0`, contradicting `r‚Çô ‚âÝ 0` from `n ‚àà S`. Hence `¬¨(n < k)`, and NAT-order's at-least-one trichotomy at `(k, n)` gives `k < n ‚à® k = n ‚à® n < k`; the excluded disjunct drops out, and NAT-order's defining clause `k ‚â§ n ‚ü∫ k < n ‚à® k = n` folds the remainder to `k ‚â§ n`. Hence `(A n ‚àà S :: k ‚â§ n)`, and ActionPoint's uniqueness clause names `k` as the action point: **`actionPoint(a ‚äñ w) = k = zpd(a, w)`**.

When `zpd(a, w)` is undefined, the Definition fixes `a ‚äñ w = [0, ‚Ä¶, 0]` of length `L`, so every component satisfies `r·µ¢ = 0` for `1 ‚â§ i ‚â§ L`. Using the postcondition `#(a ‚äñ w) = L` to retype the index range, the universal `(A i ‚àà ‚Ñï : 1 ‚â§ i ‚â§ #(a ‚äñ w) : r·µ¢ = 0)` holds, discharging **`Zero(a ‚äñ w)`** (TA-Pos). TA-Pos's complementarity `Pos(t) ‚ü∫ ¬¨Zero(t)` then yields `¬¨Pos(a ‚äñ w)`, so the previous paragraph's Pos witness and ActionPoint identification are inapplicable here; ActionPoint's `Pos(w)` precondition fails at `w := a ‚äñ w`, leaving `actionPoint(a ‚äñ w)` unspecified.

*Formal Contract:*
- *Preconditions:* a ‚àà T, w ‚àà T, a ‚â• w (T1).
- *Depends:*
  - T0 (CarrierSetDefinition) ‚Äî carrier membership `a ‚äñ w ‚àà T` and per-operand length bounds `#a ‚â• 1`, `#w ‚â• 1`.
  - T1 (LexicographicOrder) ‚Äî precondition ordering `a ‚â• w`; trichotomy derives `w < a`; T1 case (ii) is eliminated in Divergence case (i) (its prefix-agreement range `1 ‚â§ i ‚â§ #w` instantiated at `i := k` would force `w‚Çñ = a‚Çñ`, contradicting `w‚Çñ ‚âÝ a‚Çñ`) and in Divergence case (ii) sub-case (ii-b) (it yields `a < w`, contradicting `w < a`); in Divergence case (i), T1 case (i)'s witness `j` satisfies `w‚±º < a‚±º` and, once identified with `k` via Divergence's uniqueness, supplies `w‚Çñ < a‚Çñ`.
  - T3 (CanonicalRepresentation) ‚Äî contrapositive `a ‚âÝ b ‚ü∫ #a ‚âÝ #b ‚à® (‚àÉ j : 1 ‚â§ j ‚â§ #a : a‚±º ‚âÝ b‚±º)` instantiated at `(a, w)` discharges `a ‚âÝ w` from the not-zero-padded-equal hypothesis by case analysis on the padded-disagreement index `i`: case (A) (`i ‚â§ #a ‚àß i ‚â§ #w`) fires the existential disjunct via ZPD's padded-projection equality (`√¢·µ¢ = a·µ¢`, `≈µ·µ¢ = w·µ¢` lift `√¢·µ¢ ‚âÝ ≈µ·µ¢` to `a·µ¢ ‚âÝ w·µ¢`); case (B) (`i` in a padding zone) fires the length disjunct via NAT-order's trichotomy forcing `#a ‚âÝ #w`.

  - ZPD ‚Äî defines `zpd(a, w)`; padded-projection equality clauses `√¢·µ¢ = a·µ¢` (for `1 ‚â§ i ‚â§ #a`) and `≈µ·µ¢ = w·µ¢` (for `1 ‚â§ i ‚â§ #w`) used in case (A) of the precondition's `a ‚âÝ w` derivation to lift padded disagreement to native; padding clauses `√¢·µ¢ = 0` (for `#a < i ‚â§ L`) and `≈µ·µ¢ = 0` (for `#w < i ‚â§ L`) used in case (B) to force the disagreement index into the longer operand's native domain; Relationship-to-Divergence identifies `zpd = divergence` under case (i); case-split and minimality under case (ii).

The left-identity clause is instantiated at `n := ≈µ‚Çñ` to rewrite `0 + ≈µ‚Çñ = ≈µ‚Çñ` in the Pos derivation, bridging the supposition `r‚Çñ = 0` to the contradiction `≈µ‚Çñ = √¢‚Çñ`.
- *Definition:* NAT-order's trichotomy on `(#a, #w)` selects exactly one of: (Œ±) `#a = #w`, `L = #a`; (Œ≤) `#a < #w`, `L = #w`; (Œ≥) `#w < #a`, `L = #a`. a ‚äñ w is computed by case analysis on k = zpd(a, w) (ZPD) using ZPD's padded projections `√¢`, `≈µ` on `{1, ..., L}` for every component reference at indices that may exceed the native domain: r·µ¢ = 0 for i < k, r‚Çñ = √¢‚Çñ ‚àí ≈µ‚Çñ, r·µ¢ = √¢·µ¢ for i > k; when zpd(a, w) is undefined, a ‚äñ w = [0, ‚Ä¶, 0]; #(a ‚äñ w) = L.
- *Postconditions:* a ‚äñ w ‚àà T, #(a ‚äñ w) = L (the longer of `#a` and `#w`, named by NAT-order trichotomy per the Definition); when zpd(a, w) is defined: Pos(a ‚äñ w) (TA-Pos), actionPoint(a ‚äñ w) = zpd(a, w) (ActionPoint); when zpd(a, w) is undefined: Zero(a ‚äñ w) (TA-Pos).
