### NAT-zero's "minimum reading" is stated in `<`/`=` form rather than `≤`
**Class**: OBSERVE
**ASN**: NAT-zero axiom: `(A n ∈ ℕ :: 0 < n ∨ 0 = n)`. *Consequence*: `(A n ∈ ℕ :: ¬(n < 0))`.
**Issue**: NAT-order's *Definition* introduces `≤` exactly as `m < n ∨ m = n`, and NAT-zero depends on NAT-order. The axiom clause `0 < n ∨ 0 = n` is therefore literally `0 ≤ n` under the definitions in scope; writing it expanded reads as if `≤` were unavailable here, even though the depend chain makes it available. Stating the axiom as `(A n ∈ ℕ :: 0 ≤ n)` would let the prose name `0` as "the minimum" using the relation that already carries that meaning.
