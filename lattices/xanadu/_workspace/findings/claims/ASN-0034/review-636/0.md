### NAT-addbound's Depends entries recapitulate body proof steps
**Class**: REVISE
**Foundation**: NAT-addbound (NatAdditionDominatesOperands)
**ASN**: The four Depends entries in NAT-addbound's Formal Contract. Example (NAT-zero entry):
> "NAT-zero (NatZeroMinimum) — supplies the minimality clause `(A k ∈ ℕ :: 0 < k ∨ 0 = k)`, instantiated at `k := m` in the right-dominance derivation (yielding `0 ≤ m` once NAT-order's `≤`-definition unfolds the disjunction) and at `k := n` in the left-dominance derivation (yielding `0 ≤ n` by the same unfolding)."

And the NAT-order entry:
> "supplies the defining equivalence `a ≤ b ⟺ a < b ∨ a = b` (used at `(0, m)` and `(0, n)` to convert NAT-zero's disjunctions to `0 ≤ m` and `0 ≤ n` respectively) and `a ≥ b ⟺ b ≤ a` (used at `(m + n, n)` to convert `n ≤ m + n` to `m + n ≥ n` and at `(m + n, m)` to convert `m ≤ m + n` to `m + n ≥ m`)."

**Issue**: Each Depends entry walks the same instantiation steps and intermediate forms that the body's *Right dominance* and *Left dominance* proofs already establish. The yielded forms (`0 ≤ m`, `0 + n ≤ m + n`, `n ≤ m + n`) and the unfolding/conversion sequence are the body's reasoning, restated inside the structural slot. This is the same body↔structural-slot doubling pattern flagged on NAT-order's *Consequence* slot in the previous cycle, only inverted: the body is the load-bearing derivation, and the Depends slot mirrors it. A precise reader cross-checking the proof must read the body, then re-read the Depends entries to confirm they say the same thing; a future sharpening of the proof would have to update both surfaces. The Depends slot's purpose is to declare which clause each dep supplies and (coarsely) where it lands — not to re-derive the proof.

**What needs resolving**: NAT-addbound's Depends entries should be trimmed to state which clause/equivalence each dep supplies and which derivation (right-dominance, left-dominance) consumes it, without recapitulating the intermediate forms (`0 ≤ m`, `0 + n ≤ m + n`, etc.) or the unfolding/conversion mechanics. The body proofs remain the single home for those steps.
