**T4a (SyntacticEquivalence).** T4's field-segment constraint ‚ÄĒ stated positionally as (i) no two zeros are adjacent, (ii) `t‚āĀ ‚Č† 0`, (iii) `t_{#t} ‚Č† 0` ‚ÄĒ is equivalent to the condition that every *field segment* of `t` is non-empty, where the field segments are the `zeros(t) + 1` maximal contiguous sub-sequences of non-zero positions delimited by the zeros.

Let `t ‚ąą T` be a tumbler with `zeros(t) ‚Č§ 3`. Set `k = zeros(t)`; T4's Exhaustion Consequence ‚ÄĒ universally quantified as `(A t ‚ąą T : zeros(t) ‚Č§ 3 : zeros(t) ‚ąą {0, 1, 2, 3})`, with hypothesis matching T4a's precondition pointwise ‚ÄĒ instantiates at the local `t` to pin `k ‚ąą {0, 1, 2, 3}` directly. Enumerate the zeros of `t` at positions `s‚āĀ < s‚āā < ‚Ä¶ < s_k` (strict increase by NAT-order; the length `k` of this enumeration equals `zeros(t)` by NAT-card's enumeration characterisation of `|¬∑|` applied at the zero-index subset `{i : 1 ‚Č§ i ‚Č§ #t ‚ąß tŠĶĘ = 0} ‚äÜ {1, ‚Ä¶, #t}`, whose cardinality T4 identifies with `zeros(t)`). Set `s‚āÄ = 0` and `s_{k+1} = #t + 1` as sentinels. The arithmetic in what follows ‚ÄĒ the numerals `2` and `3`, the sums `s_i + 1`, `s_i + 2`, and `#t + 1`, and the last-segment upper bound `s_{k+1} ‚ąí 1` ‚ÄĒ is grounded thus: NAT-closure posits `1 ‚ąą ‚Ąē` and closes ‚Ąē under addition, so `2 := 1 + 1 ‚ąą ‚Ąē`, `3 := 2 + 1 ‚ąą ‚Ąē`, and each of `s_i + 1`, `s_i + 2`, `#t + 1` lies in ‚Ąē. NAT-sub's right-telescoping clause `(m + n) ‚ąí n = m`, instantiated at `m = #t, n = 1`, reduces `s_{k+1} ‚ąí 1 = (#t + 1) ‚ąí 1` to `#t` in ‚Ąē, so the last segment occupies the ‚Ąē-interval `[s_k + 1, #t]`. T0's Axiom includes the nonemptiness clause `(A a ‚ąą T :: 1 ‚Č§ #a)`; instantiating its universally quantified `a` at `t` (licensed by `t ‚ąą T`) yields `#t ‚Č• 1` directly as a first-class axiom citation, and this bound underwrites the `k = 0` branches of Forward Conditions (ii) and (iii), where the single segment equals `t` itself. The `k + 1` field segments of `t` are the index ranges `(s_i, s_{i+1})` for `i = 0, 1, ‚Ä¶, k` ‚ÄĒ i.e. the component sub-sequences `t[s_i + 1 .. s_{i+1} - 1]`. A segment is non-empty precisely when `s_{i+1} ‚Č• s_i + 2`; for the last segment, whose index range is `[s_k + 1, #t]` as shown above, this specialises to the native `+1` form `#t ‚Č• s_k + 1` ‚ÄĒ equivalently `s_k + 1 ‚Č§ #t` ‚ÄĒ which is what the Reverse Last-segment argument lands on directly via NAT-discrete.

*Forward.* Assume every field segment is non-empty. We derive each positional condition.

*Condition (ii): `t‚āĀ ‚Č† 0`.* The first field segment is `t[1 .. s‚āĀ - 1]` when `k ‚Č• 1`, or `t[1 .. #t]` when `k = 0`; in either case its first index is 1. Segment non-emptiness gives `s‚āĀ ‚Č• 2` (when `k ‚Č• 1`) or `#t ‚Č• 1` (when `k = 0`), so index 1 lies in the segment and `t‚āĀ` is a non-zero component. Hence `t‚āĀ ‚Č† 0`.

*Condition (iii): `t_{#t} ‚Č† 0`.* The last field segment is `t[s_k + 1 .. #t]` when `k ‚Č• 1`, or `t[1 .. #t]` when `k = 0`; in either case its last index is `#t`. Segment non-emptiness, in the last-segment `+1` specialisation stated above, gives `s_k + 1 ‚Č§ #t` (when `k ‚Č• 1`) or `#t ‚Č• 1` (when `k = 0`), so index `#t` lies in the segment and `t_{#t}` is a non-zero component. Hence `t_{#t} ‚Č† 0`.

*Condition (i): no adjacent zeros.* Suppose for contradiction that `tŠĶĘ = 0 ‚ąß tŠĶĘ‚āä‚āĀ = 0` for some `i` with `1 ‚Č§ i < #t`. Then `i` and `i + 1` are both zero positions, so `i = s_j` and `i + 1 = s_{j'}` for some enumeration indices `j < j'` (strict monotonicity from `i < i + 1`, where the strict successor inequality `i < i + 1` is NAT-addcompat's third axiom clause `(A n ‚ąą ‚Ąē :: n < n + 1)` instantiated at `n := i`); since `i + 1 ‚Č§ #t` places the second zero inside the enumerated range `{s_1, ‚Ä¶, s_k}`, we have `j' ‚Č§ k`, hence `j + 1 ‚Č§ j' ‚Č§ k` ‚ÄĒ so `s_{j+1}` is a real enumeration entry (not the sentinel `s_{k+1} = #t + 1`). We claim `s_{j+1} = i + 1`: if instead `j + 1 < j'`, strict monotonicity would give `s_j < s_{j+1} < s_{j'}`, i.e., `i < s_{j+1} < i + 1`, and NAT-discrete's no-interval Consequence `m ‚Č§ n < m + 1 ‚üĻ n = m`, instantiated at `(m, n) := (i, s_{j+1})`, would force `s_{j+1} = i`, contradicting `s_j < s_{j+1}`. Hence `s_{j+1} = i + 1`, and the interior field segment `t[s_j + 1 .. s_{j+1} - 1] = t[i + 1 .. i]` has no indices, so it is empty. This contradicts segment non-emptiness. Hence no two zeros are adjacent.

*Reverse.* Assume conditions (i), (ii), and (iii) hold. We show every field segment is non-empty.

*First segment (`i = 0`).* The segment occupies indices 1 through `s‚āĀ - 1` (or 1 through `#t` when `k = 0`). If `k = 0` the segment equals `t` itself, which has `#t ‚Č• 1` indices. If `k ‚Č• 1`, condition (ii) forces `t‚āĀ ‚Č† 0`, so index 1 is not a zero position, and therefore `s‚āĀ ‚Č• 2` ‚ÄĒ the segment has at least one index.

*Last segment (`i = k`).* The segment occupies indices `s_k + 1` through `#t` (when `k ‚Č• 1`); non-emptiness of this index range is the `+1` inequality `s_k + 1 ‚Č§ #t`. Condition (iii) forces `t_{#t} ‚Č† 0`, so index `#t` is not among the zero positions `s‚āĀ < ‚Ä¶ < s_k`, giving `s_k ‚Č† #t`. Every zero position `s_j` is an index of `t`, hence `s_k ‚Č§ #t`; unfolding this by NAT-order's definition `s_k ‚Č§ #t ‚üļ s_k < #t ‚ą® s_k = #t` and excluding the `s_k = #t` branch by `s_k ‚Č† #t` leaves `s_k < #t`. NAT-discrete's axiom `m < n ‚üĻ m + 1 ‚Č§ n`, instantiated at `(m, n) := (s_k, #t)`, promotes this strict inequality directly to its native `+1` output `s_k + 1 ‚Č§ #t` ‚ÄĒ the segment has at least one index.

*Interior segments (`1 ‚Č§ i < k`).* Each such segment occupies indices `s_i + 1` through `s_{i+1} - 1`. Condition (i) forbids `s_{i+1} = s_i + 1`, and since the zero positions are strictly increasing we already have `s_{i+1} ‚Č• s_i + 1`; together these give `s_{i+1} ‚Č• s_i + 2`, so the segment has at least one index.

All segments ‚ÄĒ first, interior, and last ‚ÄĒ are non-empty. ‚ąé
