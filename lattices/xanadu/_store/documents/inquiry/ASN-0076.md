title: EDITLINK Operation
out_of_scope: INSERT, DELETE, COPY, REARRANGE operation mechanics, document forking,
  version DAG structure, replication and inter-server protocol (BEBE)
question: Nelson (LM 4/79) lists 'editable links' as a desired feature. The current
  spec makes links immutable once created (L12). Editing a link should follow the
  same permanence pattern as editing a document — the original is never destroyed.
  A new link is created with modified endsets, and the relationship between old and
  new is recorded. The original link's guarantees are unaffected.
