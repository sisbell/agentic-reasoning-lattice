### TA1 statement quantifier domain unbounded
**Class**: OBSERVE
**ASN**: TA1 (OrderPreservationUnderAddition): "`(A a, b, w : a < b ∧ Pos(w) ∧ actionPoint(w) ≤ min(#a, #b) : a ⊕ w ≤ b ⊕ w)`"
**Issue**: The outer universal quantifier does not restrict `a, b, w` to `T`, unlike sibling claims (TA0 uses `(A a, w ∈ T : ... )`, TA-Pos uses `(A t ∈ T :: ...)`, ActionPoint uses "For `w ∈ T` with `Pos(w)`"). The Formal Contract Preconditions correctly list `a ∈ T, b ∈ T, w ∈ T`, and the inner predicates `<`, `Pos(·)`, `⊕` are only defined over `T`, so this is a framing issue rather than a soundness gap. Worth aligning with the surrounding style.

VERDICT: OBSERVE
