### Quantifier-typing notation drifts between T0 and TA-Pos
**Class**: OBSERVE
**ASN**: T0 comprehension axiom `(A i : 1 ≤ i ≤ p : tᵢ = r(i))` and extensionality `(A i : 1 ≤ i ≤ #a : aᵢ = bᵢ)` omit `i ∈ ℕ`; TA-Pos writes `(A i ∈ ℕ : 1 ≤ i ≤ #t : tᵢ = 0)` with the type marker.
**Issue**: Within a single ASN that pays close attention to grounding `≤` and `1` in NAT-order/NAT-closure, the silent-type binder `(A i : 1 ≤ i ≤ p : ...)` reads as informally typed. Sound by convention (`1 ≤ i ≤ p` forces `i` to a natural in context), but the inconsistency invites a reader to check whether the missing `i ∈ ℕ` is significant.
