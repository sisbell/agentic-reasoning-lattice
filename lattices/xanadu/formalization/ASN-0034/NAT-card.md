**NAT-card (NatFiniteSetCardinality).** For every `n ∈ ℕ` and every subset `S ⊆ {1, 2, …, n} ⊆ ℕ`, the cardinality `|S| ∈ ℕ` is the unique `k ∈ ℕ` such that `S` admits a strictly increasing enumeration `s₁ < s₂ < … < s_k` under `<` on ℕ with `S = {s₁, s₂, …, s_k}` (with `k = 0` and the empty enumeration when `S = ∅`, so `|∅| = 0`). The length `k` is uniquely determined by `S` because two strictly increasing enumerations of the same set must agree element-by-element under NAT-order's trichotomy; hence `|·|` is a well-defined total function on subsets of every initial segment `{1, …, n}` of ℕ with codomain ℕ, and `|S| ≤ n` for every such `S`. The operator `|·|` is distinct from T0's tumbler-length `#· : T → ℕ`, which acts on sequences rather than sets.

*Formal Contract:*
- *Axiom:* `(A n ∈ ℕ : S ⊆ {1, 2, …, n} :: |S| ∈ ℕ)` (conditional codomain); `|S|` is the unique `k ∈ ℕ` such that `S` admits a strictly increasing enumeration `s₁ < s₂ < … < s_k` under `<` on ℕ with `S = {s₁, s₂, …, s_k}` (empty enumeration when `k = 0`, giving `|∅| = 0`) (enumeration characterisation); `(A n ∈ ℕ : S ⊆ {1, 2, …, n} :: |S| ≤ n)` (upper bound). `|·|` acts on subsets of ℕ and is distinct from T0's tumbler-length `#·`, which acts on sequences.
