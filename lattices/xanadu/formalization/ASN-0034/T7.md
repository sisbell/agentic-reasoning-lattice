## First element-field component distinguishes element-level tumblers

Within a document's element space, the first component after the third zero delimiter identifies the *subspace*: 1 for text, 2 for links. Nelson also mentions that the link subspace "could be further subdivided."

**T7 (FirstElementFieldDistinction).** If two T4-valid element-level tumblers have different first element-field components, they are distinct tumblers:

  `(A a, b вҲҲ T : a, b satisfy T4 вҲ§ zeros(a) = zeros(b) = 3 : a.EвӮҒ вү  b.EвӮҒ вҹ№ a вү  b)`

*Proof.* We are given two T4-valid tumblers `a, b вҲҲ T` with `zeros(a) = zeros(b) = 3`, so by T4b `a.EвӮҒ` and `b.EвӮҒ` denote well-defined components, and `a.EвӮҒ вү  b.EвӮҒ`. We must show `a вү  b`.

By T4, every zero in either tumbler is a field separator. By T4a, T4's positional field-segment constraints force each of the four field segments вҖ” node, user, document, element вҖ” to be non-empty (so `Оұ, ОІ, Оі, Оҙ вүҘ 1` and `Оұ', ОІ', Оі', Оҙ' вүҘ 1`). By T0, every component lies in в„•, so every non-separator component is strictly positive.

Write the field lengths of `a` as `(Оұ, ОІ, Оі, Оҙ)`, so the three separators sit at positions `Оұ + 1`, `Оұ + ОІ + 2`, `Оұ + ОІ + Оі + 3`, and `a.EвӮҒ` occupies position `pвӮҗ = Оұ + ОІ + Оі + 4`. Analogously for `b`, write `(Оұ', ОІ', Оі', Оҙ')` with `p_b = Оұ' + ОІ' + Оі' + 4`.

*Case 1* (`pвӮҗ = p_b`). Both first element-field components sit at position `p`. By hypothesis `a[p] вү  b[p]`, so by T3, `a вү  b`.

*Case 2* (`pвӮҗ вү  p_b`).

*Sub-case 2a* (`#a вү  #b`). By T3, `a вү  b`.

*Sub-case 2b* (`#a = #b`). Suppose for contradiction the separator-position sets `{Оұ + 1, Оұ + ОІ + 2, Оұ + ОІ + Оі + 3}` and `{Оұ' + 1, Оұ' + ОІ' + 2, Оұ' + ОІ' + Оі' + 3}` coincide. Match them pairwise in ascending order.

From `Оұ + 1 = Оұ' + 1`: by NAT-cancel (right cancellation at `m = 1`), `Оұ = Оұ'`.

From `Оұ + ОІ + 2 = Оұ + ОІ' + 2`: by NAT-addassoc (at `m = Оұ, n = ОІ, p = 2`, and with `ОІ'`), rewrite as `(Оұ + ОІ) + 2 = (Оұ + ОІ') + 2`; by NAT-cancel right cancellation at `m = 2`, `Оұ + ОІ = Оұ + ОІ'`; by NAT-cancel left cancellation at `m = Оұ`, `ОІ = ОІ'`.

From `Оұ + ОІ + Оі + 3 = Оұ + ОІ + Оі' + 3`: by NAT-addassoc (at `m = Оұ + ОІ, n = Оі, p = 3`, and with `Оі'`), rewrite as `(Оұ + ОІ + Оі) + 3 = (Оұ + ОІ + Оі') + 3`; by NAT-cancel right cancellation at `m = 3`, `Оұ + ОІ + Оі = Оұ + ОІ + Оі'`; by NAT-cancel left cancellation at `m = Оұ + ОІ`, `Оі = Оі'`.

Then `pвӮҗ = Оұ + ОІ + Оі + 4 = Оұ' + ОІ' + Оі' + 4 = p_b`, contradicting `pвӮҗ вү  p_b`.

So the separator-position sets differ: there exists a position `j` that is a separator in one tumbler but not the other. At `j`, one tumbler has `0` and the other has a non-separator component, strictly positive by T0 and T4's role-assignment. Hence `a[j] вү  b[j]`, and by T3, `a вү  b`. вҲҺ

*Formal Contract:*
- *Preconditions:* `a, b вҲҲ T` satisfy the T4 constraints вҖ” at most three zero-valued components, no two zeros adjacent, `aвӮҒ вү  0`, `a_{#a} вү  0` (and likewise for `b`) вҖ” and have `zeros(a) = zeros(b) = 3`.
- *Depends:*
  - T0 (CarrierSetDefinition) вҖ” components lie in в„•; supplies strict positivity of non-separator components.
  - T3 (CanonicalRepresentation) вҖ” tumblers are equal iff same length and agree at every position; converts positional/length disagreement to tumbler inequality.
  - T4 (HierarchicalParsing) вҖ” constrains the bound variables; supplies the role-assignment under which zeros are separators.
  - T4a (SyntacticEquivalence) вҖ” converts T4's positional clauses to segment non-emptiness, fixing `Оұ, ОІ, Оі, Оҙ вүҘ 1`.
  - T4b (UniqueParse) вҖ” licenses the well-definedness of `t.EвӮҒ` on T4-valid inputs with `zeros(t) = 3`.
  - NAT-cancel (NatAdditionCancellation) вҖ” right/left cancellation discharges the three pairwise-matching extractions `Оұ = Оұ'`, `ОІ = ОІ'`, `Оі = Оі'`.
  - NAT-addassoc (NatAdditionAssociative) вҖ” regroups `Оұ + ОІ + 2` and `Оұ + ОІ + Оі + 3` so NAT-cancel applies at `m = 2` and `m = 3`.
- *Postconditions:* `a.EвӮҒ вү  b.EвӮҒ вҹ№ a вү  b`.

The ordering T1 places all text addresses (subspace 1) before all link addresses (subspace 2) within the same document, because `1 < 2` at the subspace position вҖ” a consequence of the lexicographic order, not an assumption.
