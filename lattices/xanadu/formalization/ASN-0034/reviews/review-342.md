### T1 part (c) Case `k₂ < k₁` case-(ii) uses the contrapositive of NAT-discrete, not the forward direction
**Foundation**: NAT-discrete (NatDiscreteness) — stated as `m < n ⟹ m + 1 ≤ n`, with T0 introducing this as the "forward direction" content of the axiom.
**ASN**: T1's Depends entry: "NAT-discrete (NatDiscreteness) — forward direction `m < n ⟹ m + 1 ≤ n`, used in Case 3 to supply the successor-comparison step and in part (c) Case `k₂ < k₁` case-(ii) branch to obtain `k₂ ≤ m` from `k₂ < m + 1`." T1's proof text at that site: "Since `k₂ < k₁` and `a` has components below `k₁`, `k₂ ≤ m`" (implicitly discharging `k₂ < m + 1 ⟹ k₂ ≤ m`, since the case-(ii) branch of `a < b` sets `k₁ = m + 1`).
**Issue**: The step "`k₂ ≤ m` from `k₂ < m + 1`" is *not* the forward direction `m < n ⟹ m + 1 ≤ n` applied directly — that application (at `x = k₂, y = m + 1`) would yield `k₂ + 1 ≤ m + 1`, from which stripping the `+ 1` to reach `k₂ ≤ m` requires a non-cited monotonicity or cancellation step. The clean derivation is the *contrapositive* of the forward direction at `(x = m, y = k₂)`: from `m < k₂ ⟹ m + 1 ≤ k₂`, contrapose with NAT-order trichotomy to obtain `k₂ < m + 1 ⟹ k₂ ≤ m`. T1's Depends therefore mis-labels this use-site the same way the cycle-2 regional finding flags for Case 1. Even though a companion finding already addresses Case 1's mis-labeled contrapositive use, part (c)'s use-site is labeled "forward direction" in the same sentence and is a distinct site the Case-1 finding does not touch; resolving that finding by adding Case 1 to a "contrapositive" use-list would still leave part (c) mis-listed as a forward-direction consumer.
**What needs resolving**: Either re-label part (c) Case `k₂ < k₁` case-(ii)'s use-site as the contrapositive of NAT-discrete's forward direction (alongside Case 1 under whatever scope entry the Case-1 resolution adopts), or rewrite the proof text at that site so that `k₂ ≤ m` is obtained by a route that genuinely uses the forward direction — e.g., chaining the forward direction with an explicitly cited stripping step (NAT-sub's right telescoping + strict monotonicity, or NAT-cancel's right cancellation composed with the `≤` disjunct split).
