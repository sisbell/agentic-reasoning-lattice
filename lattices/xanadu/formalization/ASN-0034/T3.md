## Sequence equality

Equality of tumblers must mean component-wise identity.

**T3 (SequenceEqualityIsComponentwise).** `(A a, b ∈ T : a = b ≡ #a = #b ∧ (A i : 1 ≤ i ≤ #a : aᵢ = bᵢ))`.

Two tumblers are equal exactly when they have the same length and agree at every position. Conversely, if they differ in any component or in length, they are distinct.

This is the extensional characterization of equality for finite sequences. T0 commits to `T` as the set of nonempty finite sequences over ℕ with length operator `#·` and component projection `·ᵢ`, but it does not by itself supply the principle that two sequences sharing length and agreeing at every index are equal — that is not among T0's clauses. We therefore posit the principle here as the extensionality axiom for the carrier, on which every downstream use of "`a = b`" within `T` depends: addressing, span containment, and allocation all lean on equality being literal, not "up to some external identification." T3 takes the standard extensional reading of finite sequences as primitive for `T`, rather than deriving it from a weaker substrate.

T3 does not assert that each abstract value has a unique representative. Inside the carrier `T`, the sequences `[0, 0, 5]` and `[0, 7]` are already distinct elements: no normalization is imposed, and none is needed to distinguish them. If the system later wishes to regard different sequences as denoting the same value — for instance, by collapsing leading-zero variants — that is a separate construction requiring a denotation relation on sequences and a normalization function selecting a canonical witness, with its own proof obligations. Those obligations, together with any argument about comparison-function correctness under normalization (e.g., behavior of a zero-detection routine when inspecting only the first mantissa slot), lie outside T3 and must be discharged by a distinct claim.

*Formal Contract:*
- *Axiom:* For all `a, b ∈ T`, `a = b ≡ #a = #b ∧ (A i : 1 ≤ i ≤ #a : aᵢ = bᵢ)`.
- *Depends:*
  - T0 (CarrierSetDefinition) — supplies the carrier `T`, the length operator `#·`, and the component projection `·ᵢ` with its index domain `{1, …, #a}`, furnishing the vocabulary in which the axiom is stated. T3 does not derive extensional equality from T0; it axiomatizes it over the carrier T0 introduces.
