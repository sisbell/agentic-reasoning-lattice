**NAT-order (NatStrictTotalOrder).** The binary relation `<` on ‚Ąē is a strict total order, with non-strict companion `‚Č§` defined by `m ‚Č§ n ‚üļ m < n ‚ą® m = n` and reverse companions `‚Č•` and `>` defined by `m ‚Č• n ‚üļ n ‚Č§ m` and `m > n ‚üļ n < m`.

Strict total order on ‚Ąē means three clauses hold jointly:
- Irreflexivity: `¬¨(n < n)` for every `n ‚ąą ‚Ąē`
- Transitivity: `m < n ‚ąß n < p ‚üĻ m < p` for every `m, n, p ‚ąą ‚Ąē`
- At-least-one trichotomy: for any `m, n ‚ąą ‚Ąē`, at least one of `m < n`, `m = n`, `n < m` holds

Together the three clauses yield *exactly-one trichotomy*: for any `m, n ‚ąą ‚Ąē`, exactly one of `m < n`, `m = n`, `n < m` holds. Exactly-one is the at-least-one disjunction conjoined with three pairwise mutual-exclusion clauses. `¬¨(m < n ‚ąß n < m)` follows from transitivity ‚ÄĒ which chains `m < n` and `n < m` to `m < m` ‚ÄĒ against irreflexivity. `¬¨(m < n ‚ąß m = n)` follows by substituting `m = n` into `m < n` via indiscernibility of `=`, rewriting to `m < m`, which irreflexivity at `n := m` rules out. `¬¨(m = n ‚ąß n < m)` follows by the same substitution applied to `n < m`: rewriting under `m = n` yields `m < m`, again against irreflexivity.

The non-strict companion `‚Č§` inherits transitivity from `<` and `=` jointly: `m ‚Č§ n ‚ąß n ‚Č§ p ‚üĻ m ‚Č§ p`. Unfolding each hypothesis against the defining disjunction `x ‚Č§ y ‚üļ x < y ‚ą® x = y` yields four cases, and each discharges to `m ‚Č§ p`. If both are strict ‚ÄĒ `m < n ‚ąß n < p` ‚ÄĒ `<`-transitivity gives `m < p`, so `m ‚Č§ p` by the definition. If the first is strict and the second is equality ‚ÄĒ `m < n ‚ąß n = p` ‚ÄĒ indiscernibility of `=` substitutes `n = p` into `m < n` to yield `m < p`. The symmetric case `m = n ‚ąß n < p` substitutes `m = n` into `n < p` to yield `m < p`. If both are equalities ‚ÄĒ `m = n ‚ąß n = p` ‚ÄĒ transitivity of `=` gives `m = p`, so `m ‚Č§ p`. We export `‚Č§`-transitivity as a Consequence so consumers can chain non-strict bounds without re-deriving the case split.

*Formal Contract:*
- *Axiom:* `< ‚äÜ ‚Ąē √ó ‚Ąē` (`<` is a binary relation on ‚Ąē); `(A n ‚ąą ‚Ąē :: ¬¨(n < n))` (irreflexivity); `(A m, n, p ‚ąą ‚Ąē : m < n ‚ąß n < p : m < p)` (transitivity); `(A m, n ‚ąą ‚Ąē :: m < n ‚ą® m = n ‚ą® n < m)` (at-least-one trichotomy).
- *Consequence:* Exactly-one trichotomy: `(A m, n ‚ąą ‚Ąē :: (m < n ‚ą® m = n ‚ą® n < m) ‚ąß ¬¨(m < n ‚ąß n < m) ‚ąß ¬¨(m < n ‚ąß m = n) ‚ąß ¬¨(m = n ‚ąß n < m))`. The disjunction is the at-least-one axiom clause directly; `¬¨(m < n ‚ąß n < m)` follows from transitivity and irreflexivity; `¬¨(m < n ‚ąß m = n)` follows by substituting `m = n` into `m < n` via indiscernibility of `=`, rewriting to `m < m` against irreflexivity at `n := m`; `¬¨(m = n ‚ąß n < m)` follows by the same substitution applied to `n < m`.
- *Consequence:* `‚Č§`-transitivity: `(A m, n, p ‚ąą ‚Ąē : m ‚Č§ n ‚ąß n ‚Č§ p : m ‚Č§ p)`. Unfolding each hypothesis by the definition `x ‚Č§ y ‚üļ x < y ‚ą® x = y` yields four cases. `m < n ‚ąß n < p` gives `m < p` by `<`-transitivity, hence `m ‚Č§ p`. `m < n ‚ąß n = p` gives `m < p` by substituting `n = p` into `m < n` via indiscernibility of `=`, hence `m ‚Č§ p`. `m = n ‚ąß n < p` gives `m < p` by substituting `m = n` into `n < p`, hence `m ‚Č§ p`. `m = n ‚ąß n = p` gives `m = p` by transitivity of `=`, hence `m ‚Č§ p`.
- *Definition:* `(A m, n ‚ąą ‚Ąē :: m ‚Č§ n ‚üļ m < n ‚ą® m = n)`; `(A m, n ‚ąą ‚Ąē :: m ‚Č• n ‚üļ n ‚Č§ m)`; `(A m, n ‚ąą ‚Ąē :: m > n ‚üļ n < m)`.
- *Depends:* (none).
