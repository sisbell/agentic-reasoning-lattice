**NAT-order (NatStrictTotalOrder).** The binary relation `<` on в„• is a strict total order, with non-strict companion `вүӨ` defined by `m вүӨ n вҹә m < n вҲЁ m = n` and reverse companions `вүҘ` and `>` defined by `m вүҘ n вҹә n вүӨ m` and `m > n вҹә n < m`.

Strict total order on в„• means three clauses hold jointly:
- Irreflexivity: `В¬(n < n)` for every `n вҲҲ в„•`
- Transitivity: `m < n вҲ§ n < p вҹ№ m < p` for every `m, n, p вҲҲ в„•`
- At-least-one trichotomy: for any `m, n вҲҲ в„•`, at least one of `m < n`, `m = n`, `n < m` holds

The three clauses jointly export *exactly-one trichotomy* as a Consequence: for any `m, n вҲҲ в„•`, exactly one of `m < n`, `m = n`, `n < m` holds. Exactly-one is the at-least-one disjunction conjoined with three pairwise mutual-exclusion clauses. `В¬(m < n вҲ§ n < m)` follows from transitivity вҖ” which chains `m < n` and `n < m` to `m < m` вҖ” against irreflexivity. `В¬(m < n вҲ§ m = n)` follows by substituting `m = n` into `m < n` via indiscernibility of `=` вҖ” a logical property of equality available throughout, not a property of `<` вҖ” rewriting to `m < m`, which irreflexivity at `n := m` rules out. `В¬(m = n вҲ§ n < m)` follows by the same substitution applied to `n < m`: rewriting under `m = n` yields `m < m`, again against irreflexivity. The familiar disjointness form `m < n вҹ№ m вү  n` is the contrapositive of the mutual-exclusion conjunct `В¬(m < n вҲ§ m = n)` вҖ” a derivable restatement of that conjunct, not a separately exported Consequence; a consumer wanting the implicational form unfolds it from the exactly-one trichotomy bullet at the point of use. Stating it directly in the axiom slot would still launder the irreflexivity-plus-indiscernibility derivation through a non-minimal axiom clause, which is why the contract leaves it as a derivable restatement rather than naming it independently.

The axiom slot introduces `<` before constraining it: the first clause `< вҠҶ в„• Г— в„•` posits `<` as a binary relation on в„•, and the three strict-total-order clauses that follow then constrain that relation. NAT-closure follows the same register for the arithmetic primitive, opening its axiom slot with the signature `+ : в„• Г— в„• вҶ’ в„•` before the unit-membership and left-identity clauses.

The Definition slot introduces the non-strict companion `вүӨ` from `<` and logical equality, and the reverse companions `вүҘ` and `>` as the converses of `вүӨ` and `<` respectively. These are notational definitions, not additional axioms: every downstream occurrence of `m вүҘ n`, `m > n` unfolds to `n вүӨ m`, `n < m` and inherits the strict-total-order properties through that unfolding.
