TumblerAdd's constructive definition determines each component of the result from exactly one input. This makes the operation left-cancellative.

**TA-LC (LeftCancellation).** If a έΛΧ x = a έΛΧ y with both sides well-defined (TA0 satisfied for both), then x = y.

Let `kέΓΒ` be the action point of `x` and `kέΓΓ` the action point of `y`. Both exist because TA0 requires `Pos(x)` and `Pos(y)`, so each has at least one nonzero component. We eliminate both strict orderings.

**Case kέΓΒ < kέΓΓ.** Since `kέΓΒ < kέΓΓ` and the action point is the first nonzero component, every component of `y` before position `kέΓΓ` is zero έΑΦ in particular `y_{kέΓΒ} = 0`. In the addition `a έΛΧ x`, position `kέΓΒ` is the action point itself, so TumblerAdd gives `(a έΛΧ x)_{kέΓΒ} = a_{kέΓΒ} + x_{kέΓΒ}`. From `a έΛΧ x = a έΛΧ y` we obtain `a_{kέΓΒ} + x_{kέΓΒ} = a_{kέΓΒ}`, and summand absorption on έΕΧ έΑΦ `m + n = m έθ╣ n = 0` (NAT-cancel) έΑΦ instantiated with `m = a_{kέΓΒ}` and `n = x_{kέΓΒ}` gives `x_{kέΓΒ} = 0`. **Case kέΓΓ < kέΓΒ.** Since `kέΓΓ < kέΓΒ` and the action point is the first nonzero component, every component of `x` before position `kέΓΒ` is zero έΑΦ in particular `x_{kέΓΓ} = 0`. Both strict orderings are impossible. **Positions i < k.** Both `x` and `y` have action point `k`, so by definition of action point every component before `k` is zero: `xά╡λ = 0` and `yά╡λ = 0`. Therefore `xά╡λ = yά╡λ = 0`.

TumblerAdd is *left-cancellative*: the start position can be "divided out" from equal results, recovering the displacement uniquely. This is a direct consequence of TumblerAdd's constructive definition έΑΦ each component of the result is determined by exactly one input, so equality of results propagates back to equality of inputs.

*Worked example.* Let a = [2, 5] and suppose a έΛΧ x = a έΛΧ y = [2, 8]. We recover x and y uniquely. First, the action points must agree. Suppose k_x = 1: then (a έΛΧ x)έΓΒ = aέΓΒ + xέΓΒ = 2 + xέΓΒ = 2, giving xέΓΒ = 0, which contradicts k_x = 1 being the first nonzero component.
