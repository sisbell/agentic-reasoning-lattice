## The total order

We require a total order on T. Nelson describes the "tumbler line" as a single linear sequence: "In a sense the tumbler line is like the real line, i.e., the line of integers and all the numbers in between." The system maps a hierarchical tree вҖ” servers containing accounts containing documents containing elements вҖ” onto this flat line via depth-first traversal, which produces a total order: for any two nodes, traversal visits one before the other. The ordering rule is lexicographic:

**T1 (LexicographicOrder).** For tumblers `a = aвӮҒ. ... .aвӮҳ` and `b = bвӮҒ. ... .bвӮҷ`, define `a < b` iff there exists `k вүҘ 1` such that `(A i : 1 вүӨ i < k : aбөў = bбөў)` and either:

  (i) `k вүӨ m вҲ§ k вүӨ n` and `aвӮ– < bвӮ–`, or

  (ii) `k = m + 1 вүӨ n` (that is, `a` is a proper prefix of `b`).

The prefix convention вҖ” a prefix is less than any proper extension вҖ” is what makes depth-first traversal work. The server address `2` is less than every address within server `2`'s subtree, because every such address extends the prefix `2` with further components. Server `2`'s subtree begins immediately after `2` in the order and extends until some address whose first component exceeds `2`.

*Proof.* We show that `<` is a strict total order on T by establishing irreflexivity, trichotomy, and transitivity. The argument relies on `<` on в„• (NAT-order) and on T3 (CanonicalRepresentation): tumblers with the same length and identical components at every position are equal.

*(a) Irreflexivity.* We must show: `(A a вҲҲ T :: В¬(a < a))`.

Suppose `a < a` for some `a вҲҲ T` with `#a = m`. Then there exists `k вүҘ 1` with `aбөў = aбөў` for all `1 вүӨ i < k` (vacuously satisfied) and either (i) `k вүӨ m` and `aвӮ– < aвӮ–`, or (ii) `k = m + 1 вүӨ m`. Case (i) violates NAT-order's irreflexivity. Case (ii) requires `m + 1 вүӨ m`; NAT-addcompat gives `m < m + 1`, and NAT-order's trichotomy then excludes both `m + 1 < m` and `m + 1 = m`. Both cases produce contradictions.

*(b) Trichotomy.* We must show: `(A a, b вҲҲ T :: (a < b вҲЁ a = b вҲЁ b < a) вҲ§ В¬(a < b вҲ§ a = b) вҲ§ В¬(a < b вҲ§ b < a) вҲ§ В¬(a = b вҲ§ b < a))`.

Let `a, b вҲҲ T` with `#a = m` and `#b = n`. A *divergence position* is an index `k вүҘ 1` satisfying one of the clauses (Оұ) `k вүӨ m вҲ§ k вүӨ n вҲ§ aвӮ– вү  bвӮ–`, (ОІ) `k = m + 1 вҲ§ k вүӨ n`, or (Оі) `k = n + 1 вҲ§ k вүӨ m`. The proof splits first on whether any divergence position exists; when at least one does, NAT-wellorder applied to the nonempty set of such positions delivers a least element вҖ” the *first divergence position* `k` вҖ” after which a sub-split on which clause `k` satisfies completes the analysis. The three branches below are exhaustive.

*Case 1: no divergence position exists.* Then `aбөў = bбөў` for all `i` with `1 вүӨ i вүӨ m вҲ§ i вүӨ n` (no index satisfies clause (Оұ)), and the two exhaustion-shape divergences are absent too: `В¬(m + 1 вүӨ n)` and `В¬(n + 1 вүӨ m)` (no index satisfies (ОІ) or (Оі)). NAT-discrete's forward direction `m < n вҹ№ m + 1 вүӨ n` contraposes with the first to yield `В¬(m < n)`; symmetrically, `В¬(n < m)`. NAT-order's trichotomy at `(m, n)` then forces `m = n`. The shared-position equalities now range over all `1 вүӨ i вүӨ m`, so `a = b` by T3. Part (a) gives `В¬(a < a)` and `В¬(a > a)`.

*Case 2: the first divergence position `k` satisfies clause (Оұ)* вҖ” `k вүӨ m вҲ§ k вүӨ n вҲ§ aвӮ– вү  bвӮ–`. Minimality of `k` gives `aбөў = bбөў` for all `i < k`, but `aвӮ– вү  bвӮ–`, so `a вү  b` by T3. By NAT-order's trichotomy, exactly one of `aвӮ– < bвӮ–` or `bвӮ– < aвӮ–` holds. If `aвӮ– < bвӮ–`, then `k` witnesses `a < b` via T1(i); if `bвӮ– < aвӮ–`, then `k` witnesses `b < a` via T1(i). No reverse witness `k'` exists. If `k' > k`, agreement fails at position `k`. If `k' = k`, case (i) requires the opposite inequality at `k`, excluded by NAT-order's trichotomy, and case (ii) requires `k = n + 1` (resp. `m + 1`), contradicting `k вүӨ n` (resp. `k вүӨ m`). If `k' < k`, minimality of `k` gives `a_{k'} = b_{k'}`, so case (i) is excluded by NAT-order's trichotomy at `(a_{k'}, b_{k'})`, and case (ii) requires `k' = n + 1` (or `m + 1`), contradicting `k' < k вүӨ n` (respectively `k вүӨ m`).

*Case 3: the first divergence position `k` satisfies clause (ОІ) or (Оі).* Minimality of `k` rules out any shared-position mismatch: under clause (ОІ), `k = m + 1`, so any shared-position index `j` (one with `j вүӨ m вҲ§ j вүӨ n`) satisfies `j вүӨ m < k` via NAT-addcompat's `m < m + 1`; under clause (Оі), `k = n + 1`, so symmetrically any shared-position `j` satisfies `j вүӨ n < k`. In either case, an (Оұ)-mismatch at such `j` would witness a divergence position strictly below `k`, contradicting minimality. So `aбөў = bбөў` for all `i` with `1 вүӨ i вүӨ m вҲ§ i вүӨ n`. Both clauses force `m вү  n`: (ОІ) gives `m + 1 вүӨ n`, hence `m < n` via NAT-addcompat's `m < m + 1`; (Оі) gives `n < m` symmetrically. So `a вү  b` by T3. NAT-order's trichotomy at `(m, n)` resolves `m вү  n` into `m < n вҲЁ n < m`. If `m < n`, then `k = m + 1 вүӨ n`, so `a` is a proper prefix of `b` and `k` witnesses `a < b` via T1(ii). No reverse witness for `b < a` exists: case (i) would require `bвұј < aвұј` at some shared position `j`, excluded by NAT-order's trichotomy at `(aвұј, bвұј)` given `aвұј = bвұј`; case (ii) would require `n + 1 вүӨ m`, which with NAT-addcompat's `n < n + 1` gives `n < m`, contradicting `m < n`. If `n < m`, the mirrored argument gives `b < a`.

These three cases partition `T Г— T`, and in each case exactly one of the three relations holds.

*(c) Transitivity.* We must show: `(A a, b, c вҲҲ T : a < b вҲ§ b < c : a < c)`.

Let `kвӮҒ` witness `a < b` and `kвӮӮ` witness `b < c`, with `#a = m`, `#b = n`, `#c = p`. For all `i < kвӮҒ вҲ§ i < kвӮӮ`, `aбөў = bбөў = cбөў` by transitivity of equality. Case analysis on `kвӮҒ` and `kвӮӮ`.

*Case kвӮҒ < kвӮӮ.* Position `kвӮҒ` lies in the agreement range of `b < c`, so `bвӮ–вӮҒ = cвӮ–вӮҒ`. If `a < b` via T1(i): `aвӮ–вӮҒ < bвӮ–вӮҒ = cвӮ–вӮҒ` with `kвӮҒ вүӨ m`, and the existence of `cвӮ–вӮҒ` gives `kвӮҒ вүӨ p`; position `kвӮҒ` witnesses `a < c` via T1(i). If `a < b` via T1(ii): `kвӮҒ = m + 1 вүӨ n`, and `cвӮ–вӮҒ` exists, so `p вүҘ m + 1`; `kвӮҒ` witnesses `a < c` via T1(ii).

*Case kвӮӮ < kвӮҒ.* Position `kвӮӮ` lies in the agreement range of `a < b`, so `aвӮ–вӮӮ = bвӮ–вӮӮ`. If `b < c` were via T1(ii), then `kвӮӮ = n + 1`. But `a < b` requires `kвӮҒ вүӨ n` (directly in case (i), via `kвӮҒ = m + 1 вүӨ n` in case (ii)), and NAT-addcompat gives `n + 1 > n`, so `kвӮӮ > n вүҘ kвӮҒ` by NAT-order transitivity, contradicting `kвӮӮ < kвӮҒ`. Therefore `b < c` is via T1(i): `bвӮ–вӮӮ < cвӮ–вӮӮ` with `kвӮӮ вүӨ n вҲ§ kвӮӮ вүӨ p`. Since `kвӮӮ < kвӮҒ` and `a` has components below `kвӮҒ`, `kвӮӮ вүӨ m`. Then `aвӮ–вӮӮ = bвӮ–вӮӮ < cвӮ–вӮӮ`, and `kвӮӮ` witnesses `a < c` via T1(i).

*Case kвӮҒ = kвӮӮ = k.* We have `aбөў = cбөў` for all `i < k`.

*Sub-case (i, i):* `aвӮ– < bвӮ– < cвӮ–` with `k вүӨ m вҲ§ k вүӨ p`. NAT-order transitivity gives `aвӮ– < cвӮ–`; T1(i) witnesses `a < c`.

*Sub-case (ii, i):* `k = m + 1 вүӨ n` and `k вүӨ p`, so `m + 1 вүӨ p`; T1(ii) witnesses `a < c`.

*Sub-case (i, ii):* `k вүӨ n` and `k = n + 1 вүӨ p`, so `k > n`; contradiction.

*Sub-case (ii, ii):* `k = m + 1 вүӨ n` and `k = n + 1 вүӨ p`. Then `m + 1 = n + 1`, so `m = n` by NAT-cancel. From `m + 1 вүӨ n` with NAT-addcompat's `m < m + 1`, we derive `m < n` (via NAT-order transitivity or substitution), contradicting `m = n` by NAT-order's trichotomy.

In every realizable combination, a witness for `a < c` under T1 is produced. вҲҺ

The strict total order `<` admits the customary non-strict companions: `a вүӨ b` abbreviates `a < b вҲЁ a = b`, and `a вүҘ b` abbreviates `b вүӨ a`.

*Formal Contract:*
- *Definition:* `a < b` iff `вҲғ k вүҘ 1` with `(A i : 1 вүӨ i < k : aбөў = bбөў)` and either (i) `k вүӨ #a вҲ§ k вүӨ #b вҲ§ aвӮ– < bвӮ–`, or (ii) `k = #a+1 вүӨ #b`.
- *Abbreviations:* `a вүӨ b` abbreviates `a < b вҲЁ a = b`; `a вүҘ b` abbreviates `b вүӨ a`.
- *Depends:*
  - T0 (CarrierSetDefinition) вҖ” length `#a` and component projection `aвӮ–` for `a вҲҲ T`.
  - T3 (CanonicalRepresentation) вҖ” bridge between component-level agreement and tumbler equality; Case 1 concludes `a = b`, Cases 2 and 3 conclude `a вү  b`.
  - NAT-order (NatStrictTotalOrder) вҖ” irreflexivity, trichotomy, and transitivity of `<` on в„•; `вүӨ`-defining clause `m вүӨ n вҹә m < n вҲЁ m = n` for composing strict with non-strict bounds.
  - NAT-addcompat (NatAdditionOrderAndSuccessor) вҖ” strict successor inequality `n < n + 1`.
  - NAT-cancel (NatAdditionCancellation) вҖ” right cancellation at `1`, used in sub-case (ii, ii) to pass from `m + 1 = n + 1` to `m = n`.
  - NAT-discrete (NatDiscreteness) вҖ” forward direction `m < n вҹ№ m + 1 вүӨ n`, used contrapositively in Case 1 to rule out `m < n` and `n < m` from the exhaustion-shape negations `В¬(m + 1 вүӨ n) вҲ§ В¬(n + 1 вүӨ m)` and in part (c) Case `kвӮӮ < kвӮҒ` case-(ii) branch to obtain `kвӮӮ вүӨ m` from `kвӮӮ < m + 1`.
  - NAT-wellorder (NatWellOrdering) вҖ” least-element principle, invoked in part (b) within the branch where at least one divergence position exists, to define the first divergence position `k`.
- *Postconditions:* (a) Irreflexivity вҖ” `(A a вҲҲ T :: В¬(a < a))`. (b) Trichotomy вҖ” `(A a,b вҲҲ T :: (a < b вҲЁ a = b вҲЁ b < a) вҲ§ В¬(a < b вҲ§ a = b) вҲ§ В¬(a < b вҲ§ b < a) вҲ§ В¬(a = b вҲ§ b < a))`. (c) Transitivity вҖ” `(A a,b,c вҲҲ T : a < b вҲ§ b < c : a < c)`.

Nelson's assertion that the tumbler line is total вҖ” that two addresses are never incomparable вҖ” is architecturally load-bearing. Spans are defined as contiguous regions on the tumbler line: "A span in the tumbler line, represented by two tumblers, refers to a subtree of the entire docuverse." If two addresses were incomparable, the interval between them would be undefined, and the entire machinery of span-sets, link endsets, and content reference would collapse.
