## The total order

We require a total order on T. Nelson describes the "tumbler line" as a single linear sequence: "In a sense the tumbler line is like the real line, i.e., the line of integers and all the numbers in between." The system maps a hierarchical tree вАФ servers containing accounts containing documents containing elements вАФ onto this flat line via depth-first traversal, which produces a total order: for any two nodes, traversal visits one before the other.   (ii) `k = m + 1 вЙ§ n` (that is, `a` is a proper prefix of `b`).

The prefix convention вАФ a prefix is less than any proper extension вАФ is what makes depth-first traversal work. The server address `2` is less than every address within server `2`'s subtree, because every such address extends the prefix `2` with further components. Server `2`'s subtree begins immediately after `2` in the order and extends until some address whose first component exceeds `2`.

*(b) Trichotomy.* We must show: `(A a, b вИИ T :: (a < b вИ® a = b вИ® b < a) вИІ ¬ђ(a < b вИІ a = b) вИІ ¬ђ(a < b вИІ b < a) вИІ ¬ђ(a = b вИІ b < a))`.

The strict total order `<` admits the customary non-strict companions: `a вЙ§ b` abbreviates `a < b вИ® a = b`, and `a вЙ• b` abbreviates `b вЙ§ a`.

*Formal Contract:*
- *Definition:* `a < b` iff `вИГ k вЙ• 1` with `(A i : 1 вЙ§ i < k : aбµҐ = bбµҐ)` and either (i) `k вЙ§ #a вИІ k вЙ§ #b вИІ aвВЦ < bвВЦ`, or (ii) `k = #a+1 вЙ§ #b`.
- *Abbreviations:* `a вЙ§ b` abbreviates `a < b вИ® a = b`; `a вЙ• b` abbreviates `b вЙ§ a`.
- *Depends:*

- *Postconditions:* (a) Irreflexivity вАФ `(A a вИИ T :: ¬ђ(a < a))`. (b) Trichotomy вАФ `(A a,b вИИ T :: (a < b вИ® a = b вИ® b < a) вИІ ¬ђ(a < b вИІ a = b) вИІ ¬ђ(a < b вИІ b < a) вИІ ¬ђ(a = b вИІ b < a))`. (c) Transitivity вАФ `(A a,b,c вИИ T : a < b вИІ b < c : a < c)`.

Nelson's assertion that the tumbler line is total вАФ that two addresses are never incomparable вАФ is architecturally load-bearing. Spans are defined as contiguous regions on the tumbler line: "A span in the tumbler line, represented by two tumblers, refers to a subtree of the entire docuverse." If two addresses were incomparable, the interval between them would be undefined, and the entire machinery of span-sets, link endsets, and content reference would collapse.
