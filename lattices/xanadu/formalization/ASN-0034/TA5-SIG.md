**TA5-SIG (LastSignificantPosition).** We define the *last significant position* of a tumbler `t ∈ T`, written `sig(t)`.

When `t` has at least one nonzero component — that is, `(E i : 1 ≤ i ≤ #t : tᵢ ≠ 0)` — we set `sig(t)` to the rightmost index at which `t` is nonzero. Let `S = {i : 1 ≤ i ≤ #t ∧ tᵢ ≠ 0}`; the existence hypothesis makes `S` nonempty, and every `i ∈ S` is bounded above by `#t`.

The maximum of a bounded nonempty ℕ-subset is not delivered directly by NAT-wellorder, which states only the *least-element* principle; ℕ has no greatest element, so the principle does not dualize unconditionally, and greatest elements are guaranteed only on subsets bounded above. We derive `max(S)` from the least-element principle with `#t` as the explicit boundedness witness. Form the upper-bound set `U = {u ∈ ℕ : (A i ∈ S :: i ≤ u)}`; every `i ∈ S` satisfies `i ≤ #t`, so `#t ∈ U` and `U ≠ ∅`. NAT-wellorder applied to `U` delivers a least upper bound `m`. We claim `m ∈ S`. Suppose otherwise: every `i ∈ S` satisfies `i ≤ m` (since `m ∈ U`) and `i ≠ m` (since `m ∉ S`), so NAT-order's `≤`-defining clause `i ≤ m ⟺ i < m ∨ i = m` forces `i < m`, and NAT-discrete's forward direction then gives `i + 1 ≤ m` for every `i ∈ S`. Fixing any `i₀ ∈ S` — `S ≠ ∅` supplies such a witness — we have `i₀ ≥ 1` (since `S ⊆ {1, …, #t}`), and `i₀ + 1 ≤ m` then yields `m ≥ 2 ≥ 1`; NAT-sub's conditional closure at `m ≥ 1` supplies `m − 1 ∈ ℕ`. The value `m − 1` is strictly below `m`: NAT-sub's right-inverse gives `(m − 1) + 1 = m`, and NAT-addcompat's `(m − 1) < (m − 1) + 1` rewrites via that identity to `m − 1 < m`. And `m − 1` bounds `S`: each `i ∈ S` satisfies `i + 1 ≤ m`, which splits via NAT-order's `≤`-defining clause into `i + 1 = m` (then NAT-sub's right telescoping `(i + 1) − 1 = i` gives `i = m − 1`) or `i + 1 < m` (then NAT-sub's strict monotonicity at `p = 1`, with both `i + 1 ≥ 1` and `m ≥ 1`, gives `(i + 1) − 1 < m − 1`, i.e., `i < m − 1`); in both cases `i ≤ m − 1`. So `m − 1 ∈ U` with `m − 1 < m`, contradicting minimality of `m` in `U`. Therefore `m ∈ S`, and we set `sig(t) = m = max(S)`.

When every component of `t` is zero — that is, `(A i : 1 ≤ i ≤ #t : tᵢ = 0)` — we set `sig(t) = #t`.

In both cases `1 ≤ sig(t) ≤ #t`, since `#t ≥ 1` for every `t ∈ T`.

*Formal Contract:*
- *Preconditions:* `t ∈ T` (any tumbler with `#t ≥ 1`).
- *Definition:* `sig(t) = max({i : 1 ≤ i ≤ #t ∧ tᵢ ≠ 0})` when `(E i : 1 ≤ i ≤ #t : tᵢ ≠ 0)`; `sig(t) = #t` when `(A i : 1 ≤ i ≤ #t : tᵢ = 0)`.
- *Depends:*
  - T0 (CarrierSetDefinition) — supplies `t ∈ T` as finite ℕ-sequences with `#t ≥ 1`, component projection `tᵢ`, the length `#t` used as the fallback value and index bound, and the boundedness witness `#t ∈ U` that makes the upper-bound set `U` nonempty for the `max(S)` derivation.
  - NAT-wellorder (NatWellOrdering) — least-element principle, applied to the upper-bound set `U = {u ∈ ℕ : (A i ∈ S :: i ≤ u)}` (nonempty via `#t ∈ U`) to produce the least upper bound `m` of `S = {i : 1 ≤ i ≤ #t ∧ tᵢ ≠ 0}`; the contradiction argument then shows `m ∈ S`, making `m = max(S)`.
  - NAT-order (NatStrictTotalOrder) — `≤`-defining clause `p ≤ q ⟺ p < q ∨ p = q`, used (a) in the contradiction branch `m ∉ S` to split each bound `i ≤ m` (for `i ∈ S`) into `i < m`, and (b) in the `m − 1 ∈ U` step to split each `i + 1 ≤ m` into `i + 1 = m` or `i + 1 < m`.
  - NAT-discrete (NatDiscreteness) — forward direction `i < m ⟹ i + 1 ≤ m`, used in the contradiction branch to lift each `i < m` (for `i ∈ S`) to the successor bound `i + 1 ≤ m`, the premise for constructing `m − 1` as a smaller upper bound.
  - NAT-sub (NatPartialSubtraction) — conditional closure `m ≥ 1 ⟹ m − 1 ∈ ℕ` (discharged from `1 ≤ i₀ + 1 ≤ m` for any `i₀ ∈ S`); right-inverse `(m − 1) + 1 = m`; right telescoping `(i + 1) − 1 = i`; strict monotonicity at `p = 1` rewriting `i + 1 < m` to `i < m − 1`; together establishing `m − 1 ∈ ℕ`, `m − 1 < m`, and `i ≤ m − 1` for every `i ∈ S`.
  - NAT-addcompat (NatAdditionOrderAndSuccessor) — strict successor inequality `n < n + 1`, instantiated at `n = m − 1` and combined with NAT-sub's right-inverse `(m − 1) + 1 = m` to conclude `m − 1 < m`.
- *Postconditions:* `1 ≤ sig(t) ≤ #t`.
