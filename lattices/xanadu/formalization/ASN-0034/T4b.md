**T4b (UniqueParse).** Under the constraints of T4 — at most three zero-valued components, no two zeros adjacent, `t₁ ≠ 0`, `t_{#t} ≠ 0` — the partial function

  `fields : T ⇀ Seq(ℕ⁺) × Seq(ℕ⁺) × Seq(ℕ⁺) × Seq(ℕ⁺)`

that decomposes a tumbler into its node, user, document, and element sub-sequences is well-defined and uniquely determined by `t` alone on the T4-valid subdomain, with absent fields represented by the empty sequence `ε` in the corresponding tuple position. Outside that subdomain — tumblers with `zeros(t) ≥ 4` or tumblers violating the field-segment constraint such as `[0, 1, 0, 2]` (leading zero) or `[1, 0, 0, 2]` (adjacent zeros) — `fields(t)` is not assigned a value by this corollary, and every caller must discharge T4-validity before invoking it.

We read `fields(t)` as the 4-tuple `(N(t), U(t), D(t), E(t))` and fix the projections `N, U, D, E : T ⇀ Seq(ℕ⁺)` accordingly — each a partial map sharing the T4-valid subdomain of `fields`, since a projection of a partial function is itself partial with the same domain. Here `ℕ⁺ = {n ∈ ℕ : n > 0}` is the subcarrier of strictly positive naturals introduced in T0. Each projection returns a finite sequence over `ℕ⁺` whenever it is defined: NAT-zero supplies `0 ≤ tᵢ` as a lower bound on every component of `t` (which lies in ℕ by T0), NAT-discrete at `m = 0` rules out the interval `0 ≤ tᵢ < 1` when `tᵢ ≠ 0`, so every non-zero component is strictly positive, and T4 assigns zero the role of separator only, so no field-content position carries the value zero — the value domain of the projections is therefore exactly T0's `ℕ⁺`. A projection may be the empty sequence `ε` — this is the representation chosen for an *absent* field on the T4-valid subdomain (distinct from the projection being undefined, which occurs only outside that subdomain). The representation is unambiguous: T4a (SyntacticEquivalence) records that every *present* field segment is non-empty under T4's field-segment constraint, so for each `X ∈ {N, U, D, E}` and every T4-valid `t`, `X(t) = ε` holds if and only if field `X` is absent in `t`.

The absence pattern of `fields(t)` is fixed by case analysis on `zeros(t)`, which by T4 lies in `{0, 1, 2, 3}`:

  - `zeros(t) = 0`: `N(t) = (t₁, ..., t_{#t})` (non-empty since `#t ≥ 1` by T0), `U(t) = D(t) = E(t) = ε`.
  - `zeros(t) = 1`: `N(t)` and `U(t)` are non-empty, `D(t) = E(t) = ε`.
  - `zeros(t) = 2`: `N(t)`, `U(t)`, and `D(t)` are non-empty, `E(t) = ε`.
  - `zeros(t) = 3`: all four projections are non-empty.

The component-access notation `t.X₁` (for `X ∈ {N, U, D, E}`) denotes the first component of `X(t)` and is therefore defined exactly when `X(t) ≠ ε`; by the table above, this reduces to a threshold check on `zeros(t)`. In particular, for every T4-valid `t ∈ T`, `t.N₁` is defined; `t.U₁` is defined iff `zeros(t) ≥ 1`; `t.D₁` is defined iff `zeros(t) ≥ 2`; `t.E₁` is defined iff `zeros(t) = 3`. Where `t.X₁` appears in downstream properties, the containing claim is understood to carry the corresponding threshold on `zeros` as a precondition.

The argument for well-definedness turns on a single observation: under T4's definitional role-assignment, the separator positions are exactly recoverable from the raw tumbler. We claim that a position `i` satisfies `tᵢ = 0` if and only if `i` is a field separator. The forward direction holds by construction: every separator has value 0 in the T4 address format `N₁. ... .Nₐ . 0 . U₁. ... .Uᵦ . 0 . D₁. ... .Dᵧ . 0 . E₁. ... .Eδ`. For the reverse direction: T4 assigns zero-valued positions the role of separators, so if `tᵢ = 0` then `i` is a separator position; the non-zero positions, which by NAT-zero (`0 ≤ tᵢ`) and NAT-discrete (`m ≤ tᵢ < m + 1 ⟹ tᵢ = m`, at `m = 0`) on T0's carrier ℕ are strictly positive, carry field components. Therefore the set `{i : 1 ≤ i ≤ #t ∧ tᵢ = 0}` is exactly the set of separator positions — computable by a single scan of `t`.

Given the separator positions, the field boundaries follow deterministically. Let the zero positions in increasing order be `s₁ < s₂ < ... < s_k`, where `k = zeros(t) ∈ {0, 1, 2, 3}`; NAT-order (strict total order on ℕ) licenses the strictly increasing enumeration of those positions. We compute `fields(t)` case by case, exhibiting each projection explicitly:

  - *Case k = 0.* `N(t) = (t₁, ..., t_{#t})` and `U(t) = D(t) = E(t) = ε`. The node field occupies the entire tumbler; no separator carves off further fields. Non-emptiness of `N(t)` follows from `#t ≥ 1` (T0).
  - *Case k = 1.* `N(t) = (t₁, ..., t_{s₁ - 1})`, `U(t) = (t_{s₁ + 1}, ..., t_{#t})`, and `D(t) = E(t) = ε`. The single separator at `s₁` partitions the non-zero positions into two field segments. Non-emptiness of both is the field-segment constraint: `t₁ ≠ 0` forces `s₁ ≥ 2`, so `N(t)` has at least one component; `t_{#t} ≠ 0` forces `s₁ ≤ #t - 1`, so `U(t)` has at least one component.
  - *Case k = 2.* `N(t) = (t₁, ..., t_{s₁ - 1})`, `U(t) = (t_{s₁ + 1}, ..., t_{s₂ - 1})`, `D(t) = (t_{s₂ + 1}, ..., t_{#t})`, and `E(t) = ε`. The two separators at `s₁ < s₂` partition the non-zero positions into three field segments. Non-emptiness of the boundary segments follows as in case `k = 1`; non-emptiness of the middle segment `U(t)` follows from the no-adjacent-zeros clause (`s₂ ≥ s₁ + 2`), which forces `s₁ + 1 ≤ s₂ - 1`.
  - *Case k = 3.* `N(t) = (t₁, ..., t_{s₁ - 1})`, `U(t) = (t_{s₁ + 1}, ..., t_{s₂ - 1})`, `D(t) = (t_{s₂ + 1}, ..., t_{s₃ - 1})`, and `E(t) = (t_{s₃ + 1}, ..., t_{#t})`. The three separators partition the non-zero positions into four field segments. Non-emptiness of all four is the full field-segment constraint: the two boundary clauses handle `N(t)` and `E(t)`, and the two instances of no-adjacent-zeros (`s₂ ≥ s₁ + 2`, `s₃ ≥ s₂ + 2`) handle the interior segments `U(t)` and `D(t)`.

T4 directly asserts that no two separators are adjacent and that the tumbler neither begins nor ends with zero, and the four cases above have each discharged segment non-emptiness by direct appeal to those positional clauses (together with `#t ≥ 1` from T0 in the case `k = 0`) — no case routes through T4a's segment-non-emptiness equivalence. Consequently, in every case above, every projection prescribed to be non-empty has at least one component, and every projection prescribed to be `ε` has no source positions to draw from. The canonical representation of T3 ensures that the components of `t` are fixed — there is no alternative encoding of the same tumbler with different component values. The separator positions are therefore uniquely determined, the field boundaries are uniquely determined, and the sub-sequences assigned to each field are uniquely determined.

Two distinct decompositions of `t` would require two distinct sets of separator positions. But there is exactly one such set. Therefore `fields(t)` is well-defined and unique. ∎
