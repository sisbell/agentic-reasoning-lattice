**T10a.7 (EnumerationInjectivity).** For every allocator A with domain `dom(A) = {t‚āô : n ‚Č• 0}` ‚ÄĒ `t‚āÄ` the base address and `t‚āô‚āä‚āĀ = inc(t‚āô, 0)` ‚ÄĒ the indexing map `n ‚Ü¶ t‚āô` is injective: for all `m, n ‚Č• 0` with `m ‚Č† n`, `t‚āė ‚Č† t‚āô`.

Without this property, a single element of `dom(A)` could be reached by two distinct indices, and any predicate phrased over the enumeration index would receive ambiguous values even when the witnessing allocator is already unique.

*Proof.* Let `m, n ‚Č• 0` with `m ‚Č† n`. By trichotomy of `<` on ‚Ąē, either `m < n` or `n < m`; assume without loss of generality that `m < n`. We show `t‚āė < t‚āô` by induction on the gap `d = n - m ‚Č• 1`.

*Base case* (`d = 1`). Here `n = m + 1`, so `t‚āô = inc(t‚āė, 0)`. By TA5(a), `inc(t‚āė, 0) > t‚āė`, hence `t‚āė < t‚āô`.

*Inductive step* (from `d` to `d + 1`). Since `d ‚Č• 1`, the index `n - 1` satisfies `m < n - 1 < n` with gap `(n - 1) - m = d`. The inductive hypothesis gives `t‚āė < t‚āô‚āč‚āĀ`. By TA5(a), `t‚āô = inc(t‚āô‚āč‚āĀ, 0) > t‚āô‚āč‚āĀ`. By T1(c) transitivity, `t‚āė < t‚āô`.

By T1(a) irreflexivity, `t‚āė < t‚āô` excludes `t‚āė = t‚āô`; hence `t‚āė ‚Č† t‚āô`. ‚ąé

*Formal Contract:*
- *Precondition:* Allocator A conforming to T10a, with domain `dom(A) = {t‚āô : n ‚Č• 0}` where `t‚āÄ` is the base address and `t‚āô‚āä‚āĀ = inc(t‚āô, 0)`.
- *Postcondition:* The map `n ‚Ü¶ t‚āô` is injective: `(A m, n ‚Č• 0 : m ‚Č† n : t‚āė ‚Č† t‚āô)`. Equivalently, `(A m, n ‚Č• 0 : m < n : t‚āė < t‚āô)`.
- *Depends:*
  - T10a (AllocatorDiscipline) ‚ÄĒ supplies the enumeration `t‚āô‚āä‚āĀ = inc(t‚āô, 0)`.
  - TA5 (HierarchicalIncrement), postcondition (a) ‚ÄĒ strict monotonicity `inc(t‚āô, 0) > t‚āô`.
  - T1 (LexicographicOrder), postcondition (c) ‚ÄĒ transitivity of `<`.
  - T1 (LexicographicOrder), postcondition (a) ‚ÄĒ irreflexivity of `<`.
