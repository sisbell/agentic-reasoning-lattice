**T10a.7 (EnumerationInjectivity).** For every allocator A with domain `dom(A) = {t‚Çô : n ‚â• 0}` ‚Äî `t‚ÇÄ` the base address and `t‚Çô‚Çä‚ÇÅ = inc(t‚Çô, 0)` ‚Äî the indexing map `n ‚Ü¶ t‚Çô` is injective: for all `m, n ‚â• 0` with `m ‚âÝ n`, `t‚Çò ‚âÝ t‚Çô`.

Without this property, a single element of `dom(A)` could be reached by two distinct indices, and any predicate phrased over the enumeration index would receive ambiguous values even when the witnessing allocator is already unique.

*Proof.* Let `m, n ‚â• 0` with `m ‚âÝ n`. By NAT-order's trichotomy on ‚Ñï, exactly one of `m < n`, `m = n`, or `n < m` holds; the hypothesis `m ‚âÝ n` rules out equality, leaving `m < n ‚à® n < m`. Assume without loss of generality that `m < n`. We show `t‚Çò < t‚Çô` by induction on the gap `d = n - m ‚â• 1`.

*Base case* (`d = 1`). Here `n = m + 1`, so `t‚Çô = inc(t‚Çò, 0)`. By TA5(a), `inc(t‚Çò, 0) > t‚Çò`, hence `t‚Çò < t‚Çô`.

*Inductive step* (from `d` to `d + 1`). Since `d ‚â• 1`, the index `n - 1` satisfies `m < n - 1 < n` with gap `(n - 1) - m = d`. The inductive hypothesis gives `t‚Çò < t‚Çô‚Çã‚ÇÅ`. By TA5(a), `t‚Çô = inc(t‚Çô‚Çã‚ÇÅ, 0) > t‚Çô‚Çã‚ÇÅ`. By T1(c) transitivity, `t‚Çò < t‚Çô`.

By T1(a) irreflexivity, `t‚Çò < t‚Çô` excludes `t‚Çò = t‚Çô`; hence `t‚Çò ‚âÝ t‚Çô`. ‚àé

*Formal Contract:*
- *Precondition:* Allocator A conforming to T10a, with domain `dom(A) = {t‚Çô : n ‚â• 0}` where `t‚ÇÄ` is the base address and `t‚Çô‚Çä‚ÇÅ = inc(t‚Çô, 0)`.
- *Postcondition:* The map `n ‚Ü¶ t‚Çô` is injective: `(A m, n ‚â• 0 : m ‚âÝ n : t‚Çò ‚âÝ t‚Çô)`. Equivalently, `(A m, n ‚â• 0 : m < n : t‚Çò < t‚Çô)`.
- *Depends:*
  - T10a (AllocatorDiscipline) ‚Äî supplies the enumeration `t‚Çô‚Çä‚ÇÅ = inc(t‚Çô, 0)`.
  - TA5 (HierarchicalIncrement), postcondition (a) ‚Äî strict monotonicity `inc(t‚Çô, 0) > t‚Çô`.
  - T1 (LexicographicOrder), postcondition (c) ‚Äî transitivity of `<`.
  - T1 (LexicographicOrder), postcondition (a) ‚Äî irreflexivity of `<`.
  - NAT-order (NatStrictTotalOrder) ‚Äî trichotomy on ‚Ñï indices ‚Äî resolves `m ‚âÝ n` into `m < n ‚à® n < m`.
