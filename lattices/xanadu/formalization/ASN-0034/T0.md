**T0 (CarrierSetDefinition).** `T` is the set of nonempty finite sequences over ℕ, written `d₁.d₂. ... .dₙ` with each `dᵢ ∈ ℕ`. For each `a ∈ T`, write `#a` for the length of `a` and `aᵢ` for the value of the `i`-th component of `a`. The component positions of `a` form the index domain `{j ∈ ℕ : 1 ≤ j ≤ #a}`, and `aᵢ ∈ ℕ` at each `i ∈ {j ∈ ℕ : 1 ≤ j ≤ #a}`. `(A a ∈ T :: 1 ≤ #a)` forces each tumbler to have at least one component, so the index domain is nonempty. Elements of T are *extensional*: two tumblers with equal length and identical components at every position are the same tumbler — there is no hidden structure beyond the length-and-components data.

*Formal Contract:*
- *Axiom:* T is the set of finite sequences `a` over ℕ satisfying `1 ≤ #a`, equipped with length `#· : T → ℕ` and component projection `·ᵢ` whose index domain for each `a ∈ T` is `{j ∈ ℕ : 1 ≤ j ≤ #a}`, with `aᵢ ∈ ℕ` at each `i ∈ {j ∈ ℕ : 1 ≤ j ≤ #a}`. Extensionality: `(A a, b ∈ T : #a = #b ∧ (A i : 1 ≤ i ≤ #a : aᵢ = bᵢ) : a = b)` — elements of T with equal length and pointwise-equal components are identical.
- *Depends:*
  - NAT-closure (NatArithmeticClosureAndIdentity, this ASN) — supplies `1 ∈ ℕ` for the lower bound of the nonemptiness clause `1 ≤ #a`.
  - NAT-order (NatStrictTotalOrder) — supplies the non-strict relation `≤` on ℕ appearing in the nonemptiness clause `1 ≤ #a`.
