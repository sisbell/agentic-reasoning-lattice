**T0 (CarrierSetDefinition).** `T` is the set of nonempty finite sequences over ℕ, written `d₁.d₂. ... .dₙ` with each `dᵢ ∈ ℕ`. For each `a ∈ T`, write `#a` for the length of `a` and `aᵢ` for the value of the `i`-th component of `a`. The component positions of `a` form the index domain `{j ∈ ℕ : 1 ≤ j ≤ #a}`, and `aᵢ ∈ ℕ` at each `i ∈ {j ∈ ℕ : 1 ≤ j ≤ #a}`. The nonemptiness of each `a ∈ T` is formalized as `(A a ∈ T :: 1 ≤ #a)`: every tumbler has at least one component. The numeral `1` bounding the length from below is the `1 ∈ ℕ` posited by NAT-closure; the relation `≤` is the non-strict order on ℕ defined by NAT-order, so `{j ∈ ℕ : 1 ≤ j ≤ #a}` is nonempty.

*Formal Contract:*
- *Axiom:* T is the set of finite sequences `a` over ℕ satisfying `1 ≤ #a`, equipped with length `#· : T → ℕ` and component projection `·ᵢ` whose index domain for each `a ∈ T` is `{j ∈ ℕ : 1 ≤ j ≤ #a}`, with `aᵢ ∈ ℕ` at each `i ∈ {j ∈ ℕ : 1 ≤ j ≤ #a}`.
- *Depends:*
  - NAT-closure (NatArithmeticClosureAndIdentity, this ASN) — supplies `1 ∈ ℕ` for the lower bound of the nonemptiness clause `1 ≤ #a`.
  - NAT-order (NatStrictTotalOrder) — supplies the non-strict relation `≤` on ℕ appearing in the nonemptiness clause `1 ≤ #a`.
