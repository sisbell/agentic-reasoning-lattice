**T0 (CarrierSetDefinition).** `T` is the set of nonempty finite sequences over ℕ, written `d₁.d₂. ... .dₙ` with each `dᵢ ∈ ℕ`. For each `a ∈ T`, write `#a` for the length of `a` and `aᵢ` for the value of the `i`-th component of `a`. The component positions of `a` form the index domain `{1, …, #a} ⊆ ℕ`, and `aᵢ ∈ ℕ` at each `i ∈ {1, …, #a}`. The nonemptiness of each `a ∈ T` is formalized as `(A a ∈ T :: 1 ≤ #a)`: every tumbler has at least one component. The numeral `1` bounding the length from below is the `1 ∈ ℕ` posited by NAT-closure; the relation `≤` is the non-strict order on ℕ defined by NAT-order. Reflexivity of `≤` follows from the equality branch of NAT-order's definition `m ≤ n ⟺ m < n ∨ m = n` at `m := n := 1`, so `1 ≤ 1` holds; together with `1 ≤ #a` this places `1` in the index set `{j ∈ ℕ : 1 ≤ j ≤ #a}`, so the index domain `{1, …, #a}` is never empty and bounded quantifiers of the form `(Q i : 1 ≤ i ≤ #a : …)` range over a nonempty set rather than collapsing to vacuity.

*Formal Contract:*
- *Axiom:* T is the set of finite sequences `a` over ℕ satisfying `1 ≤ #a`, equipped with length `#· : T → ℕ` and component projection `·ᵢ` whose index domain for each `a ∈ T` is `{1, …, #a} ⊆ ℕ`, with `aᵢ ∈ ℕ` at each `i ∈ {1, …, #a}`.
- *Depends:*
  - NAT-closure (NatArithmeticClosureAndIdentity, this ASN) — supplies `1 ∈ ℕ` for the lower bound of the nonemptiness clause `1 ≤ #a`.
  - NAT-order (NatStrictTotalOrder) — supplies the non-strict relation `≤` on ℕ appearing in the nonemptiness clause `1 ≤ #a`; reflexivity of `≤` follows from the equality branch of its definition `m ≤ n ⟺ m < n ∨ m = n`, giving `1 ≤ 1` and hence nonemptiness of the index set `{j ∈ ℕ : 1 ≤ j ≤ #a}`.
