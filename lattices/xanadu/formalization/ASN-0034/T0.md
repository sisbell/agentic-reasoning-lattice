**T0 (CarrierSetDefinition).** `T` is the set of nonempty finite sequences over ℕ, written `d₁.d₂. ... .dₙ` with each `dᵢ ∈ ℕ`. For each `a ∈ T`, write `#a` for the length of `a` and `aᵢ` for the value of the `i`-th component of `a`. The component positions of `a` form the index domain `{1, …, #a} ⊆ ℕ`, and `aᵢ ∈ ℕ` at each `i ∈ {1, …, #a}`. The nonemptiness of each `a ∈ T` is formalized as `(A a ∈ T :: #a ≥ 1)`: every tumbler has at least one component. The numeral `1` bounding the length from below is the `1 ∈ ℕ` posited by NAT-closure; the relation `≥` is the converse of the non-strict order `≤` on ℕ defined by NAT-order. The inequality `#a ≥ 1` is thus well-typed within ℕ, and with it the index domain `{1, …, #a}` is never empty, so bounded quantifiers of the form `(Q i : 1 ≤ i ≤ #a : …)` range over a nonempty set rather than collapsing to vacuity.

*Formal Contract:*
- *Axiom:* T is the set of finite sequences over ℕ, equipped with length `#· : T → ℕ` and component projection `·ᵢ` whose index domain for each `a ∈ T` is `{1, …, #a} ⊆ ℕ`, with `aᵢ ∈ ℕ` at each `i ∈ {1, …, #a}`; and `(A a ∈ T :: #a ≥ 1)`.
- *Depends:*
  - NAT-closure (NatArithmeticClosureAndIdentity) — supplies `1 ∈ ℕ` for the lower bound of the nonemptiness clause `#a ≥ 1`.
  - NAT-order (NatStrictTotalOrder) — supplies `≤` on ℕ, whose converse `≥` appears in the nonemptiness clause `#a ≥ 1`.
