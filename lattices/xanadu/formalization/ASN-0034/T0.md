**T0 (CarrierSetDefinition).** `T = {d₁.d₂. ... .dₙ : each dᵢ ∈ ℕ, n ≥ 1}`. For each `a ∈ T`, write `#a` for the length of `a` (so `#a ≥ 1`) and `aᵢ` for the value of the `i`-th component, defined for `1 ≤ i ≤ #a` with `aᵢ ∈ ℕ`.

The standard properties of ℕ that downstream proofs cite are stated as separate axioms, each established as its own foundation object under both a hyphenated short label and a PascalCase canonical name:

- NAT-closure (NatArithmeticClosureAndIdentity): membership `1 ∈ ℕ`, closure under successor and addition, and the additive identity `0 + n = n`.
- NAT-order (NatStrictTotalOrder): strict total order on ℕ.
- NAT-discrete (NatDiscreteness): discreteness, including `m < n ⟹ m + 1 ≤ n`.
- NAT-addcompat (NatAdditionOrderAndSuccessor): left and right order compatibility of addition with `≥`, and the strict successor inequality `n < n + 1`.
- NAT-wellorder (NatWellOrdering): well-ordering of ℕ.
- NAT-zero (NatZeroMinimum): membership `0 ∈ ℕ` together with the lower-bound inequality `(A n ∈ ℕ :: 0 ≤ n)`.
- NAT-sub (NatPartialSubtraction): conditional closure, inverse characterisations, strict positivity, right telescoping, strict monotonicity.
- NAT-cancel (NatAdditionCancellation): left cancellation, right cancellation, and summand absorption in both forms `m + n = m ⟹ n = 0` and `n + m = m ⟹ n = 0`.
- NAT-addassoc (NatAdditionAssociative): associativity of addition.

The hyphenated and PascalCase forms denote the same axiom; either resolves to the corresponding foundation object.

*Formal Contract:*
- *Axiom:* T is the set of all finite sequences over ℕ with length ≥ 1, equipped with length `#· : T → ℕ` satisfying `#a ≥ 1` for all `a ∈ T`, and component projection `·ᵢ` yielding `aᵢ ∈ ℕ` for each `1 ≤ i ≤ #a`.
