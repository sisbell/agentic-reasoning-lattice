**Divergence (Divergence).** For tumblers `a, b вИИ T` with `a вЙ† b`, the *divergence* `divergence(a, b)` is defined by two cases corresponding to the two cases of T1.

  (i) If there exists `k` with `1 вЙ§ k вИІ k вЙ§ #a вИІ k вЙ§ #b` such that `aвВЦ вЙ† bвВЦ` and `(A i : 1 вЙ§ i < k : aбµҐ = bбµҐ)`, then `divergence(a, b) = k` вАФ component divergence at a shared position.

In sub-case (ii-a), `#a < #b`, so `i вЙ§ #a` entails `i вЙ§ #b` by NAT-order transitivity and the range `1 вЙ§ i вЙ§ #a вИІ i вЙ§ #b` reduces to `1 вЙ§ i вЙ§ #a`, yielding `(A i : 1 вЙ§ i вЙ§ #a : aбµҐ = bбµҐ)`, whence `divergence(a, b) = #a + 1`. In either sub-case the divergence lies one position past the shorter tumbler's last component вАФ prefix divergence, where one tumbler is a proper prefix of the other.

Exactly one case applies for any `a вЙ† b`. Mutual exclusivity: if case (i) holds, some `aвВЦ вЙ† bвВЦ` with `k вЙ§ #a вИІ k вЙ§ #b` falsifies case (ii)'s universal agreement at shared positions. Exhaustiveness: if neither case applies, all shared components agree and `#a = #b`, so by T3, `a = b`, contradicting `a вЙ† b`.

The function is symmetric: `divergence(a, b) = divergence(b, a)`. In case (i), the qualifying conjunction is invariant under operand swap вАФ `1 вЙ§ k` mentions neither operand, `k вЙ§ #a вИІ k вЙ§ #b` by `вИІ`-commutativity, `aвВЦ вЙ† bвВЦ` by `вЙ†`-symmetry, `(A i : 1 вЙ§ i < k : aбµҐ = bбµҐ)` by `=`-symmetry вАФ so the same `k` witnesses case (i) under swap. In case (ii), swapping `(a, b)` exchanges sub-cases (ii-a) and (ii-b); both select one-plus the shorter tumbler's length.
