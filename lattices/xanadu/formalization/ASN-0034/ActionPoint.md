*Derivation.* The set S = {i : 1 вүӨ i вүӨ #w вҲ§ wбөў вү  0} is a nonempty subset of в„•: nonempty by TA-Pos, and a subset of в„• by T0's commitment that the index domain `{1, вҖҰ, #w}` of w lies in в„•. By NAT-wellorder, there exists m вҲҲ S with (A n вҲҲ S :: m вүӨ n). Such m is unique: if mвӮҒ and mвӮӮ both satisfy the clause, then mвӮҒ вүӨ mвӮӮ and mвӮӮ вүӨ mвӮҒ. Unfolding each via NAT-order's definition `m вүӨ n вҹә m < n вҲЁ m = n` yields `mвӮҒ < mвӮӮ вҲЁ mвӮҒ = mвӮӮ` from the first and `mвӮӮ < mвӮҒ вҲЁ mвӮӮ = mвӮҒ` from the second, so the conjunction distributes into four disjunct pairings, which we discharge using only NAT-order's irreflexivity `(A n вҲҲ в„• :: В¬(n < n))` and transitivity `(A m, n, p вҲҲ в„• : m < n вҲ§ n < p : m < p)`. If `mвӮҒ < mвӮӮ` and `mвӮӮ < mвӮҒ`, transitivity gives `mвӮҒ < mвӮҒ`, contradicting irreflexivity. If `mвӮҒ < mвӮӮ` and `mвӮӮ = mвӮҒ`, substituting `mвӮҒ` for `mвӮӮ` in `mвӮҒ < mвӮӮ` gives `mвӮҒ < mвӮҒ`, again contradicting irreflexivity. The case `mвӮҒ = mвӮӮ` with `mвӮӮ < mвӮҒ` is handled analogously: substituting `mвӮҒ` for `mвӮӮ` in `mвӮӮ < mвӮҒ` gives `mвӮҒ < mвӮҒ`, contradicting irreflexivity. The remaining pairing `mвӮҒ = mвӮӮ` with `mвӮӮ = mвӮҒ` asserts the equality directly. So `mвӮҒ = mвӮӮ`. So actionPoint(w) names this element unambiguously, and actionPoint(w) вҲҲ S, giving 1 вүӨ actionPoint(w) вүӨ #w. For any i with 1 вүӨ i < actionPoint(w), wбөў = 0. Suppose otherwise, for contradiction: then wбөў вү  0, and to conclude i вҲҲ S we must also establish the two remaining membership clauses `1 вүӨ i` and `i вүӨ #w`; the first holds by hypothesis, so only `i вүӨ #w` needs discharge. From actionPoint(w) вҲҲ S we have actionPoint(w) вүӨ #w, which NAT-order's definition `m вүӨ n вҹә m < n вҲЁ m = n` unfolds as `actionPoint(w) < #w вҲЁ actionPoint(w) = #w`. If `actionPoint(w) < #w`, NAT-order's transitivity applied to `i < actionPoint(w)` and `actionPoint(w) < #w` gives `i < #w`. If `actionPoint(w) = #w`, rewriting `i < actionPoint(w)` by this equality gives `i < #w`. Either way, i < #w, and NAT-order's definition `i вүӨ #w вҹә i < #w вҲЁ i = #w` then licenses `i вүӨ #w` from its left disjunct. So i вҲҲ S. Instantiating the universal `(A n вҲҲ S :: actionPoint(w) вүӨ n)` at n = i gives `actionPoint(w) вүӨ i`, which NAT-order's definition `m вүӨ n вҹә m < n вҲЁ m = n` unfolds as `actionPoint(w) < i вҲЁ actionPoint(w) = i`. If `actionPoint(w) < i`, NAT-order's transitivity applied to `actionPoint(w) < i` and `i < actionPoint(w)` gives `actionPoint(w) < actionPoint(w)`, contradicting irreflexivity. If `actionPoint(w) = i`, substituting `actionPoint(w)` for `i` in `i < actionPoint(w)` gives `actionPoint(w) < actionPoint(w)`, again contradicting irreflexivity. Both disjuncts contradict, so the supposition `wбөў вү  0` fails; hence `wбөў = 0`. For 1 вүӨ w_{actionPoint(w)}: membership of actionPoint(w) in S gives w_{actionPoint(w)} вү  0. Instantiating NAT-zero's disjunction axiom `(A n вҲҲ в„• :: 0 < n вҲЁ 0 = n)` at n = w_{actionPoint(w)} yields 0 < w_{actionPoint(w)} вҲЁ 0 = w_{actionPoint(w)}, and w_{actionPoint(w)} вү  0 excludes the equality, leaving 0 < w_{actionPoint(w)}. NAT-discrete's forward direction m < n вҹ№ m + 1 вүӨ n at m = 0, n = w_{actionPoint(w)} yields 0 + 1 вүӨ w_{actionPoint(w)}. NAT-closure posits 1 вҲҲ в„• directly, licensing its additive identity (A n вҲҲ в„• :: 0 + n = n) to be instantiated at n = 1; this gives the equality 0 + 1 = 1, and rewriting 0 + 1 вүӨ w_{actionPoint(w)} by it yields 1 вүӨ w_{actionPoint(w)}. вҲҺ
