**NAT-discrete (NatDiscreteness).** For every `m, n ‚ąą ‚Ąē`, `m < n ‚üĻ m + 1 ‚Č§ n`.

No natural number lies strictly between `n` and its successor `n + 1`: whenever `m < n`, the successor `m + 1` is already bounded above by `n`, so `m + 1 ‚Č§ n`. This is the discreteness of ‚Ąē. The axiom body invokes two symbols beyond ‚Ąē's primitive `<` and `=`: the non-strict companion `‚Č§`, *defined* in NAT-order by `m ‚Č§ n ‚üļ m < n ‚ą® m = n`, and the successor term `m + 1`, whose summand `1 ‚ąą ‚Ąē` and closure `m + 1 ‚ąą ‚Ąē` are posited by NAT-closure. Both are declared in the Depends slot so that the axiom body can be read without silently importing foundations.

The axiom has a familiar no-interval reformulation `(A m, n ‚ąą ‚Ąē :: m ‚Č§ n < m + 1 ‚üĻ n = m)` ‚ÄĒ "nothing strictly between `m` and `m + 1`" ‚ÄĒ which we record as a *Consequence:* rather than a second axiom form, since its equivalence to the axiom body is not free from the body alone: each direction leans on NAT-order clauses beyond the `‚Č§`-definition. Forward (axiom ‚üĻ no-interval): assume `m ‚Č§ n < m + 1`; unfolding `m ‚Č§ n` by the NAT-order definition `m ‚Č§ n ‚üļ m < n ‚ą® m = n` splits into two cases, each of which we walk to `n = m`. In the case `m < n`, the axiom gives `m + 1 ‚Č§ n`, which unfolds to `m + 1 < n ‚ą® m + 1 = n`; paired with the hypothesis `n < m + 1`, either disjunct is contradictory ‚ÄĒ `m + 1 < n` together with `n < m + 1` is excluded by NAT-order's exactly-one trichotomy (specifically its `¬¨(a < b ‚ąß b < a)` clause, instantiated at `(m + 1, n)`), while `m + 1 = n` rewrites `n < m + 1` to `n < n`, contradicting irreflexivity. The case `m < n` is therefore impossible; it contributes nothing to the conclusion and is discharged by contradiction. In the case `m = n`, symmetry of `=` gives `n = m` directly. Both branches of the split terminate in `n = m`, closing the case analysis. Reverse (no-interval ‚üĻ axiom): assume `m < n` and suppose, for contradiction, that `¬¨(m + 1 ‚Č§ n)`; by exactly-one trichotomy at `(m + 1, n)`, exactly one of `m + 1 < n`, `m + 1 = n`, `n < m + 1` holds, and the first two together constitute `m + 1 ‚Č§ n` ‚ÄĒ which we have ruled out ‚ÄĒ so `n < m + 1`. But then `(m, n)` satisfies `m ‚Č§ n < m + 1` (`m < n` gives `m ‚Č§ n` by the NAT-order definition), and the no-interval form forces `n = m`, contradicting NAT-order's disjointness clause `m < n ‚üĻ m ‚Č† n`. Hence `m + 1 ‚Č§ n`. ‚ąé

*Formal Contract:*
- *Axiom:* `(A m, n ‚ąą ‚Ąē :: m < n ‚üĻ m + 1 ‚Č§ n)` (discreteness).
- *Consequence:* `(A m, n ‚ąą ‚Ąē :: m ‚Č§ n < m + 1 ‚üĻ n = m)` (no-interval form). Forward: unfold `m ‚Č§ n` into its two branches `m < n` and `m = n` and walk each to `n = m`. The `m < n` branch yields `m + 1 ‚Č§ n` via the axiom, whose disjuncts `m + 1 < n` and `m + 1 = n` both collide with `n < m + 1` ‚ÄĒ the first by NAT-order's exactly-one-trichotomy clause `¬¨(a < b ‚ąß b < a)` at `(m + 1, n)`, the second by irreflexivity `¬¨(n < n)` after rewriting ‚ÄĒ so this branch is impossible and discharged by contradiction. The `m = n` branch yields `n = m` directly by symmetry of `=`. Both branches close the case analysis at `n = m`. Reverse: if `m < n` yet `¬¨(m + 1 ‚Č§ n)`, exactly-one trichotomy at `(m + 1, n)` leaves `n < m + 1`; the no-interval form at `(m, n)` then forces `n = m`, contradicting NAT-order's disjointness clause `m < n ‚üĻ m ‚Č† n`.
- *Depends:*
  - NAT-order (NatStrictTotalOrder) ‚ÄĒ supplies the non-strict companion `‚Č§` (defined by `m ‚Č§ n ‚üļ m < n ‚ą® m = n`) used in the axiom's consequent `m + 1 ‚Č§ n`, and three further clauses cited in the Consequence's bidirectional derivation: the exactly-one-trichotomy clause `¬¨(a < b ‚ąß b < a)` instantiated at `(m + 1, n)`, irreflexivity `¬¨(n < n)`, and the disjointness clause `m < n ‚üĻ m ‚Č† n`.
  - NAT-closure (NatArithmeticClosureAndIdentity, this ASN) ‚ÄĒ posits `1 ‚ąą ‚Ąē` and closes ‚Ąē under addition, so the successor `m + 1 ‚ąą ‚Ąē` and the inequalities `m + 1 ‚Č§ n` and `n < m + 1` are comparisons of two ‚Ąē-elements.
