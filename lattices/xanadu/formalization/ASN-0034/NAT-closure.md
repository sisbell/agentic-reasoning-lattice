**NAT-closure (NatArithmeticClosureAndIdentity).** The numeral `1` is in ℕ, and ℕ is closed under successor and addition, with `0` as the additive identity.

The numeral `1` is posited directly as a natural number: `1 ∈ ℕ`. For every `n ∈ ℕ`, the successor `n + 1 ∈ ℕ`. For every `m, n ∈ ℕ`, the sum `m + n ∈ ℕ`. The additive identity holds: `0 + n = n` for every `n ∈ ℕ`.

*Formal Contract:*
- *Axiom:* `1 ∈ ℕ` (one is a natural number); `(A n ∈ ℕ :: n + 1 ∈ ℕ)` (successor closure); `(A m, n ∈ ℕ :: m + n ∈ ℕ)` (addition closure); `(A n ∈ ℕ :: 0 + n = n)` (additive identity).
