# N-Endset Generalization — ASN-0043

Nelson (LM 4/79) lists "4-sets, 5-sets ... n-sets supported in link
storage and search" as a desired feature.

The current link structure is a fixed triple of endsets (from, to, type).
Generalize to a sequence of N ≥ 2 endsets. The triple becomes a convention
for the standard case, not a structural limit.

The three-endset convention (slot 1 = from, slot 2 = to, slot 3 = type)
should be preserved as the default but not enforced structurally.
