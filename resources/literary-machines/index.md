## Concept Relationships

```
DOCUVERSE (global address space)
    │
    ├── DOCUMENTS (owned units)
    │       │
    │       ├── contain SPANS (content regions)
    │       │       └── bounded by TUMBLERS
    │       │
    │       └── have VERSIONS (arrangement history)
    │
    └── LINKS (connections)
            │
            └── contain ENDSETS (span collections)
                    │
                    ├── from-set (source spans)
                    ├── to-set (target spans)
                    └── type (link semantics)

ADDRESSING:
    HUMBER (variable-length number)
        └── used in TUMBLER (structured address)
                └── addresses SPANS in DOCUVERSE
```

```bash
# Find pages mentioning "tumbler"
grep -il "tumbler" raw/*.txt | sort -V

# Read the definition page
cat raw/page-183.txt
```

1. Check **Term Index** above for primary pages

3. Then read surrounding pages for context

- `page-NNN.png` - Scanned page image
- `page-NNN.txt` - OCR text

**Page mapping:** See `table-of-contents.md` for complete raw→chapter mapping.

```bash
# Find pages with term
grep -il "TERM" raw/*.txt | sort -V
```
