# Counterpart Link

Source: Literary Machines, 4/53 (page 220)

## Semantic Intent

### What It Means

A counterpart link is a specific link type that explicitly marks equivalence between documents, sections, or passages. When two pieces of content correspond—two translations of the same text, two versions of the same design, parallel passages in different works—a counterpart link makes that relationship explicit and navigable.

Nelson also calls this a "collateral link" or "correspondence link." Unlike automatic correspondence tracking within a single document's version history (where the system knows which parts evolved from which), a counterpart link is a user-created assertion that two distinct things are equivalent in some meaningful way.

The counterpart link typically has a comment attached explaining the nature of the correspondence—why these two things are counterparts, what the relationship means.

### User Guarantee

**You can explicitly mark equivalent content across any documents.** When you recognize that two passages (in different documents, different authors' works, different languages) are counterparts of each other, you can create a link that says so. Others can then navigate between them and understand the relationship.

### Principle Served

Counterpart links serve the principle that **relationships should be explicit and navigable**. Much of scholarship involves recognizing correspondences—parallel passages, translations, adaptations, equivalent formulations. Traditionally these relationships exist only in scholars' heads or in footnotes. Counterpart links make them first-class navigable connections in the docuverse.

This also supports **intercomparison across documents**. While the versioning system handles correspondence within a document's history, counterpart links extend the same capability across unrelated documents—enabling users to "scroll through" equivalent passages in different works.

### How Users Experience It

- Recognize that a passage in one document corresponds to a passage in another
- Create a counterpart link between them, with a comment explaining the relationship
- Others reading either passage can discover the counterpart and navigate to it
- Build webs of equivalence across translations, adaptations, parallel treatments
- Use counterpart links to support scholarly comparison and cross-referencing

### Nelson's Words

> "The counterpart link shows that there are correspondences between two equivalent documents, sections or passages. (This has also been called a collateral or correspondence link. Note that a counterpart link will typically have a comment attached to it." (4/53)
