## Standards

Apply these standards when fixing issues:

1. **No broken references** — Every citation must resolve to a real foundation property
2. **No orphaned text** — Removing a proof must not leave dangling prose or half-sentences
3. **No silent dependencies** — If a proof used intermediate steps from the removed
   derivation, the dependency must be explicit via the foundation citation
4. **Registry consistency** — Labels and status must match between body and registry
5. **Downstream coherence** — Proofs that cited the old property must work with the
   new citation
