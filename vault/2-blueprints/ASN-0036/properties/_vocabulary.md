# Vocabulary — ASN-0036

- **#runs(d)** — count of correspondence runs in the current arrangement of document d
- **S** — subspace identifier: first component of the element-field V-position, S = subspace(v) = v₁
- **V_S(d)** — set of V-positions in subspace S of document d: {v ∈ dom(M(d)) : subspace(v) = S}
- **ValidInsertionPosition** — a V-position v is a valid insertion position in subspace S of document d when: (non-empty case) v = shift(min(V_S(d)), j) for some 0 ≤ j ≤ N, or (empty case) v = [S, 1, ..., 1] of depth m ≥ 2
- **a + k** — ordinal displacement applied to I-address element ordinal: a + 0 = a; a + k = shift(a, k) = a ⊕ δ(k, #a) for k ≥ 1
- **consecutive V-positions** — positions within a subspace differing only at the ordinal (last) component: s.x is followed by s.(x+1)
- **correspondence run** — triple (v, a, n) with v a V-position, a an I-address, n ≥ 1, satisfying M(d)(v+k) = a+k for all 0 ≤ k < n
- **correspondence run (vⱼ, aⱼ, nⱼ)** — triple representing a contiguous block: starting V-position vⱼ, starting A-address aⱼ, length nⱼ
- **dom(Σ.C)** — set of I-addresses at which content has been stored: {a ∈ T : Σ.C(a) is defined}
- **dom(Σ.M(d))** — set of V-positions currently active in d: {v ∈ T : Σ.M(d)(v) is defined}
- **ord(v)** — ordinal extraction: tumbler of length #v − 1 obtained by stripping the subspace identifier from a V-position v; ord(v) = [v₂, ..., vₘ]
- **origin(a)** — document-level prefix of I-address a: (fields(a).node).0.(fields(a).user).0.(fields(a).document), the full document tumbler N.0.U.0.D
- **ran(Σ.M(d))** — set of I-addresses that d currently references: {Σ.M(d)(v) : v ∈ dom(Σ.M(d))}
- **sig(v)** — index of the last nonzero component of v: sig(v) = max({i : 1 ≤ i ≤ m ∧ vᵢ ≠ 0})
- **v + k** — ordinal displacement applied to V-positions: v + 0 = v (identity); v + k = shift(v, k) for k ≥ 1
- **vpos(S, o)** — position reconstruction: prepends subspace identifier S to ordinal o to form a vpos tuple [S, o₁, ..., oₖ]
- **w_ord** — ordinal projection of displacement w: w_ord = [w₂, ..., wₘ] where m = #w, defined when w₁ = 0 and #w ≥ 2
- **Σ.C** — content store: partial function from tumblers to content values, Σ.C : T ⇀ Val
- **Σ.M(d)** — arrangement of document d: partial function from V-position tumblers to I-address tumblers, Σ.M(d) : T ⇀ T
