**T10a.2 (NonNestingSiblingPrefixes).** Distinct siblings from the same allocator are prefix-incomparable: `tᵢ ⋠ tⱼ ∧ tⱼ ⋠ tᵢ`.

Let `tᵢ` and `tⱼ` be distinct siblings from the same allocator with `i < j`. We must show neither is a prefix of the other.

First, `tᵢ ≠ tⱼ`. By TA5(a), each application of `inc(·, 0)` produces a strictly greater tumbler under T1, so `tₙ < tₙ₊₁` for every `0 ≤ n < j`. For adjacent siblings (`j = i + 1`) the conclusion `tᵢ < tⱼ` is immediate from this one-step increase. For non-adjacent siblings (`j > i + 1`), the chain `tᵢ < tᵢ₊₁ < ··· < tⱼ` requires `j − i − 1` applications of T1(c) (transitivity) to yield `tᵢ < tⱼ`. By T1(a) (irreflexivity), `tᵢ ≠ tⱼ`.

Second, `#tᵢ = #tⱼ` by T10a.1, since both are siblings of the same allocator.

Now suppose for contradiction that `tᵢ ≼ tⱼ`. Since `tᵢ ≠ tⱼ`, this is a proper prefix `tᵢ ≺ tⱼ`, whence `#tᵢ < #tⱼ` by the Prefix derived postcondition — contradicting `#tᵢ = #tⱼ`. The symmetric argument excludes `tⱼ ≼ tᵢ`: if `tⱼ ≼ tᵢ` with `tⱼ ≠ tᵢ`, then `tⱼ ≺ tᵢ` and `#tⱼ < #tᵢ` by the same postcondition, again contradicting `#tᵢ = #tⱼ`.

Therefore `tᵢ ⋠ tⱼ ∧ tⱼ ⋠ tᵢ` — the sibling prefixes are non-nesting. The full tree-wide guarantee — that allocators not in an ancestor-descendant relationship also have prefix-incomparable output domains — is established in T10a.5 (CrossAllocatorIncomparability). ∎

*Formal Contract:*
- *Precondition:* `tᵢ`, `tⱼ` are distinct siblings from the same allocator (`i ≠ j`).
- *Postcondition:* `tᵢ ⋠ tⱼ ∧ tⱼ ⋠ tᵢ` — neither is a prefix of the other.
- *Depends:* T10a.1 (UniformSiblingLength) establishes `#tᵢ = #tⱼ`; Prefix (PrefixRelation) establishes that equal-length tumblers are prefix-related only if identical. TA5 (HierarchicalIncrement), postcondition (a) — `inc(t, 0) > t` — supplies the per-step strict increase `tₙ < tₙ₊₁` invoked for adjacent siblings (`j = i + 1`) and at every step of the chain for non-adjacent siblings. T1 (LexicographicOrder), postcondition (c) transitivity — lifts that per-step increase through the `j − i − 1` applications named explicitly in the proof to yield `tᵢ < tⱼ` for non-adjacent siblings; postcondition (a) irreflexivity — converts the strict inequality `tᵢ < tⱼ` into distinctness `tᵢ ≠ tⱼ`. Together: distinct siblings have equal length, so a proper prefix relation would require `#tᵢ < #tⱼ` or `#tⱼ < #tᵢ`, contradicting equality.
