## The total order

We require a total order on T. Nelson describes the "tumbler line" as a single linear sequence: "In a sense the tumbler line is like the real line, i.e., the line of integers and all the numbers in between." The system maps a hierarchical tree ‚Äî servers containing accounts containing documents containing elements ‚Äî onto this flat line via depth-first traversal. The traversal inherently produces a total order: for any two nodes in a tree, depth-first traversal visits one before the other. The ordering rule is lexicographic:

**T1 (LexicographicOrder).** For tumblers `a = a‚ÇÅ. ... .a‚Çò` and `b = b‚ÇÅ. ... .b‚Çô`, define `a < b` iff there exists `k ‚â• 1` such that `(A i : 1 ‚â§ i < k : a·µ¢ = b·µ¢)` and either:

  (i) `k ‚â§ min(m, n)` and `a‚Çñ < b‚Çñ`, or

  (ii) `k = m + 1 ‚â§ n` (that is, `a` is a proper prefix of `b`).

The prefix convention ‚Äî a prefix is less than any proper extension ‚Äî is what makes depth-first traversal work. The server address `2` is less than every address within server `2`'s subtree, because every such address extends the prefix `2` with further components. This means server `2`'s subtree begins immediately after `2` in the order and extends until some address whose first component exceeds `2`.

*Proof.* We show that `<` as defined is a strict total order on T by establishing irreflexivity, trichotomy, and transitivity. The argument relies on the corresponding properties of `<` on ‚Ñï (NAT-order) and on T3 (CanonicalRepresentation): tumblers with the same length and identical components at every position are equal. T3 is a consequence of T0's extensional characterisation of T as a set of finite sequences ‚Äî two finite sequences are equal exactly when they agree in length and at every position ‚Äî so it is a derived theorem, not an independent postulate.

*(a) Irreflexivity.* We must show: `(A a ‚àà T :: ¬¨(a < a))`.

Suppose `a < a` for some `a ‚àà T` with `#a = m`. Then there exists `k ‚â• 1` with `a·µ¢ = a·µ¢` for all `1 ‚â§ i < k` (vacuously satisfied) and either (i) `k ‚â§ m` and `a‚Çñ < a‚Çñ`, or (ii) `k = m + 1 ‚â§ m`. Case (i) requires `a‚Çñ < a‚Çñ`, violating NAT-order's irreflexivity of `<` on ‚Ñï. Case (ii) requires `m + 1 ‚â§ m`. NAT-addcompat's strict successor inequality gives `m < m + 1`; NAT-order's trichotomy on ‚Ñï then excludes both `m + 1 < m` and `m + 1 = m`, so the disjunction `m + 1 ‚â§ m` (which abbreviates `m + 1 < m ‚à® m + 1 = m`) is false. Both cases produce contradictions, so no witness `k` exists and `¬¨(a < a)`.

*(b) Trichotomy.* We must show: `(A a, b ‚àà T :: exactly one of a < b, a = b, b < a)`.

Let `a, b ‚àà T` with `#a = m` and `#b = n`. Define the *first divergence position* `k` as the least positive integer at which `a` and `b` disagree ‚Äî either because `a‚Çñ ‚âÝ b‚Çñ` at some `k ‚â§ min(m, n)`, or because one tumbler is exhausted at position `k = min(m, n) + 1` while the other continues. (This same concept is formalized downstream as the corollary Divergence, whose two cases correspond to the two disjuncts here; the definitions are identical by construction, and any tightening of either must be mirrored in the other. The least-element claim ‚Äî that `k` is well-defined whenever `a ‚âÝ b` ‚Äî rests on the well-ordering of ‚Ñï applied to the nonempty subset of positions at which the two sequences fail to match, and is discharged from NAT-wellorder.) Three cases are exhaustive.

*Case 1: no divergence exists.* Then `m = n` and `a·µ¢ = b·µ¢` for all `1 ‚â§ i ‚â§ m`, so `a = b` by T3. By part (a), `¬¨(a < a)` and `¬¨(a > a)`, giving equality as the unique outcome.

*Case 2: `k ‚â§ min(m, n)` and `a‚Çñ ‚âÝ b‚Çñ`.* Since `a·µ¢ = b·µ¢` for all `i < k` but `a‚Çñ ‚âÝ b‚Çñ`, we have `a ‚âÝ b` by the reverse direction of T3 (tumblers that differ in any component are distinct). By NAT-order's trichotomy on ‚Ñï, exactly one of `a‚Çñ < b‚Çñ` or `b‚Çñ < a‚Çñ` holds. If `a‚Çñ < b‚Çñ`, then `k` witnesses `a < b` via T1 case (i); if `b‚Çñ < a‚Çñ`, then `k` witnesses `b < a` via T1 case (i). We confirm that no witness establishes the reverse. Any witness `k'` for the opposite ordering requires `a·µ¢ = b·µ¢` for all `i < k'`. If `k' > k`, this fails at position `k` where `a‚Çñ ‚âÝ b‚Çñ`. If `k' = k`, case (i) requires the opposite inequality at position `k`, contradicting NAT-order's trichotomy, and case (ii) requires `k = n + 1` (or `k = m + 1`), contradicting `k ‚â§ min(m, n) ‚â§ n` (respectively `‚â§ m`). If `k' < k`, the minimality of `k` gives `a_{k'} = b_{k'}`, so case (i) fails on equal components and case (ii) requires `k' = n + 1` (or `m + 1`), but `k' < k ‚â§ min(m, n)` gives `k' < n` and `k' < m`, contradicting both. No witness exists; exactly one ordering holds.

*Case 3: `k = min(m, n) + 1` ‚Äî all shared positions agree but `m ‚âÝ n`.* Since `a·µ¢ = b·µ¢` for all `1 ‚â§ i ‚â§ min(m, n)` but `m ‚âÝ n`, we have `a ‚âÝ b` by T3 (distinct lengths). If `m < n`, then `k = m + 1 ‚â§ n`, so `a` is a proper prefix of `b` and `k` witnesses `a < b` via T1 case (ii). No witness for `b < a` exists: case (i) would require `b‚±º < a‚±º` at some position `j ‚â§ min(m, n)`, but all such positions have equal components; case (ii) would require `b` to be a proper prefix of `a`, i.e., `n < m`, contradicting `m < n`. If `m > n`, the symmetric argument gives `b < a` as the unique outcome.

These three cases partition all pairs in `T √ó T`, and in each case exactly one of the three relations holds.

*(c) Transitivity.* We must show: `(A a, b, c ‚àà T : a < b ‚àß b < c : a < c)`.

Let `k‚ÇÅ` witness `a < b` and `k‚ÇÇ` witness `b < c`, with `#a = m`, `#b = n`, `#c = p`. For all `i < min(k‚ÇÅ, k‚ÇÇ)`, the agreement conditions of the two hypotheses give `a·µ¢ = b·µ¢` (since `i < k‚ÇÅ`) and `b·µ¢ = c·µ¢` (since `i < k‚ÇÇ`), hence `a·µ¢ = c·µ¢` by transitivity of equality. We produce a witness for `a < c` by case analysis on the relationship between `k‚ÇÅ` and `k‚ÇÇ`.

*Case k‚ÇÅ < k‚ÇÇ.* Position `k‚ÇÅ` lies in the agreement range of `b < c`, so `b‚Çñ‚ÇÅ = c‚Çñ‚ÇÅ`. If `a < b` via case (i): `a‚Çñ‚ÇÅ < b‚Çñ‚ÇÅ` with `k‚ÇÅ ‚â§ min(m, n)`, and since `b‚Çñ‚ÇÅ = c‚Çñ‚ÇÅ` we have `a‚Çñ‚ÇÅ < c‚Çñ‚ÇÅ`; the existence of `c‚Çñ‚ÇÅ` gives `k‚ÇÅ ‚â§ p`, so `k‚ÇÅ ‚â§ min(m, p)`, and position `k‚ÇÅ` witnesses `a < c` via T1 case (i). If `a < b` via case (ii): `k‚ÇÅ = m + 1 ‚â§ n`, and since `b‚Çñ‚ÇÅ = c‚Çñ‚ÇÅ` the component `c‚Çñ‚ÇÅ` exists, giving `p ‚â• k‚ÇÅ = m + 1`; thus `a` is a proper prefix of `c` and `k‚ÇÅ` witnesses `a < c` via T1 case (ii).

*Case k‚ÇÇ < k‚ÇÅ.* Position `k‚ÇÇ` lies in the agreement range of `a < b`, so `a‚Çñ‚ÇÇ = b‚Çñ‚ÇÇ`. We first show `b < c` must be via case (i). If `b < c` were via case (ii), then `k‚ÇÇ = n + 1`. But `a < b` requires `k‚ÇÅ ‚â§ n` ‚Äî in case (i), `k‚ÇÅ ‚â§ min(m, n) ‚â§ n`; in case (ii), `k‚ÇÅ = m + 1 ‚â§ n` ‚Äî and NAT-addcompat's strict successor inequality gives `n + 1 > n`, so `k‚ÇÇ = n + 1 > n ‚â• k‚ÇÅ`, which by NAT-order's transitivity yields `k‚ÇÇ > k‚ÇÅ`, contradicting `k‚ÇÇ < k‚ÇÅ`. Therefore `b < c` is via case (i): `b‚Çñ‚ÇÇ < c‚Çñ‚ÇÇ` with `k‚ÇÇ ‚â§ min(n, p)`. Since `k‚ÇÇ < k‚ÇÅ` and `a` has components at all positions below `k‚ÇÅ`, we have `k‚ÇÇ ‚â§ m`, giving `a‚Çñ‚ÇÇ = b‚Çñ‚ÇÇ < c‚Çñ‚ÇÇ` with `k‚ÇÇ ‚â§ min(m, p)`. Position `k‚ÇÇ` witnesses `a < c` via T1 case (i).

*Case k‚ÇÅ = k‚ÇÇ = k.* Both divergences occur at position `k`, and we have `a·µ¢ = c·µ¢` for all `i < k`. The two-case structure of T1 gives four combinations for the pair of witnesses; we show that exactly two are realizable.

*Sub-case (i, i):* `a‚Çñ < b‚Çñ` with `k ‚â§ min(m, n)` and `b‚Çñ < c‚Çñ` with `k ‚â§ min(n, p)`. NAT-order's transitivity of `<` on ‚Ñï gives `a‚Çñ < c‚Çñ`. Since `k ‚â§ m` and `k ‚â§ p`, we have `k ‚â§ min(m, p)`, and position `k` witnesses `a < c` via T1 case (i).

*Sub-case (ii, i):* `k = m + 1 ‚â§ n` and `b‚Çñ < c‚Çñ` with `k ‚â§ min(n, p)`. The bound `k ‚â§ p` gives `m + 1 = k ‚â§ p`, so `a` is a proper prefix of `c` and position `k` witnesses `a < c` via T1 case (ii).

*Sub-case (i, ii):* `a‚Çñ < b‚Çñ` with `k ‚â§ min(m, n)` requires `k ‚â§ n`, while `k = n + 1 ‚â§ p` requires `k > n`. The conjunction `k ‚â§ n` and `k > n` is a contradiction; this sub-case cannot arise.

*Sub-case (ii, ii):* `k = m + 1 ‚â§ n` and `k = n + 1 ‚â§ p`. Then `m + 1 = n + 1`, and NAT-cancel's right cancellation at the fixed summand `1` gives `m = n`. But the first condition requires `m + 1 ‚â§ n`, which by NAT-discrete's equivalence between strict and non-strict successor comparisons (`m + 1 ‚â§ n ‚ü∫ m < n`) means `m < n`, contradicting `m = n`. This sub-case cannot arise.

In every realizable combination, a witness for `a < c` under T1 is produced. ‚àé

The strict total order `<` admits the customary non-strict companions. For `a, b ‚àà T`, we write `a ‚â§ b` to mean `a < b ‚à® a = b`, and `a ‚â• b` to mean `b ‚â§ a`. The well-definedness of these abbreviations is immediate from trichotomy (part (b)): exactly one of `a < b`, `a = b`, `b < a` holds for any pair in `T`, so each disjunction is decidable.

*Formal Contract:*
- *Definition:* `a < b` iff `‚àÉ k ‚â• 1` with `(A i : 1 ‚â§ i < k : a·µ¢ = b·µ¢)` and either (i) `k ‚â§ min(#a,#b) ‚àß a‚Çñ < b‚Çñ`, or (ii) `k = #a+1 ‚â§ #b`.
- *Depends:* T0 (CarrierSetDefinition) ‚Äî the definition uses length `#a` and component projection `a‚Çñ` for `a ‚àà T`, which T0 introduces. NAT-order (NatStrictTotalOrder) ‚Äî the proof invokes the strict-total-order structure of `<` on ‚Ñï at four distinct sites: irreflexivity (part (a), to contradict `a‚Çñ < a‚Çñ`), trichotomy (part (a) Case (ii), to exclude `m + 1 < m` and `m + 1 = m` once `m < m + 1` is in hand; and part (b) Case 2, to resolve the ordering of disagreeing components `a‚Çñ` and `b‚Çñ`), and transitivity (part (c), Case `k‚ÇÇ < k‚ÇÅ` to chain `k‚ÇÇ > n ‚â• k‚ÇÅ` into `k‚ÇÇ > k‚ÇÅ`, and sub-case (i,i) to compose `a‚Çñ < b‚Çñ` and `b‚Çñ < c‚Çñ` into `a‚Çñ < c‚Çñ`), each a clause of NAT-order's Axiom. NAT-addcompat (NatAdditionOrderCompatibility) ‚Äî the proof invokes the strict successor inequality `n < n + 1` at two distinct sites: part (a) Irreflexivity, Case (ii), to supply `m < m + 1` so that NAT-order's trichotomy can refute `m + 1 ‚â§ m`; and part (c) Transitivity, Case `k‚ÇÇ < k‚ÇÅ` under the hypothetical Case (ii) of `b < c`, to derive `k‚ÇÇ = n + 1 > n` and chain (via NAT-order's transitivity) to a contradiction with `k‚ÇÇ < k‚ÇÅ`. NAT-cancel (NatAdditionCancellation) ‚Äî part (c) Transitivity, sub-case (ii, ii), invokes right cancellation at the fixed summand `1` to pass from `m + 1 = n + 1` to `m = n`; this successor-injectivity step is what produces the equality used to contradict the strict comparison derived in the same sub-case. NAT-discrete (NatDiscreteness) ‚Äî part (c) Transitivity, sub-case (ii, ii), invokes the equivalence `m + 1 ‚â§ n ‚ü∫ m < n` (the contrapositive form of discreteness, that no natural number lies strictly between `m` and `m + 1`) to convert the case hypothesis `m + 1 ‚â§ n` into the strict comparison `m < n`, which then contradicts the `m = n` produced by the NAT-cancel step. NAT-wellorder (NatWellOrdering) ‚Äî part (b) Trichotomy defines the *first divergence position* `k` as the least positive integer at which `a` and `b` disagree, invoking the well-ordering of ‚Ñï applied to the subset `{i ‚àà ‚Ñï : 1 ‚â§ i ‚â§ min(m, n) ‚àß a·µ¢ ‚âÝ b·µ¢} ‚à™ ({min(m, n) + 1} when m ‚âÝ n, else ‚àÖ) ‚äÜ ‚Ñï` ‚Äî nonempty in Cases 2 and 3 by construction (Case 1 covers the empty-set situation, where `m = n` and `a·µ¢ = b·µ¢` for all `i`, and the proof reduces to equality by T3 without invoking a divergence position) ‚Äî to supply the least element that the "first divergence position" names; Case 2's rebuttal "if `k' < k`, the minimality of `k` gives `a_{k'} = b_{k'}`" and Case 3's hypothesis that `a·µ¢ = b·µ¢` for all `1 ‚â§ i ‚â§ min(m, n)` (which holds precisely because `k = min(m, n) + 1` was chosen minimal, so no smaller position belongs to the divergence set) both rest on this minimality, so the least-element claim is discharged from NAT-wellorder rather than left implicit. T3 (CanonicalRepresentation) ‚Äî postcondition (b) requires the bridge from component-level agreement to tumbler equality: Case 1 invokes T3 to conclude `a = b` from `m = n` and `a·µ¢ = b·µ¢` for all `i`; Case 2 invokes its reverse direction (tumblers that differ in any component are distinct) to conclude `a ‚âÝ b` from `a‚Çñ ‚âÝ b‚Çñ`, which is load-bearing because trichotomy's "exactly one" outcome requires ruling out `a = b` as a third concurrent option alongside the produced witness for `a < b` or `b < a`; Case 3 invokes its contrapositive to conclude `a ‚âÝ b` from `m ‚âÝ n`.
- *Postconditions:* (a) Irreflexivity ‚Äî `(A a ‚àà T :: ¬¨(a < a))`. (b) Trichotomy ‚Äî `(A a,b ‚àà T :: exactly one of a < b, a = b, b < a)`. (c) Transitivity ‚Äî `(A a,b,c ‚àà T : a < b ‚àß b < c : a < c)`. (d) Non-strict order ‚Äî `a ‚â§ b ‚ü∫ a < b ‚à® a = b`; `a ‚â• b ‚ü∫ b ‚â§ a`.

Nelson's assertion that the tumbler line is total ‚Äî that two addresses are never incomparable ‚Äî is architecturally load-bearing. Spans are defined as contiguous regions on the tumbler line: "A span in the tumbler line, represented by two tumblers, refers to a subtree of the entire docuverse." If two addresses were incomparable, the interval between them would be undefined, and the entire machinery of span-sets, link endsets, and content reference would collapse.
