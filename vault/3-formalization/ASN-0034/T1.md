## The total order

We require a total order on T. Nelson describes the "tumbler line" as a single linear sequence: "In a sense the tumbler line is like the real line, i.e., the line of integers and all the numbers in between." The system maps a hierarchical tree ‚ÄĒ servers containing accounts containing documents containing elements ‚ÄĒ onto this flat line via depth-first traversal. The traversal inherently produces a total order: for any two nodes in a tree, depth-first traversal visits one before the other. The ordering rule is lexicographic:

**T1 (LexicographicOrder).** For tumblers `a = a‚āĀ. ... .a‚āė` and `b = b‚āĀ. ... .b‚āô`, define `a < b` iff there exists `k ‚Č• 1` such that `(A i : 1 ‚Č§ i < k : aŠĶĘ = bŠĶĘ)` and either:

  (i) `k ‚Č§ min(m, n)` and `a‚āĖ < b‚āĖ`, or

  (ii) `k = m + 1 ‚Č§ n` (that is, `a` is a proper prefix of `b`).

The prefix convention ‚ÄĒ a prefix is less than any proper extension ‚ÄĒ is what makes depth-first traversal work. The server address `2` is less than every address within server `2`'s subtree, because every such address extends the prefix `2` with further components. This means server `2`'s subtree begins immediately after `2` in the order and extends until some address whose first component exceeds `2`.

*Proof.* We show that `<` as defined is a strict total order on T by establishing irreflexivity, trichotomy, and transitivity. The argument relies on the corresponding properties of `<` on ‚Ąē (NAT-order) and on T3 (CanonicalRepresentation): tumblers with the same length and identical components at every position are equal. T3 is a consequence of T0's extensional characterisation of T as a set of finite sequences ‚ÄĒ two finite sequences are equal exactly when they agree in length and at every position ‚ÄĒ so it is a derived theorem, not an independent postulate.

*(a) Irreflexivity.* We must show: `(A a ‚ąą T :: ¬¨(a < a))`.

Suppose `a < a` for some `a ‚ąą T` with `#a = m`. Then there exists `k ‚Č• 1` with `aŠĶĘ = aŠĶĘ` for all `1 ‚Č§ i < k` (vacuously satisfied) and either (i) `k ‚Č§ m` and `a‚āĖ < a‚āĖ`, or (ii) `k = m + 1 ‚Č§ m`. Case (i) requires `a‚āĖ < a‚āĖ`, violating NAT-order's irreflexivity of `<` on ‚Ąē. Case (ii) requires `m + 1 ‚Č§ m`. NAT-addcompat's strict successor inequality gives `m < m + 1`; NAT-order's trichotomy on ‚Ąē then excludes both `m + 1 < m` and `m + 1 = m`, so the disjunction `m + 1 ‚Č§ m` (which abbreviates `m + 1 < m ‚ą® m + 1 = m`) is false. Both cases produce contradictions, so no witness `k` exists and `¬¨(a < a)`.

*(b) Trichotomy.* We must show: `(A a, b ‚ąą T :: exactly one of a < b, a = b, b < a)`.

Let `a, b ‚ąą T` with `#a = m` and `#b = n`. Define the *first divergence position* `k` as the least positive integer at which `a` and `b` disagree ‚ÄĒ either because `a‚āĖ ‚Č† b‚āĖ` at some `k ‚Č§ min(m, n)`, or because one tumbler is exhausted at position `k = min(m, n) + 1` while the other continues. (This same concept is formalized downstream as the corollary Divergence, whose two cases correspond to the two disjuncts here; the definitions are identical by construction, and any tightening of either must be mirrored in the other. The least-element claim ‚ÄĒ that `k` is well-defined whenever `a ‚Č† b` ‚ÄĒ rests on the well-ordering of ‚Ąē applied to the nonempty subset of positions at which the two sequences fail to match, and is discharged from NAT-wellorder.) Three cases are exhaustive.

*Case 1: no divergence exists.* Then `m = n` and `aŠĶĘ = bŠĶĘ` for all `1 ‚Č§ i ‚Č§ m`, so `a = b` by T3. By part (a), `¬¨(a < a)` and `¬¨(a > a)`, giving equality as the unique outcome.

*Case 2: `k ‚Č§ min(m, n)` and `a‚āĖ ‚Č† b‚āĖ`.* Since `aŠĶĘ = bŠĶĘ` for all `i < k` but `a‚āĖ ‚Č† b‚āĖ`, we have `a ‚Č† b` by the reverse direction of T3 (tumblers that differ in any component are distinct). By NAT-order's trichotomy on ‚Ąē, exactly one of `a‚āĖ < b‚āĖ` or `b‚āĖ < a‚āĖ` holds. If `a‚āĖ < b‚āĖ`, then `k` witnesses `a < b` via T1 case (i); if `b‚āĖ < a‚āĖ`, then `k` witnesses `b < a` via T1 case (i). We confirm that no witness establishes the reverse. Any witness `k'` for the opposite ordering requires `aŠĶĘ = bŠĶĘ` for all `i < k'`. If `k' > k`, this fails at position `k` where `a‚āĖ ‚Č† b‚āĖ`. If `k' = k`, case (i) requires the opposite inequality at position `k`, contradicting NAT-order's trichotomy, and case (ii) requires `k = n + 1` (or `k = m + 1`), contradicting `k ‚Č§ min(m, n) ‚Č§ n` (respectively `‚Č§ m`). If `k' < k`, the minimality of `k` gives `a_{k'} = b_{k'}`, so case (i) fails on equal components and case (ii) requires `k' = n + 1` (or `m + 1`), but `k' < k ‚Č§ min(m, n)` gives `k' < n` and `k' < m`, contradicting both. No witness exists; exactly one ordering holds.

*Case 3: `k = min(m, n) + 1` ‚ÄĒ all shared positions agree but `m ‚Č† n`.* Since `aŠĶĘ = bŠĶĘ` for all `1 ‚Č§ i ‚Č§ min(m, n)` but `m ‚Č† n`, we have `a ‚Č† b` by T3 (distinct lengths). If `m < n`, then `k = m + 1 ‚Č§ n`, so `a` is a proper prefix of `b` and `k` witnesses `a < b` via T1 case (ii). No witness for `b < a` exists: case (i) would require `b‚Īľ < a‚Īľ` at some position `j ‚Č§ min(m, n)`, but all such positions have equal components; case (ii) would require `b` to be a proper prefix of `a`, i.e., `n < m`, contradicting `m < n`. If `m > n`, the symmetric argument gives `b < a` as the unique outcome.

These three cases partition all pairs in `T √ó T`, and in each case exactly one of the three relations holds.

*(c) Transitivity.* We must show: `(A a, b, c ‚ąą T : a < b ‚ąß b < c : a < c)`.

Let `k‚āĀ` witness `a < b` and `k‚āā` witness `b < c`, with `#a = m`, `#b = n`, `#c = p`. For all `i < min(k‚āĀ, k‚āā)`, the agreement conditions of the two hypotheses give `aŠĶĘ = bŠĶĘ` (since `i < k‚āĀ`) and `bŠĶĘ = cŠĶĘ` (since `i < k‚āā`), hence `aŠĶĘ = cŠĶĘ` by transitivity of equality. We produce a witness for `a < c` by case analysis on the relationship between `k‚āĀ` and `k‚āā`.

*Case k‚āĀ < k‚āā.* Position `k‚āĀ` lies in the agreement range of `b < c`, so `b‚āĖ‚āĀ = c‚āĖ‚āĀ`. If `a < b` via case (i): `a‚āĖ‚āĀ < b‚āĖ‚āĀ` with `k‚āĀ ‚Č§ min(m, n)`, and since `b‚āĖ‚āĀ = c‚āĖ‚āĀ` we have `a‚āĖ‚āĀ < c‚āĖ‚āĀ`; the existence of `c‚āĖ‚āĀ` gives `k‚āĀ ‚Č§ p`, so `k‚āĀ ‚Č§ min(m, p)`, and position `k‚āĀ` witnesses `a < c` via T1 case (i). If `a < b` via case (ii): `k‚āĀ = m + 1 ‚Č§ n`, and since `b‚āĖ‚āĀ = c‚āĖ‚āĀ` the component `c‚āĖ‚āĀ` exists, giving `p ‚Č• k‚āĀ = m + 1`; thus `a` is a proper prefix of `c` and `k‚āĀ` witnesses `a < c` via T1 case (ii).

*Case k‚āā < k‚āĀ.* Position `k‚āā` lies in the agreement range of `a < b`, so `a‚āĖ‚āā = b‚āĖ‚āā`. We first show `b < c` must be via case (i). If `b < c` were via case (ii), then `k‚āā = n + 1`. But `a < b` requires `k‚āĀ ‚Č§ n` ‚ÄĒ in case (i), `k‚āĀ ‚Č§ min(m, n) ‚Č§ n`; in case (ii), `k‚āĀ = m + 1 ‚Č§ n` ‚ÄĒ and NAT-addcompat's strict successor inequality gives `n + 1 > n`, so `k‚āā = n + 1 > n ‚Č• k‚āĀ`, which by NAT-order's transitivity yields `k‚āā > k‚āĀ`, contradicting `k‚āā < k‚āĀ`. Therefore `b < c` is via case (i): `b‚āĖ‚āā < c‚āĖ‚āā` with `k‚āā ‚Č§ min(n, p)`. Since `k‚āā < k‚āĀ` and `a` has components at all positions below `k‚āĀ`, we have `k‚āā ‚Č§ m`, giving `a‚āĖ‚āā = b‚āĖ‚āā < c‚āĖ‚āā` with `k‚āā ‚Č§ min(m, p)`. Position `k‚āā` witnesses `a < c` via T1 case (i).

*Case k‚āĀ = k‚āā = k.* Both divergences occur at position `k`, and we have `aŠĶĘ = cŠĶĘ` for all `i < k`. The two-case structure of T1 gives four combinations for the pair of witnesses; we show that exactly two are realizable.

*Sub-case (i, i):* `a‚āĖ < b‚āĖ` with `k ‚Č§ min(m, n)` and `b‚āĖ < c‚āĖ` with `k ‚Č§ min(n, p)`. NAT-order's transitivity of `<` on ‚Ąē gives `a‚āĖ < c‚āĖ`. Since `k ‚Č§ m` and `k ‚Č§ p`, we have `k ‚Č§ min(m, p)`, and position `k` witnesses `a < c` via T1 case (i).

*Sub-case (ii, i):* `k = m + 1 ‚Č§ n` and `b‚āĖ < c‚āĖ` with `k ‚Č§ min(n, p)`. The bound `k ‚Č§ p` gives `m + 1 = k ‚Č§ p`, so `a` is a proper prefix of `c` and position `k` witnesses `a < c` via T1 case (ii).

*Sub-case (i, ii):* `a‚āĖ < b‚āĖ` with `k ‚Č§ min(m, n)` requires `k ‚Č§ n`, while `k = n + 1 ‚Č§ p` requires `k > n`. The conjunction `k ‚Č§ n` and `k > n` is a contradiction; this sub-case cannot arise.

*Sub-case (ii, ii):* `k = m + 1 ‚Č§ n` and `k = n + 1 ‚Č§ p`. Then `m + 1 = n + 1`, and NAT-cancel's right cancellation at the fixed summand `1` gives `m = n`. But the first condition requires `m + 1 ‚Č§ n`, which by NAT-discrete's equivalence between strict and non-strict successor comparisons (`m + 1 ‚Č§ n ‚üļ m < n`) means `m < n`, contradicting `m = n`. This sub-case cannot arise.

In every realizable combination, a witness for `a < c` under T1 is produced. ‚ąé

The strict total order `<` admits the customary non-strict companions. For `a, b ‚ąą T`, we write `a ‚Č§ b` to mean `a < b ‚ą® a = b`, and `a ‚Č• b` to mean `b ‚Č§ a`. The well-definedness of these abbreviations is immediate from trichotomy (part (b)): exactly one of `a < b`, `a = b`, `b < a` holds for any pair in `T`, so each disjunction is decidable.

*Formal Contract:*
- *Definition:* `a < b` iff `‚ąÉ k ‚Č• 1` with `(A i : 1 ‚Č§ i < k : aŠĶĘ = bŠĶĘ)` and either (i) `k ‚Č§ min(#a,#b) ‚ąß a‚āĖ < b‚āĖ`, or (ii) `k = #a+1 ‚Č§ #b`.
- *Depends:* T0 (CarrierSetDefinition) ‚ÄĒ the definition uses length `#a` and component projection `a‚āĖ` for `a ‚ąą T`, which T0 introduces. NAT-order (NatStrictTotalOrder) ‚ÄĒ the proof invokes the strict-total-order structure of `<` on ‚Ąē at nine distinct sites, each a clause of NAT-order's Axiom. Irreflexivity is invoked once, at part (a) Case (i), to contradict `a‚āĖ < a‚āĖ`. Trichotomy is invoked six times: part (a) Case (ii), to exclude `m + 1 < m` and `m + 1 = m` once `m < m + 1` is in hand; part (b) Case 2's opening step, to resolve the ordering of disagreeing components `a‚āĖ` and `b‚āĖ` into `a‚āĖ < b‚āĖ` or `b‚āĖ < a‚āĖ`; part (b) Case 2's `k' = k` reverse-witness rebuttal via the reverse witness's case (i), to exclude the opposite inequality at position `k` against the forward witness already fixed by the opening step; part (b) Case 2's `k' = k` reverse-witness rebuttal via the reverse witness's case (ii), to exclude `n + 1 < n` and `n + 1 = n` (respectively `m + 1 < m` and `m + 1 = m`) once NAT-addcompat supplies `n < n + 1` (resp. `m < m + 1`), thereby refuting the sub-case's `k = n + 1 ‚Č§ n` (resp. `k = m + 1 ‚Č§ m`); part (b) Case 2's `k' < k` reverse-witness rebuttal via the reverse witness's case (ii), to exclude `n + 1 < n` (resp. `m + 1 < m`) once NAT-addcompat supplies `n < n + 1` (resp. `m < m + 1`), thereby refuting the sub-case's `k' = n + 1 < n` (resp. `k' = m + 1 < m`) under the agreement bound `k' < k ‚Č§ min(m, n)`; and part (c) Transitivity sub-case (i, ii), to refute the conjunction `k ‚Č§ n ‚ąß k > n` once NAT-addcompat supplies `n < n + 1` and the substitution `k = n + 1` delivers the strict inequality `n < k`. Transitivity is invoked twice: part (c) Transitivity Case `k‚āā < k‚āĀ`, to chain `k‚āā > n ‚Č• k‚āĀ` into `k‚āā > k‚āĀ`; and sub-case (i, i), to compose `a‚āĖ < b‚āĖ` and `b‚āĖ < c‚āĖ` into `a‚āĖ < c‚āĖ`. NAT-addcompat (NatAdditionOrderCompatibility) ‚ÄĒ the proof invokes the strict successor inequality `n < n + 1` at five distinct sites: part (a) Irreflexivity, Case (ii), to supply `m < m + 1` so that NAT-order's trichotomy can refute `m + 1 ‚Č§ m`; part (b) Trichotomy, Case 2's `k' = k` reverse-witness rebuttal under the reverse witness's case (ii), to supply `n < n + 1` (resp. `m < m + 1`) so that NAT-order's trichotomy can refute `n + 1 ‚Č§ n` (resp. `m + 1 ‚Č§ m`) ‚ÄĒ the inequality produced by substituting the sub-case's `k = n + 1` (resp. `k = m + 1`) into the bound `k ‚Č§ min(m, n) ‚Č§ n` (resp. `k ‚Č§ m`); part (b) Trichotomy, Case 2's `k' < k` reverse-witness rebuttal under the reverse witness's case (ii), to supply `n < n + 1` (resp. `m < m + 1`) so that NAT-order's trichotomy can refute `n + 1 < n` (resp. `m + 1 < m`) ‚ÄĒ the strictly sharper inequality produced by substituting the sub-case's `k' = n + 1` (resp. `k' = m + 1`) into the agreement bound `k' < k ‚Č§ min(m, n)`, which yields `k' < n` (resp. `k' < m`); part (c) Transitivity, Case `k‚āā < k‚āĀ` under the hypothetical Case (ii) of `b < c`, to derive `k‚āā = n + 1 > n` and chain (via NAT-order's transitivity) to a contradiction with `k‚āā < k‚āĀ`; and part (c) Transitivity, sub-case (i, ii), to supply `n < n + 1` so that substituting the sub-case's `k = n + 1` yields `n < k` (equivalently `k > n`), which together with the case (i) requirement `k ‚Č§ min(m, n) ‚Č§ n` produces the conjunction `k ‚Č§ n ‚ąß k > n` that NAT-order's trichotomy then forecloses. NAT-cancel (NatAdditionCancellation) ‚ÄĒ part (c) Transitivity, sub-case (ii, ii), invokes right cancellation at the fixed summand `1` to pass from `m + 1 = n + 1` to `m = n`; this successor-injectivity step is what produces the equality used to contradict the strict comparison derived in the same sub-case. NAT-discrete (NatDiscreteness) ‚ÄĒ the proof invokes the equivalence `m + 1 ‚Č§ n ‚üļ m < n` (the contrapositive form of discreteness, that no natural number lies strictly between `m` and `m + 1`) at three distinct sites: part (b) Trichotomy, Case 3, uses the forward direction `m < n ‚üĻ m + 1 ‚Č§ n` to pass from the case hypothesis `m < n` to `k = m + 1 ‚Č§ n` ‚ÄĒ the successor-comparison step that underwrites "`a` is a proper prefix of `b`" and therefore T1 case (ii)'s witness for `a < b` (the symmetric `m > n` branch of the same Case 3 uses the mirrored instance `n < m ‚üĻ n + 1 ‚Č§ m` at the corresponding site); part (c) Transitivity, Case `k‚āā < k‚āĀ` under the case-(ii) branch of `a < b`, uses the forward direction in its contrapositive form `k‚āā < m + 1 ‚üĻ k‚āā ‚Č§ m` to translate the narrative step "`a` has components at all positions below `k‚āĀ`" ‚ÄĒ which reduces in this branch to `k‚āā < k‚āĀ = m + 1 ‚üĻ k‚āā ‚Č§ m` since `k‚āĀ = m + 1` by case (ii) of `a < b` ‚ÄĒ into the bound `k‚āā ‚Č§ m` required for position `k‚āā`'s T1 case-(i) witness for `a < c` to satisfy `k‚āā ‚Č§ min(m, p)` (the sibling case-(i) branch of `a < b`, where `k‚āĀ ‚Č§ min(m, n) ‚Č§ m`, yields `k‚āā ‚Č§ m` from `k‚āā < k‚āĀ ‚Č§ m` via NAT-order alone, so the discreteness appeal is specific to the case-(ii) branch); and part (c) Transitivity, sub-case (ii, ii), uses the backward direction `m + 1 ‚Č§ n ‚üĻ m < n` to convert the case hypothesis `m + 1 ‚Č§ n` into the strict comparison `m < n`, which then contradicts the `m = n` produced by the NAT-cancel step. NAT-wellorder (NatWellOrdering) ‚ÄĒ part (b) Trichotomy defines the *first divergence position* `k` as the least positive integer at which `a` and `b` disagree, invoking the well-ordering of ‚Ąē applied to the subset `{i ‚ąą ‚Ąē : 1 ‚Č§ i ‚Č§ min(m, n) ‚ąß aŠĶĘ ‚Č† bŠĶĘ} ‚ą™ ({min(m, n) + 1} when m ‚Č† n, else ‚ąÖ) ‚äÜ ‚Ąē` ‚ÄĒ nonempty in Cases 2 and 3 by construction (Case 1 covers the empty-set situation, where `m = n` and `aŠĶĘ = bŠĶĘ` for all `i`, and the proof reduces to equality by T3 without invoking a divergence position) ‚ÄĒ to supply the least element that the "first divergence position" names; Case 2's rebuttal "if `k' < k`, the minimality of `k` gives `a_{k'} = b_{k'}`" and Case 3's hypothesis that `aŠĶĘ = bŠĶĘ` for all `1 ‚Č§ i ‚Č§ min(m, n)` (which holds precisely because `k = min(m, n) + 1` was chosen minimal, so no smaller position belongs to the divergence set) both rest on this minimality, so the least-element claim is discharged from NAT-wellorder rather than left implicit. T3 (CanonicalRepresentation) ‚ÄĒ postcondition (b) requires the bridge from component-level agreement to tumbler equality: Case 1 invokes T3 to conclude `a = b` from `m = n` and `aŠĶĘ = bŠĶĘ` for all `i`; Case 2 invokes its reverse direction (tumblers that differ in any component are distinct) to conclude `a ‚Č† b` from `a‚āĖ ‚Č† b‚āĖ`, which is load-bearing because trichotomy's "exactly one" outcome requires ruling out `a = b` as a third concurrent option alongside the produced witness for `a < b` or `b < a`; Case 3 invokes its contrapositive to conclude `a ‚Č† b` from `m ‚Č† n`.
- *Postconditions:* (a) Irreflexivity ‚ÄĒ `(A a ‚ąą T :: ¬¨(a < a))`. (b) Trichotomy ‚ÄĒ `(A a,b ‚ąą T :: exactly one of a < b, a = b, b < a)`. (c) Transitivity ‚ÄĒ `(A a,b,c ‚ąą T : a < b ‚ąß b < c : a < c)`. (d) Non-strict order ‚ÄĒ `a ‚Č§ b ‚üļ a < b ‚ą® a = b`; `a ‚Č• b ‚üļ b ‚Č§ a`.

Nelson's assertion that the tumbler line is total ‚ÄĒ that two addresses are never incomparable ‚ÄĒ is architecturally load-bearing. Spans are defined as contiguous regions on the tumbler line: "A span in the tumbler line, represented by two tumblers, refers to a subtree of the entire docuverse." If two addresses were incomparable, the interval between them would be undefined, and the entire machinery of span-sets, link endsets, and content reference would collapse.
