**T10a.7 (EnumerationInjectivity).** For every allocator A with domain `dom(A) = {tₙ : n ≥ 0}` — `t₀` the base address and `tₙ₊₁ = inc(tₙ, 0)` — the indexing map `n ↦ tₙ` is injective: for all `m, n ≥ 0` with `m ≠ n`, `tₘ ≠ tₙ`.

Without this property, a single element of `dom(A)` could be reached by two distinct indices, and any predicate phrased over the enumeration index — `allocated_before(a, b)` in T9 — would receive ambiguous values even when the witnessing allocator is already unique. T10a.6 (DomainDisjointness) secures uniqueness of the witnessing allocator across allocators; T10a.7 secures uniqueness of the enumeration index within a fixed allocator. Together they make the indices `i, j` with `a = tᵢ, b = tⱼ` single-valued functions of the pair `(a, b)` whenever `same_allocator(a, b)` holds.

*Proof.* Let `m, n ≥ 0` with `m ≠ n`. By trichotomy of `<` on ℕ, either `m < n` or `n < m`, and the conclusion `tₘ ≠ tₙ` is symmetric in `m` and `n`; assume without loss of generality that `m < n`. We show `tₘ < tₙ` by induction on the gap `d = n - m ≥ 1`.

*Base case* (`d = 1`). Here `n = m + 1`, so `tₙ = inc(tₘ, 0)`. By TA5(a), `inc(tₘ, 0) > tₘ`, hence `tₘ < tₙ`.

*Inductive step* (from `d` to `d + 1`, assuming the result holds for gap `d`). We must show `tₘ < tₙ` when `n - m = d + 1`. Since `d ≥ 1`, the index `n - 1` satisfies `m < n - 1 < n` with gap `(n - 1) - m = d`. The inductive hypothesis gives `tₘ < tₙ₋₁`. By TA5(a), `tₙ = inc(tₙ₋₁, 0) > tₙ₋₁`, so `tₙ₋₁ < tₙ`. By T1 (c) transitivity of the strict order, `tₘ < tₙ`.

This completes the induction: whenever `m < n`, `tₘ < tₙ`. By T1 (a) irreflexivity, `tₘ < tₙ` excludes `tₘ = tₙ`; hence `tₘ ≠ tₙ`. The symmetric case `n < m` yields `tₙ ≠ tₘ` by the same argument with the roles of `m` and `n` exchanged. In either case `tₘ ≠ tₙ`. ∎

*Formal Contract:*
- *Precondition:* Allocator A conforming to T10a, with domain `dom(A) = {tₙ : n ≥ 0}` where `t₀` is the base address and `tₙ₊₁ = inc(tₙ, 0)`.
- *Postcondition:* The map `n ↦ tₙ` is injective: `(A m, n ≥ 0 : m ≠ n : tₘ ≠ tₙ)`. Equivalently (by T1 trichotomy, applied to the strict inequality established in the proof), `(A m, n ≥ 0 : m < n : tₘ < tₙ)` — the enumeration is strictly increasing under the tumbler order T1.
- *Depends:* T10a (AllocatorDiscipline) — supplies the enumeration `tₙ₊₁ = inc(tₙ, 0)` whose injectivity is the claim. TA5 (HierarchicalIncrement), postcondition (a) — strict monotonicity `inc(tₙ, 0) > tₙ`, providing the per-step increase at the base case and in the inductive step. T1 (LexicographicOrder), postcondition (c) transitivity — lifts the per-step increase to `tₘ < tₙ` across arbitrary gaps; postcondition (a) irreflexivity — converts the strict inequality `tₘ < tₙ` into distinctness `tₘ ≠ tₙ`.
