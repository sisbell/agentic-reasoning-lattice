**Definition (ZeroPaddedDivergence).** For tumblers `a, w ∈ T`, the *zero-padded divergence* `zpd(a, w)` is defined over the zero-padded extensions of both operands to a common length `L`, selected by NAT-order's trichotomy applied to the length pair `(#a, #w)`: in sub-case (α) `#a = #w`, `L = #a` (equivalently `#w`) and no padding is required; in sub-case (β) `#a < #w`, `L = #w`; in sub-case (γ) `#w < #a`, `L = #a`. Trichotomy's exactly-one clause ensures precisely one sub-case applies, so `L` — the longer of `#a` and `#w` — is named by the selected sub-case without appeal to any primitive binary-maximum operator on ℕ, matching Divergence's analogous length-pair dispatch. Introduce the *padded projections* `â`, `ŵ` on the common domain `{1, ..., L}` by `âᵢ = aᵢ` for `1 ≤ i ≤ #a` (agreeing with T0's projection on its native domain) and `âᵢ = 0` for `#a < i ≤ L` (filling the padding region past `a`'s last position), and symmetrically `ŵᵢ = wᵢ` for `1 ≤ i ≤ #w` together with `ŵᵢ = 0` for `#w < i ≤ L`; `â`, `ŵ` are ASN-internal operators that extend T0's component projection from its native domain `{1, ..., #a}` (resp. `{1, ..., #w}`) to the common domain `{1, ..., L}` by combining it with the literal `0` — which NAT-zero supplies as an element of ℕ — on the padding region, so that every `âᵢ` and `ŵᵢ` is ℕ-valued on `{1, ..., L}`. In sub-case (α) the padding region is empty for both operands, so `â = a` and `ŵ = w`; in sub-case (β) only `â` carries a nontrivial padding region (positions `#a + 1` through `L = #w`) and `ŵ = w`; in sub-case (γ) only `ŵ` carries a nontrivial padding region (positions `#w + 1` through `L = #a`) and `â = a`. If the padded projections agree at every position — `(A i : 1 ≤ i ≤ L : âᵢ = ŵᵢ)` — then `zpd(a, w)` is *undefined*; we say `a` and `w` are *zero-padded-equal*. Otherwise, `zpd(a, w)` is the least `k` with `1 ≤ k ≤ L` such that `âₖ ≠ ŵₖ` and `(A i : 1 ≤ i < k : âᵢ = ŵᵢ)` — the first position at which the padded projections disagree.

The function is partial. It is undefined precisely when the padded projections `â`, `ŵ` agree everywhere on `{1, ..., L}` — as occurs when one operand is a proper prefix of the other with all trailing components zero (e.g., `a = [3, 0]` and `w = [3]` yield `L = 2` under sub-case (γ), with `â = [3, 0]` and `ŵ = [3, 0]` coinciding on both positions). Two equal tumblers have an empty padding region — `â = a` and `ŵ = w = a` — so they are trivially zero-padded-equal and `zpd(a, a)` is always undefined.

**Relationship to Divergence.** When `a ≠ w`, the formal Divergence and `zpd` may disagree. NAT-order's trichotomy on `(#a, #w)` partitions the analysis into the same three sub-cases (α), (β), (γ) as the Definition. In Divergence case (i) — component divergence at a shared position `k` with `k ≤ #a ∧ k ≤ #w` — the padded projections coincide with the native projections throughout `1, ..., k` (since `âᵢ = aᵢ` holds on `{1, ..., #a}` and `ŵᵢ = wᵢ` holds on `{1, ..., #w}`, with `k` satisfying both bounds), so no padding component is consulted and `zpd(a, w) = divergence(a, w) = k`. The shared-position bound is stated as the conjunction `k ≤ #a ∧ k ≤ #w` rather than `k ≤ min(#a, #w)` so that no primitive binary-minimum operator on ℕ is invoked, and the conclusion holds uniformly across sub-cases (α), (β), and (γ) since the conjunction is independent of which of `#a`, `#w` is smaller. In Divergence case (ii) — one tumbler is a proper prefix of the other, which requires `#a ≠ #w` and therefore falls in sub-case (β) or (γ), excluding (α). In sub-case (β), where `#a < #w`, Divergence reports the boundary `#a + 1` (replacing `min(#a, #w) + 1` for this sub-case), whereas `zpd` scans through the padded components `âᵢ = 0` for `#a < i ≤ #w` against `ŵᵢ = wᵢ` on its native domain (since `#w = L`): if `w` has a nonzero component at some position `i` with `#a < i ≤ #w`, then `âᵢ = 0 ≠ wᵢ = ŵᵢ` at that `i` and `zpd(a, w)` equals the least such `i`, which is at least `#a + 1 = divergence(a, w)`; if all components of `w` at positions `#a < i ≤ #w` are zero, then `âᵢ = ŵᵢ = 0` throughout the padding region, the padded projections agree everywhere, and `zpd(a, w)` is undefined. In sub-case (γ), where `#w < #a`, the situation is symmetric: Divergence reports the boundary `#w + 1`, `ŵ` carries the nontrivial padding with `ŵᵢ = 0` for `#w < i ≤ #a` while `â = a` on its native domain, and the same dichotomy applies to the components of `a` at positions `#w < i ≤ #a`. In either sub-case the distinction reduces to whether the longer operand carries any nonzero component past the shorter operand's last position.

*Formal Contract:*
- *Domain:* a ∈ T, w ∈ T
- *Definition:* NAT-order's trichotomy on `(#a, #w)` selects exactly one of: (α) `#a = #w`, `L = #a`; (β) `#a < #w`, `L = #w`; (γ) `#w < #a`, `L = #a`. Define padded projections `â`, `ŵ` on `{1, ..., L}` by `âᵢ = aᵢ` for `1 ≤ i ≤ #a` and `âᵢ = 0` for `#a < i ≤ L`, and `ŵᵢ = wᵢ` for `1 ≤ i ≤ #w` and `ŵᵢ = 0` for `#w < i ≤ L`. If `(A i : 1 ≤ i ≤ L : âᵢ = ŵᵢ)`, `zpd(a, w)` is undefined. Otherwise, `zpd(a, w) = min {k : 1 ≤ k ≤ L ∧ âₖ ≠ ŵₖ}`.
- *Depends:* T0 (CarrierSetDefinition) — the membership `a, w ∈ T` in the Domain, together with the lengths `#a`, `#w` (used in the trichotomy case-split that selects the common length `L` and in the index conditions `#a < i ≤ L`, `#w < i ≤ L` that delimit the padding region) and the component projections `aᵢ` for `1 ≤ i ≤ #a` and `wᵢ` for `1 ≤ i ≤ #w` on their respective native domains (feeding the agreeing clause of the padded projections, `âᵢ = aᵢ` for `1 ≤ i ≤ #a` and `ŵᵢ = wᵢ` for `1 ≤ i ≤ #w`), all come from T0's characterisation of T as finite sequences over ℕ with length ≥ 1; T0's component projection is declared only on these native domains, and the positions `#a < i ≤ L` (resp. `#w < i ≤ L`) in the padding region lie outside T0's domain — the extension to the common domain `{1, ..., L}` is performed by the ASN-internal padded projections `â`, `ŵ` introduced in the Definition, which combine T0's projection on its native domain with the literal `0` (supplied by NAT-zero, below) on the padding region, rather than being attributed to T0. The agreement predicate `(A i : 1 ≤ i ≤ L : âᵢ = ŵᵢ)` and disagreement predicate `âₖ ≠ ŵₖ` rest on T0's ℕ-valuation of native-domain components combined with NAT-zero's supply of `0 ∈ ℕ` for the padding region, making the (in)equality a well-formed comparison in ℕ rather than an uninterpreted symbol — this citation discharges the carrier-set, length, and native-domain component-projection operators from T0 without overstating T0's reach into the padding region, matching the per-step convention established in ActionPoint, Divergence, and TA-Pos. Divergence (Divergence) — the Relationship to Divergence postconditions consume Divergence's two-case structure (case (i): component divergence at `k` with `k ≤ #a ∧ k ≤ #b`, case (ii): prefix divergence at `#a + 1` in sub-case (ii-a) or `#b + 1` in sub-case (ii-b) per NAT-order trichotomy at `(#a, #b)`) and its domain restriction `a ≠ b`, which gates the postcondition's guard `a ≠ w`.
- *Codomain:* When defined, `zpd(a, w) ∈ {1, ..., L}`, where `L` is the common length selected by NAT-order trichotomy on `(#a, #w)`: `L = #a` in sub-cases (α) and (γ), `L = #w` in sub-case (β).
- *Partiality:* zpd(a, w) is undefined iff a and w are zero-padded-equal.
- *Postconditions (Symmetry):* `zpd(a, w)` is defined iff `zpd(w, a)` is defined, and when defined, `zpd(a, w) = zpd(w, a)`. Padding both operands to the common length `L` selected by NAT-order trichotomy on `(#a, #w)` is symmetric under exchange — sub-case (α) is self-symmetric (no padding), while sub-cases (β) and (γ) swap under exchange of operands, with the roles of `â` and `ŵ` swapping accordingly, and both orientations yield the same `L` (the longer of `#a`, `#w`) — and the disagreement predicate is symmetric: `âₖ ≠ ŵₖ` iff `ŵₖ ≠ âₖ`.
- *Postconditions (Relationship to Divergence):* For `a ≠ w`: in Divergence case (i) — component divergence at `k` with `k ≤ #a ∧ k ≤ #w` (the conjunction replacing `k ≤ min(#a, #w)`) — the padded projections coincide with the native projections throughout `1, ..., k`, so no padding component is consulted and `zpd(a, w) = divergence(a, w)`; in Divergence case (ii) — one operand is a proper prefix of the other, occurring under NAT-order trichotomy sub-case (β) (`#a < #w`) or (γ) (`#w < #a`) — if the longer operand has a nonzero component at some position beyond the shorter operand's last (position `#a` under (β) or position `#w` under (γ)), then `zpd(a, w)` is defined and `zpd(a, w) ≥ divergence(a, w)` (≥ `#a + 1` under (β), ≥ `#w + 1` under (γ), each replacing `min(#a, #w) + 1` for its sub-case); if all such trailing components are zero, then the padded projections agree everywhere and `zpd(a, w)` is undefined (the operands are zero-padded-equal despite being distinct).
