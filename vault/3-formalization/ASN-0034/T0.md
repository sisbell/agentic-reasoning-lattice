**T0 (CarrierSetDefinition).** `T = {d₁.d₂. ... .dₙ : each dᵢ ∈ ℕ, n ≥ 1}`. This is an axiom: we posit the carrier set by definition, not by derivation. The natural numbers ℕ are taken with their standard properties, including closure under successor and addition, and discreteness: for `m, n ∈ ℕ`, `m ≤ n < m + 1` implies `n = m`.

*Formal Contract:*
- *Axiom:* T is the set of all finite sequences over ℕ with length ≥ 1; ℕ is closed under successor and addition and satisfies discreteness: `(A m, n ∈ ℕ :: m ≤ n < m + 1 ⟹ n = m)`.
