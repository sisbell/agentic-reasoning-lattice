**T0 (CarrierSetDefinition).** `T = {d₁.d₂. ... .dₙ : each dᵢ ∈ ℕ, n ≥ 1}`. This is an axiom: we posit the carrier set by definition, not by derivation. For each `a ∈ T`, write `#a` for the number of components in `a` — the length of the sequence — so that `#a ≥ 1`, and write `aᵢ` for the value of the `i`-th component, defined for `1 ≤ i ≤ #a` and satisfying `aᵢ ∈ ℕ`. Write `ℕ⁺ = {n ∈ ℕ : n > 0}` for the subcarrier of strictly positive naturals — downstream properties (notably T4b and T6) use this notation as the value domain of non-zero components, distinct from the full carrier ℕ which also admits zero in the role of field separator.

The carrier set and its operators are stated here. The standard properties of ℕ that downstream proofs cite — closure under successor and addition, strict total order, discreteness, order compatibility of addition, well-ordering, zero as lower bound of ℕ, partial subtraction (with its conditional closure, inverse characterisations, strict positivity, right telescoping, and strict monotonicity clauses), additive cancellation (left, right, and summand absorption), and associativity of addition — are stated as separate axioms, each established as its own foundation object under both a hyphenated short label and a PascalCase canonical name: NAT-closure (NatArithmeticClosure), NAT-order (NatStrictTotalOrder), NAT-discrete (NatDiscreteness), NAT-addcompat (NatAdditionOrderCompatibility), NAT-wellorder (NatWellOrdering), NAT-zero (NatZeroMinimum), NAT-sub (NatPartialSubtraction), NAT-cancel (NatAdditionCancellation), NAT-addassoc (NatAdditionAssociative). The two name forms denote the same axiom in each case — the hyphenated form is the file-local label, the PascalCase form is the canonical identifier — so a downstream Depends entry that cites either form (or both, as in the conventional pairing `NAT-zero (NatZeroMinimum)`) resolves unambiguously to the corresponding foundation object. This list is exhaustive: the ASN states no NAT-* axiom outside it. Each proof cites only the ℕ facts it actually uses.

*Formal Contract:*
- *Axiom:* T is the set of all finite sequences over ℕ with length ≥ 1, equipped with length `#· : T → ℕ` satisfying `#a ≥ 1` for all `a ∈ T`, and component projection `·ᵢ` yielding `aᵢ ∈ ℕ` for each `1 ≤ i ≤ #a`. The subcarrier `ℕ⁺ = {n ∈ ℕ : n > 0}` denotes the strictly positive naturals, supplying the value domain of non-zero components that downstream properties cite.
