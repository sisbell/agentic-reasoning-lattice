**T0 (CarrierSetDefinition).** `T = {d₁.d₂. ... .dₙ : each dᵢ ∈ ℕ, n ≥ 1}`. This is an axiom: we posit the carrier set by definition, not by derivation. For each `a ∈ T`, write `#a` for the number of components in `a` — the length of the sequence — so that `#a ≥ 1`, and write `aᵢ` for the value of the `i`-th component, defined for `1 ≤ i ≤ #a` and satisfying `aᵢ ∈ ℕ`. The natural numbers ℕ are taken with the following standard properties, enumerated here because downstream proofs cite them directly. First, ℕ is closed under successor and addition, with `0` as the additive identity — `0 + n = n` for every `n ∈ ℕ`. Second, the binary relation `<` on ℕ is a strict total order: (i) irreflexivity — `¬(n < n)` for every `n ∈ ℕ`; (ii) transitivity — `m < n ∧ n < p ⟹ m < p` for `m, n, p ∈ ℕ`; (iii) trichotomy — for any `m, n ∈ ℕ`, exactly one of `m < n`, `m = n`, `n < m` holds. The non-strict companion `≤` is defined by `m ≤ n ⟺ m < n ∨ m = n`, which is the relation used in the discreteness axiom. Third, ℕ satisfies discreteness: for `m, n ∈ ℕ`, `m ≤ n < m + 1` implies `n = m`. Fourth, addition is compatible with the order: for `m, n, p ∈ ℕ`, `n ≥ p ⟹ m + n ≥ m + p` (order-compatibility of `+`), and `n < n + 1` for every `n ∈ ℕ` (strict successor inequality).

*Formal Contract:*
- *Axiom:* T is the set of all finite sequences over ℕ with length ≥ 1, equipped with length `#· : T → ℕ` satisfying `#a ≥ 1` for all `a ∈ T`, and component projection `·ᵢ` yielding `aᵢ ∈ ℕ` for each `1 ≤ i ≤ #a`. ℕ is closed under successor and addition, with `(A n ∈ ℕ :: 0 + n = n)` (additive identity). The relation `<` on ℕ is a strict total order: `(A n ∈ ℕ :: ¬(n < n))` (irreflexivity); `(A m, n, p ∈ ℕ : m < n ∧ n < p : m < p)` (transitivity); `(A m, n ∈ ℕ :: exactly one of m < n, m = n, n < m)` (trichotomy). The non-strict relation is defined by `m ≤ n ⟺ m < n ∨ m = n`. ℕ satisfies discreteness: `(A m, n ∈ ℕ :: m ≤ n < m + 1 ⟹ n = m)`. Addition is compatible with the order: `(A m, n, p ∈ ℕ : n ≥ p : m + n ≥ m + p)` (order-compatibility of `+`); and `(A n ∈ ℕ :: n < n + 1)` (strict successor inequality).
