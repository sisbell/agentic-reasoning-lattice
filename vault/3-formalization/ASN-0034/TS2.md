**TS2 (ShiftInjectivity).**

`(A v₁, v₂, n : n ≥ 1 ∧ #v₁ = #v₂ = m : shift(v₁, n) = shift(v₂, n) ⟹ v₁ = v₂)`

*Proof.* Fix v₁, v₂ ∈ T with #v₁ = #v₂ = m, and fix n ≥ 1. Assume shift(v₁, n) = shift(v₂, n). Since v₁, v₂ ∈ T, T0 (CarrierSetDefinition) gives m = #v₁ = #v₂ ≥ 1, so no empty-sequence case arises.

At the v₂ invocation: `v ∈ T` transfers directly from TS2's own hypothesis `v₂ ∈ T` (one of TS2's own preconditions) under the substitution `v = v₂`; `n ∈ ℕ` transfers directly from TS2's same precondition `n ∈ ℕ` (n is fixed across both invocations); and `n ≥ 1` transfers directly from TS2's same precondition `n ≥ 1` (likewise). We apply TA-MTO (ManyToOne) with w = δ(n, m), a = v₁, b = v₂, verifying its six preconditions one-for-one in the order TA-MTO's contract states them — `w ∈ T, Pos(w), a ∈ T, b ∈ T, #a ≥ actionPoint(w), #b ≥ actionPoint(w)` — with each of the three membership preconditions and each of the two length bounds discharged separately rather than fused into a composite item:

(ii) Pos(δ(n, m)) — by OrdinalDisplacement's exported postcondition `Pos(δ(n, m))`.
