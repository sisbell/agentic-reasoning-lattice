**TS5 (ShiftAmountMonotonicity).**

`(A v, n‚āĀ, n‚āā, m : v ‚ąą T ‚ąß n‚āĀ ‚ąą ‚Ąē ‚ąß n‚āā ‚ąą ‚Ąē ‚ąß n‚āĀ ‚Č• 1 ‚ąß n‚āā > n‚āĀ ‚ąß #v = m : shift(v, n‚āĀ) < shift(v, n‚āā))`

The dummy `m` in the binder is a local abbreviation for `#v`: for any fixed `v ‚ąą T`, the range predicate `#v = m` determines `m` uniquely as the length of `v` (with `m ‚ąą ‚Ąē` following by T0's length operator typing `#¬∑: T ‚Üí ‚Ąē` applied to `v ‚ąą T`, and `m ‚Č• 1` following by T0's length axiom `#a ‚Č• 1 for all a ‚ąą T`), so the universal over `m` collapses to the single witness `m = #v` ‚ÄĒ an abbreviation pattern matching the one OrdinalShift's own Definition clause `shift(v, n) = v ‚äē őī(n, m) where m = #v` uses at its point of introduction, and matching TS3's and TS4's binder prefaces at their analogous dummy-`m` sites.

*Proof.* We show that shifting a tumbler by a larger amount produces a strictly greater result: if n‚āā exceeds n‚āĀ, then the shift by n‚āā overshoots the shift by n‚āĀ.

Fix v ‚ąą T, and let m abbreviate #v so that the range predicate `#v = m` binds m as the length of v. Fix n‚āĀ, n‚āā ‚ąą ‚Ąē with n‚āĀ ‚Č• 1 and n‚āā > n‚āĀ. The carrier-typings `n‚āĀ ‚ąą ‚Ąē` and `n‚āā ‚ąą ‚Ąē` are named separately in the preconditions because each is consumed downstream as the named source for NAT-sub's three ‚Ąē-typing clauses, for TS3's `a ‚ąą ‚Ąē` precondition under the substitution `a = n‚āĀ`, and (transitively, via NAT-sub's conditional closure on the difference `d = n‚āā ‚ąí n‚āĀ`) for TS3's `b ‚ąą ‚Ąē` precondition under `b = d` and TS4's `n ‚ąą ‚Ąē` precondition under `n = d` ‚ÄĒ the order facts `n‚āĀ ‚Č• 1` and `n‚āā > n‚āĀ` presuppose rather than establish carrier membership (the convention OrdinalDisplacement's own proof articulates at its m-th-position-typing site, and TS3 re-cites at each of its three OrdinalShift invocation sites). We must prove shift(v, n‚āĀ) < shift(v, n‚āā).

Define d = n‚āā ‚ąí n‚āĀ. Three NAT-sub clauses participate at this site, each discharged to named sources. The ‚Ąē-typing `d ‚ąą ‚Ąē` follows from NAT-sub's conditional closure axiom `(A m, n ‚ąą ‚Ąē : m ‚Č• n : m ‚ąí n ‚ąą ‚Ąē)` instantiated at `m = n‚āā, n = n‚āĀ`: NAT-sub's two ‚Ąē-typing preconditions transfer directly from TS5's own preconditions `n‚āā ‚ąą ‚Ąē` and `n‚āĀ ‚ąą ‚Ąē`, and the order precondition `n‚āā ‚Č• n‚āĀ` follows from TS5's hypothesis `n‚āā > n‚āĀ` via NAT-order's defining clause `m ‚Č§ n ‚üļ m < n ‚ą® m = n` read right-to-left at the strict disjunct. The order fact `d ‚Č• 1` follows from NAT-sub's strict positivity axiom `(A m, n ‚ąą ‚Ąē : m > n : m ‚ąí n ‚Č• 1)` instantiated at `m = n‚āā, n = n‚āĀ`: the same two ‚Ąē-typing preconditions transfer from TS5's preconditions, and the strict order precondition `n‚āā > n‚āĀ` is exactly TS5's hypothesis. The rewrite `n‚āā = n‚āĀ + d` follows from NAT-sub's left-inverse characterisation `(A m, n ‚ąą ‚Ąē : m ‚Č• n : n + (m ‚ąí n) = m)` instantiated at `m = n‚āā, n = n‚āĀ`: again the two ‚Ąē-typing preconditions transfer from TS5's preconditions, the order precondition `n‚āā ‚Č• n‚āĀ` is supplied by the same NAT-order conversion, and the resulting equation `n‚āĀ + (n‚āā ‚ąí n‚āĀ) = n‚āā`, with `n‚āā ‚ąí n‚āĀ` abbreviated as `d`, becomes `n‚āĀ + d = n‚āā`. So we have n‚āā = n‚āĀ + d with both n‚āĀ ‚Č• 1 (TS5's precondition) and d ‚Č• 1 (just established).

We invoke TS3 (ShiftComposition), which states that for any tumbler u with #u = m and any pair of shift amounts a, b ‚ąą ‚Ąē with a ‚Č• 1, b ‚Č• 1: shift(shift(u, a), b) = shift(u, a + b). Here u = v, a = n‚āĀ, b = d. TS3's six preconditions are discharged to named sources: `v ‚ąą T` (TS5's precondition, discharging TS3's first-argument carrier-typing); `n‚āĀ ‚ąą ‚Ąē` (TS5's precondition, discharging TS3's `a ‚ąą ‚Ąē` under `a = n‚āĀ`); `d ‚ąą ‚Ąē` (just established via NAT-sub's conditional closure, discharging TS3's `b ‚ąą ‚Ąē` under `b = d`); `n‚āĀ ‚Č• 1` (TS5's precondition); `d ‚Č• 1` (just established via NAT-sub's strict positivity); and `#v = m` (TS5's precondition). Therefore shift(shift(v, n‚āĀ), d) = shift(v, n‚āĀ + d) = shift(v, n‚āā), the last equality by the NAT-sub left-inverse rewrite above. This expresses the larger shift as a composition: first shift by n‚āĀ, then shift the result by d.

Let u = shift(v, n‚āĀ). By OrdinalShift, u is a tumbler with #u = m (shift preserves length). Then shift(v, n‚āā) = shift(u, d). We invoke TS4 (ShiftStrictIncrease), which states that for any tumbler u with #u = m and any n ‚ąą ‚Ąē with n ‚Č• 1: shift(u, n) > u. Here u = shift(v, n‚āĀ) and n = d. TS4's four preconditions are discharged to named sources: `u ‚ąą T` (from OrdinalShift's exported carrier-typing of `shift(v, n‚āĀ)` just cited); `d ‚ąą ‚Ąē` (just established via NAT-sub's conditional closure, discharging TS4's `n ‚ąą ‚Ąē` under `n = d`); `d ‚Č• 1` (just established via NAT-sub's strict positivity); and `#u = m` (just noted). Therefore shift(u, d) > u.

Substituting back: shift(v, n‚āā) = shift(u, d) > u = shift(v, n‚āĀ), that is, shift(v, n‚āĀ) < shift(v, n‚āā). ‚ąé

*Worked example.* Let v = [2, 3, 7] (m = 3) with n‚āĀ = 4 and n‚āā = 7, so n‚āā > n‚āĀ ‚Č• 1. Then őī(4, 3) = [0, 0, 4] and őī(7, 3) = [0, 0, 7], each with action point 3. TA0: k = 3 ‚Č§ 3 = #v at both invocations. By TumblerAdd: shift(v, 4) = [2, 3, 7 + 4] = [2, 3, 11] and shift(v, 7) = [2, 3, 7 + 7] = [2, 3, 14]. The first two components agree and 11 < 14 at position 3, so by T1's lexicographic ordering [2, 3, 11] < [2, 3, 14], that is, shift(v, 4) < shift(v, 7). ‚úď
