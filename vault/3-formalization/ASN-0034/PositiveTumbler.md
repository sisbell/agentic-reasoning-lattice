## Zero tumblers and positivity

Under T3, the tumblers `[0]`, `[0, 0]`, `[0, 0, 0]`, etc., are *distinct* elements of T ‚Äî they have different lengths. Under T1, they form a chain: `[0] < [0, 0] < [0, 0, 0] < ...` by the prefix rule. There is no single "zero tumbler"; there are infinitely many all-zero tumblers. (T2 (IntrinsicComparison), which sits between T1 and T3 in this ASN's numbering, is stated elsewhere in this ASN and is not invoked here; the T1 ‚Üí T3 jump in this property's citations reflects TA-Pos's proof needs, not a numbering gap.)

**TA-Pos (PositiveTumbler).** A tumbler `t ‚àà T` is *positive*, written `Pos(t)`, iff at least one of its components is nonzero: `(E i : 1 ‚â§ i ‚â§ #t : t·µ¢ ‚âÝ 0)`. A tumbler is a *zero tumbler* iff every component is zero: `(A i : 1 ‚â§ i ‚â§ #t : t·µ¢ = 0)`.

The predicate `Pos(t)` is not written `t > 0`, because `>` already denotes T1's lexicographic ordering, and the two notions diverge. The all-zero tumbler `[0, 0]` exhibits the divergence concretely: under T1, `[0] < [0, 0]` by case (ii) ‚Äî the prefix rule, since `[0]` is a proper prefix of `[0, 0]` ‚Äî yet `¬¨Pos([0, 0])`, since every component of `[0, 0]` is zero. The converse direction does hold: `Pos(t)` implies `t` is T1-greater than every zero tumbler, as the following proof establishes.

Every positive tumbler is greater than every zero tumbler under T1.

*Proof.* Let `t ‚àà T` with `Pos(t)` and let `z ‚àà T` be a zero tumbler; we show `z < t`. Since `Pos(t)`, the set `{i : 1 ‚â§ i ‚â§ #t ‚àß t·µ¢ ‚âÝ 0}` is a nonempty subset of ‚Ñï ‚Äî nonempty by `Pos(t)`'s existential, a subset of ‚Ñï because its elements are indices in the range `1 ‚â§ i ‚â§ #t` ‚Äî so NAT-wellorder's well-ordering of ‚Ñï furnishes a smallest element `k` with `1 ‚â§ k ‚â§ #t` and `t‚Çñ ‚âÝ 0`; for all `i < k` we have `t·µ¢ = 0`, by minimality of `k`. Since `z` is a zero tumbler, `z·µ¢ = 0` for all `i ‚â§ #z`. We distinguish two cases by the relationship between `#z` and `k`.

*Case `#z ‚â• k`.* For `1 ‚â§ i < k` we have `z·µ¢ = 0 = t·µ¢`, establishing the T1 agreement condition. The case hypothesis `#z ‚â• k` supplies `k ‚â§ #z`, and the construction of `k` supplies `k ‚â§ #t`, establishing the pair `k ‚â§ #z ‚àß k ‚â§ #t` required by T1 case (i). We now establish `z‚Çñ = 0 < t‚Çñ`: NAT-zero furnishes `0 ‚â§ t‚Çñ` since `t‚Çñ ‚àà ‚Ñï`, which NAT-order's definition `m ‚â§ n ‚ü∫ m < n ‚à® m = n` instantiated at `m = 0, n = t‚Çñ` unfolds to `0 < t‚Çñ ‚à® 0 = t‚Çñ`; the construction of `k` supplies `t‚Çñ ‚âÝ 0`, excluding the equality disjunct and leaving `0 < t‚Çñ`. By T1 case (i) with witness `k`, `z < t`.

*Case `#z < k`.* For `1 ‚â§ i ‚â§ #z` we have `i < k` (since `i ‚â§ #z < k`), whence `t·µ¢ = 0 = z·µ¢`, establishing the T1 agreement condition at every position of `z`. From `#z < k ‚â§ #t` we obtain `#z + 1 ‚â§ #t`, so by T1 case (ii) with witness `#z + 1`, `z < t`. ‚àé

*Formal Contract:*
- *Definition:* `Pos(t)` (positive) iff `(E i : 1 ‚â§ i ‚â§ #t : t·µ¢ ‚âÝ 0)`. Zero tumbler: `(A i : 1 ‚â§ i ‚â§ #t : t·µ¢ = 0)`.
- *Depends:* T0 (CarrierSetDefinition) ‚Äî the carrier set `T`, the length `#t`, and the component projection `t·µ¢` used in the Definition (`Pos(t)` iff `(E i : 1 ‚â§ i ‚â§ #t : t·µ¢ ‚âÝ 0)`) and in the zero-tumbler companion definition (`(A i : 1 ‚â§ i ‚â§ #t : t·µ¢ = 0)`) all come from T0's characterisation of T as finite sequences over ‚Ñï with length ‚â• 1. NAT-wellorder (NatWellOrdering) ‚Äî the postcondition proof's step that `Pos(t)` furnishes a smallest index `k` with `t‚Çñ ‚âÝ 0` invokes the well-ordering of ‚Ñï applied to the nonempty subset `{i : 1 ‚â§ i ‚â§ #t ‚àß t·µ¢ ‚âÝ 0} ‚äÜ ‚Ñï`, so the least-element claim is discharged from NAT-wellorder rather than left implicit. NAT-zero (NatZeroMinimum) ‚Äî the postcondition proof's Case `#z ‚â• k` step `z‚Çñ = 0 < t‚Çñ` requires `0 ‚â§ t‚Çñ` as a premise so that NAT-order's definition `m ‚â§ n ‚ü∫ m < n ‚à® m = n` at `m = 0, n = t‚Çñ` can unfold it to `0 < t‚Çñ ‚à® 0 = t‚Çñ`; no other listed NAT-* axiom supplies this lower bound: NAT-order's trichotomy compares `0` and `t‚Çñ` without fixing the winner, and NAT-wellorder's least-element principle gives ‚Ñï a minimum without naming it `0`. NAT-zero's axiom `(A n ‚àà ‚Ñï :: 0 ‚â§ n)` discharges exactly this lower-bound premise, after which the construction of `k`'s `t‚Çñ ‚âÝ 0` excludes the equality disjunct and leaves `0 < t‚Çñ`. NAT-discrete (NatDiscreteness) ‚Äî the postcondition proof's Case `#z < k` step deriving `#z + 1 ‚â§ #t` from `#z < k ‚â§ #t` is licensed by NAT-discrete in its contrapositive form ‚Äî for `m, n ‚àà ‚Ñï`, `m < n ‚üπ m + 1 ‚â§ n`, since the contrary assumption `¬¨(m + 1 ‚â§ n)` gives `n < m + 1` by NAT-order's trichotomy, combined with `m < n` yields `m ‚â§ n < m + 1`, whence NAT-discrete forces `n = m`, contradicting `m < n` by NAT-order's irreflexivity ‚Äî instantiated at `m = #z, n = k` and then chained with `k ‚â§ #t` by NAT-order's transitivity of `‚â§` this yields the T1 case (ii) witness `#z + 1 ‚â§ #t`, so the `m < n ‚üπ m + 1 ‚â§ n` inference is discharged from NAT-discrete (with NAT-order) rather than left implicit. NAT-order (NatStrictTotalOrder) ‚Äî supplies (i) the definition of `‚â§` used in Case `#z ‚â• k` to unfold `0 ‚â§ t‚Çñ` into `0 < t‚Çñ ‚à® 0 = t‚Çñ`, whose equality disjunct the construction of `k`'s `t‚Çñ ‚âÝ 0` then excludes, leaving `0 < t‚Çñ` directly ‚Äî the step that replaces the former `t‚Çñ ‚â• 1` detour through NAT-discrete; and (ii) the trichotomy, irreflexivity, and `‚â§`-transitivity steps composed with NAT-discrete in the Case `#z < k` contrapositive argument above. T1 (LexicographicOrder) ‚Äî the postcondition proof invokes T1 case (i) when `#z ‚â• k` to conclude `z < t` from `z‚Çñ = 0 < t‚Çñ`, and T1 case (ii) when `#z < k` to conclude `z < t` from `z` being a proper prefix of `t`.
- *Postconditions:* `(A t ‚àà T, z ‚àà T : Pos(t) ‚àß (A i : 1 ‚â§ i ‚â§ #z : z·µ¢ = 0) :: z < t)` ‚Äî every positive tumbler is strictly greater under T1 than every zero tumbler of any length.
