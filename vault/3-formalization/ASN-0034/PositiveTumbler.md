## Zero tumblers and positivity

Under T3, the tumblers `[0]`, `[0, 0]`, `[0, 0, 0]`, etc., are *distinct* elements of T — they have different lengths. Under T1, they form a chain: `[0] < [0, 0] < [0, 0, 0] < ...` by the prefix rule. There is no single "zero tumbler"; there are infinitely many all-zero tumblers. (This distinctness remark is motivational commentary, not a load-bearing claim: the postcondition's universal quantification over zero tumblers `z ∈ T` would hold whether length-varying zero sequences denote distinct elements of T or collapse to a single representative — if two `z`'s coincided as elements of T, the claim `z < t` would still need to be discharged pointwise, and the proof below would still discharge it. T3 is therefore referenced here for motivation only and is not cited in the formal Depends below.)

**TA-Pos (PositiveTumbler).** A tumbler `t ∈ T` is *positive*, written `Pos(t)`, iff at least one of its components is nonzero: `(E i : 1 ≤ i ≤ #t : tᵢ ≠ 0)`. A tumbler is a *zero tumbler* iff every component is zero: `(A i : 1 ≤ i ≤ #t : tᵢ = 0)`.

The predicate `Pos(t)` is not written `t > 0`, because `>` already denotes T1's lexicographic ordering, and the two notions diverge. The all-zero tumbler `[0, 0]` exhibits the divergence concretely: under T1, `[0] < [0, 0]` by case (ii) — the prefix rule, since `[0]` is a proper prefix of `[0, 0]` — yet `¬Pos([0, 0])`, since every component of `[0, 0]` is zero. The converse direction does hold: `Pos(t)` implies `t` is T1-greater than every zero tumbler, as the following proof establishes.

Every positive tumbler is greater than every zero tumbler under T1.

*Proof.* Let `t ∈ T` with `Pos(t)` and let `z ∈ T` be a zero tumbler; we show `z < t`. Since `Pos(t)`, the set `{i : 1 ≤ i ≤ #t ∧ tᵢ ≠ 0}` is a nonempty subset of ℕ — nonempty by `Pos(t)`'s existential, a subset of ℕ because its elements are indices in the range `1 ≤ i ≤ #t` — so NAT-wellorder's well-ordering of ℕ furnishes a smallest element `k` with `1 ≤ k ≤ #t` and `tₖ ≠ 0`; for all `i < k` we have `tᵢ = 0`, by minimality of `k`. Since `z` is a zero tumbler, `zᵢ = 0` for all `i ≤ #z`. We distinguish two cases by the relationship between `#z` and `k`.

*Case `#z ≥ k`.* For `1 ≤ i < k` we have `zᵢ = 0 = tᵢ`, establishing the T1 agreement condition. The case hypothesis `#z ≥ k` supplies `k ≤ #z`, and the construction of `k` supplies `k ≤ #t`, establishing the pair `k ≤ #z ∧ k ≤ #t` required by T1 case (i). We now establish `zₖ = 0 < tₖ`: NAT-zero furnishes `0 ≤ tₖ` since `tₖ ∈ ℕ`, which NAT-order's definition `m ≤ n ⟺ m < n ∨ m = n` instantiated at `m = 0, n = tₖ` unfolds to `0 < tₖ ∨ 0 = tₖ`; the construction of `k` supplies `tₖ ≠ 0`, excluding the equality disjunct and leaving `0 < tₖ`. By T1 case (i) with witness `k`, `z < t`.

*Case `#z < k`.* For `1 ≤ i ≤ #z` we have `i < k` (since `i ≤ #z < k`), whence `tᵢ = 0 = zᵢ`, establishing the T1 agreement condition at every position of `z`. From `#z < k ≤ #t` we obtain `#z + 1 ≤ #t`, so by T1 case (ii) with witness `#z + 1`, `z < t`. ∎

*Formal Contract:*
- *Definition:* `Pos(t)` (positive) iff `(E i : 1 ≤ i ≤ #t : tᵢ ≠ 0)`. Zero tumbler: `(A i : 1 ≤ i ≤ #t : tᵢ = 0)`.
- *Depends:* T0 (CarrierSetDefinition) — the carrier set `T`, the length `#t`, and the component projection `tᵢ` used in the Definition (`Pos(t)` iff `(E i : 1 ≤ i ≤ #t : tᵢ ≠ 0)`) and in the zero-tumbler companion definition (`(A i : 1 ≤ i ≤ #t : tᵢ = 0)`) all come from T0's characterisation of T as finite sequences over ℕ with length ≥ 1. NAT-wellorder (NatWellOrdering) — the postcondition proof's step that `Pos(t)` furnishes a smallest index `k` with `tₖ ≠ 0` invokes the well-ordering of ℕ applied to the nonempty subset `{i : 1 ≤ i ≤ #t ∧ tᵢ ≠ 0} ⊆ ℕ`, so the least-element claim is discharged from NAT-wellorder rather than left implicit. NAT-zero (NatZeroMinimum) — the postcondition proof's Case `#z ≥ k` step `zₖ = 0 < tₖ` requires `0 ≤ tₖ` as a premise so that NAT-order's definition `m ≤ n ⟺ m < n ∨ m = n` at `m = 0, n = tₖ` can unfold it to `0 < tₖ ∨ 0 = tₖ`; no other listed NAT-* axiom supplies this lower bound: NAT-order's trichotomy compares `0` and `tₖ` without fixing the winner, and NAT-wellorder's least-element principle gives ℕ a minimum without naming it `0`. NAT-zero's axiom `(A n ∈ ℕ :: 0 ≤ n)` discharges exactly this lower-bound premise, after which the construction of `k`'s `tₖ ≠ 0` excludes the equality disjunct and leaves `0 < tₖ`. NAT-discrete (NatDiscreteness) — the postcondition proof's Case `#z < k` step deriving `#z + 1 ≤ #t` from `#z < k ≤ #t` is licensed by NAT-discrete in its contrapositive form — for `m, n ∈ ℕ`, `m < n ⟹ m + 1 ≤ n`, since the contrary assumption `¬(m + 1 ≤ n)` gives `n < m + 1` by NAT-order's trichotomy, combined with `m < n` yields `m ≤ n < m + 1`, whence NAT-discrete forces `n = m`, contradicting `m < n` by NAT-order's irreflexivity — instantiated at `m = #z, n = k` and then chained with `k ≤ #t` by NAT-order's transitivity of `≤` this yields the T1 case (ii) witness `#z + 1 ≤ #t`, so the `m < n ⟹ m + 1 ≤ n` inference is discharged from NAT-discrete (with NAT-order) rather than left implicit. NAT-order (NatStrictTotalOrder) — supplies (i) the definition of `≤` used in Case `#z ≥ k` to unfold `0 ≤ tₖ` into `0 < tₖ ∨ 0 = tₖ`, whose equality disjunct the construction of `k`'s `tₖ ≠ 0` then excludes, leaving `0 < tₖ` directly — the step that replaces the former `tₖ ≥ 1` detour through NAT-discrete; and (ii) the trichotomy, irreflexivity, and `≤`-transitivity steps composed with NAT-discrete in the Case `#z < k` contrapositive argument above. T1 (LexicographicOrder) — the postcondition proof invokes T1 case (i) when `#z ≥ k` to conclude `z < t` from `zₖ = 0 < tₖ`, and T1 case (ii) when `#z < k` to conclude `z < t` from `z` being a proper prefix of `t`.
- *Postconditions:* `(A t ∈ T, z ∈ T : Pos(t) ∧ (A i : 1 ≤ i ≤ #z : zᵢ = 0) :: z < t)` — every positive tumbler is strictly greater under T1 than every zero tumbler of any length.
