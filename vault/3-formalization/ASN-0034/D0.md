### Displacement identities

Given two positions a and b on the tumbler line, a natural question is whether b ‚äñ a yields a displacement w such that a ‚äï w faithfully recovers b. We establish the well-definedness condition for such displacement recovery and the round-trip identity that guarantees faithfulness.

From TumblerAdd, a ‚äï w acts at the action point k of w: it copies a‚ÇÅ..a‚Çñ‚Çã‚ÇÅ, advances a‚Çñ by w‚Çñ, and replaces the tail with w's tail. So if a ‚äï w = b, then a and b agree on components 1..k‚àí1 and diverge at k, with b‚Çñ = a‚Çñ + w‚Çñ and b·µ¢ = w·µ¢ for i > k. Reading off the width:

  w·µ¢ = 0  for i < k,    w‚Çñ = b‚Çñ ‚àí a‚Çñ,    w·µ¢ = b·µ¢  for i > k

where k = divergence(a, b). The component formulas match TumblerSub's definition of b ‚äñ a, but the lengths may differ: the reading-off produces a tumbler of length #b = #w (by TumblerAdd's result-length identity), whereas TumblerSub yields #(b ‚äñ a) = max(#a, #b). When #a ‚â§ #b the lengths coincide and the identification is exact. When #a > #b, TumblerSub extends the result with trailing zeros to length #a, and by T3 (CanonicalRepresentation) the extended tumbler is distinct from the original w. We write w = b ‚äñ a ‚Äî an exact identity when #a ‚â§ #b ‚Äî and call it the *displacement from a to b*. The displacement is well-defined when:

**D0 (DisplacementWellDefined).** a < b, and the divergence k of a and b satisfies k ‚â§ #a.

*Proof.* We show that under the hypotheses `a, b ‚àà T`, `a < b`, and `divergence(a, b) ‚â§ #a`, the displacement `w = b ‚äñ a` is a well-defined positive tumbler whose action point equals `divergence(a, b)`, and the addition `a ‚äï w` is well-defined. We also identify the boundary condition for round-trip faithfulness.

Let `k = divergence(a, b)`. We show that D0's hypotheses eliminate Divergence case (ii). That case requires `#a ‚âÝ #b` with all shared components agreeing, giving `k = min(#a, #b) + 1`. Two sub-cases arise. If `#a < #b`, then `min(#a, #b) = #a`, so `k = #a + 1 > #a`, contradicting `k ‚â§ #a`. If `#b < #a`, then `min(#a, #b) = #b` and `a·µ¢ = b·µ¢` for all `1 ‚â§ i ‚â§ #b`, so `b` is a proper prefix of `a`; by T1 case (ii), `b < a`, contradicting the hypothesis `a < b`. Both sub-cases yield contradictions, so Divergence case (ii) is impossible. We are therefore in case (i): `k ‚â§ min(#a, #b)`, with `a·µ¢ = b·µ¢` for all `i < k` and `a‚Çñ ‚âÝ b‚Çñ`. Since `a < b`, T1 case (i) gives the direction: `a‚Çñ < b‚Çñ`.

When `a` is a proper prefix of `b`, the Divergence definition gives case (ii) with `k = #a + 1 > #a`, violating D0's hypothesis. No displacement exists for prefix-related pairs ‚Äî the subtraction is defined but the round-trip addition is not, because the action point would exceed `#a`.

**Well-definedness of the subtraction.** Since `a < b` entails `b ‚â• a`, the subtraction `w = b ‚äñ a` is a well-defined tumbler in T by TA2. We now compute `w` explicitly. By TumblerSub, zero-pad both operands to length `max(#b, #a)` and scan for the first position at which the padded sequences disagree. Since we are in Divergence case (i), `k ‚â§ min(#a, #b)`, so both `a` and `b` possess actual (non-padded) components at every position `1, ..., k`. The zero-padded extensions therefore agree with the unpadded sequences through position `k ‚àí 1` and disagree at `k`, whence `zpd(b, a) = divergence(a, b) = k` (ZPD, Relationship to Divergence, case (i)). TumblerSub, parameterized by `zpd(b, a) = k`, yields:

  `w·µ¢ = 0` for `i < k`, `w‚Çñ = b‚Çñ ‚àí a‚Çñ`, `w·µ¢ = b·µ¢` for `i > k`

with `#w = max(#b, #a)`. The component `w‚Çñ = b‚Çñ ‚àí a‚Çñ` is well-defined and non-negative because `b‚Çñ > a‚Çñ` (established above).

**Positivity.** The displacement `w` is positive: `w‚Çñ = b‚Çñ ‚àí a‚Çñ ‚â• 1` since `a‚Çñ < b‚Çñ` and both are natural numbers. All components before position `k` are zero, so `w` is not the zero tumbler.

**Action point.** The action point of `w` is `k`: every component `w·µ¢ = 0` for `i < k`, and `w‚Çñ > 0`, so `k` is the first positive component of `w`.

**Well-definedness of the addition.** TA0 requires `Pos(w)` (established) and `actionPoint(w) ‚â§ #a`. The action point is `k`, and the hypothesis gives `k ‚â§ #a`, so TA0 is satisfied. The addition `a ‚äï w` is a well-defined tumbler in T.

**Round-trip boundary.** The displacement has length `#w = max(#a, #b)`. By the result-length identity (TumblerAdd), `#(a ‚äï w) = #w`. When `#a > #b`, this gives `#(a ‚äï w) = #a > #b`, so `a ‚äï w ‚âÝ b` by T3 (CanonicalRepresentation) ‚Äî the round-trip fails on length alone. Round-trip faithfulness requires the additional condition `#a ‚â§ #b`, under which `#w = #b` and the component-by-component recovery succeeds (D1). ‚àé

*Formal Contract:*
- *Preconditions:* a ‚àà T, b ‚àà T, a < b, divergence(a, b) ‚â§ #a
- *Postconditions:* b ‚äñ a ‚àà T, Pos(b ‚äñ a) (PositiveTumbler), actionPoint(b ‚äñ a) = divergence(a, b), #(b ‚äñ a) = max(#a, #b), a ‚äï (b ‚äñ a) ‚àà T, #a > #b ‚Üí a ‚äï (b ‚äñ a) ‚âÝ b
