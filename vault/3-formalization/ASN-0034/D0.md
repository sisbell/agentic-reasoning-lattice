### Displacement identities

Given two positions a and b on the tumbler line, a natural question is whether b ⊖ a yields a displacement w such that a ⊕ w faithfully recovers b. We establish the well-definedness condition for such displacement recovery and the round-trip identity that guarantees faithfulness.

From TumblerAdd, a ⊕ w acts at the action point k of w: it copies a₁..aₖ₋₁, advances aₖ by wₖ, and replaces the tail with w's tail. So if a ⊕ w = b, then a and b agree on components 1..k−1 and diverge at k, with bₖ = aₖ + wₖ and bᵢ = wᵢ for i > k. Reading off the width (the step `wₖ = bₖ − aₖ` is licensed by NAT-sub, whose conditional-closure clause requires `bₖ ≥ aₖ`, supplied here by `bₖ = aₖ + wₖ ≥ aₖ` via NAT-addcompat with NAT-closure's additive identity):

  wᵢ = 0  for i < k,    wₖ = bₖ − aₖ,    wᵢ = bᵢ  for i > k

where k = divergence(a, b). The component formulas match TumblerSub's definition of b ⊖ a, but the lengths may differ: the reading-off produces a tumbler of length #b = #w (by TumblerAdd's result-length identity), whereas TumblerSub yields `#(b ⊖ a) = L` with `L` named by the length-pair dispatch at `(#b, #a)` — `L = #b` under sub-cases (α) `#b = #a` and (γ) `#a < #b`, `L = #a` under sub-case (β) `#b < #a`, rather than via any primitive binary-maximum on ℕ. When `#a ≤ #b` the dispatch selects sub-case (α) or (γ), giving `L = #b`, so the lengths coincide and the identification is exact. When `#a > #b` (equivalently `#b < #a`) the dispatch selects sub-case (β), giving `L = #a`, so TumblerSub extends the result with trailing zeros to length `#a`, and by T3 (CanonicalRepresentation) the extended tumbler is distinct from the original w. We write w = b ⊖ a — an exact identity when #a ≤ #b — and call it the *displacement from a to b*. The displacement is well-defined when:

**D0 (DisplacementWellDefined).** a < b, and the divergence k of a and b satisfies k ≤ #a.

*Proof.* We show that under the hypotheses `a, b ∈ T`, `a < b`, and `divergence(a, b) ≤ #a`, the displacement `w = b ⊖ a` is a well-defined positive tumbler whose action point equals `divergence(a, b)`, and the addition `a ⊕ w` is well-defined. We also identify the boundary condition for round-trip faithfulness.

Let `k = divergence(a, b)`. We show that D0's hypotheses eliminate Divergence case (ii). That case requires `#a ≠ #b` with all shared components agreeing, and NAT-order's trichotomy at `(#a, #b)` — consumed inside Divergence to rule out the `#a = #b` branch — leaves exactly one of two sub-cases as Divergence states them. Sub-case (ii-a) has `#a < #b` with `divergence(a, b) = #a + 1`, whence `k = #a + 1`; substituting into the precondition `k ≤ #a` yields `#a + 1 ≤ #a`, which against NAT-addcompat's strict successor inequality `#a < #a + 1` is refuted by NAT-order's trichotomy at `(#a, #a + 1)` — its exactly-one clause excluding both `#a + 1 < #a` and `#a + 1 = #a` once `#a < #a + 1` is fixed. Sub-case (ii-b) has `#b < #a` with `divergence(a, b) = #b + 1` and `aᵢ = bᵢ` for all `1 ≤ i ≤ #b`, so `b` is a proper prefix of `a`; by T1 case (ii), `b < a`, contradicting the hypothesis `a < b`. Both sub-cases yield contradictions, so Divergence case (ii) is impossible. We are therefore in case (i): `k ≤ #a ∧ k ≤ #b` (the shared-bound conjunction Divergence states in place of any primitive binary-minimum on ℕ), with `aᵢ = bᵢ` for all `i < k` and `aₖ ≠ bₖ`. Since `a < b`, T1 case (i) gives the direction: `aₖ < bₖ`.

When `a` is a proper prefix of `b`, the Divergence definition gives case (ii) sub-case (ii-a) with `k = #a + 1`; NAT-addcompat's strict successor inequality `#a < #a + 1` then yields `k > #a`, violating D0's hypothesis. No displacement exists for prefix-related pairs — the subtraction is defined but the round-trip addition is not, because the action point would exceed `#a`.

**Well-definedness of the subtraction.** Since `a < b` entails `b ≥ a`, the subtraction `w = b ⊖ a` is a well-defined tumbler in T by TA2. Its length is named by TumblerSub's length-pair dispatch at `(#b, #a)` — NAT-order's trichotomy selecting sub-case (α) `#b = #a` with `L = #b`, sub-case (β) `#b < #a` with `L = #a`, or sub-case (γ) `#a < #b` with `L = #b`, rather than via any primitive binary-maximum on ℕ — so `#w = L`. We now compute `w` explicitly. By TumblerSub, zero-pad both operands to the common length `L` and scan for the first position at which the padded sequences disagree. Since we are in Divergence case (i), `k ≤ #a ∧ k ≤ #b` (the shared-bound conjunction Divergence states in place of any primitive binary-minimum on ℕ), so both `a` and `b` possess actual (non-padded) components at every position `1, ..., k`. The zero-padded extensions therefore agree with the unpadded sequences through position `k − 1` and disagree at `k`, whence `zpd(b, a) = divergence(b, a)` (ZPD, Relationship to Divergence, case (i)); by Divergence symmetry, `divergence(b, a) = divergence(a, b) = k`. TumblerSub, parameterized by `zpd(b, a) = k`, yields:

  `wᵢ = 0` for `i < k`, `wₖ = bₖ − aₖ`, `wᵢ = bᵢ` for `i > k`

with `#w = L` as named above by the length-pair dispatch. The component `wₖ = bₖ − aₖ ∈ ℕ` is well-defined by NAT-sub (conditional closure) — its precondition `bₖ ≥ aₖ` is supplied by the strict inequality `bₖ > aₖ` established above combined with NAT-order's definition of `≤` from `<` — and NAT-sub's strict-positivity clause additionally yields `wₖ ≥ 1`.

**Positivity.** Since `zpd(b, a)` is defined (established above), TumblerSub's conditional postcondition gives `Pos(w)`.

**Action point.** TumblerSub's conditional postcondition further gives `actionPoint(w) = zpd(b, a)`. Since `zpd(b, a) = divergence(a, b) = k` (established above), `actionPoint(w) = k`.

**Well-definedness of the addition.** TA0 requires `Pos(w)` (established) and `actionPoint(w) ≤ #a`. The action point is `k`, and the hypothesis gives `k ≤ #a`, so TA0 is satisfied. The addition `a ⊕ w` is a well-defined tumbler in T.

**Round-trip boundary.** The displacement's length `#w = L` is named by TumblerSub's length-pair dispatch at `(#b, #a)`: `L = #b` under sub-cases (α) `#b = #a` and (γ) `#a < #b`, `L = #a` under sub-case (β) `#b < #a`. By the result-length identity (TumblerAdd), `#(a ⊕ w) = #w = L`. When `#a > #b` (equivalently `#b < #a`, placing the dispatch in sub-case (β)), `L = #a`, so `#(a ⊕ w) = #a > #b`, whence `a ⊕ w ≠ b` by T3 (CanonicalRepresentation) — the round-trip fails on length alone. Round-trip faithfulness requires the additional condition `#a ≤ #b`, under which the dispatch selects sub-case (α) or (γ), giving `L = #b` and `#w = #b`, and the component-by-component recovery succeeds (D1). ∎
