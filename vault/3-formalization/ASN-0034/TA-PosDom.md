**TA-PosDom (PositiveDominatesZero).** `(A t ‚àà T, z ‚àà T : Pos(t) ‚àß Zero(z) :: z < t)` ‚Äî every positive tumbler is strictly greater under T1 than every zero tumbler of any length.

Under T3, the tumblers `[0]`, `[0, 0]`, `[0, 0, 0]`, etc., are *distinct* elements of T ‚Äî they have different lengths. Under T1, they form a chain: `[0] < [0, 0] < [0, 0, 0] < ...` by the prefix rule. There is no single zero tumbler; there are infinitely many `z` with `Zero(z)`. (This distinctness remark is motivational commentary, not a load-bearing claim: the postcondition's universal quantification over `z ‚àà T` with `Zero(z)` would hold whether length-varying zero sequences denote distinct elements of T or collapse to a single representative ‚Äî if two `z`'s coincided as elements of T, the claim `z < t` would still need to be discharged pointwise, and the proof below would still discharge it. T3 is therefore referenced here for motivation only and is not cited in the formal Depends below.)

*Proof.* Let `t ‚àà T` with `Pos(t)` ‚Äî where `Pos(t)` is TA-Pos's existential `(E i : 1 ‚â§ i ‚â§ #t : t·µ¢ ‚âÝ 0)` ‚Äî and let `z ‚àà T` with `Zero(z)`; we show `z < t`. Since `Pos(t)`, ActionPoint applies to `t` ‚Äî its sole precondition is `Pos(w)`, which `t` satisfies ‚Äî and supplies the index `actionPoint(t)` together with three postconditions: (i) `1 ‚â§ actionPoint(t) ‚â§ #t`; (ii) `t·µ¢ = 0` for all `1 ‚â§ i < actionPoint(t)`; (iii) `t_{actionPoint(t)} ‚â• 1`. Earlier drafts of this proof reconstructed these three facts from a direct NAT-wellorder appeal to the set `{i : 1 ‚â§ i ‚â§ #t ‚àß t·µ¢ ‚âÝ 0}` ‚Äî least element via well-ordering, minimality-yields-zeros-below corollary, and nonzero-at-the-least-index ‚Äî but that reconstruction duplicates ActionPoint's construction verbatim (same set, same axiom, same minimality-yields-zeros-below corollary), so the proof below instead consumes `actionPoint(t)` as the T1 divergence witness and delegates the construction to ActionPoint, paralleling TA-strict's delegation of its ordering postcondition to TumblerAdd rather than quoting TumblerAdd's four-postcondition list. Write `k` for `actionPoint(t)`. Unpacking `Zero(z)` via its TA-Pos defining clause `(A i : 1 ‚â§ i ‚â§ #z : z·µ¢ = 0)` gives `z·µ¢ = 0` for all `i` with `1 ‚â§ i ‚â§ #z` ‚Äî the per-index component facts the cases below consume. We distinguish two cases by the relationship between `#z` and `k`.

*Case `#z ‚â• k`.* For `1 ‚â§ i < k` we have `z·µ¢ = 0` (unpacked from `Zero(z)` above; the hypothesis `i < k ‚â§ #z` places `i` within `Zero(z)`'s index range `1 ‚â§ i ‚â§ #z`) and `t·µ¢ = 0` (ActionPoint's postcondition (ii)), so `z·µ¢ = t·µ¢`, establishing the T1 agreement condition. The case hypothesis `#z ‚â• k` supplies `k ‚â§ #z`, and ActionPoint's postcondition (i) supplies `k ‚â§ #t`, establishing the pair `k ‚â§ #z ‚àß k ‚â§ #t` required by T1 case (i). We now establish `z‚Çñ = 0 < t‚Çñ`: `z‚Çñ = 0` by unpacking `Zero(z)` at `i = k` (the hypothesis `k ‚â§ #z` placing `k` within `Zero(z)`'s index range), and ActionPoint's postcondition (iii) supplies `1 ‚â§ t‚Çñ`. NAT-addcompat's strict successor inequality `n < n + 1` instantiated at `n = 0` supplies `0 < 0 + 1`; NAT-closure's additive identity `0 + n = n` instantiated at `n = 1` rewrites the right-hand side `0 + 1` to `1`, yielding `0 < 1` ‚Äî the same per-step rewrite TumblerAdd's dominance sub-case `a‚Çñ > 0` applies when NAT-closure's additive identity `0 + w‚Çñ = w‚Çñ` rewrites the right-hand side of `a‚Çñ + w‚Çñ ‚â• 0 + w‚Çñ` to `w‚Çñ`, rather than elided as a syntactic identity. NAT-order's definition `m ‚â§ n ‚ü∫ m < n ‚à® m = n` instantiated at `m = 1, n = t‚Çñ` unfolds `1 ‚â§ t‚Çñ` to `1 < t‚Çñ ‚à® 1 = t‚Çñ`, and the two disjuncts compose with `0 < 1` to deliver `0 < t‚Çñ` ‚Äî strict disjunct via NAT-order's transitivity of `<` on ‚Ñï composing `0 < 1 < t‚Çñ` into `0 < t‚Çñ`, equality disjunct via substitution of `1 = t‚Çñ` into `0 < 1`. By T1 case (i) with witness `k`, `z < t`.

*Case `#z < k`.* For `1 ‚â§ i ‚â§ #z` we have `i < k` (since `i ‚â§ #z < k`), whence `t·µ¢ = 0` (ActionPoint's postcondition (ii)) and `z·µ¢ = 0` (unpacked from `Zero(z)` at index `i`, which lies in `Zero(z)`'s index range `1 ‚â§ i ‚â§ #z` by hypothesis), so `t·µ¢ = 0 = z·µ¢`, establishing the T1 agreement condition at every position of `z`. From `#z < k` and ActionPoint's postcondition (i)'s `k ‚â§ #t` ‚Äî composed via NAT-order's `<`/`‚â§` transitivity (the strict disjunct `k < #t` of the unpacked `k ‚â§ #t` composes with `#z < k` by NAT-order's transitivity of `<` to yield `#z < #t`; the equality disjunct `k = #t` substitutes into `#z < k` to yield `#z < #t` directly) ‚Äî we obtain `#z < #t`. NAT-discrete's axiom `m ‚â§ n < m + 1 ‚üπ n = m` in its contrapositive form ‚Äî for `m, n ‚àà ‚Ñï`, `m < n ‚üπ m + 1 ‚â§ n`, since the contrary assumption `¬¨(m + 1 ‚â§ n)` gives `n < m + 1` by NAT-order's trichotomy, combined with `m < n` yields `m ‚â§ n < m + 1`, whence NAT-discrete forces `n = m`, contradicting `m < n` by NAT-order's irreflexivity ‚Äî instantiated at `m = #z, n = #t` yields `#z + 1 ‚â§ #t`. By T1 case (ii) with witness `#z + 1`, `z < t`. ‚àé
