## Tumbler subtraction

**TumblerSub (TumblerSub).** Given two tumblers `a` (minuend) and `w` (subtrahend), compute their component-wise difference at the first point of zero-padded divergence. The primary downstream use is computing displacements between positions — given positions `a` and `b` with `b ≥ a`, the subtraction `b ⊖ a` yields the component-wise difference from `a` to `b`. This difference is a broader object than a displacement consumable by TumblerAdd: when `a` is a proper sequence-prefix of `b`, the action point of `b ⊖ a` satisfies `actionPoint(b ⊖ a) = zpd(b, a) ≥ #a + 1 > #a`, violating TumblerAdd's precondition `actionPoint(w) ≤ #a`. A second obstruction is independent of the prefix case: when `#a > #b`, the Definition's NAT-order trichotomy dispatch (below) — applied to the length pair `(#b, #a)` of the operand pair of `b ⊖ a` — places the pair in its `#b < #a` sub-case, whose longer length `L = #a` furnishes the result length, so the subtraction produces `#(b ⊖ a) = #a`; TumblerAdd's result-length identity gives `#(a ⊕ (b ⊖ a)) = #a > #b`, so `a ⊕ (b ⊖ a) ≠ b` by T3 — the round-trip fails on length alone even when the zpd constraint is satisfied. The roundtrip `a ⊕ (b ⊖ a) = b` therefore requires two independently necessary constraints: `divergence(a, b) ≤ #a` (ensuring TumblerAdd's precondition) and `#a ≤ #b` (ensuring the result length equals `#b`). D0 (DisplacementWellDefined) takes the first as a precondition and proves the necessity of the second via its negative postcondition (`#a > #b → a ⊕ (b ⊖ a) ≠ b`); the positive round-trip identity under the full conjunction is established by D1 (DisplacementRoundTrip). When the operands have different lengths, we conceptually zero-pad the shorter to the length of the longer — writing `aᵢ = 0` for `i > #a` and `wᵢ = 0` for `i > #w` — and use these padded values throughout the procedure. Reproducing ZPD's length-pair dispatch for the same operand pair, NAT-order's trichotomy on `(#a, #w)` selects exactly one of the sub-cases (α) `#a = #w` with `L = #a` (equivalently `#w`), (β) `#a < #w` with `L = #w`, or (γ) `#w < #a` with `L = #a`; `L` — the longer of `#a` and `#w` — is furnished by the selected sub-case rather than by any primitive binary-maximum operator on ℕ, matching the convention that every operator in the Definition must be discharged by a Depends entry. When the zero-padded sequences agree at every position (no divergence exists), the result is the zero tumbler of length `L`: `a ⊖ w = [0, ..., 0]`. Otherwise, `zpd(a, w)` is defined (ZPD) — write `k = zpd(a, w)` for the first position at which the padded sequences disagree. The result is:

```
         ⎧ 0             if i < k        (these levels matched — zero them)
rᵢ   =  ⎨ aₖ - wₖ      if i = k        (subtract at the divergence point)
         ⎩ aᵢ           if i > k        (copy from minuend, zero-padded)
```

The result has length `L`.

**Precondition:** `a ≥ w` (T1). We prove that when `zpd(a, w)` is defined, this entails `aₖ > wₖ` at `k = zpd(a, w)`. Since zpd is defined, `a` and `w` are not zero-padded-equal (ZPD), so in particular `a ≠ w`; combined with `a ≥ w`, this yields `w < a` (T1). Two Divergence cases arise for the pair `(w, a)` with `w ≠ a`:

  (i) Component divergence at position `k` with `k ≤ #w ∧ k ≤ #a` (the conjunction replacing `k ≤ min(#w, #a)` so that no primitive binary-minimum operator on ℕ is invoked, matching Divergence's case (i)) and `wₖ ≠ aₖ`. The ZPD–Divergence relationship (ZPD) gives `zpd(a, w) = divergence(a, w) = k`. Since `w < a` via T1 case (i), `wₖ < aₖ`, whence `aₖ > wₖ`.

  (ii) Prefix divergence — `w` is a proper prefix of `a` (the direction forced by `w < a` via T1 case (ii)), so `#w < #a` and `wᵢ = aᵢ` for all `1 ≤ i ≤ #w`. The padded extension sets `wₖ = 0` for `k > #w`. Since zpd is defined, the longer operand `a` has some nonzero component beyond `#w`; at `k = zpd(a, w)`, `aₖ ≠ 0 = wₖ`, so `aₖ > 0 = wₖ`.

In both cases `aₖ > wₖ` at `k = zpd(a, w)`. When zpd is undefined, the precondition `a ≥ w` continues to hold by hypothesis; the consequence `aₖ > wₖ` is vacuous because there is no `k = zpd(a, w)` at which to evaluate it, and the constructive definition produces the zero tumbler with no subtraction at a divergence point.  ∎

Each component of the result is a natural number: for `i < k`, `rᵢ = 0 ∈ ℕ`; at the divergence point, `rₖ = aₖ − wₖ ∈ ℕ` by NAT-sub (conditional closure), whose precondition `aₖ ≥ wₖ` is discharged by the precondition consequence `aₖ > wₖ` at the zpd point (proved above) combined with NAT-order; for `i > k`, `rᵢ` is the zero-padded value of `a`, which is either `aᵢ ∈ ℕ` (when `i ≤ #a`) or `0 ∈ ℕ` (when `i > #a`). In the equal case (no divergence), every component is `0 ∈ ℕ`. The result is therefore a finite sequence over ℕ with length `L ≥ 1` — since `a, w ∈ T` requires both `#a ≥ 1` and `#w ≥ 1`, and `L` is named by the trichotomy dispatch as one of `#a` or `#w`, each of which is at least 1 — hence **`a ⊖ w ∈ T`** by T0.

When `zpd(a, w)` is defined — write `k = zpd(a, w)` — the result is positive. Components before position `k` are zero by construction, and `rₖ = aₖ − wₖ ≥ 1` by NAT-sub (strict positivity), whose precondition `aₖ > wₖ` is the precondition consequence proved above. The result is therefore not the zero tumbler, whence **`Pos(a ⊖ w)`** (TA-Pos). By ActionPoint, `actionPoint(a ⊖ w) = min({i : 1 ≤ i ≤ #(a ⊖ w) ∧ rᵢ ≠ 0}) = k`, since `rᵢ = 0` for `i < k` and `rₖ ≥ 1`: **`actionPoint(a ⊖ w) = zpd(a, w)`**. When `zpd(a, w)` is undefined, the result is the zero tumbler and neither conclusion holds.

*Formal Contract:*
- *Preconditions:* a ∈ T, w ∈ T, a ≥ w (T1). Consequence: when zpd(a, w) is defined, aₖ > wₖ at k = zpd(a, w).
- *Depends:* T0 (CarrierSetDefinition) — membership `a ⊖ w ∈ T` is concluded via T0's characterisation of T as finite sequences over ℕ with length ≥ 1. T1 (LexicographicOrder) — the precondition `a ≥ w` is a T1 ordering; the precondition consequence derives `w < a` from `a ≥ w ∧ a ≠ w` via T1 trichotomy. Divergence (Divergence) — the precondition consequence proceeds by case analysis on Divergence's two cases for the pair `(w, a)`. ZPD (ZPD) — defines `zpd(a, w)` and supplies the ZPD–Divergence relationship identifying `zpd(a, w) = divergence(a, w) = k` in case (i). TA-Pos (PositiveTumbler) — defines `Pos` for the conditional postcondition `Pos(a ⊖ w)`. ActionPoint (ActionPoint) — supplies the action-point identification for the conditional postcondition `actionPoint(a ⊖ w) = zpd(a, w)`. NAT-sub (NatPartialSubtraction) — conditional closure discharges `rₖ = aₖ − wₖ ∈ ℕ` at the divergence point (under `aₖ ≥ wₖ`, supplied by the precondition consequence), and strict positivity discharges `rₖ ≥ 1` under the strict form `aₖ > wₖ`; without NAT-sub the definition's divergence-point component would use partial subtraction on ℕ as an unsourced primitive. NAT-discrete (NatDiscreteness) — the precondition consequence's case (ii) step `aₖ ≠ 0 ⟹ aₖ > 0` (deriving `aₖ > 0 = wₖ` at `k = zpd(a, w)` from the prefix-divergence observation that `aₖ ≠ 0` while `wₖ = 0` by zero-padding) is licensed by NAT-discrete's axiom `m ≤ n < m + 1 ⟹ n = m` instantiated at `m = 0` (no `n ∈ ℕ` with `0 ≤ n < 1` other than `0`), so the "nonzero ⇒ `> 0`" inference is discharged from NAT-discrete rather than left implicit, matching the per-step convention TA-Pos and ActionPoint already follow for this same inference. NAT-order (NatStrictTotalOrder) — invoked at two distinct ℕ-pairs for two distinct roles, both instances of the same trichotomy axiom applied at distinct points. *Length pair `(#a, #w)` (internal to the Definition)*: trichotomy's exactly-one clause discharges the common-length selection, leaving exactly one of sub-case (α) `#a = #w` (`L = #a`), sub-case (β) `#a < #w` (`L = #w`), or sub-case (γ) `#w < #a` (`L = #a`); `L` — the longer of `#a` and `#w`, which furnishes the result length `#(a ⊖ w) = L` in the Definition, the Postconditions, and the `L ≥ 1` membership check that concludes `a ⊖ w ∈ T` — is named by the selected sub-case rather than by a primitive binary-maximum operator on ℕ, reproducing ZPD's analogous length-pair dispatch for the same operand pair `(a, w)` at which both definitions evaluate and matching the convention that every operator in the Definition must be discharged by a Depends entry. The same length-pair dispatch also names the longer length `#a` in the motivational observation `#(b ⊖ a) = #a` (under `#a > #b`) applied to the reversed operand pair of `b ⊖ a`. *Component pair `(aₖ, wₖ)` (internal to the NAT-sub discharge)*: converts the strict inequality `aₖ > wₖ` of the precondition consequence into the weak `aₖ ≥ wₖ` required by NAT-sub's conditional-closure clause, via the definition of `≤` from `<`. Both roles are enumerated together under this single NAT-order citation, matching the per-step convention ZPD and Divergence apply when a single axiom is invoked at multiple ℕ-pairs.
- *Definition:* NAT-order's trichotomy on `(#a, #w)` selects exactly one of: (α) `#a = #w`, `L = #a`; (β) `#a < #w`, `L = #w`; (γ) `#w < #a`, `L = #a`. a ⊖ w is computed by case analysis on k = zpd(a, w) (ZPD), all component references using zero-padded values (aᵢ = 0 for i > #a, wᵢ = 0 for i > #w); rᵢ = 0 for i < k, rₖ = aₖ − wₖ, rᵢ = aᵢ (zero-padded) for i > k; when zpd(a, w) is undefined, a ⊖ w = [0, …, 0]; #(a ⊖ w) = L (the longer length named by the trichotomy dispatch above, replacing `max(#a, #w)`).
- *Postconditions:* a ⊖ w ∈ T, #(a ⊖ w) = L (the longer of `#a` and `#w`, named by NAT-order trichotomy on `(#a, #w)` per the Definition: `L = #a` in sub-cases (α) and (γ), `L = #w` in sub-case (β)); when zpd(a, w) is defined: Pos(a ⊖ w) (TA-Pos), actionPoint(a ⊖ w) = zpd(a, w) (ActionPoint)
