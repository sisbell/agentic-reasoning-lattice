## Tumbler subtraction

**TumblerSub (TumblerSub).** Given two tumblers `a` (minuend) and `w` (subtrahend), compute their component-wise difference at the first point of zero-padded divergence. The primary downstream use is computing displacements between positions ‚ÄĒ given positions `a` and `b` with `b ‚Č• a`, the subtraction `b ‚äĖ a` yields the component-wise difference from `a` to `b`. This difference is a broader object than a displacement consumable by TumblerAdd: when `a` is a proper sequence-prefix of `b`, the action point of `b ‚äĖ a` satisfies `actionPoint(b ‚äĖ a) = zpd(b, a) ‚Č• #a + 1 > #a`, violating TumblerAdd's precondition `actionPoint(w) ‚Č§ #a`. A second obstruction is independent of the prefix case: when `#a > #b`, the Definition's NAT-order trichotomy dispatch (below) ‚ÄĒ applied to the length pair `(#b, #a)` of the operand pair of `b ‚äĖ a` ‚ÄĒ places the pair in its `#b < #a` sub-case, whose longer length `L = #a` furnishes the result length, so the subtraction produces `#(b ‚äĖ a) = #a`; TumblerAdd's result-length identity gives `#(a ‚äē (b ‚äĖ a)) = #a > #b`, so `a ‚äē (b ‚äĖ a) ‚Č† b` by T3 ‚ÄĒ the round-trip fails on length alone even when the zpd constraint is satisfied. The roundtrip `a ‚äē (b ‚äĖ a) = b` therefore requires two independently necessary constraints: `divergence(a, b) ‚Č§ #a` (ensuring TumblerAdd's precondition) and `#a ‚Č§ #b` (ensuring the result length equals `#b`). D0 (DisplacementWellDefined) takes the first as a precondition and proves the necessity of the second via its negative postcondition (`#a > #b ‚Üí a ‚äē (b ‚äĖ a) ‚Č† b`); the positive round-trip identity under the full conjunction is established by D1 (DisplacementRoundTrip). When the operands have different lengths, we conceptually zero-pad the shorter to the length of the longer ‚ÄĒ writing `aŠĶĘ = 0` for `i > #a` and `wŠĶĘ = 0` for `i > #w`, with the literal `0` supplied as an element of ‚Ąē by NAT-zero so that the padded values extend T0's ‚Ąē-valued component-typing uniformly onto the padding region ‚ÄĒ and use these padded values throughout the procedure. Reproducing ZPD's length-pair dispatch for the same operand pair, NAT-order's trichotomy on `(#a, #w)` selects exactly one of the sub-cases (őĪ) `#a = #w` with `L = #a` (equivalently `#w`), (ő≤) `#a < #w` with `L = #w`, or (ő≥) `#w < #a` with `L = #a`; `L` ‚ÄĒ the longer of `#a` and `#w` ‚ÄĒ is furnished by the selected sub-case rather than by any primitive binary-maximum operator on ‚Ąē, matching the convention that every operator in the Definition must be discharged by a Depends entry. When the zero-padded sequences agree at every position (no divergence exists), the result is the zero tumbler of length `L`: `a ‚äĖ w = [0, ..., 0]`. Otherwise, `zpd(a, w)` is defined (ZPD) ‚ÄĒ write `k = zpd(a, w)` for the first position at which the padded sequences disagree. The result is:

```
         ‚éß 0             if i < k        (these levels matched ‚ÄĒ zero them)
rŠĶĘ   =  ‚é® a‚āĖ - w‚āĖ      if i = k        (subtract at the divergence point)
         ‚é© aŠĶĘ           if i > k        (copy from minuend, zero-padded)
```

The result has length `L`.

**Precondition:** `a ‚Č• w` (T1). We prove that when `zpd(a, w)` is defined, this entails `a‚āĖ > w‚āĖ` at `k = zpd(a, w)`. Since zpd is defined, `a` and `w` are not zero-padded-equal (ZPD), so in particular `a ‚Č† w`; combined with `a ‚Č• w`, this yields `w < a` (T1). Two Divergence cases arise for the pair `(w, a)` with `w ‚Č† a`:

  (i) Component divergence at position `k` with `k ‚Č§ #w ‚ąß k ‚Č§ #a` (the conjunction replacing `k ‚Č§ min(#w, #a)` so that no primitive binary-minimum operator on ‚Ąē is invoked, matching Divergence's case (i)) and `w‚āĖ ‚Č† a‚āĖ`. The ZPD‚ÄďDivergence relationship (ZPD) gives `zpd(a, w) = divergence(a, w) = k`. Since `w < a` via T1 case (i), `w‚āĖ < a‚āĖ`, whence `a‚āĖ > w‚āĖ`.

  (ii) Prefix divergence ‚ÄĒ `w` is a proper prefix of `a` (the direction forced by `w < a` via T1 case (ii)), so `#w < #a` and `wŠĶĘ = aŠĶĘ` for all `1 ‚Č§ i ‚Č§ #w`. The padded extension sets `w‚āĖ = 0` for `k > #w`, with `0 ‚ąą ‚Ąē` supplied by NAT-zero so that the equality is a well-formed ‚Ąē-valued comparison. Since zpd is defined, the longer operand `a` has some nonzero component beyond `#w`; at `k = zpd(a, w)`, `a‚āĖ ‚Č† 0 = w‚āĖ`. The step `a‚āĖ ‚Č† 0 ‚üĻ a‚āĖ > 0` is discharged by NAT-zero's lower bound `0 ‚Č§ a‚āĖ` (instantiated at `n = a‚āĖ`, with `a‚āĖ ‚ąą ‚Ąē` supplied by T0's component-typing), NAT-order's defining clause `m ‚Č§ n ‚üļ m < n ‚ą® m = n` at `m = 0, n = a‚āĖ` unfolding `0 ‚Č§ a‚āĖ` to `0 < a‚āĖ ‚ą® 0 = a‚āĖ`, and the divergence observation `a‚āĖ ‚Č† 0` excluding the equality disjunct to leave `0 < a‚āĖ`; hence `a‚āĖ > 0 = w‚āĖ`.

In both cases `a‚āĖ > w‚āĖ` at `k = zpd(a, w)`. When zpd is undefined, the precondition `a ‚Č• w` continues to hold by hypothesis; the consequence `a‚āĖ > w‚āĖ` is vacuous because there is no `k = zpd(a, w)` at which to evaluate it, and the constructive definition produces the zero tumbler with no subtraction at a divergence point.  ‚ąé

Each component of the result is a natural number: for `i < k`, `rŠĶĘ = 0 ‚ąą ‚Ąē` by NAT-zero; at the divergence point, `r‚āĖ = a‚āĖ ‚ąí w‚āĖ ‚ąą ‚Ąē` by NAT-sub (conditional closure), whose precondition `a‚āĖ ‚Č• w‚āĖ` is discharged by the precondition consequence `a‚āĖ > w‚āĖ` at the zpd point (proved above) combined with NAT-order; for `i > k`, `rŠĶĘ` is the zero-padded value of `a`, which is either `aŠĶĘ ‚ąą ‚Ąē` (when `i ‚Č§ #a`) or `0 ‚ąą ‚Ąē` by NAT-zero (when `i > #a`). In the equal case (no divergence), every component is `0 ‚ąą ‚Ąē` by NAT-zero. The result is therefore a finite sequence over ‚Ąē with length `L ‚Č• 1` ‚ÄĒ since `a, w ‚ąą T` requires both `#a ‚Č• 1` and `#w ‚Č• 1`, and `L` is named by the trichotomy dispatch as one of `#a` or `#w`, each of which is at least 1 ‚ÄĒ hence **`a ‚äĖ w ‚ąą T`** by T0.

When `zpd(a, w)` is defined ‚ÄĒ write `k = zpd(a, w)` ‚ÄĒ the result is positive. Components before position `k` are zero by construction, and `r‚āĖ = a‚āĖ ‚ąí w‚āĖ ‚Č• 1` by NAT-sub (strict positivity), whose precondition `a‚āĖ > w‚āĖ` is the precondition consequence proved above. The result is therefore not the zero tumbler, whence **`Pos(a ‚äĖ w)`** (TA-Pos). By ActionPoint, `actionPoint(a ‚äĖ w) = min({i : 1 ‚Č§ i ‚Č§ #(a ‚äĖ w) ‚ąß rŠĶĘ ‚Č† 0}) = k`, since `rŠĶĘ = 0` for `i < k` and `r‚āĖ ‚Č• 1`: **`actionPoint(a ‚äĖ w) = zpd(a, w)`**. When `zpd(a, w)` is undefined, the result is the zero tumbler and neither conclusion holds.
