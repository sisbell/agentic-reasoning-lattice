## Tumbler subtraction

**TumblerSub (TumblerSub).** Given two tumblers `a` (minuend) and `w` (subtrahend), compute their component-wise difference at the first point of zero-padded divergence. The primary downstream use is computing displacements between positions вҖ” given positions `a` and `b` with `b вүҘ a`, the subtraction `b вҠ– a` yields the component-wise difference from `a` to `b`. This difference is a broader object than a displacement consumable by TumblerAdd: when `a` is a proper sequence-prefix of `b`, the action point of `b вҠ– a` satisfies `actionPoint(b вҠ– a) = zpd(b, a) вүҘ #a + 1 > #a`, violating TumblerAdd's precondition `actionPoint(w) вүӨ #a`. A second obstruction is independent of the prefix case: when `#a > #b`, the subtraction produces `#(b вҠ– a) = max(#a, #b) = #a`, and TumblerAdd's result-length identity gives `#(a вҠ• (b вҠ– a)) = #a > #b`, so `a вҠ• (b вҠ– a) вү  b` by T3 вҖ” the round-trip fails on length alone even when the zpd constraint is satisfied. The roundtrip `a вҠ• (b вҠ– a) = b` therefore requires two independently necessary constraints: `divergence(a, b) вүӨ #a` (ensuring TumblerAdd's precondition) and `#a вүӨ #b` (ensuring the result length equals `#b`). D0 (DisplacementWellDefined) takes the first as a precondition and proves the necessity of the second via its negative postcondition (`#a > #b вҶ’ a вҠ• (b вҠ– a) вү  b`); the positive round-trip identity under the full conjunction is established by D1 (DisplacementRoundTrip). When the operands have different lengths, we conceptually zero-pad the shorter to the length of the longer вҖ” writing `aбөў = 0` for `i > #a` and `wбөў = 0` for `i > #w` вҖ” and use these padded values throughout the procedure. When the zero-padded sequences agree at every position (no divergence exists), the result is the zero tumbler of length `max(#a, #w)`: `a вҠ– w = [0, ..., 0]`. Otherwise, `zpd(a, w)` is defined (ZPD) вҖ” write `k = zpd(a, w)` for the first position at which the padded sequences disagree. The result is:

```
         вҺ§ 0             if i < k        (these levels matched вҖ” zero them)
rбөў   =  вҺЁ aвӮ– - wвӮ–      if i = k        (subtract at the divergence point)
         вҺ© aбөў           if i > k        (copy from minuend, zero-padded)
```

The result has length `max(#a, #w)`.

**Precondition:** `a вүҘ w` (T1). We prove that when `zpd(a, w)` is defined, this entails `aвӮ– > wвӮ–` at `k = zpd(a, w)`. Since zpd is defined, `a` and `w` are not zero-padded-equal (ZPD), so in particular `a вү  w`; combined with `a вүҘ w`, this yields `w < a` (T1). Two Divergence cases arise for the pair `(w, a)` with `w вү  a`:

  (i) Component divergence at position `k вүӨ min(#w, #a)` with `wвӮ– вү  aвӮ–`. The ZPDвҖ“Divergence relationship (ZPD) gives `zpd(a, w) = divergence(a, w) = k`. Since `w < a` via T1 case (i), `wвӮ– < aвӮ–`, whence `aвӮ– > wвӮ–`.

  (ii) Prefix divergence вҖ” `w` is a proper prefix of `a` (the direction forced by `w < a` via T1 case (ii)), so `#w < #a` and `wбөў = aбөў` for all `1 вүӨ i вүӨ #w`. The padded extension sets `wвӮ– = 0` for `k > #w`. Since zpd is defined, the longer operand `a` has some nonzero component beyond `#w`; at `k = zpd(a, w)`, `aвӮ– вү  0 = wвӮ–`, so `aвӮ– > 0 = wвӮ–`.

In both cases `aвӮ– > wвӮ–` at `k = zpd(a, w)`. When zpd is undefined, the precondition `a вүҘ w` continues to hold by hypothesis; the consequence `aвӮ– > wвӮ–` is vacuous because there is no `k = zpd(a, w)` at which to evaluate it, and the constructive definition produces the zero tumbler with no subtraction at a divergence point.  вҲҺ

Each component of the result is a natural number: for `i < k`, `rбөў = 0 вҲҲ в„•`; at the divergence point, `rвӮ– = aвӮ– вҲ’ wвӮ– вҲҲ в„•` since `a вүҘ w` entails `aвӮ– > wвӮ–` at the zpd point (proved above); for `i > k`, `rбөў` is the zero-padded value of `a`, which is either `aбөў вҲҲ в„•` (when `i вүӨ #a`) or `0 вҲҲ в„•` (when `i > #a`). In the equal case (no divergence), every component is `0 вҲҲ в„•`. The result is therefore a finite sequence over в„• with length `max(#a, #w) вүҘ 1` вҖ” since `a, w вҲҲ T` requires `#a вүҘ 1` and `#w вүҘ 1` вҖ” hence **`a вҠ– w вҲҲ T`** by T0.

When `zpd(a, w)` is defined вҖ” write `k = zpd(a, w)` вҖ” the result is positive. Components before position `k` are zero by construction, and `rвӮ– = aвӮ– вҲ’ wвӮ– вүҘ 1` since the precondition consequence gives `aвӮ– > wвӮ–`. The result is therefore not the zero tumbler, whence **`Pos(a вҠ– w)`** (TA-Pos). By ActionPoint, `actionPoint(a вҠ– w) = min({i : 1 вүӨ i вүӨ #(a вҠ– w) вҲ§ rбөў вү  0}) = k`, since `rбөў = 0` for `i < k` and `rвӮ– вүҘ 1`: **`actionPoint(a вҠ– w) = zpd(a, w)`**. When `zpd(a, w)` is undefined, the result is the zero tumbler and neither conclusion holds.
