## Tumbler subtraction

**TumblerSub (TumblerSub).** Given two tumblers `a` (minuend) and `w` (subtrahend), compute their component-wise difference at the first point of zero-padded divergence. The primary downstream use is computing displacements between positions — given positions `a` and `b` with `b ≥ a`, the subtraction `b ⊖ a` yields the component-wise difference from `a` to `b`. This difference is a broader object than a displacement consumable by TumblerAdd: when `a` is a proper sequence-prefix of `b`, the proper-prefix hypothesis splits into two sub-cases according to whether `zpd(b, a)` is defined, and both sub-cases leave `a ⊕ (b ⊖ a)` undefined through different clauses of TumblerAdd's precondition. (a) *Sub-case with `b` having a nonzero component at some position `i > #a`:* the padded sequences agree on `1, ..., #a` by the prefix hypothesis `aᵢ = bᵢ` and disagree at that `i > #a` (where `aᵢ = 0` by the zero-padding of the shorter operand `a` and `bᵢ ≠ 0` by hypothesis), so `zpd(b, a)` is defined; the action point of `b ⊖ a` then satisfies `actionPoint(b ⊖ a) = zpd(b, a) ≥ #a + 1 > #a`, violating TumblerAdd's precondition `actionPoint(w) ≤ #a`. (b) *Sub-case with every component of `b` at positions `i > #a` equal to zero:* the padded sequences coincide at every position (at `1, ..., #a` by the prefix hypothesis, at `i > #a` by the shared value `0` — from `b`'s hypothesized trailing zeros and from `a`'s zero-padding), so `zpd(b, a)` is undefined; the Definition's no-divergence branch (below) produces `b ⊖ a = [0, …, 0]` — the zero tumbler of length `L` — for which `Pos(b ⊖ a)` fails by TA-Pos's characterisation (the existential `(E i : 1 ≤ i ≤ #(b ⊖ a) : (b ⊖ a)ᵢ ≠ 0)` is vacuously false with every component zero), violating TumblerAdd's *other* precondition `Pos(w)` rather than its action-point bound. D0's precondition `divergence(a, b) ≤ #a` excludes both sub-cases uniformly: Divergence's case (ii) assigns `divergence(a, b) = #a + 1 > #a` to every proper-prefix pair — the assignment depends only on the shared-prefix agreement `aᵢ = bᵢ` for `1 ≤ i ≤ #a` and the length inequality `#a < #b`, neither of which is sensitive to whether `b`'s trailing components are zero. A second obstruction is independent of the prefix case: when `#a > #b`, the Definition's NAT-order trichotomy dispatch (below) — applied to the length pair `(#b, #a)` of the operand pair of `b ⊖ a` — places the pair in its `#b < #a` sub-case, whose longer length `L = #a` furnishes the result length, so the subtraction produces `#(b ⊖ a) = #a`; TumblerAdd's result-length identity gives `#(a ⊕ (b ⊖ a)) = #a > #b`, so `a ⊕ (b ⊖ a) ≠ b` by T3 — the round-trip fails on length alone even when the zpd constraint is satisfied. The roundtrip `a ⊕ (b ⊖ a) = b` therefore requires two independently necessary constraints: `divergence(a, b) ≤ #a` (ensuring TumblerAdd's precondition) and `#a ≤ #b` (ensuring the result length equals `#b`). D0 (DisplacementWellDefined) takes the first as a precondition and proves the necessity of the second via its negative postcondition (`#a > #b → a ⊕ (b ⊖ a) ≠ b`); the positive round-trip identity under the full conjunction is established by D1 (DisplacementRoundTrip). When the operands have different lengths, we conceptually zero-pad the shorter to the length of the longer — writing `aᵢ = 0` for `i > #a` and `wᵢ = 0` for `i > #w`, with the literal `0` supplied as an element of ℕ by NAT-zero so that the padded values extend T0's ℕ-valued component-typing uniformly onto the padding region — and use these padded values throughout the procedure. Reproducing ZPD's length-pair dispatch for the same operand pair, NAT-order's trichotomy on `(#a, #w)` selects exactly one of the sub-cases (α) `#a = #w` with `L = #a` (equivalently `#w`), (β) `#a < #w` with `L = #w`, or (γ) `#w < #a` with `L = #a`; `L` — the longer of `#a` and `#w` — is furnished by the selected sub-case rather than by any primitive binary-maximum operator on ℕ, matching the convention that every operator in the Definition must be discharged by a Depends entry. When the zero-padded sequences agree at every position (no divergence exists), the result is the zero tumbler of length `L`: `a ⊖ w = [0, ..., 0]`. Otherwise, `zpd(a, w)` is defined (ZPD) — write `k = zpd(a, w)` for the first position at which the padded sequences disagree. The result is:

```
         ⎧ 0             if i < k        (these levels matched — zero them)
rᵢ   =  ⎨ aₖ - wₖ      if i = k        (subtract at the divergence point)
         ⎩ aᵢ           if i > k        (copy from minuend, zero-padded)
```

The result has length `L`.

**Precondition:** `a ≥ w` (T1). We prove that when `zpd(a, w)` is defined, this entails `aₖ > wₖ` at `k = zpd(a, w)`. Since zpd is defined, `a` and `w` are not zero-padded-equal (ZPD), so in particular `a ≠ w`; combined with `a ≥ w`, this yields `w < a` (T1). Two Divergence cases arise for the pair `(w, a)` with `w ≠ a`:

  (i) Component divergence at position `k` with `k ≤ #w ∧ k ≤ #a` (the conjunction replacing `k ≤ min(#w, #a)` so that no primitive binary-minimum operator on ℕ is invoked, matching Divergence's case (i)) and `wₖ ≠ aₖ`. The ZPD–Divergence relationship (ZPD) gives `zpd(a, w) = divergence(a, w) = k`. Since `w < a` via T1 case (i), `wₖ < aₖ`, whence `aₖ > wₖ`.

  (ii) Prefix divergence — Divergence case (ii) applied to the pair `(w, a)` splits via NAT-order's trichotomy on the length pair `(#w, #a)` (ruling out `#w = #a`) into sub-case (ii-a) `#w < #a` with `divergence(w, a) = #w + 1` and `wᵢ = aᵢ` for `1 ≤ i ≤ #w`, placing `w` as a proper prefix of `a`, and sub-case (ii-b) `#a < #w` with `divergence(w, a) = #a + 1` and `wᵢ = aᵢ` for `1 ≤ i ≤ #a`, placing `a` as a proper prefix of `w`. Sub-case (ii-b) is eliminated by T1 direction: its prefix hypothesis — `a` a proper prefix of `w` — yields `a < w` via T1 case (ii), contradicting the derived `w < a` — the same mechanism D0's sub-case (ii-b) elimination uses. We are therefore in sub-case (ii-a): `w` is a proper prefix of `a`, so `#w < #a` and `wᵢ = aᵢ` for all `1 ≤ i ≤ #w`. The padded extension sets `wᵢ = 0` for `i > #w`, with `0 ∈ ℕ` supplied by NAT-zero so that the equality is a well-formed ℕ-valued comparison. Since zpd is defined, the padded sequences disagree at some position (ZPD case-split, contrapositive); combined with the prefix agreement `wᵢ = aᵢ` on `1 ≤ i ≤ #w` — where the padded values coincide with the originals and therefore agree — the disagreement must occur at some `i > #w`, where `wᵢ = 0` by the zero-padding forces `aᵢ ≠ 0`; the longer operand `a` therefore has some nonzero component beyond `#w`. ZPD additionally names `k = zpd(a, w)` as the *least* padded-divergence position; since the same prefix agreement rules out every `i` with `1 ≤ i ≤ #w` as a padded-divergence position, the minimal divergence position must lie strictly beyond `#w`, so `k > #w`. At the named `k`, the zero-padding gives `wₖ = 0` (via the rule `wᵢ = 0` for `i > #w` instantiated at `i = k`), and `k`'s status as a padded-divergence position gives `aₖ ≠ wₖ`; combining, `aₖ ≠ 0 = wₖ` at `k = zpd(a, w)`. The step `aₖ ≠ 0 ⟹ aₖ > 0` is discharged by NAT-zero's lower bound `0 ≤ aₖ` (instantiated at `n = aₖ`, with `aₖ ∈ ℕ` supplied by T0's component-typing), NAT-order's defining clause `m ≤ n ⟺ m < n ∨ m = n` at `m = 0, n = aₖ` unfolding `0 ≤ aₖ` to `0 < aₖ ∨ 0 = aₖ`, and the divergence observation `aₖ ≠ 0` excluding the equality disjunct to leave `0 < aₖ`; hence `aₖ > 0 = wₖ`.

In both cases `aₖ > wₖ` at `k = zpd(a, w)`. When zpd is undefined, the precondition `a ≥ w` continues to hold by hypothesis; the consequence `aₖ > wₖ` is vacuous because there is no `k = zpd(a, w)` at which to evaluate it, and the constructive definition produces the zero tumbler with no subtraction at a divergence point.  ∎

Each component of the result is a natural number: for `i < k`, `rᵢ = 0 ∈ ℕ` by NAT-zero; at the divergence point, `rₖ = aₖ − wₖ ∈ ℕ` by NAT-sub (conditional closure), whose precondition `aₖ ≥ wₖ` is discharged by the precondition consequence `aₖ > wₖ` at the zpd point (proved above) combined with NAT-order; for `i > k`, `rᵢ` is the zero-padded value of `a`, which is either `aᵢ ∈ ℕ` (when `i ≤ #a`) or `0 ∈ ℕ` by NAT-zero (when `i > #a`). In the equal case (no divergence), every component is `0 ∈ ℕ` by NAT-zero. The result is therefore a finite sequence over ℕ with length `L ≥ 1` — since `a, w ∈ T` requires both `#a ≥ 1` and `#w ≥ 1`, and `L` is named by the trichotomy dispatch as one of `#a` or `#w`, each of which is at least 1 — hence **`a ⊖ w ∈ T`** by T0.

When `zpd(a, w)` is defined — write `k = zpd(a, w)` — the result is positive. Components before position `k` are zero by construction, and `rₖ = aₖ − wₖ ≥ 1` by NAT-sub (strict positivity), whose precondition `aₖ > wₖ` is the precondition consequence proved above. The result is therefore not the zero tumbler, whence **`Pos(a ⊖ w)`** (TA-Pos). By ActionPoint, `actionPoint(a ⊖ w) = min({i : 1 ≤ i ≤ #(a ⊖ w) ∧ rᵢ ≠ 0}) = k`, since `rᵢ = 0` for `i < k` and `rₖ ≥ 1`: **`actionPoint(a ⊖ w) = zpd(a, w)`**. When `zpd(a, w)` is undefined, the result is the zero tumbler and neither conclusion holds.
