## Tumbler subtraction

**TumblerSub (TumblerSub).** Given two tumblers `a` (minuend) and `w` (subtrahend), compute their component-wise difference at the first point of zero-padded divergence. The primary downstream use is computing displacements between positions ‚Äî given positions `a` and `b` with `b ‚â• a`, the subtraction `b ‚äñ a` yields the component-wise difference from `a` to `b`. This difference is a broader object than a displacement consumable by TumblerAdd: when `a` is a proper sequence-prefix of `b`, the action point of `b ‚äñ a` satisfies `actionPoint(b ‚äñ a) = zpd(b, a) ‚â• #a + 1 > #a`, violating TumblerAdd's precondition `actionPoint(w) ‚â§ #a`. A second obstruction is independent of the prefix case: when `#a > #b`, the Definition's NAT-order trichotomy dispatch (below) ‚Äî applied to the length pair `(#b, #a)` of the operand pair of `b ‚äñ a` ‚Äî places the pair in its `#b < #a` sub-case, whose longer length `L = #a` furnishes the result length, so the subtraction produces `#(b ‚äñ a) = #a`; TumblerAdd's result-length identity gives `#(a ‚äï (b ‚äñ a)) = #a > #b`, so `a ‚äï (b ‚äñ a) ‚âÝ b` by T3 ‚Äî the round-trip fails on length alone even when the zpd constraint is satisfied. The roundtrip `a ‚äï (b ‚äñ a) = b` therefore requires two independently necessary constraints: `divergence(a, b) ‚â§ #a` (ensuring TumblerAdd's precondition) and `#a ‚â§ #b` (ensuring the result length equals `#b`). D0 (DisplacementWellDefined) takes the first as a precondition and proves the necessity of the second via its negative postcondition (`#a > #b ‚Üí a ‚äï (b ‚äñ a) ‚âÝ b`); the positive round-trip identity under the full conjunction is established by D1 (DisplacementRoundTrip). When the operands have different lengths, we conceptually zero-pad the shorter to the length of the longer ‚Äî writing `a·µ¢ = 0` for `i > #a` and `w·µ¢ = 0` for `i > #w`, with the literal `0` supplied as an element of ‚Ñï by NAT-zero so that the padded values extend T0's ‚Ñï-valued component-typing uniformly onto the padding region ‚Äî and use these padded values throughout the procedure. Reproducing ZPD's length-pair dispatch for the same operand pair, NAT-order's trichotomy on `(#a, #w)` selects exactly one of the sub-cases (Œ±) `#a = #w` with `L = #a` (equivalently `#w`), (Œ≤) `#a < #w` with `L = #w`, or (Œ≥) `#w < #a` with `L = #a`; `L` ‚Äî the longer of `#a` and `#w` ‚Äî is furnished by the selected sub-case rather than by any primitive binary-maximum operator on ‚Ñï, matching the convention that every operator in the Definition must be discharged by a Depends entry. When the zero-padded sequences agree at every position (no divergence exists), the result is the zero tumbler of length `L`: `a ‚äñ w = [0, ..., 0]`. Otherwise, `zpd(a, w)` is defined (ZPD) ‚Äî write `k = zpd(a, w)` for the first position at which the padded sequences disagree. The result is:

```
         ‚éß 0             if i < k        (these levels matched ‚Äî zero them)
r·µ¢   =  ‚é® a‚Çñ - w‚Çñ      if i = k        (subtract at the divergence point)
         ‚é© a·µ¢           if i > k        (copy from minuend, zero-padded)
```

The result has length `L`.

**Precondition:** `a ‚â• w` (T1). We prove that when `zpd(a, w)` is defined, this entails `a‚Çñ > w‚Çñ` at `k = zpd(a, w)`. Since zpd is defined, `a` and `w` are not zero-padded-equal (ZPD), so in particular `a ‚âÝ w`; combined with `a ‚â• w`, this yields `w < a` (T1). Two Divergence cases arise for the pair `(w, a)` with `w ‚âÝ a`:

  (i) Component divergence at position `k` with `k ‚â§ #w ‚àß k ‚â§ #a` (the conjunction replacing `k ‚â§ min(#w, #a)` so that no primitive binary-minimum operator on ‚Ñï is invoked, matching Divergence's case (i)) and `w‚Çñ ‚âÝ a‚Çñ`.   (ii) Prefix divergence ‚Äî `w` is a proper prefix of `a` (the direction forced by `w < a` via T1 case (ii)), so `#w < #a` and `w·µ¢ = a·µ¢` for all `1 ‚â§ i ‚â§ #w`. The padded extension sets `w‚Çñ = 0` for `k > #w`, with `0 ‚àà ‚Ñï` supplied by NAT-zero so that the equality is a well-formed ‚Ñï-valued comparison. Since zpd is defined, the longer operand `a` has some nonzero component beyond `#w`; at `k = zpd(a, w)`, `a‚Çñ ‚âÝ 0 = w‚Çñ`. The step `a‚Çñ ‚âÝ 0 ‚üπ a‚Çñ > 0` is discharged by NAT-zero's lower bound `0 ‚â§ a‚Çñ` (instantiated at `n = a‚Çñ`, with `a‚Çñ ‚àà ‚Ñï` supplied by T0's component-typing), NAT-order's defining clause `m ‚â§ n ‚ü∫ m < n ‚à® m = n` at `m = 0, n = a‚Çñ` unfolding `0 ‚â§ a‚Çñ` to `0 < a‚Çñ ‚à® 0 = a‚Çñ`, and the divergence observation `a‚Çñ ‚âÝ 0` excluding the equality disjunct to leave `0 < a‚Çñ`; hence `a‚Çñ > 0 = w‚Çñ`.

In both cases `a‚Çñ > w‚Çñ` at `k = zpd(a, w)`. When zpd is undefined, the precondition `a ‚â• w` continues to hold by hypothesis; the consequence `a‚Çñ > w‚Çñ` is vacuous because there is no `k = zpd(a, w)` at which to evaluate it, and the constructive definition produces the zero tumbler with no subtraction at a divergence point.  ‚àé

Each component of the result is a natural number: for `i < k`, `r·µ¢ = 0 ‚àà ‚Ñï` by NAT-zero; at the divergence point, `r‚Çñ = a‚Çñ ‚àí w‚Çñ ‚àà ‚Ñï` by NAT-sub (conditional closure), whose precondition `a‚Çñ ‚â• w‚Çñ` is discharged by the precondition consequence `a‚Çñ > w‚Çñ` at the zpd point (proved above) combined with NAT-order; for `i > k`, `r·µ¢` is the zero-padded value of `a`, which is either `a·µ¢ ‚àà ‚Ñï` (when `i ‚â§ #a`) or `0 ‚àà ‚Ñï` by NAT-zero (when `i > #a`).
