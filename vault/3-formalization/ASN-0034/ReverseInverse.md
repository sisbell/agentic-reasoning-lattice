**ReverseInverse (ReverseInverse).** `(A a, w : a ‚â• w ‚àß Pos(w) ‚àß k = #a ‚àß #w = k ‚àß (A i : 1 ‚â§ i < k : a·µ¢ = 0) : (a ‚äñ w) ‚äï w = a)`, where `k` is the action point of `w`.

*Proof.* We show that subtracting `w` from `a` and then adding `w` back recovers `a` exactly, under conditions that make the two operations mutually inverse. Throughout, `k` denotes the action point of `w` ‚Äî the least position with `w‚Çñ > 0` ‚Äî so by definition `w·µ¢ = 0` for all `i < k`.

**Step 1: the structure of `y = a ‚äñ w`.** By TumblerSub, subtraction scans `a` and `w` for the first position where they differ, zero-padding the shorter to length `max(#a, #w)`. Since `#a = k = #w` (given), no padding is needed. At each position `i < k`, both `a·µ¢ = 0` (by the zero-prefix precondition) and `w·µ¢ = 0` (by definition of action point), so the operands agree before position `k`.

Two cases arise at position `k`. If `a‚Çñ = w‚Çñ`, then `a` and `w` agree at every position ‚Äî there are no positions beyond `k` since both have length `k` ‚Äî and TumblerSub produces the zero tumbler of length `k`. If `a‚Çñ > w‚Çñ` (the only alternative, since `a ‚â• w` excludes `a‚Çñ < w‚Çñ`), then `k` is the first divergence, and TumblerSub produces `y·µ¢ = 0` for `i < k`, `y‚Çñ = a‚Çñ - w‚Çñ > 0` (the membership `a‚Çñ - w‚Çñ ‚àà ‚Ñï` by NAT-sub's conditional closure under `a‚Çñ ‚â• w‚Çñ` from NAT-order; the strict positivity `a‚Çñ - w‚Çñ > 0` by NAT-sub's strict-positivity clause under `a‚Çñ > w‚Çñ`), and no components beyond `k` (since `max(#a, #w) = k`). In either case, `y` has three properties we record for later use:

- (Y1) `#y = k`
- (Y2) `y·µ¢ = 0` for all `1 ‚â§ i < k`
- (Y3) `y‚Çñ = a‚Çñ - w‚Çñ`

**Step 2: TA4 applies to `y` and `w`.** TA4 (Partial inverse) requires four preconditions: `Pos(w)` (given), `k = #y` (by Y1), `#w = k` (given), and `(A i : 1 ‚â§ i < k : y·µ¢ = 0)` (by Y2). All four hold, so TA4 yields:

`(y ‚äï w) ‚äñ w = y`  ‚Äî (‚ÄÝ)

**Step 3: `y ‚äï w = a` by contradiction via TA3-strict.** Assume for contradiction that `y ‚äï w ‚âÝ a`. We verify the preconditions of TA3-strict (Order preservation under subtraction, strict), which requires strict ordering between two tumblers, both `‚â• w`, and equal length.

*Equal length.* By the result-length identity (TumblerAdd), `#(y ‚äï w) = #w`. The preconditions give `#w = k` and `k = #a`, so `#(y ‚äï w) = #a`.

*`a ‚â• w`.* Given as a precondition of ReverseInverse.

*`y ‚äï w > w`.* By TumblerAdd, for `i < k`: `(y ‚äï w)·µ¢ = y·µ¢ = 0 = w·µ¢` (using Y2 and the definition of action point). At position `k`: `(y ‚äï w)‚Çñ = y‚Çñ + w‚Çñ`. Since `#(y ‚äï w) = k = #w`, there are no positions beyond `k`, so the two tumblers `y ‚äï w` and `w` agree at all positions except possibly `k`. We show `y‚Çñ > 0`. If `y‚Çñ = 0`, then by Y3, `a‚Çñ = w‚Çñ`. Combined with `a·µ¢ = w·µ¢ = 0` for all `i < k` and `#a = #w = k`, this gives `a = w` by T3 (CanonicalRepresentation). Then `y = a ‚äñ w = w ‚äñ w`, which is the zero tumbler of length `k`, and `y ‚äï w` has `(y ‚äï w)‚Çñ = 0 + w‚Çñ = w‚Çñ` (NAT-closure's additive identity, instantiated at `n = w‚Çñ`) with zeros before `k`, so `y ‚äï w = w = a` ‚Äî contradicting our assumption. Therefore `y‚Çñ > 0`. We promote this hypothesis to the strict component inequality `(y ‚äï w)‚Çñ = y‚Çñ + w‚Çñ > w‚Çñ` via the same four-step chain consuming NAT-addcompat, NAT-closure, NAT-cancel, and NAT-order together that TumblerAdd's dominance sub-case `a‚Çñ > 0` applies at its structurally identical site ‚Äî TumblerAdd's exported dominance postcondition `a ‚äï w ‚â• w` is non-strict, so the strict form this step consumes to feed T1 case (i) must be derived per-step rather than read off transitively. NAT-order's defining clause `m ‚â§ n ‚ü∫ m < n ‚à® m = n` instantiated at `m = 0, n = y‚Çñ` weakens the strict `0 < y‚Çñ` to the non-strict `0 ‚â§ y‚Çñ`. NAT-addcompat's right order-compatibility (`n ‚â• p ‚üπ n + m ‚â• p + m`, instantiated at `n = y‚Çñ, p = 0, m = w‚Çñ`) lifts `0 ‚â§ y‚Çñ` to the non-strict `y‚Çñ + w‚Çñ ‚â• 0 + w‚Çñ`. NAT-closure's additive identity `0 + w‚Çñ = w‚Çñ` rewrites the right-hand side, yielding `y‚Çñ + w‚Çñ ‚â• w‚Çñ`. NAT-cancel's summand absorption in its symmetric form `n + m = m ‚üπ n = 0` (instantiated at `n = y‚Çñ, m = w‚Çñ`) rules out the equality `y‚Çñ + w‚Çñ = w‚Çñ` ‚Äî that equality would force `y‚Çñ = 0` and contradict `y‚Çñ > 0` via NAT-order's irreflexivity (substituting `y‚Çñ = 0` into `0 < y‚Çñ` yields `0 < 0`). NAT-order's defining clause `m ‚â§ n ‚ü∫ m < n ‚à® m = n` instantiated at `m = w‚Çñ, n = y‚Çñ + w‚Çñ`, applied to the non-strict `y‚Çñ + w‚Çñ ‚â• w‚Çñ` together with the just-discharged non-equality, returns the strict `y‚Çñ + w‚Çñ > w‚Çñ`, i.e., `(y ‚äï w)‚Çñ > w‚Çñ`. The two tumblers agree before `k` and first differ at `k` with `(y ‚äï w)‚Çñ > w‚Çñ`, so by T1, `y ‚äï w > w`.

*Strict ordering between `y ‚äï w` and `a`.* By T1 (trichotomy), since `y ‚äï w ‚âÝ a`, exactly one of `y ‚äï w < a` or `y ‚äï w > a` holds. We derive a contradiction from each.

*Case `y ‚äï w > a`:* We have `a < y ‚äï w`, `a ‚â• w`, `y ‚äï w ‚â• w` (established above, in fact strict), and `#a = #(y ‚äï w)`. TA3-strict gives `a ‚äñ w < (y ‚äï w) ‚äñ w`. The left side is `y` by definition; the right side is `y` by (‚ÄÝ). This yields `y < y`, contradicting the irreflexivity of `<` (T1).

*Case `y ‚äï w < a`:* We have `y ‚äï w < a`, `y ‚äï w ‚â• w` (strict), `a ‚â• w`, and `#(y ‚äï w) = #a`. TA3-strict gives `(y ‚äï w) ‚äñ w < a ‚äñ w`. The left side is `y` by (‚ÄÝ); the right side is `y` by definition. This yields `y < y`, again contradicting irreflexivity.

Both cases are impossible, so the assumption `y ‚äï w ‚âÝ a` is false. Therefore `(a ‚äñ w) ‚äï w = a`. ‚àé
