**TA6 (ZeroTumblers).** No zero tumbler is a valid address — no all-zero tumbler designates content. Every zero tumbler is less than every positive tumbler under T1.

  `(A t ∈ T : (A i : 1 ≤ i ≤ #t : tᵢ = 0) ⟹ t is not a valid address)`

  `(A s, t ∈ T : (A i : 1 ≤ i ≤ #s : sᵢ = 0) ∧ (E j : 1 ≤ j ≤ #t : tⱼ > 0) ⟹ s < t)`

*Proof (from T0, T4, TA-Pos, NAT-zero).* We prove each conjunct separately. Recall that T4's field-segment constraint includes `t₁ ≠ 0`, and since `t₁ ∈ ℕ` by T0, this gives `t₁ > 0` — the first component belongs to the node field. Recall also TA-Pos's postcondition `(A t ∈ T, z ∈ T : Pos(t) ∧ (A i : 1 ≤ i ≤ #z : zᵢ = 0) :: z < t)` — every positive tumbler is T1-greater than every zero tumbler of any length — established in the TA-Pos section by a case analysis over T1 cases (i) and (ii). We designate that postcondition as the *canonical* statement of the zero-tumbler-below-positive-tumbler relation and cite it in Conjunct 2 rather than reproving it, so that any future tightening of the statement happens at one site.

**Conjunct 1** (invalidity): Let `t` be a zero tumbler, so `tᵢ = 0` for all `1 ≤ i ≤ #t`. Since `t ∈ T`, T0 guarantees `#t ≥ 1`, so `t₁` is defined and `t₁ = 0`. This violates T4's requirement `t₁ ≠ 0`, so `t` is not a valid address.

**Conjunct 2** (ordering): Let `s ∈ T` be a zero tumbler — so `(A i : 1 ≤ i ≤ #s : sᵢ = 0)` — and let `t ∈ T` satisfy the conjunct's other hypothesis, `(E j : 1 ≤ j ≤ #t : tⱼ > 0)`. By T0, every tumbler component lies in ℕ, and NAT-zero's axiom `(A n ∈ ℕ :: 0 ≤ n)` therefore furnishes the lower bound `0 ≤ tⱼ` on each component; with that lower bound, the conditions `tⱼ > 0` and `tⱼ ≠ 0` coincide on ℕ-valued components — the forward direction `tⱼ > 0 ⟹ tⱼ ≠ 0` excludes equality because `0 < tⱼ` is strict, and the converse `tⱼ ≠ 0 ⟹ tⱼ > 0` combines `0 ≤ tⱼ` with `tⱼ ≠ 0` to yield `0 < tⱼ`. The hypothesis is therefore literally `(E j : 1 ≤ j ≤ #t : tⱼ ≠ 0)`, which is the definition of `Pos(t)` recorded in TA-Pos. We may therefore apply TA-Pos's postcondition, instantiated with the current `t` (playing the role of TA-Pos's `t`, since `Pos(t)` holds) and with `z := s` (playing the role of TA-Pos's `z`, since `s`'s zero-tumbler guard is exactly the postcondition's guard on `z`), to obtain `s < t` — which is the conclusion of the conjunct. ∎

*Formal Contract:*
- *Depends:* T0 (CarrierSetDefinition) — Conjunct 1 uses T0's guarantee `#t ≥ 1` for all `t ∈ T` to establish that `t₁` is defined, and Conjunct 2 uses T0's constraint `aᵢ ∈ ℕ` to place every tumbler component in ℕ, which is the precondition under which NAT-zero's lower bound supplies the equivalence `tⱼ > 0 ⟺ tⱼ ≠ 0` required to bridge the hypothesis form to the TA-Pos definition of `Pos(t)`; T0 alone is insufficient, since it supplies no ordering fact about `0`. T1 (LexicographicOrder) — invoked transitively through TA-Pos's postcondition for Conjunct 2; TA-Pos's proof uses T1 case (i) at the first positive position of `t` and T1 case (ii) when `s` is a proper prefix of `t`, and Conjunct 2 inherits this case analysis without repeating it. T4 (HierarchicalParsing) — Conjunct 1 relies on T4's boundary clause `t₁ ≠ 0` to reject zero tumblers as valid addresses. TA-Pos (PositiveTumbler) — Conjunct 2 cites TA-Pos's postcondition `(A t ∈ T, z ∈ T : Pos(t) ∧ (A i : 1 ≤ i ≤ #z : zᵢ = 0) :: z < t)` as the canonical statement of the zero-tumbler-below-positive-tumbler relation; the conjunct is now a restatement of that postcondition, with operand names swapped (TA6's `s` plays the role of TA-Pos's `z`, and TA6's `t` coincides with TA-Pos's `t`), rather than an independent reproof, which prevents drift between the two statements. NAT-zero (NatZeroMinimum) — Conjunct 2 invokes NAT-zero's axiom `(A n ∈ ℕ :: 0 ≤ n)` to license the equivalence `tⱼ > 0 ⟺ tⱼ ≠ 0` on ℕ-valued components required to bridge the TA6 hypothesis form to the TA-Pos `Pos(t)` definition: the lower bound `0 ≤ tⱼ` combines with `tⱼ ≠ 0` to yield `0 < tⱼ` (the converse direction), while strictness of `<` supplies the forward direction. Without NAT-zero, T0's placement of components in ℕ would not rule out `tⱼ < 0`, and the equivalence between `> 0` and `≠ 0` would fail; this matches the citation discipline of the sibling property TA-Pos, whose Case `#z ≥ k` invokes NAT-zero for an analogous step.
- *Postconditions:* (a) `(A t ∈ T : (A i : 1 ≤ i ≤ #t : tᵢ = 0) ⟹ t is not a valid address)`. (b) `(A s, t ∈ T : (A i : 1 ≤ i ≤ #s : sᵢ = 0) ∧ (E j : 1 ≤ j ≤ #t : tⱼ > 0) ⟹ s < t)` — a restatement of TA-Pos's postcondition with operand names swapped; TA-Pos is the canonical site, and any tightening of this statement must be reflected there.

Zero tumblers serve as *sentinels*: they mark uninitialized values, denote "unbounded" when used as span endpoints, and act as lower bounds.
