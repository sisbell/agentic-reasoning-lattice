The precondition `n ÔëÑ 1` promotes to `n Ôëá 0` by a strict-from-`ÔëÑ` chain, the same pattern the sister proofs TumblerAdd and TA-Pos discharge at their analogous sites. NAT-addcompat's strict successor inequality `(A n Ôêê Ôäò :: n < n + 1)` instantiated at `n = 0` supplies `0 < 0 + 1`, i.e., `0 < 1`; NAT-order's defining clause `m Ôëñ n Ôş║ m < n Ôê¿ m = n` instantiated at `m = 1, n = n` unfolds `n ÔëÑ 1` (i.e., `1 Ôëñ n`) into the disjunction `1 < n Ôê¿ 1 = n`; in the first disjunct NAT-order's transitivity of `<` composes `0 < 1` with `1 < n` to yield `0 < n`, and in the second disjunct substitution of `n = 1` into `0 < 1` yields `0 < n` directly; either way `0 < n`, whence `n Ôëá 0` by NAT-order's irreflexivity ÔÇö the contrary assumption `n = 0` would specialise `0 < n` to `0 < 0`, contradicting `┬¼(n < n)`. The promotion is required in two places below: TA-Pos's positivity predicate `(E i : tßÁó Ôëá 0)` requires ╬┤Ôéİ Ôëá 0 as its witness, and ActionPoint's minimum-position set `{i : ╬┤ßÁó Ôëá 0}` requires ╬┤Ôéİ Ôëá 0 to contain m.

By ActionPoint, actionPoint(╬┤(n, m)) = min({i : 1 Ôëñ i Ôëñ m Ôêğ ╬┤(n, m)ßÁó Ôëá 0}); since ╬┤(n, m)ßÁó = 0 for 1 Ôëñ i < m and ╬┤(n, m)Ôéİ = n Ôëá 0, the set `{i : 1 Ôëñ i Ôëñ m Ôêğ ╬┤(n, m)ßÁó Ôëá 0}` equals the singleton `{m}`, whose minimum is m.
