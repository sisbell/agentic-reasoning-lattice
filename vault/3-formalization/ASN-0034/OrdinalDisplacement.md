### Ordinal displacement and shift

**OrdinalDisplacement (OrdinalDisplacement).** For natural number n ≥ 1 and depth m ≥ 1, the *ordinal displacement* δ(n, m) is the tumbler [0, 0, ..., 0, n] of length m — zero at positions 1 through m − 1, and n at position m. Its action point is m.

Since δ(n, m) is a finite sequence of length m ≥ 1 over ℕ, it satisfies the carrier set criterion, so δ(n, m) ∈ T by T0; T0's "each component ∈ ℕ" clause is discharged at the m-th position by the precondition `n ≥ 1` placing `n ∈ ℕ`, and at the leading positions 1..m−1 by NAT-zero's axiom `(A n ∈ ℕ :: 0 ≤ n)`, which quantifies over ℕ and thereby presupposes `0 ∈ ℕ` — the routing T0 itself prescribes for the `0 ∈ ℕ` *membership* of padding positions, and the same routing the sister proof TA-Pos applies at its analogous `0 ≤ tₖ` site. Since n ≥ 1, the m-th component of δ(n, m) is nonzero, whence Pos(δ(n, m)) by TA-Pos. By ActionPoint, actionPoint(δ(n, m)) = min({i : 1 ≤ i ≤ m ∧ δ(n, m)ᵢ ≠ 0}); since δ(n, m)ᵢ = 0 for 1 ≤ i < m and δ(n, m)ₘ = n ≥ 1, the minimum is m. ∎

When the depth is determined by context (typically m = #v for the tumbler being shifted), we write δₙ.

*Formal Contract:*
- *Preconditions:* n ≥ 1, m ≥ 1
- *Definition:* δ(n, m) = [0, 0, …, 0, n] of length m
- *Depends:* T0 (CarrierSetDefinition) — invoked to discharge `δ(n, m) ∈ T` ("Since δ(n, m) is a finite sequence of length m ≥ 1 over ℕ, it satisfies the carrier set criterion, so δ(n, m) ∈ T by T0"), where T0's three-part membership characterisation of T as the set of finite sequences over ℕ with length ≥ 1 supplies the carrier-set criterion. NAT-zero (NatZeroMinimum) — invoked to supply `0 ∈ ℕ` for the m − 1 leading zero components of δ(n, m) at positions 1..m−1, required by T0's "each component ∈ ℕ" clause within the carrier-set criterion. NAT-zero's axiom `(A n ∈ ℕ :: 0 ≤ n)` quantifies `0` as a lower bound of ℕ and thereby presupposes `0 ∈ ℕ`, which is the routing T0 itself explicitly prescribes for the `0 ∈ ℕ` *membership* of padding positions (as distinct from the algebraic identity `0 + n = n`, which T0 reserves to NAT-closure). No other listed NAT-* axiom supplies this membership: NAT-closure's additive identity clause is an equation over ℕ that presupposes, rather than supplies, `0 ∈ ℕ`; NAT-order's trichotomy compares naturals without fixing `0 ∈ ℕ`; and NAT-wellorder names a least element of ℕ without identifying it as `0`. The m-th component's membership `n ∈ ℕ` is supplied directly by the precondition `n ≥ 1` — no NAT-zero citation is required there — so this citation discharges only the m − 1 padding-position zeros, matching the per-step citation discipline the sister proof TA-Pos enforces at its analogous `0 ≤ tₖ` site. TA-Pos (PositiveTumbler) — invoked to discharge `Pos(δ(n, m))` ("Since n ≥ 1, the m-th component of δ(n, m) is nonzero, whence Pos(δ(n, m)) by TA-Pos"), where TA-Pos's positivity predicate is satisfied by the nonzero m-th component. ActionPoint (ActionPoint) — invoked to discharge `actionPoint(δ(n, m)) = m` ("By ActionPoint, actionPoint(δ(n, m)) = min({i : 1 ≤ i ≤ m ∧ δ(n, m)ᵢ ≠ 0}); since δ(n, m)ᵢ = 0 for 1 ≤ i < m and δ(n, m)ₘ = n ≥ 1, the minimum is m"), where ActionPoint's defining minimum-position formula is computed against the explicit component pattern of δ.
- *Postconditions:* δ(n, m) ∈ T (by T0), Pos(δ(n, m)) (by TA-Pos), actionPoint(δ(n, m)) = m (by ActionPoint)
