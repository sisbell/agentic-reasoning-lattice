**TA3 (OrderPreservationUnderSubtractionWeak).** `(A a, b, w : a < b ∧ a ≥ w ∧ b ≥ w : a ⊖ w ≤ b ⊖ w)`.

*Proof.* We must show that for all `a, b, w ∈ T` with `a < b`, `a ≥ w`, and `b ≥ w`, the inequality `a ⊖ w ≤ b ⊖ w` holds under T1.

By TA2, since `a ≥ w` and `b ≥ w`, both `a ⊖ w` and `b ⊖ w` are well-formed tumblers in `T`, so the comparison is well-defined. We recall the subtraction rule (TumblerSub) for self-containment: given `x ≥ w`, zero-pad both operands to a common length `L_{x,w}` — the longer of `#x` and `#w`, named by NAT-order's trichotomy on the length pair `(#x, #w)` (sub-case (α) `#x = #w` with `L_{x,w} = #x`; sub-case (β) `#x < #w` with `L_{x,w} = #w`; sub-case (γ) `#w < #x` with `L_{x,w} = #x`) rather than by a primitive binary-maximum operator on ℕ, reproducing TumblerSub's own length-pair dispatch — and scan for the first position at which the padded sequences disagree. If no such position exists (we say `x` is *zero-padded-equal* to `w`), then `x ⊖ w` is the zero tumbler of length `L_{x,w}`. Otherwise, let `d` be the first divergence position; then `(x ⊖ w)ᵢ = 0` for `i < d`, `(x ⊖ w)_d = x_d - w_d`, and `(x ⊖ w)ᵢ = xᵢ` for `i > d` (all under zero-padding, where `xᵢ = 0` for `i > #x` and `wᵢ = 0` for `i > #w` with the literal `0` supplied as an element of ℕ by NAT-zero so that the padded values extend the operands' native ℕ-valued component-typing onto the padding region, and under the same convention every literal-`0` result component TumblerSub produces — in the zero-tumbler no-divergence branch and at pre-divergence positions `i < d` — is ℕ-valued by NAT-zero, matching TumblerSub's Definition and the per-step convention ZPD, TA2, ActionPoint, and TA-Pos follow for discharging identical `0 ∈ ℕ` premises), with result length `L_{x,w}`.

Since `a < b`, T1 provides two cases: either (i) there exists a first position `j` with `j ≤ #a ∧ j ≤ #b` (the explicit conjunction replacing `j ≤ min(#a, #b)` so that no primitive binary-minimum operator on ℕ is invoked, matching T1's case-(i) definition, Divergence's case (i), TumblerSub's precondition-consequence case (i), ZPD's Relationship-to-Divergence, and TA2's sub-case (i)) where `aⱼ < bⱼ`, or (ii) `a` is a proper prefix of `b` — `#a < #b` and `aᵢ = bᵢ` for all `i ≤ #a`. We treat each in turn, partitioning further by the divergence structure of the operands against `w`.

**Case A: `a` is a proper prefix of `b`** (T1 case (ii)). Then `#a < #b` and `aᵢ = bᵢ` for all `i ≤ #a`.

*Sub-case A1: `a = w`.* Since the operands are identical, NAT-order's trichotomy on `(#a, #w)` places the pair in sub-case (α) `#a = #w`, giving `L_{a,w} = #a`; the subtraction `a ⊖ w` yields the zero tumbler of that length. Since `a` is a proper prefix of `b` and `a = w`, we have `bᵢ = wᵢ` for all `i ≤ #w = #a`. If some component `bᵢ` with `i > #w` is nonzero, the pair `(b, w)` has a divergence beyond `#w`, making `b ⊖ w` a positive tumbler; by TA6 the zero tumbler `a ⊖ w` is strictly less. If `bᵢ = 0` for all `i > #w`, the zero-padded sequences of `b` and `w` agree everywhere, and under `#w = #a < #b` NAT-order's trichotomy on `(#b, #w)` places the pair in sub-case (γ) `#w < #b`, giving `L_{b,w} = #b`; so `b ⊖ w` is the zero tumbler of length `#b`. Both results are zero tumblers, but `#(a ⊖ w) = #a < #b = #(b ⊖ w)`, so `a ⊖ w` is a proper prefix of `b ⊖ w`, giving `a ⊖ w < b ⊖ w` by T1 case (ii).

*Sub-case A2: `a > w` with divergence.* Let `dₐ` be the first position where the zero-padded sequences of `a` and `w` disagree. We claim `dₐ ≤ #a`: if `a > w` by T1 case (i), T1 case (i) supplies `dₐ ≤ #a ∧ dₐ ≤ #w` (the explicit conjunction replacing `dₐ ≤ min(#a, #w)`), whose first conjunct `dₐ ≤ #a` is the bound claimed; if by T1 case (ii), `w` is a proper prefix of `a` and `dₐ` is the first `i > #w` with `aᵢ > 0`, so `dₐ ≤ #a`. Since `bᵢ = aᵢ` for all `i ≤ #a` and `dₐ ≤ #a`, the comparison of `b` against `w` (under zero-padding) agrees with that of `a` at all positions up through `dₐ`. So `d_b = dₐ = d`.

Apply TumblerSub to both. At positions `i < d`: both results are zero. At position `d`: both compute `a_d - w_d = b_d - w_d` — each side a natural number by NAT-sub's conditional-closure clause under `a_d ≥ w_d` and `b_d ≥ w_d` (from the divergence-point inequalities via NAT-order), and equal because `a_d = b_d` for `d ≤ #a` feeds identical arguments into the same partial operation. At positions `d < i ≤ #a`: both copy from their respective minuends, giving `aᵢ = bᵢ`. The two results agree on positions `1, ..., #a`.

Beyond position `#a`, the results may differ in both components and length. Denote the result lengths `L_{a,w}` and `L_{b,w}`, named by NAT-order's trichotomy on `(#a, #w)` and `(#b, #w)` respectively per the recall above. We first establish `L_{a,w} ≤ L_{b,w}` by enumerating NAT-order's trichotomy on `(#a, #w)`. In sub-case (α) `#a = #w` or (γ) `#w < #a`, `L_{a,w} = #a`; combined with the case hypothesis `#a < #b`, this places `(#b, #w)` in sub-case (γ) `#w < #b` (since `#w ≤ #a < #b`), giving `L_{b,w} = #b`, whence `L_{a,w} = #a < #b = L_{b,w}`. In sub-case (β) `#a < #w`, `L_{a,w} = #w`; NAT-order's trichotomy on `(#b, #w)` dispatches further into (γ) `#w < #b` with `L_{b,w} = #b > #w = L_{a,w}`, (α) `#b = #w` with `L_{b,w} = #w = L_{a,w}`, or (β) `#b < #w` with `L_{b,w} = #w = L_{a,w}`. In every combination, `L_{a,w} ≤ L_{b,w}`.

At positions `#a < i ≤ L_{a,w}` (present only in sub-case (β) of `(#a, #w)`, where `L_{a,w} = #w > #a`): `(a ⊖ w)ᵢ = 0` from `a`'s zero-padding, while `(b ⊖ w)ᵢ = bᵢ` if `i ≤ #b` (copied from the minuend since `i > d = dₐ ≤ #a < i`) and `0` if `i > #b` (from `b`'s zero-padding); in either case `(a ⊖ w)ᵢ ≤ (b ⊖ w)ᵢ`.

If a first disagreement exists at some position `p` on `1, ..., L_{a,w}`, then `p > #a` (since the results agree on `1, ..., #a`), and `(a ⊖ w)_p = 0` while `(b ⊖ w)_p ≠ 0` (since they disagree and tumbler components are non-negative, `(b ⊖ w)_p > 0`); `p` satisfies `p ≤ L_{a,w} ∧ p ≤ L_{b,w}` (the explicit conjunction, its first conjunct holding by construction and its second by `L_{a,w} ≤ L_{b,w}` just established), so T1 case (i) yields `a ⊖ w < b ⊖ w`. If no disagreement exists on `1, ..., L_{a,w}`, then `(a ⊖ w)ᵢ = (b ⊖ w)ᵢ` at every such `i`. NAT-order's trichotomy at `(L_{a,w}, L_{b,w})` applied with the already-established `L_{a,w} ≤ L_{b,w}` fixes one of `L_{a,w} < L_{b,w}` or `L_{a,w} = L_{b,w}`. If `L_{a,w} < L_{b,w}`, `a ⊖ w` is a proper prefix of `b ⊖ w`, giving `a ⊖ w < b ⊖ w` by T1 case (ii). If `L_{a,w} = L_{b,w}`, both results have the same length and agree at every position, so `a ⊖ w = b ⊖ w` by T3, and `a ⊖ w ≤ b ⊖ w` holds by equality.

*Sub-case A3: `a > w` without divergence (zero-padded equality).* Since `a > w` requires a structural difference yet the padded sequences agree everywhere, the only possibility is T1 case (ii): `w` is a proper prefix of `a` with `aᵢ = 0` for all `i > #w`. Under `#w < #a`, NAT-order's trichotomy on `(#a, #w)` places the pair in sub-case (γ), giving `L_{a,w} = #a`; the subtraction `a ⊖ w` yields the zero tumbler of that length. Since `b > a > w` and `#b > #a > #w` (combining Case A's `#a < #b` with this sub-case's `#w < #a`), `b` agrees with `w` (hence with `a`) on positions `1, ..., #a`. NAT-order's trichotomy on `(#b, #w)` under `#w < #a < #b` is sub-case (γ) with `L_{b,w} = #b > #a`. If `b ⊖ w` has any positive component, then `a ⊖ w` (all zeros) is strictly less by TA6. If `b ⊖ w` is also a zero tumbler, `#(b ⊖ w) = #b > #a = #(a ⊖ w)`, so the shorter is a proper prefix of the longer, giving `a ⊖ w < b ⊖ w` by T1 case (ii).

In all sub-cases of Case A, `a ⊖ w ≤ b ⊖ w`.

**Case B: Component divergence at `j`** (T1 case (i)). There exists a first position `j` with `j ≤ #a ∧ j ≤ #b` (the explicit conjunction replacing `j ≤ min(#a, #b)`, matching T1's case-(i) definition), `aⱼ < bⱼ`, and `aᵢ = bᵢ` for all `i < j`.

*Sub-case B1: `a` is zero-padded-equal to `w`.* The subtraction `a ⊖ w` is the zero tumbler of length `L_{a,w}`. At position `j`, zero-padded equality gives `wⱼ = aⱼ`, so `bⱼ > aⱼ = wⱼ`. The pair `(b, w)` diverges at or before `j`, making `b ⊖ w` positive. By TA6, `a ⊖ w < b ⊖ w`.

For the remaining sub-cases, `a` is not zero-padded-equal to `w`, so `dₐ = zpd(a, w)` — TumblerSub's zero-padded divergence, distinct from the formal Divergence — is well-defined. The zero-padded divergence `d_b = zpd(b, w)` is also well-defined: if `b` were zero-padded-equal to `w`, then at position `dₐ` we would have `a_{dₐ} > w_{dₐ} = b_{dₐ}` (from `a ≥ w` at the zero-padded divergence), while `aᵢ = wᵢ = bᵢ` for all `i < dₐ`. Since Case B provides `j` with `j ≤ #a ∧ j ≤ #b` and `aⱼ ≠ bⱼ`, the pre-`dₐ` agreement between `a` and `b` forces `dₐ ≤ j`; combined with Case B's `j ≤ #a ∧ j ≤ #b`, this gives `dₐ ≤ #a ∧ dₐ ≤ #b`, making `dₐ` a T1 case (i) witness for `a > b` (with the required shared-position bound in conjunction form) — contradicting `a < b`. Let `j` be the first position where `aⱼ < bⱼ`.

*Sub-case B2: `dₐ = d_b = d`.* Both operands diverge from `w` at the same position. For `i < d`, both results are zero. Since `a` and `b` agree with `w` before `d`, and `aⱼ < bⱼ`, we have `j ≥ d`. If `j = d`: NAT-sub's strict monotonicity applied to `a_d < b_d` with `a_d ≥ w_d` and `b_d ≥ w_d` (both supplied by the setup above via NAT-order) yields `a_d - w_d < b_d - w_d`, so `a ⊖ w < b ⊖ w` by T1 case (i). If `j > d`: `a_d = b_d` (since the first `a`-vs-`b` disagreement is at `j > d`), so both results agree at position `d`; at positions `d < i < j`, both copy from their minuends which agree (`aᵢ = bᵢ`); at position `j`, `(a ⊖ w)ⱼ = aⱼ < bⱼ = (b ⊖ w)ⱼ` since both are in the tail-copy phase (`j > d`). By T1 case (i), `a ⊖ w < b ⊖ w`.

*Sub-case B3: `dₐ < d_b`.* At position `dₐ`, `a_{dₐ} ≠ w_{dₐ}` but `b_{dₐ} = w_{dₐ}`. Since both `a` and `b` agree with `w` at all positions before `dₐ`, `aᵢ = bᵢ` for `i < dₐ`; Case B's `j` with `j ≤ #a ∧ j ≤ #b` and `aⱼ ≠ bⱼ` then forces `dₐ ≤ j` and hence `dₐ ≤ #a ∧ dₐ ≤ #b` (via the conjunction), so the first disagreement between `a` and `b` is at `dₐ`, giving `j = dₐ` with `a_{dₐ} < b_{dₐ} = w_{dₐ}`. But `a ≥ w` requires `a_{dₐ} ≥ w_{dₐ}` at the divergence — contradiction. This case is impossible under the preconditions.

*Sub-case B4: `dₐ > d_b`.* At position `d_b`, `b_{d_b} ≠ w_{d_b}` but `a_{d_b} = w_{d_b}`. Since both `a` and `b` agree with `w` before `d_b`, the first disagreement between `a` and `b` is at `d_b`, giving `j = d_b` with `a_{d_b} = w_{d_b} < b_{d_b}` — the inequality holds because `b ≥ w` forces `b_{d_b} > w_{d_b}` at this divergence. The result `(a ⊖ w)_{d_b} = 0` since `d_b < dₐ` falls in the pre-divergence zero phase for `a ⊖ w`. The result `(b ⊖ w)_{d_b} = b_{d_b} - w_{d_b} > 0` by NAT-sub's strict-positivity clause, whose precondition `b_{d_b} > w_{d_b}` is the strict inequality just derived. At all positions `i < d_b`, both results are zero. By T1 case (i), `a ⊖ w < b ⊖ w`.

In every case, `a ⊖ w ≤ b ⊖ w`. ∎
