**TA3 (OrderPreservationUnderSubtractionWeak).** `(A a, b, w : a < b ∧ a ≥ w ∧ b ≥ w : a ⊖ w ≤ b ⊖ w)`.

*Proof.* We must show that for all `a, b, w ∈ T` with `a < b`, `a ≥ w`, and `b ≥ w`, the inequality `a ⊖ w ≤ b ⊖ w` holds under T1.

By TA2, since `a ≥ w` and `b ≥ w`, both `a ⊖ w` and `b ⊖ w` are well-formed tumblers in `T`, so the comparison is well-defined. We recall the subtraction rule (TumblerSub) for self-containment: given `x ≥ w`, zero-pad both operands to a common length `L_{x,w}` — the longer of `#x` and `#w`, named by NAT-order's trichotomy on the length pair `(#x, #w)` (sub-case (α) `#x = #w` with `L_{x,w} = #x`; sub-case (β) `#x < #w` with `L_{x,w} = #w`; sub-case (γ) `#w < #x` with `L_{x,w} = #x`) rather than by a primitive binary-maximum operator on ℕ, reproducing TumblerSub's own length-pair dispatch — and scan for the first position at which the padded sequences disagree. If no such position exists (we say `x` is *zero-padded-equal* to `w`), then `x ⊖ w` is the zero tumbler of length `L_{x,w}`. Otherwise, let `d` be the first divergence position; then `(x ⊖ w)ᵢ = 0` for `i < d`, `(x ⊖ w)_d = x_d - w_d`, and `(x ⊖ w)ᵢ = xᵢ` for `i > d` (all under zero-padding, where `xᵢ = 0` for `i > #x` and `wᵢ = 0` for `i > #w` with the literal `0` supplied as an element of ℕ by NAT-zero so that the padded values extend the operands' native ℕ-valued component-typing onto the padding region, and under the same convention every literal-`0` result component TumblerSub produces — in the zero-tumbler no-divergence branch and at pre-divergence positions `i < d` — is ℕ-valued by NAT-zero, matching TumblerSub's Definition and the per-step convention ZPD, TA2, ActionPoint, and TA-Pos follow for discharging identical `0 ∈ ℕ` premises), with result length `L_{x,w}`.

Since `a < b`, T1 provides two cases: either (i) there exists a first position `j` with `j ≤ #a ∧ j ≤ #b` (the explicit conjunction replacing `j ≤ min(#a, #b)` so that no primitive binary-minimum operator on ℕ is invoked, matching T1's case-(i) definition, Divergence's case (i), TumblerSub's precondition-consequence case (i), ZPD's Relationship-to-Divergence, and TA2's sub-case (i)) where `aⱼ < bⱼ`, or (ii) `a` is a proper prefix of `b` — `#a < #b` and `aᵢ = bᵢ` for all `i ≤ #a`. We treat each in turn, partitioning further by the divergence structure of the operands against `w`.

**Case A: `a` is a proper prefix of `b`** (T1 case (ii)). Then `#a < #b` and `aᵢ = bᵢ` for all `i ≤ #a`.

*Sub-case A1: `a = w`.* Since the operands are identical, NAT-order's trichotomy on `(#a, #w)` places the pair in sub-case (α) `#a = #w`, giving `L_{a,w} = #a`; the subtraction `a ⊖ w` yields the zero tumbler of that length. Since `a` is a proper prefix of `b` and `a = w`, we have `bᵢ = wᵢ` for all `i ≤ #w = #a`. If some component `bᵢ` with `i > #w` is nonzero, the pair `(b, w)` has a divergence beyond `#w`, making `b ⊖ w` a positive tumbler; by TA6 the zero tumbler `a ⊖ w` is strictly less. If `bᵢ = 0` for all `i > #w`, the zero-padded sequences of `b` and `w` agree everywhere, and under `#w = #a < #b` NAT-order's trichotomy on `(#b, #w)` places the pair in sub-case (γ) `#w < #b`, giving `L_{b,w} = #b`; so `b ⊖ w` is the zero tumbler of length `#b`. Both results are zero tumblers, so `(a ⊖ w)ᵢ = 0 = (b ⊖ w)ᵢ` at every position `1 ≤ i ≤ #(a ⊖ w) = #a`; the strict length inequality `#(a ⊖ w) = #a < #b = #(b ⊖ w)` is converted by NAT-discrete's forward direction `m < n ⟹ m + 1 ≤ n` at the length pair `(#a, #b)` into the successor bound `#(a ⊖ w) + 1 = #a + 1 ≤ #b = #(b ⊖ w)`, supplying T1 case (ii)'s witness `k = #(a ⊖ w) + 1 ≤ #(b ⊖ w)`. Thus `a ⊖ w` is a proper prefix of `b ⊖ w`, giving `a ⊖ w < b ⊖ w` by T1 case (ii).

*Sub-case A2: `a > w` with divergence.* Let `dₐ` be the first position where the zero-padded sequences of `a` and `w` disagree. We claim `dₐ ≤ #a`: if `a > w` by T1 case (i), T1 case (i) supplies `dₐ ≤ #a ∧ dₐ ≤ #w` (the explicit conjunction replacing `dₐ ≤ min(#a, #w)`), whose first conjunct `dₐ ≤ #a` is the bound claimed; if by T1 case (ii), `w` is a proper prefix of `a` and `dₐ` is the first `i > #w` with `aᵢ > 0`, so `dₐ ≤ #a`. Since `bᵢ = aᵢ` for all `i ≤ #a` and `dₐ ≤ #a`, the comparison of `b` against `w` (under zero-padding) agrees with that of `a` at all positions up through `dₐ`. So `d_b = dₐ = d`.

Apply TumblerSub to both. At positions `i < d`: both results are zero. At position `d`: both compute `a_d - w_d = b_d - w_d` — each side a natural number by NAT-sub's conditional-closure clause under `a_d ≥ w_d` and `b_d ≥ w_d` (from the divergence-point inequalities via NAT-order), and equal because `a_d = b_d` for `d ≤ #a` feeds identical arguments into the same partial operation. At positions `d < i ≤ #a`: both copy from their respective minuends, giving `aᵢ = bᵢ`. The two results agree on positions `1, ..., #a`.

Beyond position `#a`, the results may differ in both components and length. Denote the result lengths `L_{a,w}` and `L_{b,w}`, named by NAT-order's trichotomy on `(#a, #w)` and `(#b, #w)` respectively per the recall above. We first establish `L_{a,w} ≤ L_{b,w}` by enumerating NAT-order's trichotomy on `(#a, #w)`. In sub-case (α) `#a = #w` or (γ) `#w < #a`, `L_{a,w} = #a`; combined with the case hypothesis `#a < #b`, this places `(#b, #w)` in sub-case (γ) `#w < #b` (since `#w ≤ #a < #b`), giving `L_{b,w} = #b`, whence `L_{a,w} = #a < #b = L_{b,w}`. In sub-case (β) `#a < #w`, `L_{a,w} = #w`; NAT-order's trichotomy on `(#b, #w)` dispatches further into (γ) `#w < #b` with `L_{b,w} = #b > #w = L_{a,w}`, (α) `#b = #w` with `L_{b,w} = #w = L_{a,w}`, or (β) `#b < #w` with `L_{b,w} = #w = L_{a,w}`. In every combination, `L_{a,w} ≤ L_{b,w}`.

At positions `#a < i ≤ L_{a,w}` (present only in sub-case (β) of `(#a, #w)`, where `L_{a,w} = #w > #a`): `(a ⊖ w)ᵢ = 0` from `a`'s zero-padding, while `(b ⊖ w)ᵢ = bᵢ` if `i ≤ #b` (copied from the minuend since `i > d = dₐ ≤ #a < i`) and `0` if `i > #b` (from `b`'s zero-padding); in either case `(a ⊖ w)ᵢ ≤ (b ⊖ w)ᵢ`.

If a first disagreement exists at some position `p` on `1, ..., L_{a,w}`, then `p > #a` (since the results agree on `1, ..., #a`), and `(a ⊖ w)_p = 0` while `(b ⊖ w)_p ≠ 0`. The step `(b ⊖ w)_p ≠ 0 ⟹ (b ⊖ w)_p > 0` is discharged by NAT-zero's lower bound `0 ≤ (b ⊖ w)_p` (instantiated at `n = (b ⊖ w)_p`, with `(b ⊖ w)_p ∈ ℕ` supplied by TA2's established membership `b ⊖ w ∈ T` — TA2's Case 2 proof places each result component in ℕ, inherited here without repeating the derivation), NAT-order's defining clause `m ≤ n ⟺ m < n ∨ m = n` at `m = 0, n = (b ⊖ w)_p` unfolding `0 ≤ (b ⊖ w)_p` to `0 < (b ⊖ w)_p ∨ 0 = (b ⊖ w)_p`, and the first-disagreement observation `(b ⊖ w)_p ≠ 0` excluding the equality disjunct to leave `0 < (b ⊖ w)_p`; hence `(b ⊖ w)_p > 0`, matching the per-step convention TumblerSub's precondition-consequence case (ii), TA2's sub-case (ii), ActionPoint's third postcondition, and TA6's Conjunct 2 apply for the structurally identical `≠ 0 ⟹ > 0` step. Position `p` satisfies `p ≤ L_{a,w} ∧ p ≤ L_{b,w}` (the explicit conjunction, its first conjunct holding by construction and its second by `L_{a,w} ≤ L_{b,w}` just established), so T1 case (i) — consuming the strict component inequality `(a ⊖ w)_p = 0 < (b ⊖ w)_p` at the shared position `p` — yields `a ⊖ w < b ⊖ w`. If no disagreement exists on `1, ..., L_{a,w}`, then `(a ⊖ w)ᵢ = (b ⊖ w)ᵢ` at every such `i`. NAT-order's trichotomy at `(L_{a,w}, L_{b,w})` applied with the already-established `L_{a,w} ≤ L_{b,w}` fixes one of `L_{a,w} < L_{b,w}` or `L_{a,w} = L_{b,w}`. If `L_{a,w} < L_{b,w}`, NAT-discrete's forward direction `m < n ⟹ m + 1 ≤ n` at the length pair `(L_{a,w}, L_{b,w})` converts this strict inequality into the successor bound `L_{a,w} + 1 ≤ L_{b,w}`, supplying T1 case (ii)'s witness `k = L_{a,w} + 1 ≤ L_{b,w}`; combined with the component-wise agreement `(a ⊖ w)ᵢ = (b ⊖ w)ᵢ` at every position `1 ≤ i ≤ L_{a,w}` just established, `a ⊖ w` is a proper prefix of `b ⊖ w`, giving `a ⊖ w < b ⊖ w` by T1 case (ii). If `L_{a,w} = L_{b,w}`, both results have the same length and agree at every position, so `a ⊖ w = b ⊖ w` by T3, and `a ⊖ w ≤ b ⊖ w` holds by equality.

*Sub-case A3: `a > w` without divergence (zero-padded equality).* Since `a > w` requires a structural difference yet the padded sequences agree everywhere, the only possibility is T1 case (ii): `w` is a proper prefix of `a` with `aᵢ = 0` for all `i > #w`. Under `#w < #a`, NAT-order's trichotomy on `(#a, #w)` places the pair in sub-case (γ), giving `L_{a,w} = #a`; the subtraction `a ⊖ w` yields the zero tumbler of that length. Since `b > a > w` and `#b > #a > #w` (combining Case A's `#a < #b` with this sub-case's `#w < #a`), `b` agrees with `w` (hence with `a`) on positions `1, ..., #a`. NAT-order's trichotomy on `(#b, #w)` under `#w < #a < #b` is sub-case (γ) with `L_{b,w} = #b > #a`. If `b ⊖ w` has any positive component, then `a ⊖ w` (all zeros) is strictly less by TA6. If `b ⊖ w` is also a zero tumbler, both results have all-zero components, so `(a ⊖ w)ᵢ = 0 = (b ⊖ w)ᵢ` at every position `1 ≤ i ≤ #(a ⊖ w) = #a`; the strict length inequality `#(a ⊖ w) = #a < #b = #(b ⊖ w)` is converted by NAT-discrete's forward direction `m < n ⟹ m + 1 ≤ n` at the length pair `(#a, #b)` into the successor bound `#(a ⊖ w) + 1 = #a + 1 ≤ #b = #(b ⊖ w)`, supplying T1 case (ii)'s witness `k = #(a ⊖ w) + 1 ≤ #(b ⊖ w)`. Thus the shorter is a proper prefix of the longer, giving `a ⊖ w < b ⊖ w` by T1 case (ii).

In all sub-cases of Case A, `a ⊖ w ≤ b ⊖ w`.

**Case B: Component divergence at `j`** (T1 case (i)). There exists a first position `j` with `j ≤ #a ∧ j ≤ #b` (the explicit conjunction replacing `j ≤ min(#a, #b)`, matching T1's case-(i) definition), `aⱼ < bⱼ`, and `aᵢ = bᵢ` for all `i < j`.

*Sub-case B1: `a` is zero-padded-equal to `w`.* The subtraction `a ⊖ w` is the zero tumbler of length `L_{a,w}`. At position `j`, zero-padded equality gives `wⱼ = aⱼ`, so `bⱼ > aⱼ = wⱼ`. The pair `(b, w)` diverges at or before `j`, making `b ⊖ w` positive. By TA6, `a ⊖ w < b ⊖ w`.

For the remaining sub-cases, `a` is not zero-padded-equal to `w`, so `dₐ = zpd(a, w)` — TumblerSub's zero-padded divergence, distinct from the formal Divergence — is well-defined. The zero-padded divergence `d_b = zpd(b, w)` is also well-defined: if `b` were zero-padded-equal to `w`, then at position `dₐ` we would have `a_{dₐ} > w_{dₐ} = b_{dₐ}` (from `a ≥ w` at the zero-padded divergence), while `aᵢ = wᵢ = bᵢ` for all `i < dₐ`. Since Case B provides `j` with `j ≤ #a ∧ j ≤ #b` and `aⱼ ≠ bⱼ`, the pre-`dₐ` agreement between `a` and `b` forces `dₐ ≤ j`; combined with Case B's `j ≤ #a ∧ j ≤ #b`, this gives `dₐ ≤ #a ∧ dₐ ≤ #b`, making `dₐ` a T1 case (i) witness for `a > b` (with the required shared-position bound in conjunction form) — contradicting `a < b`. Let `j` be the first position where `aⱼ < bⱼ`.

*Sub-case B2: `dₐ = d_b = d`.* Both operands diverge from `w` at the same position. For `i < d`, both results are zero. Since `a` and `b` agree with `w` before `d`, and `aⱼ < bⱼ`, we have `j ≥ d`. If `j = d`: NAT-sub's strict monotonicity applied to `a_d < b_d` with `a_d ≥ w_d` and `b_d ≥ w_d` (both supplied by the setup above via NAT-order) yields `a_d - w_d < b_d - w_d`, so `a ⊖ w < b ⊖ w` by T1 case (i). If `j > d`: `a_d = b_d` (since the first `a`-vs-`b` disagreement is at `j > d`), so both results agree at position `d`; at positions `d < i < j`, both copy from their minuends which agree (`aᵢ = bᵢ`); at position `j`, `(a ⊖ w)ⱼ = aⱼ < bⱼ = (b ⊖ w)ⱼ` since both are in the tail-copy phase (`j > d`). By T1 case (i), `a ⊖ w < b ⊖ w`.

*Sub-case B3: `dₐ < d_b`.* At position `dₐ`, `a_{dₐ} ≠ w_{dₐ}` but `b_{dₐ} = w_{dₐ}`. Since both `a` and `b` agree with `w` at all positions before `dₐ`, `aᵢ = bᵢ` for `i < dₐ`; Case B's `j` with `j ≤ #a ∧ j ≤ #b` and `aⱼ ≠ bⱼ` then forces `dₐ ≤ j` and hence `dₐ ≤ #a ∧ dₐ ≤ #b` (via the conjunction), so the first disagreement between `a` and `b` is at `dₐ`, giving `j = dₐ` with `a_{dₐ} < b_{dₐ} = w_{dₐ}`. But `a ≥ w` requires `a_{dₐ} ≥ w_{dₐ}` at the divergence — contradiction. This case is impossible under the preconditions.

*Sub-case B4: `dₐ > d_b`.* At position `d_b`, `b_{d_b} ≠ w_{d_b}` but `a_{d_b} = w_{d_b}`. Since both `a` and `b` agree with `w` before `d_b`, the first disagreement between `a` and `b` is at `d_b`, giving `j = d_b` with `a_{d_b} = w_{d_b} < b_{d_b}` — the inequality holds because `b ≥ w` forces `b_{d_b} > w_{d_b}` at this divergence. The result `(a ⊖ w)_{d_b} = 0` since `d_b < dₐ` falls in the pre-divergence zero phase for `a ⊖ w`. The result `(b ⊖ w)_{d_b} = b_{d_b} - w_{d_b} > 0` by NAT-sub's strict-positivity clause, whose precondition `b_{d_b} > w_{d_b}` is the strict inequality just derived. At all positions `i < d_b`, both results are zero. By T1 case (i), `a ⊖ w < b ⊖ w`.

In every case, `a ⊖ w ≤ b ⊖ w`. ∎

NAT-zero (NatZeroMinimum) — the literal `0` appears at multiple ASN-internal sites in TA3, each requiring `0 ∈ ℕ` so that (a) the padded component values `xᵢ = 0` for `i > #x` and `wᵢ = 0` for `i > #w` supplied by TumblerSub's zero-padding stipulation at the operands `a`, `b`, `w` of the subtractions `a ⊖ w` and `b ⊖ w`, and (b) the literal-`0` result components TumblerSub's piecewise Definition produces at pre-divergence positions `i < d` and in the zero-tumbler no-divergence branch, respect the ASN's ℕ-valued component-typing requirement when consumed in ℕ-valued (in)equalities and serve as valid subtrahends for NAT-sub's conditional-closure and strict-positivity clauses. The operands' native component-typing supplies membership in ℕ only for existing positions `1 ≤ i ≤ #a`, `1 ≤ i ≤ #b`, `1 ≤ i ≤ #w`, so neither the padded values at positions beyond these native domains nor the literal-`0` result components introduced by TumblerSub's Definition are covered; among the remaining NAT-* axioms cited in TA3's Depends (NAT-sub, NAT-order) none identifies which elements belong to ℕ — each quantifies over ℕ presupposing its extension rather than fixing it. NAT-zero's axiom `(A n ∈ ℕ :: 0 ≤ n)` quantifies `0` as a lower bound of ℕ and thereby presupposes `0 ∈ ℕ` as a member of its quantification domain, supplying exactly the missing membership premise at every site — this citation discharges the literal `0` from NAT-zero rather than leaving `0 ∈ ℕ` implicit, matching the per-step convention TumblerSub's Definition, ZPD, TA-Pos, ActionPoint, and TA2 already follow for discharging identical `0`-related premises from NAT-zero rather than from the operand-level ℕ-valued component-typing. (i) *Sub-case A1 — zero-tumbler results and trailing-zero test*: under `a = w`, TumblerSub's no-divergence branch produces `a ⊖ w = [0, …, 0]` of length `L_{a,w} = #a`, whose every component is the literal `0 ∈ ℕ`; the case split on `b`'s trailing components — "If some component `bᵢ` with `i > #w` is nonzero" versus "If `bᵢ = 0` for all `i > #w`" — consumes the ℕ-valued (in)equality `bᵢ = 0` (resp. `bᵢ ≠ 0`) at `i > #w`, where for `#w < i ≤ #b` the literal-`0` right operand requires `0 ∈ ℕ` and for `i > #b` the value `bᵢ = 0` is itself supplied by TumblerSub's zero-padding; the second branch's observation that the zero-padded sequences of `b` and `w` agree everywhere pairs `bᵢ = 0` with the padded `wᵢ = 0` at every `i > #w`, each side requiring `0 ∈ ℕ`; and the conclusion that `b ⊖ w` is the zero tumbler of length `#b` produces literal-`0` components throughout positions `1, …, #b`, each requiring `0 ∈ ℕ`. (ii) *Sub-case A2 — tail-position result components at `#a < i ≤ L_{a,w}` and first-disagreement lower-bound premise*: in the sub-case (β) branch of `(#a, #w)` where `L_{a,w} = #w > #a`, the tail-position equality `(a ⊖ w)ᵢ = 0` at `#a < i ≤ L_{a,w}` is the literal `0` produced by TumblerSub's tail-copy clause applied to the zero-padded minuend `aᵢ = 0` at `i > #a`, and the companion clause `(b ⊖ w)ᵢ = 0` when `i > #b` follows the same pattern — both requiring `0 ∈ ℕ` so that the pointwise inequality `(a ⊖ w)ᵢ ≤ (b ⊖ w)ᵢ` remains a well-formed ℕ-valued comparison; the first-disagreement analysis's observation "`(a ⊖ w)_p = 0` while `(b ⊖ w)_p ≠ 0`" consumes the same literal `0` against a nonzero operand, and the step deriving `(b ⊖ w)_p > 0` from `(b ⊖ w)_p ≠ 0` additionally invokes NAT-zero's lower-bound axiom `(A n ∈ ℕ :: 0 ≤ n)` at `n = (b ⊖ w)_p` (with `(b ⊖ w)_p ∈ ℕ` supplied by TA2's established membership `b ⊖ w ∈ T`) to furnish the premise `0 ≤ (b ⊖ w)_p` consumed by NAT-order's defining-clause unfolding at the same component pair (enumerated as the sixth role in NAT-order's Depends entry below) that derives `(b ⊖ w)_p > 0` from `(b ⊖ w)_p ≠ 0`, matching the per-step convention TumblerSub's precondition-consequence case (ii), TA2's sub-case (ii), ActionPoint's third postcondition, and TA6's Conjunct 2 apply for the structurally identical `≠ 0 ⟹ > 0` step — NAT-order alone (whose trichotomy compares `0` and `(b ⊖ w)_p` without fixing the winner) does not furnish this premise. (iii) *Sub-case A3 — trailing-zero hypothesis and zero-tumbler results*: the sub-case hypothesis `aᵢ = 0` for `i > #w` consumes the literal `0` at positions `#w < i ≤ #a` (where `aᵢ` is a native component in ℕ compared to literal `0 ∈ ℕ`) and at `i > #a` (where `aᵢ = 0` is itself supplied by TumblerSub's zero-padding); TumblerSub's no-divergence branch then produces `a ⊖ w = [0, …, 0]` of length `L_{a,w} = #a`, and the parallel alternative "`b ⊖ w` is also a zero tumbler" produces `b ⊖ w = [0, …, 0]` of length `L_{b,w} = #b` — every literal-`0` component of both results requiring `0 ∈ ℕ`. (iv) *Sub-case B1 — zero-tumbler result and padded equality at `j`*: under `a` zero-padded-equal to `w`, TumblerSub's no-divergence branch produces `a ⊖ w = [0, …, 0]` of length `L_{a,w}`, every component literal `0 ∈ ℕ`; the ℕ-valued equality `wⱼ = aⱼ` at the T1 case (i) witness `j` (with `j ≤ #a ∧ j ≤ #b` from Case B's preamble and potentially `j > #w`, in which case `wⱼ = 0` is the padded value) consumes `0 ∈ ℕ`. (v) *B1–B2 bridge — padded comparison at `dₐ`*: the contrapositive argument that `b` is not zero-padded-equal to `w` instantiates the ℕ-valued equality `a_{dₐ} > w_{dₐ} = b_{dₐ}` at `dₐ`, which may fall beyond `#w` (making `w_{dₐ} = 0` padded) or, under the hypothetical zero-padded equality of `b` with `w`, beyond `#b` (making `b_{dₐ} = 0` padded); the ℕ-valued comparisons `w_{dₐ} = b_{dₐ}` and `a_{dₐ} > w_{dₐ}` consume `0 ∈ ℕ` at whichever padded operand provides the literal `0`. (vi) *Sub-cases B2, B3, B4 — divergence-point comparisons with potentially padded subtrahend*: the NAT-sub discharges `a_d - w_d < b_d - w_d` (B2 `j = d` branch), the component-level (in)equalities "`a_{dₐ} ≠ w_{dₐ}` but `b_{dₐ} = w_{dₐ}`" with the derived strict chain `a_{dₐ} < b_{dₐ} = w_{dₐ}` (B3), "`b_{d_b} ≠ w_{d_b}`", "`a_{d_b} = w_{d_b}`", and the derived "`a_{d_b} = w_{d_b} < b_{d_b}`" together with the zero-valued result component `(a ⊖ w)_{d_b} = 0` and the strict-positivity discharge `(b ⊖ w)_{d_b} = b_{d_b} - w_{d_b} > 0` (B4), and the B2 `j > d` branch's agreements `a_d = b_d` and the tail-copy equalities at positions `d < i < j` all rely, whenever the relevant index exceeds the operand's native length, on the padded value `w_d = 0`, `w_{dₐ} = 0`, `w_{d_b} = 0`, `a_{d_b} = 0`, or the literal-`0` pre-divergence result component supplied by TumblerSub's Definition; NAT-sub's conditional-closure and strict-positivity clauses presuppose that the subtrahend lies in ℕ, and the ℕ-valued (in)equalities at these divergence points are well-formed only when `0 ∈ ℕ` is supplied. All six sites are enumerated together under this single NAT-zero citation, matching the per-site accounting TumblerSub (four sites), TA2 (four sites), ZPD, ActionPoint, and TA-Pos apply for discharging identical `0`-related premises from NAT-zero, and matching the per-instance convention by which branches of a case that instantiate the axiom at distinct sub-case patterns count as separate sites. NAT-order (NatStrictTotalOrder) — invoked at six distinct roles, each a clause of NAT-order's Axiom applied at a distinct ℕ-pair, ℕ-triple, or pattern of pairs, replacing uses of primitive binary operators on ℕ that are not declared elsewhere in the Depends list. *Length-pair trichotomy dispatch at `(#a, #w)`*: trichotomy's exactly-one clause selects exactly one of (α) `#a = #w` with `L_{a,w} = #a`, (β) `#a < #w` with `L_{a,w} = #w`, or (γ) `#w < #a` with `L_{a,w} = #a`; `L_{a,w}` — the longer of `#a` and `#w`, which furnishes the result length of `a ⊖ w` in the TumblerSub recall paragraph and in Sub-cases A1, A2, A3, and B1 — is named by the selected sub-case rather than by a primitive binary-maximum operator on ℕ, reproducing TumblerSub's own length-pair dispatch for the same operand pair and matching the convention that every operator used must be discharged by a Depends entry. The same trichotomy is invoked in Sub-case A1 (placing the pair in sub-case (α) since `#a = #w`), in Sub-case A2 (enumerating all three sub-cases to establish `L_{a,w} ≤ L_{b,w}` and to identify where the intermediate-position analysis applies), and in Sub-case A3 (placing the pair in sub-case (γ) since `#w < #a`). *Length-pair trichotomy dispatch at `(#b, #w)`*: the analogous trichotomy at the operand pair `(b, w)` names `L_{b,w}` symmetrically, selecting one of (α) `#b = #w` with `L_{b,w} = #b`, (β) `#b < #w` with `L_{b,w} = #w`, or (γ) `#w < #b` with `L_{b,w} = #b`. Invoked in Sub-case A1 (placing the pair in (γ) since `#w = #a < #b`), in Sub-case A2 (enumerating all three sub-cases in combination with the `(#a, #w)` trichotomy to establish `L_{a,w} ≤ L_{b,w}` in every combination), and in Sub-case A3 (placing the pair in (γ) since `#w < #a < #b`). The `(#a, #w)` and `(#b, #w)` dispatches are distinct ℕ-pairs — the same trichotomy axiom applied at different symbols — and are enumerated separately under the per-instance convention declared in T1's Depends. *Result-length trichotomy dispatch at `(L_{a,w}, L_{b,w})`*: in Sub-case A2's no-disagreement branch, trichotomy's exactly-one clause applied with the already-established `L_{a,w} ≤ L_{b,w}` fixes one of `L_{a,w} < L_{b,w}` (supporting T1 case (ii) for `a ⊖ w` as a proper prefix of `b ⊖ w`) or `L_{a,w} = L_{b,w}` (supporting T3 for equality of the two results on their common length), closing the branch. This instantiation at the result-length pair is distinct from the operand-length pairs above and earns a separate site under the same per-instance convention. *Defining-clause conversion at component pairs*: `m ≤ n ⟺ m < n ∨ m = n` converts the strict inequalities `x_d > w_d` at divergence points (from `x > w` via T1) into the weak form `x_d ≥ w_d` required by NAT-sub's monotonicity and closure preconditions. *Transitivity of `<` and `≤` on ℕ*: NAT-order's transitivity clause composes inequalities at four distinct ℕ-triples, each supplying a bound consumed downstream that is not itself a trichotomy dispatch or defining-clause conversion enumerated above. (i) *Length-triple `(#w, #a, #b)`*: Sub-case A2's derivation of `#w < #b` — required to place `(#b, #w)` in the `(#b, #w)` trichotomy's sub-case (γ) and thereby to establish `L_{a,w} ≤ L_{b,w}` in the `(α) or (γ)` branch of the `(#a, #w)` trichotomy — composes `#w ≤ #a` (supplied by those two branches, which give `#w = #a` in (α) and `#w < #a` in (γ)) with Case A's `#a < #b` through weak-over-strict transitivity (the strict disjunct `#w < #a` composed with `#a < #b` via NAT-order transitivity of `<`, the equality disjunct `#w = #a` substituted into `#a < #b` to yield `#w < #b` directly without transitivity). Sub-case A3 reuses the same triple: its hypothesis `#w < #a` (from T1 case (ii) placing `w` as a proper prefix of `a`) composes with Case A's `#a < #b` through strict-strict transitivity to feed the same `(#b, #w)` trichotomy its sub-case (γ) selection and the bound `L_{b,w} = #b > #a`. The A2 and A3 uses instantiate transitivity at the shared triple `(#w, #a, #b)` and fold into a single site under the per-instance convention declared in T1's Depends — same triple across sub-cases counts as a single site. (ii) *Bridge-triple `(dₐ, j, #a)`*: the B1–B2 bridge's derivation of `dₐ ≤ #a` — the first conjunct of the T1 case (i) shared-position bound required to construct the hypothetical case (i) witness for `a > b` at the zero-padded-divergence position `dₐ` that the bridge then refutes against `a < b` — composes `dₐ ≤ j` (forced by the pre-`dₐ` agreement `aᵢ = wᵢ = bᵢ` for `i < dₐ` under the hypothetical zero-padded equality of `b` with `w`, together with Case B's `aⱼ ≠ bⱼ` at position `j` which places the first `a`-vs-`b` disagreement at or after `dₐ`) with Case B's `j ≤ #a` (the first conjunct of T1 case (i)'s explicit conjunction `j ≤ #a ∧ j ≤ #b` for the pair `(a, b)`) through transitivity of `≤`. (iii) *Bridge-triple `(dₐ, j, #b)`*: symmetrically, the same bridge derives `dₐ ≤ #b` — the second conjunct of the same shared-position bound — by composing `dₐ ≤ j` with Case B's `j ≤ #b` (the second conjunct of the same explicit conjunction) through transitivity of `≤`. Sites (ii) and (iii) instantiate transitivity at distinct ℕ-triples — same axiom, different symbols `#a` vs `#b` at the third slot — and earn separate sites under the per-instance convention, matching T1's enumeration of Case 3's `m < n` and `n < m` mirror-branch rebuttals as separate transitivity sites under the same convention. Without this transitivity the passage from `d ≤ #a` and `#a < i` to the strict `d < i` required by the tail-copy clause is not discharged — the weak bound alone does not place `i` strictly above `d`. *Defining-clause unfolding at first-disagreement component pair `(0, (b ⊖ w)_p)` (Sub-case A2's `≠ 0 ⟹ > 0` discharge)*: the defining clause `m ≤ n ⟺ m < n ∨ m = n` at `m = 0, n = (b ⊖ w)_p` unfolds NAT-zero's lower bound `0 ≤ (b ⊖ w)_p` (supplied at NAT-zero's site (ii) enumerated above, with `(b ⊖ w)_p ∈ ℕ` inherited from TA2's postcondition `b ⊖ w ∈ T`) to `0 < (b ⊖ w)_p ∨ 0 = (b ⊖ w)_p`, after which the first-disagreement observation `(b ⊖ w)_p ≠ 0` (from `(a ⊖ w)_p = 0` together with the first-disagreement hypothesis `(a ⊖ w)_p ≠ (b ⊖ w)_p` at position `p`) excludes the equality disjunct to leave `0 < (b ⊖ w)_p`, i.e., `(b ⊖ w)_p > 0`; this strict inequality, combined with `(a ⊖ w)_p = 0`, supplies the strict component comparison `(a ⊖ w)_p < (b ⊖ w)_p` that T1 case (i)'s component-divergence witness requires at position `p` (with the shared-position bound `p ≤ L_{a,w} ∧ p ≤ L_{b,w}` supplied by the branch construction and the already-established `L_{a,w} ≤ L_{b,w}`) for the conclusion `a ⊖ w < b ⊖ w`.
