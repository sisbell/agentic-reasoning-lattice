**TA3 (OrderPreservationUnderSubtractionWeak).** `(A a, b, w : a < b ‚ąß a ‚Č• w ‚ąß b ‚Č• w : a ‚äĖ w ‚Č§ b ‚äĖ w)`.

*Proof.* We must show that for all `a, b, w ‚ąą T` with `a < b`, `a ‚Č• w`, and `b ‚Č• w`, the inequality `a ‚äĖ w ‚Č§ b ‚äĖ w` holds under T1.

By TA2, since `a ‚Č• w` and `b ‚Č• w`, both `a ‚äĖ w` and `b ‚äĖ w` are well-formed tumblers in `T`, so the comparison is well-defined. We recall the subtraction rule (TumblerSub) for self-containment: given `x ‚Č• w`, zero-pad both operands to a common length `L_{x,w}` ‚ÄĒ the longer of `#x` and `#w`, named by NAT-order's trichotomy on the length pair `(#x, #w)` (sub-case (őĪ) `#x = #w` with `L_{x,w} = #x`; sub-case (ő≤) `#x < #w` with `L_{x,w} = #w`; sub-case (ő≥) `#w < #x` with `L_{x,w} = #x`) rather than by a primitive binary-maximum operator on ‚Ąē, reproducing TumblerSub's own length-pair dispatch ‚ÄĒ and scan for the first position at which the padded sequences disagree. If no such position exists (we say `x` is *zero-padded-equal* to `w`), then `x ‚äĖ w` is the zero tumbler of length `L_{x,w}`. Otherwise, let `d` be the first divergence position; then `(x ‚äĖ w)ŠĶĘ = 0` for `i < d`, `(x ‚äĖ w)_d = x_d - w_d`, and `(x ‚äĖ w)ŠĶĘ = xŠĶĘ` for `i > d` (all under zero-padding, where `xŠĶĘ = 0` for `i > #x` and `wŠĶĘ = 0` for `i > #w` with the literal `0` supplied as an element of ‚Ąē by NAT-zero so that the padded values extend the operands' native ‚Ąē-valued component-typing onto the padding region, and under the same convention every literal-`0` result component TumblerSub produces ‚ÄĒ in the zero-tumbler no-divergence branch and at pre-divergence positions `i < d` ‚ÄĒ is ‚Ąē-valued by NAT-zero, matching TumblerSub's Definition and the per-step convention ZPD, TA2, ActionPoint, and TA-Pos follow for discharging identical `0 ‚ąą ‚Ąē` premises), with result length `L_{x,w}`.

Since `a < b`, T1 provides two cases: either (i) there exists a first position `j` with `j ‚Č§ #a ‚ąß j ‚Č§ #b` (the explicit conjunction replacing `j ‚Č§ min(#a, #b)` so that no primitive binary-minimum operator on ‚Ąē is invoked, matching T1's case-(i) definition, Divergence's case (i), TumblerSub's precondition-consequence case (i), ZPD's Relationship-to-Divergence, and TA2's sub-case (i)) where `a‚Īľ < b‚Īľ`, or (ii) `a` is a proper prefix of `b` ‚ÄĒ `#a < #b` and `aŠĶĘ = bŠĶĘ` for all `i ‚Č§ #a`. We treat each in turn, partitioning further by the divergence structure of the operands against `w`.

**Case A: `a` is a proper prefix of `b`** (T1 case (ii)). Then `#a < #b` and `aŠĶĘ = bŠĶĘ` for all `i ‚Č§ #a`.

*Sub-case A1: `a = w`.* Since the operands are identical, NAT-order's trichotomy on `(#a, #w)` places the pair in sub-case (őĪ) `#a = #w`, giving `L_{a,w} = #a`; the subtraction `a ‚äĖ w` yields the zero tumbler of that length. Since `a` is a proper prefix of `b` and `a = w`, we have `bŠĶĘ = wŠĶĘ` for all `i ‚Č§ #w = #a`. If some component `bŠĶĘ` with `i > #w` is nonzero, the pair `(b, w)` has a divergence beyond `#w`, making `b ‚äĖ w` a positive tumbler; by TA6 the zero tumbler `a ‚äĖ w` is strictly less. If `bŠĶĘ = 0` for all `i > #w`, the zero-padded sequences of `b` and `w` agree everywhere, and under `#w = #a < #b` NAT-order's trichotomy on `(#b, #w)` places the pair in sub-case (ő≥) `#w < #b`, giving `L_{b,w} = #b`; so `b ‚äĖ w` is the zero tumbler of length `#b`. Both results are zero tumblers, so `(a ‚äĖ w)ŠĶĘ = 0 = (b ‚äĖ w)ŠĶĘ` at every position `1 ‚Č§ i ‚Č§ #(a ‚äĖ w) = #a`; the strict length inequality `#(a ‚äĖ w) = #a < #b = #(b ‚äĖ w)` is converted by NAT-discrete's forward direction `m < n ‚üĻ m + 1 ‚Č§ n` at the length pair `(#a, #b)` into the successor bound `#(a ‚äĖ w) + 1 = #a + 1 ‚Č§ #b = #(b ‚äĖ w)`, supplying T1 case (ii)'s witness `k = #(a ‚äĖ w) + 1 ‚Č§ #(b ‚äĖ w)`. Thus `a ‚äĖ w` is a proper prefix of `b ‚äĖ w`, giving `a ‚äĖ w < b ‚äĖ w` by T1 case (ii).

*Sub-case A2: `a > w` with divergence.* Let `d‚āź` be the first position where the zero-padded sequences of `a` and `w` disagree. We claim `d‚āź ‚Č§ #a`: if `a > w` by T1 case (i), T1 case (i) supplies `d‚āź ‚Č§ #a ‚ąß d‚āź ‚Č§ #w` (the explicit conjunction replacing `d‚āź ‚Č§ min(#a, #w)`), whose first conjunct `d‚āź ‚Č§ #a` is the bound claimed; if by T1 case (ii), `w` is a proper prefix of `a` and `d‚āź` is the first `i > #w` with `aŠĶĘ > 0`, so `d‚āź ‚Č§ #a`. Since `bŠĶĘ = aŠĶĘ` for all `i ‚Č§ #a` and `d‚āź ‚Č§ #a`, the comparison of `b` against `w` (under zero-padding) agrees with that of `a` at all positions up through `d‚āź`. So `d_b = d‚āź = d`.

Apply TumblerSub to both. At positions `i < d`: both results are zero. At position `d`: both compute `a_d - w_d = b_d - w_d` ‚ÄĒ each side a natural number by NAT-sub's conditional-closure clause under `a_d ‚Č• w_d` and `b_d ‚Č• w_d` (from the divergence-point inequalities via NAT-order), and equal because `a_d = b_d` for `d ‚Č§ #a` feeds identical arguments into the same partial operation. At positions `d < i ‚Č§ #a`: both copy from their respective minuends, giving `aŠĶĘ = bŠĶĘ`. The two results agree on positions `1, ..., #a`.

Beyond position `#a`, the results may differ in both components and length. Denote the result lengths `L_{a,w}` and `L_{b,w}`, named by NAT-order's trichotomy on `(#a, #w)` and `(#b, #w)` respectively per the recall above. We first establish `L_{a,w} ‚Č§ L_{b,w}` by enumerating NAT-order's trichotomy on `(#a, #w)`. In sub-case (őĪ) `#a = #w` or (ő≥) `#w < #a`, `L_{a,w} = #a`; combined with the case hypothesis `#a < #b`, this places `(#b, #w)` in sub-case (ő≥) `#w < #b` (since `#w ‚Č§ #a < #b`), giving `L_{b,w} = #b`, whence `L_{a,w} = #a < #b = L_{b,w}`. In sub-case (ő≤) `#a < #w`, `L_{a,w} = #w`; NAT-order's trichotomy on `(#b, #w)` dispatches further into (ő≥) `#w < #b` with `L_{b,w} = #b > #w = L_{a,w}`, (őĪ) `#b = #w` with `L_{b,w} = #w = L_{a,w}`, or (ő≤) `#b < #w` with `L_{b,w} = #w = L_{a,w}`. In every combination, `L_{a,w} ‚Č§ L_{b,w}`.

At positions `#a < i ‚Č§ L_{a,w}` (present only in sub-case (ő≤) of `(#a, #w)`, where `L_{a,w} = #w > #a`): `(a ‚äĖ w)ŠĶĘ = 0` from `a`'s zero-padding, while `(b ‚äĖ w)ŠĶĘ = bŠĶĘ` if `i ‚Č§ #b` (copied from the minuend since `i > d = d‚āź ‚Č§ #a < i`) and `0` if `i > #b` (from `b`'s zero-padding); in either case `(a ‚äĖ w)ŠĶĘ ‚Č§ (b ‚äĖ w)ŠĶĘ`.

If a first disagreement exists at some position `p` on `1, ..., L_{a,w}`, then `p > #a` (since the results agree on `1, ..., #a`), and `(a ‚äĖ w)_p = 0` while `(b ‚äĖ w)_p ‚Č† 0` (since they disagree and tumbler components are non-negative, `(b ‚äĖ w)_p > 0`); `p` satisfies `p ‚Č§ L_{a,w} ‚ąß p ‚Č§ L_{b,w}` (the explicit conjunction, its first conjunct holding by construction and its second by `L_{a,w} ‚Č§ L_{b,w}` just established), so T1 case (i) yields `a ‚äĖ w < b ‚äĖ w`. If no disagreement exists on `1, ..., L_{a,w}`, then `(a ‚äĖ w)ŠĶĘ = (b ‚äĖ w)ŠĶĘ` at every such `i`. NAT-order's trichotomy at `(L_{a,w}, L_{b,w})` applied with the already-established `L_{a,w} ‚Č§ L_{b,w}` fixes one of `L_{a,w} < L_{b,w}` or `L_{a,w} = L_{b,w}`. If `L_{a,w} < L_{b,w}`, NAT-discrete's forward direction `m < n ‚üĻ m + 1 ‚Č§ n` at the length pair `(L_{a,w}, L_{b,w})` converts this strict inequality into the successor bound `L_{a,w} + 1 ‚Č§ L_{b,w}`, supplying T1 case (ii)'s witness `k = L_{a,w} + 1 ‚Č§ L_{b,w}`; combined with the component-wise agreement `(a ‚äĖ w)ŠĶĘ = (b ‚äĖ w)ŠĶĘ` at every position `1 ‚Č§ i ‚Č§ L_{a,w}` just established, `a ‚äĖ w` is a proper prefix of `b ‚äĖ w`, giving `a ‚äĖ w < b ‚äĖ w` by T1 case (ii). If `L_{a,w} = L_{b,w}`, both results have the same length and agree at every position, so `a ‚äĖ w = b ‚äĖ w` by T3, and `a ‚äĖ w ‚Č§ b ‚äĖ w` holds by equality.

*Sub-case A3: `a > w` without divergence (zero-padded equality).* Since `a > w` requires a structural difference yet the padded sequences agree everywhere, the only possibility is T1 case (ii): `w` is a proper prefix of `a` with `aŠĶĘ = 0` for all `i > #w`. Under `#w < #a`, NAT-order's trichotomy on `(#a, #w)` places the pair in sub-case (ő≥), giving `L_{a,w} = #a`; the subtraction `a ‚äĖ w` yields the zero tumbler of that length. Since `b > a > w` and `#b > #a > #w` (combining Case A's `#a < #b` with this sub-case's `#w < #a`), `b` agrees with `w` (hence with `a`) on positions `1, ..., #a`. NAT-order's trichotomy on `(#b, #w)` under `#w < #a < #b` is sub-case (ő≥) with `L_{b,w} = #b > #a`. If `b ‚äĖ w` has any positive component, then `a ‚äĖ w` (all zeros) is strictly less by TA6. If `b ‚äĖ w` is also a zero tumbler, both results have all-zero components, so `(a ‚äĖ w)ŠĶĘ = 0 = (b ‚äĖ w)ŠĶĘ` at every position `1 ‚Č§ i ‚Č§ #(a ‚äĖ w) = #a`; the strict length inequality `#(a ‚äĖ w) = #a < #b = #(b ‚äĖ w)` is converted by NAT-discrete's forward direction `m < n ‚üĻ m + 1 ‚Č§ n` at the length pair `(#a, #b)` into the successor bound `#(a ‚äĖ w) + 1 = #a + 1 ‚Č§ #b = #(b ‚äĖ w)`, supplying T1 case (ii)'s witness `k = #(a ‚äĖ w) + 1 ‚Č§ #(b ‚äĖ w)`. Thus the shorter is a proper prefix of the longer, giving `a ‚äĖ w < b ‚äĖ w` by T1 case (ii).

In all sub-cases of Case A, `a ‚äĖ w ‚Č§ b ‚äĖ w`.

**Case B: Component divergence at `j`** (T1 case (i)). There exists a first position `j` with `j ‚Č§ #a ‚ąß j ‚Č§ #b` (the explicit conjunction replacing `j ‚Č§ min(#a, #b)`, matching T1's case-(i) definition), `a‚Īľ < b‚Īľ`, and `aŠĶĘ = bŠĶĘ` for all `i < j`.

*Sub-case B1: `a` is zero-padded-equal to `w`.* The subtraction `a ‚äĖ w` is the zero tumbler of length `L_{a,w}`. At position `j`, zero-padded equality gives `w‚Īľ = a‚Īľ`, so `b‚Īľ > a‚Īľ = w‚Īľ`. The pair `(b, w)` diverges at or before `j`, making `b ‚äĖ w` positive. By TA6, `a ‚äĖ w < b ‚äĖ w`.

For the remaining sub-cases, `a` is not zero-padded-equal to `w`, so `d‚āź = zpd(a, w)` ‚ÄĒ TumblerSub's zero-padded divergence, distinct from the formal Divergence ‚ÄĒ is well-defined. The zero-padded divergence `d_b = zpd(b, w)` is also well-defined: if `b` were zero-padded-equal to `w`, then at position `d‚āź` we would have `a_{d‚āź} > w_{d‚āź} = b_{d‚āź}` (from `a ‚Č• w` at the zero-padded divergence), while `aŠĶĘ = wŠĶĘ = bŠĶĘ` for all `i < d‚āź`. Since Case B provides `j` with `j ‚Č§ #a ‚ąß j ‚Č§ #b` and `a‚Īľ ‚Č† b‚Īľ`, the pre-`d‚āź` agreement between `a` and `b` forces `d‚āź ‚Č§ j`; combined with Case B's `j ‚Č§ #a ‚ąß j ‚Č§ #b`, this gives `d‚āź ‚Č§ #a ‚ąß d‚āź ‚Č§ #b`, making `d‚āź` a T1 case (i) witness for `a > b` (with the required shared-position bound in conjunction form) ‚ÄĒ contradicting `a < b`. Let `j` be the first position where `a‚Īľ < b‚Īľ`.

*Sub-case B2: `d‚āź = d_b = d`.* Both operands diverge from `w` at the same position. For `i < d`, both results are zero. Since `a` and `b` agree with `w` before `d`, and `a‚Īľ < b‚Īľ`, we have `j ‚Č• d`. If `j = d`: NAT-sub's strict monotonicity applied to `a_d < b_d` with `a_d ‚Č• w_d` and `b_d ‚Č• w_d` (both supplied by the setup above via NAT-order) yields `a_d - w_d < b_d - w_d`, so `a ‚äĖ w < b ‚äĖ w` by T1 case (i). If `j > d`: `a_d = b_d` (since the first `a`-vs-`b` disagreement is at `j > d`), so both results agree at position `d`; at positions `d < i < j`, both copy from their minuends which agree (`aŠĶĘ = bŠĶĘ`); at position `j`, `(a ‚äĖ w)‚Īľ = a‚Īľ < b‚Īľ = (b ‚äĖ w)‚Īľ` since both are in the tail-copy phase (`j > d`). By T1 case (i), `a ‚äĖ w < b ‚äĖ w`.

*Sub-case B3: `d‚āź < d_b`.* At position `d‚āź`, `a_{d‚āź} ‚Č† w_{d‚āź}` but `b_{d‚āź} = w_{d‚āź}`. Since both `a` and `b` agree with `w` at all positions before `d‚āź`, `aŠĶĘ = bŠĶĘ` for `i < d‚āź`; Case B's `j` with `j ‚Č§ #a ‚ąß j ‚Č§ #b` and `a‚Īľ ‚Č† b‚Īľ` then forces `d‚āź ‚Č§ j` and hence `d‚āź ‚Č§ #a ‚ąß d‚āź ‚Č§ #b` (via the conjunction), so the first disagreement between `a` and `b` is at `d‚āź`, giving `j = d‚āź` with `a_{d‚āź} < b_{d‚āź} = w_{d‚āź}`. But `a ‚Č• w` requires `a_{d‚āź} ‚Č• w_{d‚āź}` at the divergence ‚ÄĒ contradiction. This case is impossible under the preconditions.

*Sub-case B4: `d‚āź > d_b`.* At position `d_b`, `b_{d_b} ‚Č† w_{d_b}` but `a_{d_b} = w_{d_b}`. Since both `a` and `b` agree with `w` before `d_b`, the first disagreement between `a` and `b` is at `d_b`, giving `j = d_b` with `a_{d_b} = w_{d_b} < b_{d_b}` ‚ÄĒ the inequality holds because `b ‚Č• w` forces `b_{d_b} > w_{d_b}` at this divergence. The result `(a ‚äĖ w)_{d_b} = 0` since `d_b < d‚āź` falls in the pre-divergence zero phase for `a ‚äĖ w`. The result `(b ‚äĖ w)_{d_b} = b_{d_b} - w_{d_b} > 0` by NAT-sub's strict-positivity clause, whose precondition `b_{d_b} > w_{d_b}` is the strict inequality just derived. At all positions `i < d_b`, both results are zero. By T1 case (i), `a ‚äĖ w < b ‚äĖ w`.

In every case, `a ‚äĖ w ‚Č§ b ‚äĖ w`. ‚ąé
