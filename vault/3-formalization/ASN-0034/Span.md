## Definition (Span)

**Definition (Span).** A *span* is a pair `(s, ℓ)` where `s ∈ T` is a start address and `ℓ ∈ T` is a length — a positive tumbler used as a displacement whose action point satisfies `actionPoint(ℓ) ≤ #s` — denoting the contiguous range from `s` up to but not including `s ⊕ ℓ`. The two conditions `Pos(ℓ)` and `actionPoint(ℓ) ≤ #s` are precisely TA0's preconditions for `s ⊕ ℓ ∈ T`, so under them the upper bound denotes a tumbler in `T` and the set `{t ∈ T : s ≤ t < s ⊕ ℓ}` — the span — is well-defined; pairs `(s, ℓ)` violating either condition are not spans.

*Formal Contract:*
- *Preconditions:* `s ∈ T`, `ℓ ∈ T`, `Pos(ℓ)`, `actionPoint(ℓ) ≤ #s`
- *Definition:* `span(s, ℓ) = {t ∈ T : s ≤ t < s ⊕ ℓ}`
- *Depends:* TA-Pos (PositiveTumbler) — supplies the positivity predicate `Pos(·)` used in the precondition `Pos(ℓ)`; without TA-Pos the symbol `Pos` has no source and the precondition `Pos(ℓ)` cannot be discharged. ActionPoint (ActionPoint) — supplies the action-point function `actionPoint(·)` referenced in the precondition `actionPoint(ℓ) ≤ #s`; ActionPoint licenses `actionPoint(ℓ)` as a legal index into `ℓ` (under `Pos(ℓ)`), which the bound `≤ #s` then compares against the length of the base. TumblerAdd (TumblerAdd) — supplies the operator `⊕` appearing in the upper bound `s ⊕ ℓ` of the defining set `{t ∈ T : s ≤ t < s ⊕ ℓ}`; the piecewise construction of `⊕` is what gives `s ⊕ ℓ` any meaning as a tumbler expression. TA0 (WellDefinedAddition) — supplies the postcondition `s ⊕ ℓ ∈ T` under the preconditions `Pos(ℓ)` and `actionPoint(ℓ) ≤ #s`, which are exactly TA0's preconditions; so TA0 licenses the span's upper bound to denote a legal tumbler, and the narrative's sentence "the two conditions `Pos(ℓ)` and `actionPoint(ℓ) ≤ #s` are precisely TA0's preconditions for `s ⊕ ℓ ∈ T`, so under them the upper bound denotes a tumbler in `T`" rests on this citation. T1 (LexicographicOrder) — supplies the strict total order `<` used in `t < s ⊕ ℓ` and, via T1(d)'s derived non-strict companion `a ≤ b ⟺ a < b ∨ a = b`, the relation `s ≤ t` that fixes the span's lower bound; without T1 the two inequalities bracketing the defining set are undefined symbols.
