**Definition (Divergence).** For tumblers `a, b вҲҲ T` with `a вү  b`, the *divergence* `divergence(a, b)` is defined by two cases corresponding to the two cases of T1. This corollary formalizes the *first divergence position* invoked ad hoc inside T1's trichotomy proof (part b): T1's proof must name the earliest index where two tumblers disagree to select which of its two clauses applies, and Divergence names the same index as a function so that later results (TA1, TA4, T5, TA7a, TA-LC, D0) can refer to it without re-deriving the concept. Tightening either definition obliges the other to be updated in lockstep. The definition is stated without invoking any primitive binary-minimum operator on в„• вҖ” shared positions are bounded by the explicit conjunction `k вүӨ #a вҲ§ k вүӨ #b`, and case (ii)'s value is selected by a case split on `#a < #b` versus `#b < #a` licensed by NAT-order's trichotomy вҖ” so that every operator occurring in the Definition is either T0's length `#В·`, T0's component projection, the strict order `<` supplied by NAT-order, or the successor operator `+ 1` supplied by NAT-closure's successor-closure clause `(A n вҲҲ в„• :: n + 1 вҲҲ в„•)`, which forms case (ii)'s sub-case values `#a + 1` and `#b + 1` as в„•-valued expressions from the в„•-valued lengths `#a, #b`.

  (i) If there exists `k` with `1 вүӨ k вҲ§ k вүӨ #a вҲ§ k вүӨ #b` such that `aвӮ– вү  bвӮ–` and `(A i : 1 вүӨ i < k : aбөў = bбөў)`, then `divergence(a, b) = k` вҖ” component divergence at a shared position.

  (ii) If `#a вү  #b`, NAT-order's trichotomy applied to `(#a, #b)` rules out the `#a = #b` branch and leaves exactly one of `#a < #b` or `#b < #a`. In sub-case (ii-a), `#a < #b` and `(A i : 1 вүӨ i вүӨ #a : aбөў = bбөў)`, whence `divergence(a, b) = #a + 1`. In sub-case (ii-b), `#b < #a` and `(A i : 1 вүӨ i вүӨ #b : aбөў = bбөў)`, whence `divergence(a, b) = #b + 1`. In either sub-case the divergence lies one position past the shorter tumbler's last component вҖ” prefix divergence, where one tumbler is a proper prefix of the other.

Case (i)'s value `k` is unique: the conjunction `aвӮ– вү  bвӮ– вҲ§ (A i : 1 вүӨ i < k : aбөў = bбөў)` forces `k` to be the least element of `S := {i вҲҲ в„• : 1 вүӨ i вҲ§ i вүӨ #a вҲ§ i вүӨ #b вҲ§ aбөў вү  bбөў}`, and that least element exists precisely because `S` is a nonempty subset of в„• (nonempty whenever case (i) applies) вҖ” this is the well-ordering of в„• applied directly, and is discharged from NAT-wellorder rather than assumed. Case (ii)'s value вҖ” `#a + 1` under sub-case (ii-a) or `#b + 1` under sub-case (ii-b) вҖ” is determined arithmetically from the shorter length selected by NAT-order's trichotomy and needs no minimization.

Exactly one case applies for any `a вү  b`. Mutual exclusivity is immediate: if case (i) holds, some `aвӮ– вү  bвӮ–` with `k вүӨ #a вҲ§ k вүӨ #b` falsifies case (ii)'s universal agreement at shared positions (in both sub-cases (ii-a) and (ii-b), the universal quantifier would have demanded `aвӮ– = bвӮ–` at that same `k`). For exhaustiveness, suppose neither case applies вҖ” all shared components agree and `#a = #b`. Then by T3, `a = b`, contradicting `a вү  b`. In case (i), `a` and `b` differ at a component both possess. In case (ii), they agree on all shared positions but one is longer вҖ” the divergence lies "just past" the shorter tumbler's last component.

The function is symmetric: `divergence(a, b) = divergence(b, a)` for all `a вү  b`. In case (i), the condition `aвӮ– вү  bвӮ–` with agreement on all prior components is unchanged by swapping `a` and `b`, so the least such `k` is the same. In case (ii), swapping `a` and `b` exchanges sub-cases (ii-a) and (ii-b): if the ordered pair `(a, b)` satisfies sub-case (ii-a) with `#a < #b` so that `divergence(a, b) = #a + 1`, then the ordered pair `(b, a)` satisfies sub-case (ii-b) (the first operand's length exceeds the second's), yielding `divergence(b, a) = #a + 1` as well; symmetrically if `(a, b)` satisfies (ii-b). Both orientations select one-plus the shorter tumbler's length, returning the same в„•-value.

For prefix-related pairs вҖ” those falling under case (ii) вҖ” `divergence(a, b) > #a` in sub-case (ii-a) (since `divergence(a, b) = #a + 1`) and `divergence(a, b) > #b` in sub-case (ii-b) (since `divergence(a, b) = #b + 1`). TA1-strict requires both `actionPoint(w) вүӨ #a вҲ§ actionPoint(w) вүӨ #b` вҖ” combining TA0's single-operand precondition `actionPoint(w) вүӨ #a` applied to each operand вҖ” and `actionPoint(w) вүҘ divergence(a, b)`. For prefix-related operands these constraints are jointly unsatisfiable: in sub-case (ii-a), `actionPoint(w) вүӨ #a` and `actionPoint(w) вүҘ #a + 1` force a natural number to be simultaneously вүӨ `#a` and вүҘ `#a + 1`; sub-case (ii-b) is symmetric in `#b`. The action point cannot simultaneously lie within both tumblers and reach past the shorter one's last component. TA1-strict therefore makes no claim about prefix-related pairs. TA1 covers them, showing that both results become equal and order is preserved as non-reversal.
