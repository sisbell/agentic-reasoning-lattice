**Definition (Divergence).** For tumblers `a, b ∈ T` with `a ≠ b`, the *divergence* `divergence(a, b)` is defined by two cases corresponding to the two cases of T1. This corollary formalizes the *first divergence position* invoked ad hoc inside T1's trichotomy proof (part b): T1's proof must name the earliest index where two tumblers disagree to select which of its two clauses applies, and Divergence names the same index as a function so that later results (TA1, TA4, T5, TA7a, TA-LC, D0) can refer to it without re-deriving the concept. Tightening either definition obliges the other to be updated in lockstep. The definition is stated without invoking any primitive binary-minimum operator on ℕ — shared positions are bounded by the explicit conjunction `k ≤ #a ∧ k ≤ #b`, and case (ii)'s value is selected by a case split on `#a < #b` versus `#b < #a` licensed by NAT-order's trichotomy — so that every operator occurring in the Definition is either T0's length `#·`, T0's component projection, the strict order `<` supplied by NAT-order, or the successor operator `+ 1` supplied by NAT-closure's successor-closure clause `(A n ∈ ℕ :: n + 1 ∈ ℕ)`, which forms case (ii)'s sub-case values `#a + 1` and `#b + 1` as ℕ-valued expressions from the ℕ-valued lengths `#a, #b`.

  (i) If there exists `k` with `1 ≤ k ∧ k ≤ #a ∧ k ≤ #b` such that `aₖ ≠ bₖ` and `(A i : 1 ≤ i < k : aᵢ = bᵢ)`, then `divergence(a, b) = k` — component divergence at a shared position.

  (ii) If `#a ≠ #b`, NAT-order's trichotomy applied to `(#a, #b)` rules out the `#a = #b` branch and leaves exactly one of `#a < #b` or `#b < #a`. In sub-case (ii-a), `#a < #b` and `(A i : 1 ≤ i ≤ #a : aᵢ = bᵢ)`, whence `divergence(a, b) = #a + 1`. In sub-case (ii-b), `#b < #a` and `(A i : 1 ≤ i ≤ #b : aᵢ = bᵢ)`, whence `divergence(a, b) = #b + 1`. In either sub-case the divergence lies one position past the shorter tumbler's last component — prefix divergence, where one tumbler is a proper prefix of the other.

Case (i)'s value `k` is unique: the conjunction `aₖ ≠ bₖ ∧ (A i : 1 ≤ i < k : aᵢ = bᵢ)` forces `k` to be the least element of `S := {i ∈ ℕ : 1 ≤ i ∧ i ≤ #a ∧ i ≤ #b ∧ aᵢ ≠ bᵢ}`, and that least element exists precisely because `S` is a nonempty subset of ℕ (nonempty whenever case (i) applies) — this is the well-ordering of ℕ applied directly, and is discharged from NAT-wellorder rather than assumed. Case (ii)'s value — `#a + 1` under sub-case (ii-a) or `#b + 1` under sub-case (ii-b) — is determined arithmetically from the shorter length selected by NAT-order's trichotomy and needs no minimization.

Exactly one case applies for any `a ≠ b`. Mutual exclusivity is immediate: if case (i) holds, some `aₖ ≠ bₖ` with `k ≤ #a ∧ k ≤ #b` falsifies case (ii)'s universal agreement at shared positions (in both sub-cases (ii-a) and (ii-b), the universal quantifier would have demanded `aₖ = bₖ` at that same `k`). For exhaustiveness, suppose neither case applies — all shared components agree and `#a = #b`. Then by T3, `a = b`, contradicting `a ≠ b`. In case (i), `a` and `b` differ at a component both possess. In case (ii), they agree on all shared positions but one is longer — the divergence lies "just past" the shorter tumbler's last component.

The function is symmetric: `divergence(a, b) = divergence(b, a)` for all `a ≠ b`. In case (i), the condition `aₖ ≠ bₖ` with agreement on all prior components is unchanged by swapping `a` and `b`, so the least such `k` is the same. In case (ii), swapping `a` and `b` exchanges sub-cases (ii-a) and (ii-b): if the ordered pair `(a, b)` satisfies sub-case (ii-a) with `#a < #b` so that `divergence(a, b) = #a + 1`, then the ordered pair `(b, a)` satisfies sub-case (ii-b) (the first operand's length exceeds the second's), yielding `divergence(b, a) = #a + 1` as well; symmetrically if `(a, b)` satisfies (ii-b). Both orientations select one-plus the shorter tumbler's length, returning the same ℕ-value.

For prefix-related pairs — those falling under case (ii) — `divergence(a, b) > #a` in sub-case (ii-a) (since `divergence(a, b) = #a + 1`) and `divergence(a, b) > #b` in sub-case (ii-b) (since `divergence(a, b) = #b + 1`). TA1-strict requires both `actionPoint(w) ≤ #a ∧ actionPoint(w) ≤ #b` — combining TA0's single-operand precondition `actionPoint(w) ≤ #a` applied to each operand — and `actionPoint(w) ≥ divergence(a, b)`. For prefix-related operands these constraints are jointly unsatisfiable: in sub-case (ii-a), `actionPoint(w) ≤ #a` and `actionPoint(w) ≥ #a + 1` force a natural number to be simultaneously ≤ `#a` and ≥ `#a + 1`; sub-case (ii-b) is symmetric in `#b`. The action point cannot simultaneously lie within both tumblers and reach past the shorter one's last component. TA1-strict therefore makes no claim about prefix-related pairs. TA1 covers them, showing that both results become equal and order is preserved as non-reversal.

*Formal Contract:*
- *Definition:* For `a, b ∈ T` with `a ≠ b`: (i) if `∃ k : 1 ≤ k ∧ k ≤ #a ∧ k ≤ #b` with `aₖ ≠ bₖ` and `(A i : 1 ≤ i < k : aᵢ = bᵢ)`, then `divergence(a, b) = k`; (ii) if `#a ≠ #b`, NAT-order's trichotomy on `(#a, #b)` rules out `#a = #b` and leaves exactly one of: (ii-a) `#a < #b` with `(A i : 1 ≤ i ≤ #a : aᵢ = bᵢ)`, giving `divergence(a, b) = #a + 1`; or (ii-b) `#b < #a` with `(A i : 1 ≤ i ≤ #b : aᵢ = bᵢ)`, giving `divergence(a, b) = #b + 1`. Exactly one case applies (exhaustiveness by T3: if neither case holds, `a = b`).
- *Depends:* T0 (CarrierSetDefinition) — the membership `a, b ∈ T` in the Definition, together with the lengths `#a`, `#b` and the component projections `aₖ`, `bₖ`, `aᵢ`, `bᵢ` appearing throughout — in case (i)'s predicate `aₖ ≠ bₖ ∧ (A i : 1 ≤ i < k : aᵢ = bᵢ)`, in case (ii)'s sub-case predicates `(A i : 1 ≤ i ≤ #a : aᵢ = bᵢ)` and `(A i : 1 ≤ i ≤ #b : aᵢ = bᵢ)` together with `#a ≠ #b`, and in case (i)'s bound `k ≤ #a ∧ k ≤ #b` — all come from T0's characterisation of T as finite sequences over ℕ with length ≥ 1; the case predicates' ℕ-valued component comparisons `aₖ ≠ bₖ` and `aᵢ = bᵢ` rest on T0's stipulation that each component `aᵢ, bᵢ ∈ ℕ`, making the (in)equalities well-formed comparisons in ℕ rather than uninterpreted symbols, so this citation discharges the carrier-set, length, and component-projection operators from T0 rather than leaving them implicit, matching the per-step convention established in ActionPoint, Prefix, TA-Pos, and TA5-SIG. T1 (LexicographicOrder) — Divergence formalizes the "first divergence position" that T1's trichotomy proof introduces ad hoc; the two cases here correspond to T1's two ordering clauses (case (i) with T1 case (i), case (ii) with T1 case (ii)), but the case (i) correspondence is not a direct bijection with T1's directed clause — Divergence case (i) produces only the unordered disagreement `aₖ ≠ bₖ`, whereas T1 case (i) requires a directed strict inequality at the divergent component. The bridge from the unordered disagreement to either of T1's two directed case-(i) witnesses — `aₖ < bₖ` supporting `a < b`, or `bₖ < aₖ` supporting `b < a` — is mediated by NAT-order's trichotomy at the component pair `(aₖ, bₖ)` once the prior-position equality `(A i : 1 ≤ i < k : aᵢ = bᵢ)` is in hand; this component-pair role of NAT-order is declared alongside its internal length-pair role in the NAT-order citation below. The mediation is invoked at the caller site — T1 part (b) Case 2, TA1, TA4, T5, TA7a, TA-LC, D0 — not within Divergence's own proof, which stops at naming the index, so Divergence's semantics correspond to T1 case (i) *in whichever direction the component-pair trichotomy selects* rather than to a pre-directed T1 case-(i) clause. Case (ii)'s correspondence is direct and internal: the same NAT-order trichotomy at the length pair `(#a, #b)` that splits Divergence case (ii) into sub-case (ii-a) with `#a < #b` and sub-case (ii-b) with `#b < #a` simultaneously fixes the T1 case (ii) direction (sub-case (ii-a) supplying T1 case (ii) witnessing `a < b`, sub-case (ii-b) supplying T1 case (ii) witnessing `b < a`). Downstream uses of `divergence(a, b)` rely on this narrowed correspondence — direct for case (ii), trichotomy-mediated for case (i) — to bridge between the divergence index and the order relation. T3 (CanonicalRepresentation) — exhaustiveness: if neither case (i) nor case (ii) applies, all shared components agree and `#a = #b`, whence T3 yields `a = b`, contradicting `a ≠ b`. NAT-order (NatStrictTotalOrder) — invoked at two distinct pairs for two distinct roles. *Length pair `(#a, #b)` (internal to Divergence's definition)*: trichotomy discharges case (ii)'s sub-case split — when `#a ≠ #b`, the trichotomy clause rules out the `#a = #b` branch and leaves exactly one of `#a < #b` (sub-case (ii-a)) or `#b < #a` (sub-case (ii-b)), at which the sub-case-specific arithmetic value `#a + 1` or `#b + 1` is read off. This length-pair invocation replaces any implicit reliance on a primitive binary-minimum operator on ℕ — the shorter of `#a` and `#b` is named by the selected sub-case rather than by an un-axiomatized `min(#a, #b)`, matching the convention that every operator in the Definition must be discharged by a Depends entry. *Component pair `(aₖ, bₖ)` (bridging to T1's directed case (i))*: trichotomy's exactly-one clause serves as the bridging axiom between Divergence case (i) and T1's directed case (i) — once Divergence case (i) names an index `k` with `aₖ ≠ bₖ` and the prior-position equality `(A i : 1 ≤ i < k : aᵢ = bᵢ)`, trichotomy at `(aₖ, bₖ)` selects either `aₖ < bₖ` (yielding T1 case (i) witnessing `a < b`) or `bₖ < aₖ` (yielding T1 case (i) witnessing `b < a`). This component-pair invocation is not executed within Divergence's own proof (whose definition stops at the index `k`) but is declared in Divergence's Depends so that callers converting a Divergence case (i) index into a T1 ordering witness (T1 part (b) Case 2, TA1, TA4, T5, TA7a, TA-LC, D0) have the bridging axiom in scope without re-deriving it, matching the T1 citation's narrowed case-(i) correspondence. Both roles are instances of the same NAT-order axiom — trichotomy's exactly-one clause — applied at distinct ℕ-pairs, and are enumerated together here under a single NAT-order citation. NAT-wellorder (NatWellOrdering) — case (i) selects `k` as the least index at which `aₖ ≠ bₖ`; the least-element principle applied to the nonempty subset `{i ∈ ℕ : 1 ≤ i ∧ i ≤ #a ∧ i ≤ #b ∧ aᵢ ≠ bᵢ} ⊆ ℕ` supplies that minimum, making `divergence(a, b)` well-defined as a function (rather than a relation) whenever case (i) applies. NAT-closure (NatArithmeticClosure) — case (ii)'s sub-case values `#a + 1` (sub-case (ii-a)) and `#b + 1` (sub-case (ii-b)) are formed by the successor operator `+ 1`, whose well-typing as an ℕ-valued expression is supplied by NAT-closure's successor-closure clause `(A n ∈ ℕ :: n + 1 ∈ ℕ)` — invoked at `n = #a` to conclude `#a + 1 ∈ ℕ` under sub-case (ii-a), and at `n = #b` to conclude `#b + 1 ∈ ℕ` under sub-case (ii-b), with the length operands `#a, #b ∈ ℕ` supplied by T0's length operator. The successor-closure clause's syntactic use of `1` as a term in the axiom's formulation presupposes `1 ∈ ℕ` within the axiom's ambient quantification over ℕ — the same internal-quantification reading by which NAT-zero supplies `0 ∈ ℕ` through `(A n ∈ ℕ :: 0 ≤ n)` in the companion ZPD definition — so the literal `1` appearing in `#a + 1` and `#b + 1` is discharged alongside the operator without a separate citation, matching the per-step convention T0a already follows for discharging an `M + 1` construction from NAT-closure's successor-closure clause. This citation closes the parallel gap that previously left `+ 1` and `1 ∈ ℕ` undischarged even after NAT-order and NAT-wellorder had closed the corresponding `min`/`max` and `least-element` gaps in case (ii).
- *Postconditions:* `divergence(a, b) = divergence(b, a)` for all `a ≠ b` (symmetry: in case (i), the least `k` with `aₖ ≠ bₖ` is unchanged by swapping operands; in case (ii), swapping exchanges sub-cases (ii-a) and (ii-b), and both sub-cases select one-plus the shorter length).
