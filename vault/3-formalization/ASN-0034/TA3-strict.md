**TA3-strict (OrderPreservationUnderSubtractionStrict).** `(A a, b, w : a < b Ôłž a Ôëą w Ôłž b Ôëą w Ôłž #a = #b : a ÔŐľ w < b ÔŐľ w)`.

We prove that subtracting a common lower bound from two equal-length tumblers preserves strict order: if `a` precedes `b`, both dominate `w`, and `#a = #b`, then `a ÔŐľ w` strictly precedes `b ÔŐľ w`.

*Proof.* We are given `a, b, w Ôłł T` with `a < b`, `a Ôëą w`, `b Ôëą w`, and `#a = #b`. We must show `a ÔŐľ w < b ÔŐľ w`.

**Preliminaries.** We recall the definitions on which the argument depends. T1 defines `a < b` by: there exists a least `k Ôëą 1` with `aßÁó = bßÁó` for all `i < k`, and either (i) `k ÔëĄ min(#a, #b)` with `aÔéľ < bÔéľ`, or (ii) `k = #a + 1 ÔëĄ #b` (`a` a proper prefix of `b`). TumblerSub defines `x ÔŐľ w` (for `x Ôëą w`) by zero-padding both operands to length `max(#x, #w)` and scanning for the first position where the padded sequences disagree. If no disagreement exists (*zero-padded equality*), the result is the zero tumbler of length `max(#x, #w)`. If divergence occurs at position `d`, the result `r` satisfies: `rßÁó = 0` for `i < d`, `r_d = x_d - w_d`, and `rßÁó = xßÁó` (zero-padded) for `i > d`, with `#r = max(#x, #w)`.

**The form of `a < b`.** Since `#a = #b`, T1 case (ii) is impossible ÔÇö it requires `#a < #b`. So `a < b` holds by case (i): there exists a least `j` with `1 ÔëĄ j ÔëĄ #a` such that `aßÁó = bßÁó` for all `i < j` and `aÔ▒╝ < bÔ▒╝`. We fix this `j` throughout.

**Well-formedness.** By TA2, both `a ÔŐľ w` and `b ÔŐľ w` are well-defined members of `T`.

We proceed by exhaustive case analysis on the divergence structure of the pairs `(a, w)` and `(b, w)` under zero-padding.

**Case A: `a` is zero-padded-equal to `w`.** By TumblerSub, `a ÔŐľ w` is the zero tumbler of length `max(#a, #w)`. For `i < j`: `b_i = a_i` (from T1) and `a_i = w_i` (zero-padded equality), so `b_i = w_i`. At position `j`: `w_j = a_j` (zero-padded equality) and `b_j > a_j` (from `a < b`), giving `b_j > w_j`. So `(b, w)` diverges at position `j`, and TumblerSub yields `(b ÔŐľ w)_j = b_j - w_j > 0` by NAT-sub (strict positivity), whose precondition `b_j > w_j` is the strict inequality just derived. Since `b_i = w_i` for all `i < j`, TumblerSub gives `(b ÔŐľ w)_i = 0` for `i < j`. Both results are therefore zero before position `j`, and at `j` we have `(a ÔŐľ w)_j = 0 < b_j - w_j = (b ÔŐľ w)_j`. By T1 case (i), `a ÔŐľ w < b ÔŐľ w`.

**Setup for remaining cases.** Since `a` is not zero-padded-equal to `w`, the zero-padded divergence `d_a = zpd(a, w)` (TumblerSub) is well-defined. Since `a Ôëą w` and `a` is not zero-padded-equal to `w`, we have `a > w`, and at the first padded disagreement `a_{d_a} > w_{d_a}`. We verify that `d_b = zpd(b, w)` also exists: if `b` were zero-padded-equal to `w`, then `b_{d_a} = w_{d_a} < a_{d_a}`, and since `b_i = w_i = a_i` for `i < d_a`, T1 gives `a > b` ÔÇö contradicting `a < b`. So `d_b` is well-defined, with `b_{d_b} > w_{d_b}` by the same reasoning from `b > w`.

**Case 1: `d_a = d_b = d`.** Both pairs diverge from `w` at position `d`. By TumblerSub, `(a ÔŐľ w)_i = 0` and `(b ÔŐľ w)_i = 0` for all `i < d`. Since `a` and `b` both agree with `w` before `d`, they agree with each other, so `j Ôëą d`.

*Subcase `j = d`:* `(a ÔŐľ w)_d = a_d - w_d` and `(b ÔŐľ w)_d = b_d - w_d`, both well-defined natural numbers by NAT-sub (conditional closure) under `a_d Ôëą w_d` and `b_d Ôëą w_d` (from the setup inequalities via NAT-order's definition of `ÔëĄ` from `<`). From `j = d`: `a_d < b_d`. NAT-sub's strict monotonicity, whose preconditions `a_d Ôëą w_d`, `b_d Ôëą w_d`, and `a_d < b_d` are now all in hand, yields `a_d - w_d < b_d - w_d`; strict positivity yields both differences are in fact Ôëą 1. The results agree before `d` (both zero) and first disagree at `d`. By T1 case (i), `a ÔŐľ w < b ÔŐľ w`.

*Subcase `j > d`:* `a_d = b_d` (since `j > d`), so `(a ÔŐľ w)_d = a_d - w_d = b_d - w_d = (b ÔŐľ w)_d`. For `d < i < j`: both results are in TumblerSub's tail-copy phase, giving `(a ÔŐľ w)_i = a_i` and `(b ÔŐľ w)_i = b_i`; since `a_i = b_i` (`i < j`), the results agree. At position `j`: `(a ÔŐľ w)_j = a_j` and `(b ÔŐľ w)_j = b_j` (still tail-copy), with `a_j < b_j`. The results first disagree at `j`. By T1 case (i), `a ÔŐľ w < b ÔŐľ w`.

**Case 2: `d_a < d_b`.** At position `d_a`: `a_{d_a} Ôëá w_{d_a}` but `b_{d_a} = w_{d_a}` (since `d_a < d_b`). Both agree with `w` ÔÇö hence with each other ÔÇö before `d_a`, and disagree at `d_a` (since `a_{d_a} Ôëá w_{d_a} = b_{d_a}`), so `j = d_a`. From `a < b` by T1: `a_{d_a} < b_{d_a} = w_{d_a}`. But `a_{d_a} > w_{d_a}` (from setup) ÔÇö contradiction. This case is impossible.

**Case 3: `d_a > d_b`.** At position `d_b`: `b_{d_b} Ôëá w_{d_b}` but `a_{d_b} = w_{d_b}` (since `d_b < d_a`). Both agree with `w` ÔÇö hence with each other ÔÇö before `d_b`, and disagree at `d_b` (since `b_{d_b} Ôëá w_{d_b} = a_{d_b}`), so `j = d_b`. From `a < b`: `a_{d_b} < b_{d_b}`, i.e., `w_{d_b} < b_{d_b}` ÔÇö consistent with `b_{d_b} > w_{d_b}`.

For `a ÔŐľ w`: position `d_b` falls before `d_a`, placing it in the pre-divergence zero phase, so `(a ÔŐľ w)_{d_b} = 0`. For `b ÔŐľ w`: `d_b` is the divergence point, so `(b ÔŐľ w)_{d_b} = b_{d_b} - w_{d_b} > 0` by NAT-sub (strict positivity), whose precondition `b_{d_b} > w_{d_b}` is the setup's strict inequality. At all positions `i < d_b`, both results are zero (pre-divergence for both). The first disagreement is at `d_b` with `0 < b_{d_b} - w_{d_b}`. By T1 case (i), `a ÔŐľ w < b ÔŐľ w`.

In every case, `a ÔŐľ w < b ÔŐľ w` is established. ÔłÄ
