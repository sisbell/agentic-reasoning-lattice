**TA3-strict (OrderPreservationUnderSubtractionStrict).** `(A a, b, w : a < b ‚àß a ‚â• w ‚àß b ‚â• w ‚àß #a = #b : a ‚äñ w < b ‚äñ w)`.

We prove that subtracting a common lower bound from two equal-length tumblers preserves strict order: if `a` precedes `b`, both dominate `w`, and `#a = #b`, then `a ‚äñ w` strictly precedes `b ‚äñ w`.

*Proof.* We are given `a, b, w ‚àà T` with `a < b`, `a ‚â• w`, `b ‚â• w`, and `#a = #b`. We must show `a ‚äñ w < b ‚äñ w`.

**Preliminaries.** We recall the definitions on which the argument depends. T1 defines `a < b` by: there exists a least `k ‚â• 1` with `a·µ¢ = b·µ¢` for all `i < k`, and either (i) `k ‚â§ #a ‚àß k ‚â§ #b` with `a‚Çñ < b‚Çñ` (the conjunction replacing `k ‚â§ min(#a, #b)` so that no primitive binary-minimum operator on ‚Ñï is invoked, matching T1's Definition verbatim), or (ii) `k = #a + 1 ‚â§ #b` (`a` a proper prefix of `b`). TumblerSub defines `x ‚äñ w` (for `x ‚â• w`) by zero-padding both operands to a common length `L_{x,w}` ‚Äî the longer of `#x` and `#w`, named by NAT-order's trichotomy on the length pair `(#x, #w)` (sub-case (Œ±) `#x = #w` with `L_{x,w} = #x`; sub-case (Œ≤) `#x < #w` with `L_{x,w} = #w`; sub-case (Œ≥) `#w < #x` with `L_{x,w} = #x`) rather than by a primitive binary-maximum operator on ‚Ñï, reproducing TumblerSub's own length-pair dispatch ‚Äî and scanning for the first position where the padded sequences disagree. If no disagreement exists (*zero-padded equality*), the result is the zero tumbler of length `L_{x,w}`. If divergence occurs at position `d`, the result `r` satisfies: `r·µ¢ = 0` for `i < d`, `r_d = x_d - w_d`, and `r·µ¢ = x·µ¢` (zero-padded) for `i > d`, with `#r = L_{x,w}`.

**The form of `a < b`.** Since `#a = #b`, T1 case (ii) is impossible ‚Äî it requires `#a < #b`. So `a < b` holds by case (i): there exists a least `j` with `1 ‚â§ j ‚â§ #a` such that `a·µ¢ = b·µ¢` for all `i < j` and `a‚±º < b‚±º`. We fix this `j` throughout.

**Well-formedness.** By TA2, both `a ‚äñ w` and `b ‚äñ w` are well-defined members of `T`.

We proceed by exhaustive case analysis on the divergence structure of the pairs `(a, w)` and `(b, w)` under zero-padding.

**Case A: `a` is zero-padded-equal to `w`.** By TumblerSub, `a ‚äñ w` is the zero tumbler of length `L_{a,w}` ‚Äî the longer of `#a` and `#w`, named by NAT-order's trichotomy on `(#a, #w)` per the Preliminaries' dispatch instantiated at this specific operand pair. For `i < j`: `b_i = a_i` (from T1) and `a_i = w_i` (zero-padded equality), so `b_i = w_i`. At position `j`: `w_j = a_j` (zero-padded equality) and `b_j > a_j` (from `a < b`), giving `b_j > w_j`. So `(b, w)` diverges at position `j`, and TumblerSub yields `(b ‚äñ w)_j = b_j - w_j > 0` by NAT-sub (strict positivity), whose precondition `b_j > w_j` is the strict inequality just derived. Since `b_i = w_i` for all `i < j`, TumblerSub gives `(b ‚äñ w)_i = 0` for `i < j`. Both results are therefore zero before position `j`, and at `j` we have `(a ‚äñ w)_j = 0 < b_j - w_j = (b ‚äñ w)_j`. By T1 case (i), `a ‚äñ w < b ‚äñ w`.

**Setup for remaining cases.** Since `a` is not zero-padded-equal to `w`, the zero-padded divergence `d_a = zpd(a, w)` (TumblerSub) is well-defined. **Case 3: `d_a > d_b`.** At position `d_b`: `b_{d_b} ‚âÝ w_{d_b}` but `a_{d_b} = w_{d_b}` (since `d_b < d_a`). For `a ‚äñ w`: position `d_b` falls before `d_a`, placing it in the pre-divergence zero phase, so `(a ‚äñ w)_{d_b} = 0`.
