**TA3-strict (OrderPreservationUnderSubtractionStrict).** `(A a, b, w : a < b ∧ a ≥ w ∧ b ≥ w ∧ #a = #b : a ⊖ w < b ⊖ w)`.

We prove that subtracting a common lower bound from two equal-length tumblers preserves strict order: if `a` precedes `b`, both dominate `w`, and `#a = #b`, then `a ⊖ w` strictly precedes `b ⊖ w`.

*Proof.* We are given `a, b, w ∈ T` with `a < b`, `a ≥ w`, `b ≥ w`, and `#a = #b`. We must show `a ⊖ w < b ⊖ w`.

**Preliminaries.** We recall the definitions on which the argument depends. T1 defines `a < b` by: there exists a least `k ≥ 1` with `aᵢ = bᵢ` for all `i < k`, and either (i) `k ≤ #a ∧ k ≤ #b` with `aₖ < bₖ` (the conjunction replacing `k ≤ min(#a, #b)` so that no primitive binary-minimum operator on ℕ is invoked, matching T1's Definition verbatim), or (ii) `k = #a + 1 ≤ #b` (`a` a proper prefix of `b`). TumblerSub defines `x ⊖ w` (for `x ≥ w`) by zero-padding both operands to a common length `L_{x,w}` — the longer of `#x` and `#w`, named by NAT-order's trichotomy on the length pair `(#x, #w)` (sub-case (α) `#x = #w` with `L_{x,w} = #x`; sub-case (β) `#x < #w` with `L_{x,w} = #w`; sub-case (γ) `#w < #x` with `L_{x,w} = #x`) rather than by a primitive binary-maximum operator on ℕ, reproducing TumblerSub's own length-pair dispatch — and scanning for the first position where the padded sequences disagree. If no disagreement exists (*zero-padded equality*), the result is the zero tumbler of length `L_{x,w}`. If divergence occurs at position `d`, the result `r` satisfies: `rᵢ = 0` for `i < d`, `r_d = x_d - w_d`, and `rᵢ = xᵢ` (zero-padded) for `i > d`, with `#r = L_{x,w}`.

**The form of `a < b`.** Since `#a = #b`, T1 case (ii) is impossible — it requires `#a < #b`. So `a < b` holds by case (i): there exists a least `j` with `1 ≤ j ≤ #a` such that `aᵢ = bᵢ` for all `i < j` and `aⱼ < bⱼ`. We fix this `j` throughout.

**Well-formedness.** By TA2, both `a ⊖ w` and `b ⊖ w` are well-defined members of `T`.

We proceed by exhaustive case analysis on the divergence structure of the pairs `(a, w)` and `(b, w)` under zero-padding.

**Case A: `a` is zero-padded-equal to `w`.** By TumblerSub, `a ⊖ w` is the zero tumbler of length `L_{a,w}` — the longer of `#a` and `#w`, named by NAT-order's trichotomy on `(#a, #w)` per the Preliminaries' dispatch instantiated at this specific operand pair. For `i < j`: `b_i = a_i` (from T1) and `a_i = w_i` (zero-padded equality), so `b_i = w_i`. At position `j`: `w_j = a_j` (zero-padded equality) and `b_j > a_j` (from `a < b`), giving `b_j > w_j`. So `(b, w)` diverges at position `j`, and TumblerSub yields `(b ⊖ w)_j = b_j - w_j > 0` by NAT-sub (strict positivity), whose precondition `b_j > w_j` is the strict inequality just derived. Since `b_i = w_i` for all `i < j`, TumblerSub gives `(b ⊖ w)_i = 0` for `i < j`. Both results are therefore zero before position `j`, and at `j` we have `(a ⊖ w)_j = 0 < b_j - w_j = (b ⊖ w)_j`. By T1 case (i), `a ⊖ w < b ⊖ w`.

**Setup for remaining cases.** Since `a` is not zero-padded-equal to `w`, the zero-padded divergence `d_a = zpd(a, w)` (TumblerSub) is well-defined. Since `a ≥ w` and `a` is not zero-padded-equal to `w`, we have `a > w`, and at the first padded disagreement `a_{d_a} > w_{d_a}`. We verify that `d_b = zpd(b, w)` also exists: if `b` were zero-padded-equal to `w`, then `b_{d_a} = w_{d_a} < a_{d_a}`, and since `b_i = w_i = a_i` for `i < d_a`, T1 gives `a > b` — contradicting `a < b`. So `d_b` is well-defined, with `b_{d_b} > w_{d_b}` by the same reasoning from `b > w`.

**Case 1: `d_a = d_b = d`.** Both pairs diverge from `w` at position `d`. By TumblerSub, `(a ⊖ w)_i = 0` and `(b ⊖ w)_i = 0` for all `i < d`. Since `a` and `b` both agree with `w` before `d`, they agree with each other, so `j ≥ d`.

*Subcase `j = d`:* `(a ⊖ w)_d = a_d - w_d` and `(b ⊖ w)_d = b_d - w_d`, both well-defined natural numbers by NAT-sub (conditional closure) under `a_d ≥ w_d` and `b_d ≥ w_d` (from the setup inequalities via NAT-order's definition of `≤` from `<`). From `j = d`: `a_d < b_d`. NAT-sub's strict monotonicity, whose preconditions `a_d ≥ w_d`, `b_d ≥ w_d`, and `a_d < b_d` are now all in hand, yields `a_d - w_d < b_d - w_d`; strict positivity yields both differences are in fact ≥ 1. The results agree before `d` (both zero) and first disagree at `d`. By T1 case (i), `a ⊖ w < b ⊖ w`.

*Subcase `j > d`:* `a_d = b_d` (since `j > d`), so `(a ⊖ w)_d = a_d - w_d = b_d - w_d = (b ⊖ w)_d`. For `d < i < j`: both results are in TumblerSub's tail-copy phase, giving `(a ⊖ w)_i = a_i` and `(b ⊖ w)_i = b_i`; since `a_i = b_i` (`i < j`), the results agree. At position `j`: `(a ⊖ w)_j = a_j` and `(b ⊖ w)_j = b_j` (still tail-copy), with `a_j < b_j`. The results first disagree at `j`. By T1 case (i), `a ⊖ w < b ⊖ w`.

**Case 2: `d_a < d_b`.** At position `d_a`: `a_{d_a} ≠ w_{d_a}` but `b_{d_a} = w_{d_a}` (since `d_a < d_b`). Both agree with `w` — hence with each other — before `d_a`, and disagree at `d_a` (since `a_{d_a} ≠ w_{d_a} = b_{d_a}`), so `j = d_a`. From `a < b` by T1: `a_{d_a} < b_{d_a} = w_{d_a}`. But `a_{d_a} > w_{d_a}` (from setup) — contradiction. This case is impossible.

**Case 3: `d_a > d_b`.** At position `d_b`: `b_{d_b} ≠ w_{d_b}` but `a_{d_b} = w_{d_b}` (since `d_b < d_a`). Both agree with `w` — hence with each other — before `d_b`, and disagree at `d_b` (since `b_{d_b} ≠ w_{d_b} = a_{d_b}`), so `j = d_b`. From `a < b`: `a_{d_b} < b_{d_b}`, i.e., `w_{d_b} < b_{d_b}` — consistent with `b_{d_b} > w_{d_b}`.

For `a ⊖ w`: position `d_b` falls before `d_a`, placing it in the pre-divergence zero phase, so `(a ⊖ w)_{d_b} = 0`. For `b ⊖ w`: `d_b` is the divergence point, so `(b ⊖ w)_{d_b} = b_{d_b} - w_{d_b} > 0` by NAT-sub (strict positivity), whose precondition `b_{d_b} > w_{d_b}` is the setup's strict inequality. At all positions `i < d_b`, both results are zero (pre-divergence for both). The first disagreement is at `d_b` with `0 < b_{d_b} - w_{d_b}`. By T1 case (i), `a ⊖ w < b ⊖ w`.

In every case, `a ⊖ w < b ⊖ w` is established. ∎

*Formal Contract:*
- *Preconditions:* a ∈ T, b ∈ T, w ∈ T, a < b, a ≥ w, b ≥ w, #a = #b
- *Depends:* T1 (LexicographicOrder) — invoked at the Preliminaries to recall the definition of `<` ("T1 defines `a < b` by: there exists a least `k ≥ 1` with `aᵢ = bᵢ` for all `i < k`, and either (i) `k ≤ #a ∧ k ≤ #b` with `aₖ < bₖ` (the explicit conjunction replacing `k ≤ min(#a, #b)` so that no primitive binary-minimum operator on ℕ is invoked, matching T1's Definition verbatim and TumblerSub's, TA2's, and ZPD's analogous explicit-conjunction formulations), or (ii) `k = #a + 1 ≤ #b`"), at *The form of `a < b`* to rule out case (ii) given `#a = #b` and isolate case (i)'s witness `j`, in the setup-rebuttal "since `b_i = w_i = a_i` for `i < d_a`, T1 gives `a > b`", in Case 2 ("From `a < b` by T1: `a_{d_a} < b_{d_a}`") and Case 3 ("From `a < b`: `a_{d_b} < b_{d_b}`"), and at the closing of every case ("By T1 case (i), `a ⊖ w < b ⊖ w`") to convert each first-divergence witness into the strict ordering on the differences. TumblerSub (TumblerSub) — invoked at the Preliminaries to recall the definition of `x ⊖ w` (zero-padding, divergence-discovery, and the three-region `rᵢ = 0` for `i < d`, `r_d = x_d - w_d`, `rᵢ = xᵢ` for `i > d` rule), at every component-wise computation in Cases A, 1, and 3 (e.g. "TumblerSub yields `(b ⊖ w)_j = b_j - w_j > 0`", "By TumblerSub, `(a ⊖ w)_i = 0` and `(b ⊖ w)_i = 0` for all `i < d`"), and in the subcase "`j > d`" tail-copy reasoning. ZPD (ZPD) — invoked in the *Setup for remaining cases* to fix `d_a = zpd(a, w)` and `d_b = zpd(b, w)` ("the zero-padded divergence `d_a = zpd(a, w)` (TumblerSub) is well-defined") and again to license the case split `d_a = d_b`, `d_a < d_b`, `d_a > d_b` against the ZPD function. TA2 (WellDefinedSubtraction) — invoked at *Well-formedness* to discharge `a ⊖ w, b ⊖ w ∈ T` ("By TA2, both `a ⊖ w` and `b ⊖ w` are well-defined members of `T`"); without TA2 the membership of the two differences in T (required for T1 to apply) would be unsourced. NAT-sub (NatPartialSubtraction) — conditional closure discharges the well-definedness of `a_d - w_d` and `b_d - w_d` as natural numbers at every divergence point; strict positivity discharges `b_j - w_j > 0` in Case A and `b_{d_b} - w_{d_b} > 0` in Case 3 (from the setup's `b > w` strict inequality); strict monotonicity discharges `a_d - w_d < b_d - w_d` in Case 1's subcase `j = d` (from `a_d < b_d` with both `≥ w_d`). NAT-order (NatStrictTotalOrder) — invoked at four distinct roles: three length-pair trichotomy dispatches that name the result length of each subtraction invocation in place of a primitive binary-maximum operator on ℕ, and one defining-clause conversion at a component pair. *Length pair `(#a, #b)` (Preliminaries' T1 case (i) explicit conjunction)*: trichotomy's exactly-one clause names T1 case (i)'s shared-position bound as the explicit conjunction `k ≤ #a ∧ k ≤ #b` rather than `k ≤ min(#a, #b)`, so that no primitive binary-minimum operator on ℕ is invoked, matching T1's Definition verbatim and reproducing the analogous explicit-conjunction formulations in TumblerSub's precondition-consequence case (i), TA2's sub-case (i), ZPD's Relationship-to-Divergence case (i), and Divergence's case (i); the TA3-strict precondition `#a = #b` places this pair in sub-case (α), but the conjunction form is retained for uniformity with T1's statement and with every downstream consumer that reads the conjunction directly. *Length pair `(#x, #w)` at the Preliminaries' generic TumblerSub recall, instantiated twice — at `(#a, #w)` for `a ⊖ w` (also consumed in Case A's explicit "zero tumbler of length `L_{a,w}`" framing) and at `(#b, #w)` for `b ⊖ w`*: trichotomy's exactly-one clause discharges each length-pair dispatch that names the longer length `L` of a subtraction's operand pair, selecting exactly one of sub-case (α) first-slot `=` second-slot (`L` = first), sub-case (β) first-slot `<` second-slot (`L` = second), or sub-case (γ) second-slot `<` first-slot (`L` = first); `L_{a,w}` — the longer of `#a` and `#w`, furnishing the result length of `a ⊖ w` in the Preliminaries' generic recall and in Case A's explicit framing — and `L_{b,w}` — the longer of `#b` and `#w`, furnishing the result length of `b ⊖ w` — are each named by the selected sub-case rather than by a primitive binary-maximum operator on ℕ, reproducing TumblerSub's own length-pair dispatch at each operand pair and matching the convention that every operator used must be discharged by a Depends entry. The `(#a, #w)` and `(#b, #w)` instantiations are distinct ℕ-pairs — the same trichotomy axiom applied at different symbols — and earn separate sites under the per-instance convention declared in T1's Depends (under the TA3-strict precondition `#a = #b` the two pairs coincide after substitution, but the instantiations at distinct symbols are counted separately, matching TA3's own per-pair accounting of `(#a, #w)` and `(#b, #w)` as separate trichotomy roles). *Component pair `(x_d, w_d)` (internal to NAT-sub's discharge)*: the defining clause `m ≤ n ⟺ m < n ∨ m = n` converts the strict inequalities `x_d > w_d` arising at divergence points into the weak `x_d ≥ w_d` form required by NAT-sub's preconditions, via the definition of `≤` from `<`. All four roles are enumerated together under this single NAT-order citation, matching the per-step convention TA2, TumblerSub, and ZPD apply when a single axiom is invoked at multiple ℕ-pairs.
- *Postconditions:* a ⊖ w < b ⊖ w
