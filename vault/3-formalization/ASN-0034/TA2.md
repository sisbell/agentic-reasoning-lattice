### Subtraction for width computation

Let `⊖` denote tumbler subtraction: given two positions, compute the displacement between them.

**TA2 (WellDefinedSubtraction).** For tumblers `a, w ∈ T` where `a ≥ w`, `a ⊖ w` is a well-defined tumbler in `T`.

*Proof.* We show that for all `a, w ∈ T` with `a ≥ w`, the operation `a ⊖ w` as defined by TumblerSub produces a member of `T` — a finite sequence of non-negative integers with at least one component.

By TumblerSub, subtraction zero-pads both operands to length `L` — the longer of `#a` and `#w`, named by NAT-order's trichotomy on the length pair `(#a, #w)` (sub-case (α) `#a = #w` with `L = #a`; sub-case (β) `#a < #w` with `L = #w`; sub-case (γ) `#w < #a` with `L = #a`) rather than by a primitive binary-maximum operator on ℕ, reproducing TumblerSub's own length-pair dispatch for the same operand pair — and scans for the first position at which the padded sequences disagree. Two cases arise.

*Case 1: no divergence (zero-padded equality).* The padded sequences of `a` and `w` agree at every position. TumblerSub produces the zero tumbler `[0, ..., 0]` of length `L`. Since `#a ≥ 1` and `#w ≥ 1` (both are tumblers in T), `L ≥ 1` — whichever of `#a` or `#w` the trichotomy names as `L`, the named length inherits the ≥ 1 bound from T0's minimum-length clause. Each component is `0 ∈ ℕ` by NAT-zero — T0's component-typing covers only *existing* positions of `a` and `w`, not the literal zeros freshly introduced by the zero-tumbler construction. The result is a finite sequence of non-negative integers with length at least 1 — a member of T.

*Case 2: divergence at position `k`.* The padded sequences agree at all positions `i < k` and disagree at `k`. TumblerSub defines the result `r = a ⊖ w` componentwise: `rᵢ = 0` for `i < k`, `rₖ = aₖ - wₖ`, and `rᵢ = aᵢ` for `i > k` (where all component references use zero-padded values), with `#r = L`.

We verify that each region produces non-negative integer components.

*Pre-divergence* (`i < k`): `rᵢ = 0 ∈ ℕ` by NAT-zero — the literal `0` assigned by TumblerSub's Definition at positions strictly below the divergence point is not among T0's component-typing assertions for `a` or `w`, so the ℕ-membership of each such `rᵢ` must be supplied separately.

*Divergence point* (`i = k`): We must show `aₖ ≥ wₖ` (zero-padded values) so that `rₖ = aₖ - wₖ ∈ ℕ`. Since the divergence exists, the padded sequences differ, so `a` and `w` are not zero-padded-equal. If `a = w` as tumblers (by T3: same length and components), then their padded sequences are trivially identical — no divergence exists, contradicting the case hypothesis. Therefore `a ≠ w`, and combined with `a ≥ w` this gives `a > w` under T1. Two sub-cases arise from T1's definition of strict ordering.

*Sub-case (i): T1 case (i) — component divergence.* There exists a first position `j` with `j ≤ #a ∧ j ≤ #w` (the conjunction replacing `j ≤ min(#a, #w)` so that no primitive binary-minimum operator on ℕ is invoked, matching Divergence's case (i) and TumblerSub's precondition-consequence case (i)), with `aⱼ > wⱼ` and `aᵢ = wᵢ` for all `i < j`. These positions lie within both original sequences, so the zero-padded values agree with the originals. The padded sequences therefore agree before `j` and disagree at `j`, making `j` the first padded divergence: `k = j`. At position `k`, `aₖ > wₖ`, whence `aₖ ≥ wₖ` by the definition of `≤` from `<` (NAT-order); NAT-sub's conditional-closure clause then yields `rₖ = aₖ - wₖ ∈ ℕ`.

*Sub-case (ii): T1 case (ii) — prefix relationship.* Here `w` is a proper prefix of `a`: `#w < #a` and `aᵢ = wᵢ` for all `i ≤ #w`. This places the length pair `(#a, #w)` in the trichotomy's sub-case (γ) where `L = #a`, so zero-padding extends `w` with zeros at positions `#w + 1` through `L = #a`, with `0 ∈ ℕ` supplied by NAT-zero so that the padded values extend T0's ℕ-valued component-typing uniformly onto the padding region `i > #w` and the subsequent ℕ-valued comparisons at the divergence point are well-formed. The padded sequences agree at all positions `i ≤ #w`. The divergence `k` falls at the first position `i > #w` where `aᵢ ≠ 0` — such a position must exist, for if `aᵢ = 0` at every `i > #w` the padded sequences would agree everywhere, contradicting the case hypothesis. At position `k`, `aₖ ≠ 0 = wₖ` (zero-padded, where `wₖ = 0` by the prefix zero-padding just noted, with `0 ∈ ℕ` supplied by NAT-zero making the equality a well-formed ℕ-valued comparison). The step `aₖ ≠ 0 ⟹ aₖ > 0` is discharged by NAT-zero's lower bound `0 ≤ aₖ` (instantiated at `n = aₖ`, with `aₖ ∈ ℕ` supplied by T0's component-typing), NAT-order's defining clause `m ≤ n ⟺ m < n ∨ m = n` at `m = 0, n = aₖ` unfolding `0 ≤ aₖ` to `0 < aₖ ∨ 0 = aₖ`, and the divergence observation `aₖ ≠ 0` excluding the equality disjunct to leave `0 < aₖ`; hence `aₖ > 0 = wₖ` (the `0 = wₖ` equality again using NAT-zero's `0 ∈ ℕ` for the zero-padded value of `w`), matching the per-step convention TumblerSub's precondition-consequence case (ii) and ActionPoint's third postcondition apply for the structurally identical `≠ 0 ⟹ > 0` step. Then `aₖ ≥ wₖ` by NAT-order's definition of `≤` from `<`; NAT-sub's conditional closure then yields `rₖ = aₖ - 0 = aₖ ∈ ℕ` (with the subtrahend `0 ∈ ℕ` again from NAT-zero's zero-padding supply).

*Tail* (`i > k`): `rᵢ = aᵢ` (zero-padded). If `i ≤ #a`, then `aᵢ` is a component of `a ∈ T`, hence `aᵢ ∈ ℕ` by T0's component-typing. If `i > #a`, then `aᵢ = 0 ∈ ℕ` (zero-padded) by NAT-zero — T0's component-typing is silent on positions outside `a`'s original length `{1, ..., #a}`, so the ℕ-membership of the zero-padded tail values must be supplied separately.

The result `r` has length `L ≥ 1` — by the same trichotomy dispatch that names `L` above, the named length is one of `#a` or `#w`, each ≥ 1 by T0's minimum-length clause — with every component in ℕ, a member of T.

In both cases, `a ⊖ w ∈ T` with `#(a ⊖ w) = L`. ∎

*Formal Contract:*
- *Preconditions:* a ∈ T, w ∈ T, a ≥ w
- *Depends:* TumblerSub (TumblerSub) — supplies the piecewise construction of `a ⊖ w`: zero-padding, case split on divergence, componentwise definition, and result length `L` (the longer of `#a` and `#w`, named by NAT-order trichotomy on `(#a, #w)` per TumblerSub's Definition). T0 (CarrierSetDefinition) — the proof uses T0 in two ways: (1) T0's minimum-length guarantee `#a ≥ 1` for `a ∈ T` yields `L ≥ 1` — whichever of `#a` or `#w` the NAT-order trichotomy names as `L`, the named length inherits the ≥ 1 bound from T0; (2) T0's carrier-set characterisation — membership in T as a finite sequence over ℕ with length ≥ 1 — is the criterion applied in both cases to conclude `a ⊖ w ∈ T`. T1 (LexicographicOrder) — the proof derives `a > w` from `a ≥ w ∧ a ≠ w` via T1 trichotomy, then uses T1's two cases to establish `aₖ ≥ wₖ` at the divergence point. T3 (CanonicalRepresentation) — the proof concludes `a ≠ w` from the existence of a padded divergence: if `a = w` by T3, the padded sequences would be identical, contradicting the case hypothesis. NAT-sub (NatPartialSubtraction) — the divergence-point component `rₖ = aₖ - wₖ ∈ ℕ` is discharged by NAT-sub's conditional-closure clause in both sub-cases (i) and (ii), after T1 supplies `aₖ ≥ wₖ`; without NAT-sub the step would use partial subtraction on ℕ as an unsourced primitive. NAT-zero (NatZeroMinimum) — the literal `0` appears at four ASN-internal sites in TA2, each requiring `0 ∈ ℕ` so that the componentwise membership claim `rᵢ ∈ ℕ` underlying `a ⊖ w ∈ T` respects T0's ℕ-valued component-typing requirement and the ℕ-valued (in)equalities invoked at the divergence point are well-formed. T0 supplies component-membership in ℕ only for *existing* positions `1 ≤ i ≤ #a`, `1 ≤ i ≤ #w`, so neither the padded values at positions `i > #a`, `i > #w` nor the literal-`0` components of the result are among T0's assertions; among the other NAT-* axioms cited in TA2's Depends (NAT-order, NAT-sub) none identifies which elements belong to ℕ — each quantifies over ℕ presupposing its extension rather than fixing it. NAT-zero's axiom `(A n ∈ ℕ :: 0 ≤ n)` quantifies `0` as a lower bound of ℕ and thereby presupposes `0 ∈ ℕ` as a member of its quantification domain, supplying exactly the missing membership premise at all four sites — this citation discharges the literal `0` from NAT-zero rather than leaving `0 ∈ ℕ` implicit, matching the per-step convention TumblerSub's Definition, ZPD, TA-Pos, and ActionPoint already follow for discharging identical `0`-related premises from NAT-zero rather than T0, and reconciling the ASN's convention that T0 alone does not suffice for `0 ∈ ℕ` at zero-padded positions. (i) *Case 1 no-divergence branch*: the zero tumbler `[0, ..., 0]` of length `L` has every component the literal `0`; the membership-proof step "Each component is `0 ∈ ℕ`" consumed in concluding `a ⊖ w ∈ T` in this branch is well-formed only when `0 ∈ ℕ`, matching TumblerSub's Definition clause `a ⊖ w = [0, …, 0]` in the no-divergence branch. (ii) *Case 2 pre-divergence* (`i < k`): the componentwise value `rᵢ = 0` at every position strictly below the divergence point requires `0 ∈ ℕ` so that the per-position membership claim `rᵢ = 0 ∈ ℕ` is well-formed, matching TumblerSub's Definition clause `rᵢ = 0` for `i < k`. (iii) *Case 2 sub-case (ii) prefix zero-padding and lower-bound premise*: the prefix padding of `w` places literal zeros at positions `#w + 1` through `L = #a`, furnishing the zero-padded value `wₖ = 0` at `k > #w` consumed in the ℕ-valued comparisons `aₖ ≠ 0 = wₖ` and `aₖ > 0 = wₖ` and in the divergence-point component `rₖ = aₖ − 0 = aₖ ∈ ℕ` fed to NAT-sub's conditional-closure clause; the same case additionally invokes NAT-zero's lower-bound axiom `(A n ∈ ℕ :: 0 ≤ n)` at `n = aₖ` (with `aₖ ∈ ℕ` supplied by T0's component-typing) to furnish the premise `0 ≤ aₖ` consumed by NAT-order's defining-clause unfolding that derives `aₖ > 0` from `aₖ ≠ 0`, matching the per-step convention TumblerSub's precondition-consequence case (ii) and ActionPoint's third postcondition apply for the structurally identical `≠ 0 ⟹ > 0` step — NAT-order alone (whose trichotomy compares `0` and `aₖ` without fixing the winner) does not furnish this premise. (iv) *Case 2 tail* (`i > k`, `i > #a`): the zero-padded tail value `aᵢ = 0 ∈ ℕ` at `i > #a` — consumed in the membership step for tail positions outside the minuend's original length `{1, ..., #a}` — requires `0 ∈ ℕ` so that the ℕ-membership claim is well-formed, matching TumblerSub's Definition clause `rᵢ = aᵢ` (zero-padded) for `i > k` applied at `i > #a`. NAT-order (NatStrictTotalOrder) — invoked at three distinct pairs for three distinct roles. *Length pair `(#a, #w)` (internal to the Case 2 length-dispatch)*: trichotomy's exactly-one clause discharges the length-pair dispatch that names the result length `L`, selecting exactly one of sub-case (α) `#a = #w` (`L = #a`), sub-case (β) `#a < #w` (`L = #w`), or sub-case (γ) `#w < #a` (`L = #a`); `L` — the longer of `#a` and `#w`, which furnishes the result length `#(a ⊖ w) = L` in the Case 2 construction, the Postconditions, and the `L ≥ 1` membership check consumed in both Case 1 and Case 2 — is named by the selected sub-case rather than by a primitive binary-maximum operator on ℕ, reproducing TumblerSub's length-pair dispatch for the same operand pair rather than treating `max(#a, #w)` as an unsourced primitive, and matching the convention that every operator in TumblerSub's Definition must be discharged by a Depends entry. The same dispatch also names the shared-position bound `j ≤ #a ∧ j ≤ #w` in Sub-case (i) as the explicit conjunction rather than `j ≤ min(#a, #w)`, matching Divergence's case (i) and TumblerSub's precondition-consequence case (i) so that no primitive binary-minimum operator on ℕ is invoked. *Component pair `(0, aₖ)` (sub-case (ii) `≠ 0 ⟹ > 0` discharge)*: the defining clause `m ≤ n ⟺ m < n ∨ m = n` at `m = 0, n = aₖ` unfolds NAT-zero's lower bound `0 ≤ aₖ` to `0 < aₖ ∨ 0 = aₖ`, after which the divergence observation `aₖ ≠ 0 = wₖ` (where `wₖ = 0` by the prefix zero-padding) excludes the equality disjunct to leave `0 < aₖ`, i.e., `aₖ > 0`; this NAT-zero + NAT-order discharge matches the per-step convention TumblerSub's precondition-consequence case (ii) and ActionPoint's third postcondition apply for the structurally identical `≠ 0 ⟹ > 0` inference. *Component pair `(aₖ, wₖ)` (internal to the NAT-sub discharge)*: converts the strict inequalities `aₖ > wₖ` (sub-case i) and `aₖ > 0` (sub-case ii, where `wₖ = 0` by zero-padding) into the weak form `aₖ ≥ wₖ` required by NAT-sub's conditional-closure clause, via the defining clause `m ≤ n ⟺ m < n ∨ m = n` of `≤` from `<`. All three roles are instances of the same NAT-order axiom applied at distinct ℕ-pairs, enumerated together under this single citation.
- *Postconditions:* a ⊖ w ∈ T, #(a ⊖ w) = L (the longer of `#a` and `#w`, named by NAT-order's trichotomy on `(#a, #w)` per TumblerSub's Definition: `L = #a` in sub-cases (α) and (γ), `L = #w` in sub-case (β))
