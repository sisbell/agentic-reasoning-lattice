### Subtraction for width computation

Let `‚äĖ` denote tumbler subtraction: given two positions, compute the displacement between them.

**TA2 (WellDefinedSubtraction).** For tumblers `a, w ‚ąą T` where `a ‚Č• w`, `a ‚äĖ w` is a well-defined tumbler in `T`.

*Proof.* We show that for all `a, w ‚ąą T` with `a ‚Č• w`, the operation `a ‚äĖ w` as defined by TumblerSub produces a member of `T` ‚ÄĒ a finite sequence of non-negative integers with at least one component.

By TumblerSub, subtraction zero-pads both operands to length `L` ‚ÄĒ the longer of `#a` and `#w`, named by NAT-order's trichotomy on the length pair `(#a, #w)` (sub-case (őĪ) `#a = #w` with `L = #a`; sub-case (ő≤) `#a < #w` with `L = #w`; sub-case (ő≥) `#w < #a` with `L = #a`) rather than by a primitive binary-maximum operator on ‚Ąē, reproducing TumblerSub's own length-pair dispatch for the same operand pair ‚ÄĒ and scans for the first position at which the padded sequences disagree. Two cases arise.

*Case 1: no divergence (zero-padded equality).* The padded sequences of `a` and `w` agree at every position. TumblerSub produces the zero tumbler `[0, ..., 0]` of length `L`. Since `#a ‚Č• 1` and `#w ‚Č• 1` (both are tumblers in T), `L ‚Č• 1` ‚ÄĒ whichever of `#a` or `#w` the trichotomy names as `L`, the named length inherits the ‚Č• 1 bound from T0's minimum-length clause. Each component is `0 ‚ąą ‚Ąē` by NAT-zero ‚ÄĒ T0's component-typing covers only *existing* positions of `a` and `w`, not the literal zeros freshly introduced by the zero-tumbler construction. The result is a finite sequence of non-negative integers with length at least 1 ‚ÄĒ a member of T.

*Case 2: divergence at position `k`.* The padded sequences agree at all positions `i < k` and disagree at `k`. TumblerSub defines the result `r = a ‚äĖ w` componentwise: `rŠĶĘ = 0` for `i < k`, `r‚āĖ = a‚āĖ - w‚āĖ`, and `rŠĶĘ = aŠĶĘ` for `i > k` (where all component references use zero-padded values), with `#r = L`.

We verify that each region produces non-negative integer components.

*Pre-divergence* (`i < k`): `rŠĶĘ = 0 ‚ąą ‚Ąē` by NAT-zero ‚ÄĒ the literal `0` assigned by TumblerSub's Definition at positions strictly below the divergence point is not among T0's component-typing assertions for `a` or `w`, so the ‚Ąē-membership of each such `rŠĶĘ` must be supplied separately.

*Divergence point* (`i = k`): We must show `a‚āĖ ‚Č• w‚āĖ` (zero-padded values) so that `r‚āĖ = a‚āĖ - w‚āĖ ‚ąą ‚Ąē`. Since the divergence exists, the padded sequences differ, so `a` and `w` are not zero-padded-equal. If `a = w` as tumblers (by T3: same length and components), then their padded sequences are trivially identical ‚ÄĒ no divergence exists, contradicting the case hypothesis. Therefore `a ‚Č† w`, and combined with `a ‚Č• w` this gives `a > w` under T1. Two sub-cases arise from T1's definition of strict ordering.

*Sub-case (i): T1 case (i) ‚ÄĒ component divergence.* There exists a first position `j` with `j ‚Č§ #a ‚ąß j ‚Č§ #w` (the conjunction replacing `j ‚Č§ min(#a, #w)` so that no primitive binary-minimum operator on ‚Ąē is invoked, matching Divergence's case (i) and TumblerSub's precondition-consequence case (i)), with `a‚Īľ > w‚Īľ` and `aŠĶĘ = wŠĶĘ` for all `i < j`. These positions lie within both original sequences, so the zero-padded values agree with the originals. The padded sequences therefore agree before `j` and disagree at `j`, making `j` the first padded divergence: `k = j`. At position `k`, `a‚āĖ > w‚āĖ`, whence `a‚āĖ ‚Č• w‚āĖ` by the definition of `‚Č§` from `<` (NAT-order); NAT-sub's conditional-closure clause then yields `r‚āĖ = a‚āĖ - w‚āĖ ‚ąą ‚Ąē`.

*Sub-case (ii): T1 case (ii) ‚ÄĒ prefix relationship.* Here `w` is a proper prefix of `a`: `#w < #a` and `aŠĶĘ = wŠĶĘ` for all `i ‚Č§ #w`. This places the length pair `(#a, #w)` in the trichotomy's sub-case (ő≥) where `L = #a`, so zero-padding extends `w` with zeros at positions `#w + 1` through `L = #a`, with `0 ‚ąą ‚Ąē` supplied by NAT-zero so that the padded values extend T0's ‚Ąē-valued component-typing uniformly onto the padding region `i > #w` and the subsequent ‚Ąē-valued comparisons at the divergence point are well-formed. The padded sequences agree at all positions `i ‚Č§ #w`. The divergence `k` falls at the first position `i > #w` where `aŠĶĘ ‚Č† 0` ‚ÄĒ such a position must exist, for if `aŠĶĘ = 0` at every `i > #w` the padded sequences would agree everywhere, contradicting the case hypothesis. At position `k`, `a‚āĖ ‚Č† 0 = w‚āĖ` (zero-padded, where `w‚āĖ = 0` by the prefix zero-padding just noted, with `0 ‚ąą ‚Ąē` supplied by NAT-zero making the equality a well-formed ‚Ąē-valued comparison). The step `a‚āĖ ‚Č† 0 ‚üĻ a‚āĖ > 0` is discharged by NAT-zero's lower bound `0 ‚Č§ a‚āĖ` (instantiated at `n = a‚āĖ`, with `a‚āĖ ‚ąą ‚Ąē` supplied by T0's component-typing), NAT-order's defining clause `m ‚Č§ n ‚üļ m < n ‚ą® m = n` at `m = 0, n = a‚āĖ` unfolding `0 ‚Č§ a‚āĖ` to `0 < a‚āĖ ‚ą® 0 = a‚āĖ`, and the divergence observation `a‚āĖ ‚Č† 0` excluding the equality disjunct to leave `0 < a‚āĖ`; hence `a‚āĖ > 0 = w‚āĖ` (the `0 = w‚āĖ` equality again using NAT-zero's `0 ‚ąą ‚Ąē` for the zero-padded value of `w`), matching the per-step convention TumblerSub's precondition-consequence case (ii) and ActionPoint's third postcondition apply for the structurally identical `‚Č† 0 ‚üĻ > 0` step. Then `a‚āĖ ‚Č• w‚āĖ` by NAT-order's definition of `‚Č§` from `<`; NAT-sub's conditional closure then yields `r‚āĖ = a‚āĖ - 0 ‚ąą ‚Ąē` (with the subtrahend `0 ‚ąą ‚Ąē` again from NAT-zero's zero-padding supply).

*Tail* (`i > k`): `rŠĶĘ = aŠĶĘ` (zero-padded). If `i ‚Č§ #a`, then `aŠĶĘ` is a component of `a ‚ąą T`, hence `aŠĶĘ ‚ąą ‚Ąē` by T0's component-typing. If `i > #a`, then `aŠĶĘ = 0 ‚ąą ‚Ąē` (zero-padded) by NAT-zero ‚ÄĒ T0's component-typing is silent on positions outside `a`'s original length `{1, ..., #a}`, so the ‚Ąē-membership of the zero-padded tail values must be supplied separately.

The result `r` has length `L ‚Č• 1` ‚ÄĒ by the same trichotomy dispatch that names `L` above, the named length is one of `#a` or `#w`, each ‚Č• 1 by T0's minimum-length clause ‚ÄĒ with every component in ‚Ąē, a member of T.

In both cases, `a ‚äĖ w ‚ąą T` with `#(a ‚äĖ w) = L`. ‚ąé
