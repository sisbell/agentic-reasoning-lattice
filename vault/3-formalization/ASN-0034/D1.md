**D1 (DisplacementRoundTrip).** For tumblers a, b ∈ T with a < b, divergence(a, b) ≤ #a, and #a ≤ #b:

  a ⊕ (b ⊖ a) = b

*Proof.* We must show that the displacement from a to b, when added back to a, recovers b exactly.

Let k = divergence(a, b). The preconditions give k ≤ #a and #a ≤ #b. We eliminate Divergence case (ii) by addressing each of its sub-cases as Divergence states them. Sub-case (ii-a) — `#a < #b` with `divergence(a, b) = #a + 1`, whence `k = #a + 1` — is foreclosed by combining the sub-case value with the precondition `k ≤ #a`: substitution yields `#a + 1 ≤ #a`, which against NAT-addcompat's strict successor inequality `#a < #a + 1` is refuted by NAT-order's trichotomy at the pair `(#a, #a + 1)` — its exactly-one clause excluding both `#a + 1 < #a` and `#a + 1 = #a` once `#a < #a + 1` is fixed. Sub-case (ii-b) — `#b < #a` with `divergence(a, b) = #b + 1` — is foreclosed directly by the precondition `#a ≤ #b`: NAT-order's trichotomy at the length pair `(#a, #b)` rules out `#b < #a` co-holding with either disjunct of `#a ≤ #b` (the strict `#a < #b` or the equality `#a = #b`). The remaining situation `#a = #b` (the equality disjunct of the precondition) is excluded from case (ii) at the case-hypothesis level, since Divergence case (ii) requires `#a ≠ #b`. Case (ii) is therefore eliminated on each of its sub-cases and at its case hypothesis, placing us in case (i): aᵢ = bᵢ for all i < k, and aₖ < bₖ (the strict inequality follows from a < b at the divergence point, by T1).

Define w = b ⊖ a. TumblerSub computes this via case analysis on `zpd(b, a)` (ZPD). Since we are in Divergence case (i) — component divergence at position `k` satisfying the shared-bound conjunction `k ≤ #a ∧ k ≤ #b` that Divergence case (i) states in place of any primitive binary-minimum on ℕ — the ZPD–Divergence relationship (ZPD), instantiated at the single operand pair `(b, a)` to match `zpd(b, a)`'s operand order, gives `zpd(b, a) = divergence(b, a)`; Divergence's symmetry postcondition `divergence(a, b) = divergence(b, a)` then bridges this to `k = divergence(a, b)`, yielding `zpd(b, a) = divergence(b, a) = divergence(a, b) = k` without instantiating the ZPD–Divergence relationship at two different operand pairs. The subtraction yields:

  wᵢ = 0           for i < k
  wₖ = bₖ − aₖ     (well-defined ∈ ℕ by NAT-sub conditional closure, since bₖ ≥ aₖ via NAT-order from bₖ > aₖ)
  wᵢ = bᵢ          for i > k

The result's length `#w` is named by TumblerSub's length-pair dispatch — NAT-order's trichotomy at `(#b, #a)` selecting sub-case (α) `#b = #a` with `L = #b`, sub-case (β) `#b < #a` with `L = #a`, or sub-case (γ) `#a < #b` with `L = #b`, rather than via any primitive binary-maximum on ℕ. The precondition `#a ≤ #b` forecloses sub-case (β): NAT-order's exactly-one clause at the length pair `(#a, #b)` — the same invocation already used above to exclude Divergence sub-case (ii-b) — rules out `#b < #a` co-holding with either disjunct of `#a ≤ #b`. Sub-case (α) (under the equality disjunct `#a = #b`) and sub-case (γ) (under the strict disjunct `#a < #b`) both give `L = #b`, so `#w = #b`.

We establish three properties of w needed for TumblerAdd. First, w ∈ T: by hypothesis b ∈ T and a ∈ T, and since a < b implies b ≥ a, TumblerSub's preconditions are satisfied for minuend b and subtrahend a, and its postcondition gives w = b ⊖ a ∈ T. Second, Pos(w): the component wₖ = bₖ − aₖ ≥ 1 by NAT-sub (strict positivity), whose precondition `bₖ > aₖ` is the strict inequality derived from `a < b` at the divergence point, so w is not the zero tumbler. Third, the action point of w is k: every component before position k is zero, and wₖ > 0, so k is the first positive component. Since k ≤ #a by hypothesis, the precondition of TumblerAdd (TA0) is satisfied — the action point falls within the start position's length.

Now we compute a ⊕ w by TumblerAdd's constructive definition, which builds the result component by component in three regions determined by the action point k:

*Positions i < k (prefix copy):* (a ⊕ w)ᵢ = aᵢ. By the Divergence agreement property, aᵢ = bᵢ for all i < k. So (a ⊕ w)ᵢ = bᵢ.

*Position i = k (single-component advance):* (a ⊕ w)ₖ = aₖ + wₖ = aₖ + (bₖ − aₖ) = bₖ. The final equality `aₖ + (bₖ − aₖ) = bₖ` is NAT-sub's left-inverse characterisation applied to the naturals `bₖ, aₖ ∈ ℕ` with `bₖ ≥ aₖ` (from the strict inequality `bₖ > aₖ` via NAT-order); the substitution `wₖ = bₖ − aₖ` is the definition of `w` at position `k`.

*Positions i > k (tail from displacement):* (a ⊕ w)ᵢ = wᵢ = bᵢ. These components were copied from b into w by TumblerSub, and TumblerAdd copies them from w into the result.

It remains to verify that the lengths match. By the result-length identity of TumblerAdd, #(a ⊕ w) = #w = #b. Every component of a ⊕ w equals the corresponding component of b, and both tumblers have length #b. By T3 (CanonicalRepresentation), a ⊕ w = b.  ∎

*Formal Contract:*
- *Preconditions:* a ∈ T, b ∈ T, a < b, divergence(a, b) ≤ #a, #a ≤ #b
- *Depends:* Divergence (Divergence) — case analysis eliminates case (ii) by addressing sub-cases (ii-a) and (ii-b) and the `#a = #b` case-hypothesis failure separately, establishing agreement aᵢ = bᵢ for i < k from the remaining case (i), which supplies the shared-bound conjunction `k ≤ #a ∧ k ≤ #b` in place of any primitive binary-minimum on ℕ; Divergence's symmetry postcondition `divergence(a, b) = divergence(b, a)` bridges the operand-pair mismatch between the precondition's `k = divergence(a, b)` and the ZPD–Divergence relationship's natural instantiation at `(b, a)` (matching `zpd(b, a)`'s operand order), yielding `zpd(b, a) = divergence(b, a) = divergence(a, b) = k` without any operand swap on the ZPD side. T1 (LexicographicOrder) — derives aₖ < bₖ from a < b at divergence point k. ZPD (ZPD) — identifies zpd(b, a) = divergence(b, a) via the ZPD–Divergence relationship in case (i), instantiated at the single operand pair `(b, a)` to match `zpd(b, a)`'s operand order; the onward identification with `k = divergence(a, b)` is supplied by Divergence's symmetry postcondition (cited above), not by re-instantiating the ZPD–Divergence relationship at a different operand pair. TumblerSub (TumblerSub) — supplies component formulas for w = b ⊖ a, establishes w ∈ T from b ≥ a, and supplies the length-pair dispatch at `(#b, #a)` — sub-cases (α) `#b = #a` with `L = #b`, (β) `#b < #a` with `L = #a`, (γ) `#a < #b` with `L = #b` — that names the result length `#w = L` per sub-case in place of any primitive binary-maximum on ℕ. TA-Pos (PositiveTumbler) — defines Pos(w), satisfied since wₖ = bₖ − aₖ ≥ 1. TA0 (WellDefinedAddition) — establishes a ⊕ w ∈ T from Pos(w) and actionPoint(w) = k ≤ #a. TumblerAdd (TumblerAdd) — supplies the constructive component-by-component definition and the result-length identity #(a ⊕ w) = #w. T3 (CanonicalRepresentation) — concludes a ⊕ w = b from component-wise equality and #(a ⊕ w) = #b. NAT-sub (NatPartialSubtraction) — conditional closure discharges `wₖ = bₖ − aₖ ∈ ℕ`; strict positivity discharges `wₖ ≥ 1` (from `bₖ > aₖ`), which feeds into the verification that Pos(w) holds; the left-inverse characterisation discharges `aₖ + (bₖ − aₖ) = bₖ` at position k of the single-component advance. NAT-addcompat (NatAdditionOrderAndSuccessor) — supplies the strict successor inequality `#a < #a + 1` used in the sub-case (ii-a) elimination: combined with `#a + 1 ≤ #a` (from substituting the sub-case value `k = #a + 1` into the precondition `k ≤ #a`) it produces the conjunction that NAT-order's trichotomy at `(#a, #a + 1)` forecloses, refuting sub-case (ii-a). NAT-order (NatStrictTotalOrder) — invoked at three distinct roles, each an instance of the same axiom at distinct points. First, trichotomy at the pair `(#a, #a + 1)` forecloses sub-case (ii-a) of Divergence case (ii): once NAT-addcompat supplies `#a < #a + 1`, the exactly-one clause excludes both `#a + 1 < #a` and `#a + 1 = #a`, refuting the substituted `#a + 1 ≤ #a`. Second, trichotomy at the length pair `(#a, #b)` forecloses sub-case (ii-b) of Divergence case (ii) directly from the precondition `#a ≤ #b` (excluding `#b < #a` co-holding with either disjunct of the precondition); the same length-pair invocation — reused under the shared-pair convention — also forecloses sub-case (β) of TumblerSub's length-pair dispatch at `(#b, #a)`, leaving sub-cases (α) and (γ) to give `L = #b` as the length of `w`. Third, the `<`-to-`≤` weakening converts `bₖ > aₖ` into the weak `bₖ ≥ aₖ` required by NAT-sub's closure and left-inverse preconditions.
- *Postconditions:* a ⊕ (b ⊖ a) = b
