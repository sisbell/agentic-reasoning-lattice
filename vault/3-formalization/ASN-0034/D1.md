**D1 (DisplacementRoundTrip).** For tumblers a, b ∈ T with a < b, divergence(a, b) ≤ #a, and #a ≤ #b:

  a ⊕ (b ⊖ a) = b

*Proof.* We must show that the displacement from a to b, when added back to a, recovers b exactly.

Let k = divergence(a, b). The preconditions give k ≤ #a and #a ≤ #b, so in particular k ≤ #a ≤ #b, whence k ≤ min(#a, #b). This rules out Divergence case (ii) — which would require k = min(#a, #b) + 1 > #a — and places us in case (i): aᵢ = bᵢ for all i < k, and aₖ < bₖ (the strict inequality follows from a < b at the divergence point, by T1).

Define w = b ⊖ a. TumblerSub computes this via case analysis on `zpd(b, a)` (ZPD). Since we are in Divergence case (i) — component divergence at position `k ≤ min(#a, #b)` — the ZPD–Divergence relationship (ZPD) gives `zpd(b, a) = divergence(a, b) = k`. The subtraction yields:

  wᵢ = 0           for i < k
  wₖ = bₖ − aₖ     (well-defined ∈ ℕ by NAT-sub conditional closure, since bₖ ≥ aₖ via NAT-order from bₖ > aₖ)
  wᵢ = bᵢ          for i > k

The result has length #w = max(#b, #a) = #b, since #a ≤ #b by hypothesis.

We establish three properties of w needed for TumblerAdd. First, w ∈ T: by hypothesis b ∈ T and a ∈ T, and since a < b implies b ≥ a, TumblerSub's preconditions are satisfied for minuend b and subtrahend a, and its postcondition gives w = b ⊖ a ∈ T. Second, Pos(w): the component wₖ = bₖ − aₖ ≥ 1 by NAT-sub (strict positivity), whose precondition `bₖ > aₖ` is the strict inequality derived from `a < b` at the divergence point, so w is not the zero tumbler. Third, the action point of w is k: every component before position k is zero, and wₖ > 0, so k is the first positive component. Since k ≤ #a by hypothesis, the precondition of TumblerAdd (TA0) is satisfied — the action point falls within the start position's length.

Now we compute a ⊕ w by TumblerAdd's constructive definition, which builds the result component by component in three regions determined by the action point k:

*Positions i < k (prefix copy):* (a ⊕ w)ᵢ = aᵢ. By the Divergence agreement property, aᵢ = bᵢ for all i < k. So (a ⊕ w)ᵢ = bᵢ.

*Position i = k (single-component advance):* (a ⊕ w)ₖ = aₖ + wₖ = aₖ + (bₖ − aₖ) = bₖ. The final equality `aₖ + (bₖ − aₖ) = bₖ` is NAT-sub's left-inverse characterisation applied to the naturals `bₖ, aₖ ∈ ℕ` with `bₖ ≥ aₖ` (from the strict inequality `bₖ > aₖ` via NAT-order); the substitution `wₖ = bₖ − aₖ` is the definition of `w` at position `k`.

*Positions i > k (tail from displacement):* (a ⊕ w)ᵢ = wᵢ = bᵢ. These components were copied from b into w by TumblerSub, and TumblerAdd copies them from w into the result.

It remains to verify that the lengths match. By the result-length identity of TumblerAdd, #(a ⊕ w) = #w = #b. Every component of a ⊕ w equals the corresponding component of b, and both tumblers have length #b. By T3 (CanonicalRepresentation), a ⊕ w = b.  ∎

*Formal Contract:*
- *Preconditions:* a ∈ T, b ∈ T, a < b, divergence(a, b) ≤ #a, #a ≤ #b
- *Depends:* Divergence (Divergence) — case analysis eliminates case (ii), establishing agreement aᵢ = bᵢ for i < k. T1 (LexicographicOrder) — derives aₖ < bₖ from a < b at divergence point k. ZPD (ZPD) — identifies zpd(b, a) = divergence(a, b) = k via the ZPD–Divergence relationship in case (i). TumblerSub (TumblerSub) — supplies component formulas for w = b ⊖ a and establishes w ∈ T from b ≥ a. TA-Pos (PositiveTumbler) — defines Pos(w), satisfied since wₖ = bₖ − aₖ ≥ 1. TA0 (WellDefinedAddition) — establishes a ⊕ w ∈ T from Pos(w) and actionPoint(w) = k ≤ #a. TumblerAdd (TumblerAdd) — supplies the constructive component-by-component definition and the result-length identity #(a ⊕ w) = #w. T3 (CanonicalRepresentation) — concludes a ⊕ w = b from component-wise equality and #(a ⊕ w) = #b. NAT-sub (NatPartialSubtraction) — conditional closure discharges `wₖ = bₖ − aₖ ∈ ℕ`; strict positivity discharges `wₖ ≥ 1` (from `bₖ > aₖ`), which feeds into the verification that Pos(w) holds; the left-inverse characterisation discharges `aₖ + (bₖ − aₖ) = bₖ` at position k of the single-component advance. NAT-order (NatStrictTotalOrder) — converts `bₖ > aₖ` into the weak `bₖ ≥ aₖ` required by NAT-sub's closure and left-inverse preconditions.
- *Postconditions:* a ⊕ (b ⊖ a) = b
