**D2 (DisplacementUnique).** Under D1's preconditions (a < b, divergence(a, b) ≤ #a, #a ≤ #b), for any w ∈ T with Pos(w) and actionPoint(w) ≤ #a such that a ⊕ w = b, we have w = b ⊖ a.

*Proof.* We show that any displacement w carrying a to b must equal the canonical displacement b ⊖ a. The argument has two steps: we produce a second witness for the same equation, then apply left cancellation to conclude the two witnesses are identical.

**Step 1: a second witness.** By D1 (DisplacementRoundTrip), the preconditions a < b, divergence(a, b) ≤ #a, and #a ≤ #b guarantee that a ⊕ (b ⊖ a) = b. So both w and b ⊖ a, when added to a, yield b.

**Step 2: establishing TA-LC's preconditions.** To apply left cancellation (TA-LC), we must verify that both additions satisfy TA0 — that is, both displacements are positive and their action points fall within #a.

For w: the hypotheses Pos(w) and actionPoint(w) ≤ #a are furnished directly by the statement of D2, so TA0's preconditions are satisfied without appeal to the equation a ⊕ w = b. The addition a ⊕ w is therefore well-defined and equals b by assumption.

For b ⊖ a: let k = divergence(a, b). By hypothesis k ≤ #a. Since a < b, the divergence occurs at a position where bₖ > aₖ (by T1), so (b ⊖ a)ₖ = bₖ − aₖ ≥ 1. Every component before position k is zero (TumblerSub copies the agreement prefix as zeros). Therefore Pos(b ⊖ a) with action point k, and k ≤ #a satisfies TA0.

**Step 3: cancellation.** From the hypothesis a ⊕ w = b and Step 1's a ⊕ (b ⊖ a) = b, we obtain:

  a ⊕ w = a ⊕ (b ⊖ a)

Both sides are well-defined (Step 2). By TA-LC (LeftCancellation) — which states that a ⊕ x = a ⊕ y implies x = y when both additions satisfy TA0 — we conclude w = b ⊖ a.  ∎

D1 and D2 together characterize the displacement completely: D1 says b ⊖ a recovers b, D2 says nothing else does.

When a = b, no displacement is needed; the degenerate case is handled separately since b ⊖ a produces the zero tumbler and a ⊕ (b ⊖ a) is not well-formed (TA0 requires Pos(w)). D0 ensures the displacement is well-defined; D1 ensures the round-trip is faithful when additionally #a ≤ #b.

*Worked example.* Consider a = [1, 2, 3] and b = [1, 5, 1]. We have #a = #b = 3.

*D0 check.* divergence(a, b) = 2, since a₁ = b₁ = 1 and a₂ = 2 ≠ 5 = b₂. The condition k = 2 ≤ #a = 3 is satisfied.

*Displacement.* By TumblerSub, w = b ⊖ a: w₁ = 0 (i < k), w₂ = 5 − 2 = 3 (i = k), w₃ = 1 (i > k, from b). So w = [0, 3, 1].

*Round-trip.* The action point of w is 2. By TumblerAdd, a ⊕ [0, 3, 1]: position 1 copies a₁ = 1, position 2 computes 2 + 3 = 5, position 3 copies w₃ = 1. Result: [1, 5, 1] = b.  ✓

*Uniqueness check.* Suppose some other w' also satisfies a ⊕ w' = b = [1, 5, 1]. By D2, w' = b ⊖ a = [0, 3, 1] = w. There is no alternative displacement.

The generalization to #a < #b can be seen with a' = [1, 2] and the same b = [1, 5, 1]. Here #a' = 2 < 3 = #b, the divergence is still 2 (a'₂ = 2 ≠ 5 = b₂), and k = 2 ≤ #a' = 2 satisfies D0. TumblerSub (zero-padding a' to length 3) gives the same w = [0, 3, 1] of length 3. The round-trip a' ⊕ [0, 3, 1] produces [1, 5, 1] = b — the result has length #w = 3 = #b, matching the target.

*Formal Contract:*
- *Preconditions:* a ∈ T, b ∈ T, w ∈ T, a < b, divergence(a, b) ≤ #a, #a ≤ #b, Pos(w), actionPoint(w) ≤ #a, a ⊕ w = b
- *Depends:* D1 (DisplacementRoundTrip) — Step 1 invokes D1 to supply the second witness `a ⊕ (b ⊖ a) = b`, which is the equation that left cancellation will be applied to in Step 3. D0 (Divergence/DisplacementWellDefined) — referenced in the closing remarks ("D0 ensures the displacement is well-defined") and implicit in the Step 2 reasoning that `b ⊖ a` is well-formed under `divergence(a, b) ≤ #a`. T1 (LexicographicOrder) — invoked in Step 2 to derive `bₖ > aₖ` at the divergence position from `a < b`, which establishes `(b ⊖ a)ₖ ≥ 1` and hence the positivity of `b ⊖ a`. TumblerSub (TumblerSub) — invoked in Step 2 to compute the components of `b ⊖ a` ("Every component before position k is zero (TumblerSub copies the agreement prefix as zeros)") and again in the Worked example to compute `w = [0, 3, 1]`. TumblerAdd (TumblerAdd) — invoked in the Worked example to verify the round-trip `a ⊕ [0, 3, 1] = [1, 5, 1]`. ActionPoint (ActionPoint) — supplies the action-point function used in the precondition `actionPoint(w) ≤ #a` and in Step 2's claim that `b ⊖ a` has action point `k`. TA-Pos (PositiveTumbler) — supplies the positivity predicate `Pos(·)` invoked in the precondition `Pos(w)` and in Step 2's verification that `Pos(b ⊖ a)` follows from `(b ⊖ a)ₖ ≥ 1`. TA0 (WellDefinedAddition) — Step 2 verifies "TA0's preconditions" twice (once for `w` and once for `b ⊖ a`) so that both additions are well-defined; without TA0 the equation `a ⊕ w = a ⊕ (b ⊖ a)` of Step 3 would have no licensed source for "both sides are well-defined". TA-LC (LeftCancellation) — Step 3's concluding step: "By TA-LC (LeftCancellation) … we conclude w = b ⊖ a" — TA-LC supplies the cancellation rule that converts the equation `a ⊕ w = a ⊕ (b ⊖ a)` into `w = b ⊖ a`, which is the theorem's postcondition.
- *Postconditions:* w = b ⊖ a
