### S8a's formal axiom has an unbound document variable

**Foundation**: S8-depth (`(A d ∈ D, p, q : ...)`), S8-vdepth (`(A d ∈ D, v : ...)`), S8-fin ("for every d ∈ D"), D-CTG (`(A d ∈ D, S, u, q : ...)`)
**ASN**: S8a (VPositionWellFormedness), formal axiom: `(A v ∈ dom(Σ.M(d)) :: zeros(v) = 0 ∧ v₁ ≥ 1)`
**Issue**: Every sibling invariant in ASN-0036 explicitly binds the document variable `d` — S8-depth and D-CTG bind it inside the Dijkstra quantifier `(A d ∈ D, ...)`, S8-vdepth does the same, S8-fin binds it in prose ("for every d ∈ D in every reachable state Σ"). S8a's formal statement `(A v ∈ dom(Σ.M(d)) :: zeros(v) = 0 ∧ v₁ ≥ 1)` leaves `d` as a free variable. The formal contract's axiom line says "every V-position" but never quantifies over documents. A reader (or a TLA+/Dafny encoding) must guess that `d` is implicitly universally quantified over D. This matters because D-MIN, VIP, and the S8 partition proof all cite S8a for all documents simultaneously — they need `(A d ∈ D : (A v ∈ dom(Σ.M(d)) :: zeros(v) = 0 ∧ v₁ ≥ 1))`, not the open formula. The omission is unique to S8a among the invariants; every other property that quantifies over `dom(Σ.M(d))` binds `d` explicitly.
**What needs resolving**: S8a's formal axiom must bind `d ∈ D`, matching the pattern used by S8-depth, S8-vdepth, and D-CTG. The well-formed statement is `(A d ∈ D, v : v ∈ dom(Σ.M(d)) : zeros(v) = 0 ∧ v₁ ≥ 1)`.
