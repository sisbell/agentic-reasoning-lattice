# Sub-Questions — Deletion Economics

**Inquiry:** What must the system guarantee about economic obligations — storage costs, royalty flows, ownership accounting — when content is deleted from V-space but persists in I-space?

1. [nelson] When an author removes content from a document but the system retains it permanently, must the author continue to bear the storage cost for the retained copy?
2. [nelson] If content is no longer visible in any document but persists in permanent storage, must royalty obligations to the original creator continue or may they be suspended?
3. [nelson] Must the system maintain an unbroken ownership record for content that has been removed from all documents but still exists in the permanent layer?
4. [nelson] When a document edit removes quoted content, does the transclusion royalty obligation end at the moment of removal or persist as long as the content remains permanently stored?
5. [nelson] Must the system distinguish between "content a user chose to delete" and "content the system must retain for permanence" when calculating who owes storage costs?
6. [nelson] If a link points to content that has been removed from its source document but still exists permanently, must the system preserve the economic relationship between the linking party and the content owner?
7. [nelson] Must the system guarantee that no economic obligation can be created or destroyed by the mere act of removing content from a visible document?
8. [nelson] When multiple versions of a document exist and content appears in earlier versions but not the current one, must the system account for storage costs across all versions or only the version that introduced the content?
9. [nelson] Must the system provide the content owner a way to discover all persisting economic obligations tied to content that is no longer visible in any document?
10. [nelson] If permanently stored content is later reintroduced into a new document, must the system guarantee that the economic terms match the original obligations, or may new terms be established?
